










懶石聆禪師語錄卷第三
嗣法門人真蘊錄

行繇

一日眾入方丈。請師行。實師曰。吾本蜀重慶忠州人。
父姓張母趙氏。夜夢吞月覺而生。十二歲夢感投師。
薙染方廿歲。痛念無常。迅速生死。事大若不明了。出
家奚益。發足初參。象崖和尚受具請益命。看萬法歸
一。一歸何處。話抵死參究。觸境逢緣。漸覺工夫疏塞。
做來做去。忽一日。不覺身心世界。渾然虛豁豁地。如
青天霹靂。心中似有一物。從脅下被人抽去。一般甦
醒將來。虛空大地。日月星辰。森羅萬象。蜎飛蝡動。情
與無情。人非人等。俱在一毫頭上。煥然等現。斯時心
生慘怛。不覺涕泣。值挂入室牌予禮拜師。舉世尊拈
花。迦葉微笑。汝作麼會。予曰。用會作麼。師云。世尊不
拈花。迦葉不微笑。汝又如何。予曰。兩眼對兩眼。師云。
拈花微笑之後又作麼生。予曰。赤心片片。師連打。予
連喝。師又打。予拂袖便出。復呈偈曰。根源盤自威音
外。穿過髑髏花正鮮。不借東風些子力。香風襲地燦
驢年。師看過。打一拂子。予一喝。師休去。侍師五載後。
參破師翁侍立。次一僧。請開示。翁云。此事如明鏡。當
臺漢來漢現。胡來胡現。予聞。悚然汗出。偶說偈曰。吾
有一句。冰霜皎日殊相。劣形寥落空寂。翁云。那箇是
汝空寂的。予一喝便行。後因師翁命本師。上天童掃
塔。予隨侍至漢中府靜明寺。適結制安。維那一夕同
眾。在方丈立。師舉。韓愈參大顛。擯首座公案。你諸人
作麼生會。予與師一掌。師上下覷予曰。者老漢啞卻
口。那便出。師拽杖隨追。予先至法堂打鼓。眾集。師落
堂云。且道老僧適才兩覷是甚麼意旨。予復兩掌。被
師痛打一頓。因受源流拂子。予固辭某。是何人敢當
此任。且德薄行微。兼鮮學識。和尚如此。豈不取笑諸

方。師云。收取住山去。他日自有別寶波斯。予不得已。
向前禮了九拜。至今懊悔。錯錯錯。今日無故不覺忉
怛。播吾平生之醜。玷污諸人耳目。雖然如是。且道適
才三錯是何意旨。眾禮拜。師云。錯。

法語

索我臨行句。胸中無一字。寫出不成文。念來非有義。
弗是妙蓮華。亦非祖師意。持去見諸方。眼底生荊棘。
不善打葛藤。慣用吹毛利。若作言句親。失卻自家事。
老僧與麼說。汝又如何[妳-女+口]。個中本尋常。從頭須子細。
文彩未彰前。十有九不契。靜明鐵蒺藜。一恁汝啗啐。
忽然唼破時。方敢請棒喫。

禪不須參。道不須學。碗底箸頭多脫略。歷歷騰騰耀
古今。林林總總無拘縛。碧眼胡僧難摸索。蟭螟眼上
三隻角。

此事勿論上智下愚。務在當人信得及。把得定。看個
如何是諸佛出身處。又云。東山水上行。意旨如何。試
猜猜看。與麼會得三世諸佛。借伊鼻孔出氣。歷代祖
師。共汝單上橫眠。設不爾者。辜負行腳。

書

復了庵劉居士

承索便面以質之之語。山僧到者裏。恁麼也不得。不
恁麼也不得。恁麼。不恁麼。總不得。何也。若謂有喚甚
麼作主人公。若謂無食瓔珞粥的是誰。認則眼中著
屑。不認錯過了也。請老居士兩頭撒開。中亦不立。直
下與麼道將一句來。方敢定奪。呵呵。

復月幢法兄

懸寶鏡於千峰之外。洞察幽微展慧劍。於亂煙之中。
削塵顯密。扶破沙盆。斬新條令。移風易俗。大化群迷。
非兄力量而。誰為之。弟才鄙陋。德淺行微。謹守榾柮
以待天年。何期一旦光射昆明。愧之不及。忻躍焚檀。

稽首致謝。

佛事

徹空火。以火炬打圜相云。徹空徹空。南北西東。渾無
一物。布袋裝風。打底一著。手把虛空。攛火炬云。遍界
琉璃含寶月。照破雲山千萬重。

悟空火。悟空不空。雲月是同心。無定止界。有西東。適
來快活。無處不通。助汝一把無明火。燄光騰處。脫樊
籠。

為野哲禪兄起龕。以杖敲龕云。藕斷清兮絲一斷。光
前裕後門外看。而今移入東風裏。大丈夫兒得自便。

舉火云。哲公哲公。來時無口。去亦無蹤。非出非外。塞
滿虛空。得大自在。獲大圜通。船子一脈獨肖。其公即
今奉上一把火。燒得須彌走芥中。

芥山維那火。凝然湛寂雪霜濃。一芥之山內外空。鐵
鷂沖霄藏北斗。更於何地振宗風。大眾還知此公去。
就也無擲火炬云。火裏蓮華千萬重。信之無礙一光
通。

辨通蒙堂火。呈火炬云。火雲堆裏解翻身。撞倒須彌
影不存。六月半空飛片雪。十方剎海冷如冰攛火炬
云。臨行相助一把火。送入金剛山更深。

為藥山律主起龕。以杖擊龕云。點著髑髏眼。虛空轉
落落。觸碎鐵蛇關。通身無繫縛。試看彩雲外。火裏花
灼灼。

舉火云老漢生平。百十三遍。觀山水債巳完。而今兩
腳稍空去。慧劍揮空不等閑。以火炬打圜相云。只向
者裏端然坐。無邊剎海戒珠圜。

為灼然。火灼然。灼然充塞大千。去來無相。生死何關。
甚處著腳雲月。溪山攛火炬云。助汝一把無明火。送
入蓮邦絕聖胎。

為醉月直歲。火執役叢林。千古榜樣。撒手便行。衲僧。

氣象咦東風。昨夜嚷如麻。及至曉來。落一花試。問闍
黎何所事。渾身是月照天涯。

為南天起龕。師以杖擊龕云。金風撲地吼南天。折腳
東籬沒半邊。驚起鷦鷯何處立。去來只在縱橫間。霜
花滿地無人掃。處處冰心淨月圜。

舉火以炬打○相云。者個東西皆本具。臨頭因甚不
親識。灼然個裏薦承當。跨鶴沖霄白滴滴。髑髏粉碎
一塵空。地水火風誰收拾。雲邊鐵鷂過新羅。火裏木
雞穿破壁。妙應叢中不肯住。五蘊峰前堪遊戲。與麼
無去亦無來。無去無來泯蹤跡。攛下炬云。老僧助汝
者些兒。鍛送金剛寶蓮地。

無鬚鎖


「　活鱍玲瓏心眼通。郎當廓落振宗風。乾坤世界都關卻。惟露中間一段空。」



無孔笛


「　非短非長勿舊新。頻吹清韻令人聞。聲和萬籟鈞天樂。驚惺昆池月一輪。」



無底船


「　永劫修持穩柁機。片帆時挂霧同飛。逍遙且遜清風力。明月蘆花滿載歸。」



無根樹


「　雨灑風吹葉不絲。挺然卓立過須彌。通身是個光明藏。洞徹坤維萬象枝。」



無底缽


「　個樣團團未宰功。持來眾裏萬人欽。頻盛世上鐵釘飯。供養十方羅漢僧。」



無絃琴


「　赤手攜將月下彈。五音六律最清宣。坐中誰是子期客。聽取無腔聲韻傳。」



無底籃



「　難堪秀器自成家。裏面能盛死活蛇。擲向街前人弗愛。嘶風注雨月篩華。」



無孔錘


「　硬似泥團軟似鋼。輥天輥地勢難當。拈來安置滇雲上。電閃雷轟震大方。」



無星秤


「　赤體條條絕點霞。洞山曾式三斤麻。鉤頭分兩全拋出。得意人人善作家。」



無縫塔


「　東西南北見陶然。瑞起琉璃寶月圜。意氣弗從天地得。不歡欣處也忻歡。」

宴坐


「　心空體寂固靈虛。萬象交羅本自如。洞徹玄微岐意絕。明明皎月落天湖。」



入室


「　欲叩明師破暗關。臨機貴在絕情緣。當風雪刃才挨著。死活雙忘脫苦酸。」



普請


「　梆聲未絕力行先。惟恐行門功弗全。般若無方隨所應。搬柴運水處須圜。」



經行


「　倦來何事且留神。散步林泉石上行。個裏些兒如蹋著。無生曲子調高吟。」



誦經


「　跏趺對月歷三更。靜誦空王無字經。萬象森羅齊側耳。烹茶石女甚慇懃。」



禮拜


「　掃室焚香禮拜勤。歸投句子業消冰。猛然磕破髑髏也。獨露堂堂無二人。」



道話


「　一度相逢一度親。無心話道道孤明。了然直下超方便。耀古騰今舉世聞。」



焚香


「　煙清火亙一爐香。靜裏常焚日月長。萬象森羅聞見盡。須知紅燄裏頭藏。」




掃地


「　篲柄聊舒風若神。微塵剎海淨光生。石頭瓦礫含諸象。亙古圜明質本清。」



運水


「　一滴水兮一丈波。常然泛泛滿江河。老僧今日生涯別。拈出令人止渴多。」



搬柴


「　閒來無事到雲林。肩得寒嵒樹一根。歷劫及今煨不悉。騰騰光燄灼乾坤。」



分燈

橫眉海瑞


「　黃梅峰頂漏三更。衣法雙傳月下行。石上標宗千古印。層層慧燄續賢英。」



自覺書恆


「　驀地相逢不露蹤。搖風擺雨十方空。親拈示汝龜毛拂。他日昆池點化龍。」



一休性旵


「　毫端拈起疾如風。絕相離名迥不同。萬仞峰頭剛獨步。逢場處處演吾宗。」



雲崖明峰


「　英豪氣岸兩頭脫。拄地撐天活鱍鱍。卓向雲深百尺崖。有時龍化濟人渴。」



恆念道富


「　拈起木叉體相圓。頭頭法法慧光寒。赤心片力荷擔去。白馬蓮城繼後傳。」



止鏡真蘊

呈雙桂破師翁


「　聯不成兮撥不開。成團作隊漢江臺。終南山頂月中桂。云是師翁親手栽。」



聞師翁訃音


「　違座南濱二十秋。快人肝膽解人愁。而今幸得安平貼。又謂西行征戰遊。」



示橫眉禪人


「　化日舒長春正融。亂雲深處豢獰龍。嘶風喚雨操其節。幾帶雷音吼太空。」



示如是禪人


「　如是住時如是作。四威儀內看心牛。忽然看破看的個。耕遍雲山志不休。」



示密旨禪人


「　大地撮來如粟米。明明不悉為君舉。直須猛省吞卻去。唼碎囫圇是密旨。」



示懶參禪人


「　冤有頭兮債有主。靜明鼓笛誰為伍。老僧撥轉上頭關。韻出清音契太古。」



示止鏡禪人



「　爐頭佛法聽無言。作箇忘機混沌禪。古鏡即同火爐闊。祖師心印為親傳。」



示非指禪人


「　臨濟家私夜不收。尋常高挂白雲頭。當機一喝知音少。領略還須佛祖儔。」



示一休禪人


「　覿面全提是甚麼。阿誰直下得擔荷。人前莫把風規露。且向碧雲深處坐。」



示劉居士


「　老僧無法與人傳。無法無傳透祖關。石女夢回機太活。手把瑤琴月下彈。」



示張居士


「　歷歷分明無錯謬。提弓把箭是誰能。當風疾展金剛劍。鬼面神頭喪膽魂。」



示行腳僧


「　富貴不貪衣糞草。心頭絕慮無煩惱。當求出世超凡士。毋作叢林嘗飯保。」



示醉月典座


「　從來務下合如常。久歷雲廚粥飯香。木杓笊籬齊合掌。金牛飯桶句重揚。」



哭象崖和尚


「　慈悲大煞老婆心。棒喝曾經不放輕。一片痛腸終未吐。又跨鐵鶴向西征。」



留別月幢法兄


「　宜兄宜弟席蘭谿。話到三更月滿隄。瀝腹領腸情未已。雲瓢雪笠度峰西。」



赴藩園齋


「　愛日擔雲入竹籬。園花砌玉點新枝。森森怪石如人立。影浴天池醒舊眉。」



送嘯石南行



「　聒聒雁高卷白雲。一聲啼斷古江城。平空展出摩霄翼。駕海騎魚水上行。」



送不會監院行腳


「　數年輔弼整叢林。著著臨機意最深。莫道功勳空費也。池澂月現見天心。」



送存無律師主雲霧山


「　三冬聚首春初別。蹋遍黑龍江底月。無口缽盂吞佛祖。木叉掃盡千峰雪。」



送密言律師入山


「　處處青山堪駐錫。云何直指野雲巢。藤蘿月挂枝梢上。只恐霜花笑寂寥。」



寄玄素法侄


「　五龍瑞氣接天池。點滴藏鋒不露絲。只待風雲雷雨作。乾坤黯黑用全施。」



寄石龍禪師


「　清河話別巳多󵊮。幾跨須彌躡大千。諸子案頭譚不一。癡人繇認井中天。」



寄雲巢


「　忽地相逢又別離。水山窮處看雲飛。五龍峰頂三更月。影落曹源任指撝。」



見僧頂箕雨中行


「　簸箕戴在僧頭上。恰似將軍出戰時。手裏雖無三尺劍。衝風直入始為奇。」



寄甸舒弟


「　高提祖印踞滇南。一鏃當風用在先。捉得新羅鐵鷂子留將千古繼宗傳。」



送枕石弟歸楚


「　羨君乘興去𨹁山。楚水江頭把釣竿。奪得驪龍珠一顆。孤明迥迥洞人寒。」



白塔子



「　誰將玉管豎東皋。半臥江干洗兔毫。驚起魚龍奔上下。墨花隨影點天橋。」



靜明塔


「　卓立無私萬古存。撐天拄地倚雲門。長安風月惟君壽。架海金梁柱一根。」



豎旛竿


「　剎竿頭上仰蓮心。見者應須著眼青。弟應兄呼傳底事。靜明今日又重新。」



雲霧山


「　千峰挺出越紅塵。峭拔孤嶙繫白雲。月過西巖風凜凜。藤床竹榻冷光生。」



壽南溟


「　歷遍阿僧祗劫來。何云今日為初度。真風永扇芥城空。不受陰陽花甲數。」



壽一休


「　端陽烈燄引清風。一點靈花燦正紅。處世節操塵不染。驢年香噴地天中。」



補窗


「　剪雲補個破虛空。幸得無痕縫不通。沒量大人頻踞地。孤明千古縱橫中。」



賞釣


「　絲綸初展漢江頭。徹困金鱗水上浮。香餌時餂幾喪命。清波不犯五湖秋。」



風鈴


「　風吹鈴鐸響雲中。鐵鷂流星過海東。見者心隳聞者喪。木人隊隊舞寒空。」



聞讀


「　一字疏通爛葛藤。百千妙義此中存。不須點劃分權實。清淨本然法上塵。」



野宿



「　山寺晚風遞遠鐘。漁燒新火滿江紅。松巢鶴醒驚初夢。啼上枝頭月正濃。」



秋日


「　人人有月性天週。莫向中宵外妄求。不信但看跟腳下。孤明歷歷照瓊樓。」



峨眉


「　頻遊駕象獲圓通。影現峨眉七級中。萬億河山留不住。為憐普眼處相通。」



夜聞


「　夜清何處玉簫森。按指聲驚木偶人。吹去旻天雲散盡。珊瑚枕上月華新。」



藤橋


「　布水攙雲也怪哉。溪頭抱日結良胎。度驢度馬奚終用。自有擎天一段才。」



登眺


「　重陽九日忽登樓。四顧煙波一目收。唯有就中些子別。菊花原要倩誰酬。」



佛誕


「　倒騎香象降王宮。足底蓮生八面風。頂上圓光輝萬國。遍虛空界示金容。」



偶成


「　商山野老弄春風。鼓動昆池魚化龍。霹靂聲驚三峽水。和雲合雨釣江東。」



風箏


「　者樣東西怪又癡。麻纏紙裹逐風吹。單絲繫著懸空上。大似流鶯戲小雞。」



上梁


「　雙日雙時雙鳳朝。東西南北碧雲高。人天剔起摩醯眼。試看雙龍挂九霄。」



除夕



「　井市堆頭一歲除。荊門處處貼新符。火花炮響迎佳節。破我禪心德不孤。」



示眾


「　夜來明月挂中天。立坐都盧毛骨寒。拈出無生切要句。臨機錯過話頭難。」




「　無地卓錐不惜貧。傾腸倒腹為諸人。琉璃撲碎還如故。八臂那吒提不行。」




「　尋常只把話頭疑。疑去疑來疑是誰。疑破自家疑底個。始知甕裏有風雷。」




「　月滿霜籬僧滿堂。家風平貼耀心光。老僧說法機緣別。不打前賢爛葛椿。」




「　眉毛丁結不容情。意識心緣一并傾。剔起金剛正眼看。急須覷破覷的人。」




「　僧滿禪堂月滿籬。家風峭隱鳳龍獅。腥羶氣味薰天地。不是苦心人不知。」




「　不枉靈山付囑來。心華燦燦信然開。斗光植就金剛種。結個驢年不朽胎。」




「　不枉靈山付囑來。高提寶印向天開。逢場作戲須當薦。剎海無邊總是胎。」




「　不枉靈山付囑來。宗風初展少林開。神光斷臂安心後。始覺從前錯認胎。」




「　一葉香殘一度秋。蟬鳴老樹碧雲流。西風亂吐黃金色。月上疏林輥玉毬。」



入滇


「　于木隨身七七年。頻敲風雨入雲滇。而今靠在禪床上。鐵額銅頭心也酸。」



因事


「　羨君伶俐逞聰明。笑殺東京一夥人。設犯那羅延箭毒。恐伊性命卒難存。」
