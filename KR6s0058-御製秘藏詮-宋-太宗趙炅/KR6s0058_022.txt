










御製秘藏詮幽隱律詩卷第二十二　𦘺


「　我作一篇詩」


【(詩者志之所之也子夏序云在心為志發言為詩故上以
風化下也夫聖人之為心也舉一事措一辭則雖云乎有作窮其深極其遠則寔在乎無
為縱周穆有黃竹之篇魏文有烏鵲之句以此望彼良不足云)。】

「　人間天上知」


【(王言雷動聖化風行道𢍆寰中理窮繫表故大易有言曰先天而天弗違後天而奉
天時而況於人乎而況於鬼神乎斯言不誣矣)。】

「　徒他皆解悟」


【(彼懷生之
大情也但見其四時行焉萬物生焉殊不能盡知天地生成之德乃因言立象庶得象忘
言然後使君君臣臣父父子子各遂其性咸本乎真序天地之高卑法陰陽之倚伏多識
乎前言往行輝光日新者也於是忘僥競之貪心荷成平之正化紜紜之際自有成功碌
碌之閒咸知妙本)。】

「　細認理無私」


【(天以無私而覆也地以無私而覆也
日月以無私而照也盖理本如斯乃出之於自然矣而人由是而生焉其為善者降之百
祥為不善者降之百殃亦無私之義明矣此則正陳述作立言之理所謂言忘而理暢也
不亦冝哉)。】

「　邪妄終名利」


【(邪以不正為名妄以非真為義若乃僭不道之
名竊非常之利名而無實利而無猒則伇伇勞神區區殞性如伊逐馬似彼騰猿斯本邪
思終成妄動心神馳於內形㒵奔於外內外俱弊心形兩疲名愈高而愈矜利彌奢而彌
侈故詩三百一言以蔽之曰思無邪)。】

「　顛狂勿信之」


【(背理而作即顛倒狂
亂之可知既迷其真終昧其性性之昧也真則遠焉亦由百鎰之金不足以供其失路之
費也雖欲信之其可得乎故經所謂如來為真可憐愍者)。】

「　幽深豈易見」



【(自非聖人克勤克儉惟靜惟清則道遠乎哉雖近而不見也)。】

「　逢善即為師」


【(師者䡄範為名也觸類長之何往不適擇善而從其道一也故書云學無常師主善

為師斯乃為心之師非師於心也)。】


「　有年便僻靜」


【(尊高尚齒著在格言況乎知非之年豈無便乎僻靜清虛
守道理𢍆無為冥合真心逍遙玄域矣)。】

「　我意是人知」


【(夫聖人之心若
日月之麗于天孰不知乎照臨之道焉類溟渤之紀于地孰不知乎朝宗之望矣然則廓
靜妭氛澄清沃日者亦民人之心也是知聖人無心以百姓心為心者豈不然哉)。】

「　思極通三界」


【(三界之名乃九地之通稱也得無色定者離色界染觀欲界欲
以修所成慧登解脫門脫屣迷津濯纓定沼也今思極於彼而義張於此大乎哉萬趣同
源五乘一致則何物而不可化矣)。】

「　含靈遶四維」


【(道極四維無幽不照
今且指百億日月中一日月運轉之處𢩁一四天下一金輪王所王之地即五趣雜居之
一也含靈者所謂盡含一真萬化之靈也又莊子云靈府者但畜善以為名也)。】

「　從容閑理道」


【(妙閑至理𡨋𢍆真機自在從容通融萬物故復言之道也)。】

「　動止合規儀」


【(游焉息焉造次於是動靜相生其儀不忒)。】

「　但喜民安樂」


【(書云民惟邦本本固邦寧今比戶之可封諒惟天而是則惟聖克念而民用康又詩
云將安將樂)。】

「　恒持萬載基」


【(聖政育蒼生以守位蒼生藉聖政以全生
生全而太階白平位守而宗枋永保至矣大矣雖休勿休安而持之德以御之又寧止乎
萬年而巳矣)。】


「　善人常快樂」


【(善聖之人靜唯樂道動必得時消息盈虛進退雍雅悔吝
不作快樂乃常是知天道無親唯與善人)。】

「　清淨好歡娛」


【(清淨之化既敷
淳素之風乃復雖焉知於帝力且不違於義方人神阜安禽魚咸遂熙熙然何歡娛之可
及矣)。】

「　感激情分別」


【(天之無恩而大恩生是以聖人行湛露之澤布如春
之人故不求其報而或自起分別欲謝生成盖不達至人之情強作感激之意也)。】

「　優



【(道德之域以優以遊事絕殊途情超流俗使夫澆漓之世邁於
義軒之代)。】

「　炎蒸然燧火」


【(炎炎欝蒸為熱方熾復以燧火然而継之亦猶
損不足而奉有餘且萬物負陰而抱陽乃遂生發其或剛壯之氣太盛則順益之道何有
若能高者抑之下者舉之則与天同休稱物平施不亦善乎)。】

「　世昧冷冰壺」


【(世代之間盛衰恒理柰俗之情見榮而進受辱而退殊不知倚伏交相吉凶一致當
得路之時戶可炙手及失寵之際門堪張羅是知冷然之情冰壺可諭矣)。】

「　極意誰消息」


【(上文所言至極之意誰能體解唯有道之士方可明而行之)。】

「　門深且晏如」


【(玄妙之門淵深晏靜迎之不見其首隨之不見其後豈同物變
寧逐時遷儻或修之於身則後天之壽不為遠矣)。】


「　意思何興感」


【(天真之性理本恬然因外事之忽生方感動於內意所謂
情發于中而形於言者也)。】

「　情來不暫休」


【(情觸於事事起於情情逐事流
展轉伇慮以日繫月雖休勿休)。】

「　安危陳計䇿」


【(聖人之有天下也安者謹
而守之俾萬世不拔危者扶而持之使四海一統陳乎計筞化被蒸黎伇思勞神日慎一
日)。】

「　康濟入時流」


【(教立雍熙人皆康濟從微至著日居月諸翼翼孜孜聖化
流於時矣)。】

「　細葉稀踈影」


【(夫香草美人比於君子者詩之雅道也今若貞
松勁栢經霜不凋細葉凡材先秋易落亦猶德行之者堪久姦回之者易摧歲寒然後妍
𧈪自見矣)。】

「　清風滿背秋」


【(時之安也民被穆若之化時之泰也風不鳴條
矧以葵能向陽人能守道金風倐起咸悟清涼)。】

「　凄凉悲往事」


【(春生夏長
見物景之繁華鶿去鴈來覩時光之遷易思巳往之事得不凄凉者乎)。】

「　白髮上人頭」


【(萬事交煎四時代謝苟不能洞達至道自省心機忽焉光陰不覺老之將至)。】


「　往事興人感」


【(夫情感於內而形言于外今言往事興人感者所謂三皇

之後王帝巳還純風蕩蕩修之者昌素德眀眀背之者失矣)。】

「　徘徊空寂寥」


【(至德本寂至理本幽奉之者景慕先哲詠之者化彼諸蒙今言寂寥者凝神理道之
謂也)。】

「　湛然思古道」


【(夫道者通性達情亡言𢍆理古道者即上古中古可
久可大之道也湛然思慕不亦樂乎)。】

「　群品似相邀」


【(體道化物體物成詠
致同聲者相應同氣者相求雲從龍風從虎斯之謂也)。】

「　春臺松栢靜」



【(太皡垂功萬彙成務美松栢茂堅貞之橾嗟挑李揚芳菲之榮彼此一時動息無二)。】

「　秋風海岳搖」


【(春生萬物物不知其化秋肅萬物物不知其功則知榮枯代謝
同歸於道也)。】

「　時遷何倐忽」


【(日居月諸皆言𢙣景今則不然暄凉雖遷
至理寧變也)。】

「　落葉自零飄」


【(潘岳云冬索而春敷夏茂而秋落彼不達
之者為時之憂此則不尓花暖春陽不足以娛也葉凋秋暮不足以悲也)。】


「　坐謝幾光陰」


【(人之處世也若白駒之過隙所慮者名不書於史䇿功不
布於蒸民是以大禹惜於寸陰陶公寶於分景今若居然而坐空度時光良可惜也)。】

「　周才重古今」


【(易曰聖人以智周萬物而道濟天下故曰周才也爰自開闢迨
乎聖朝古今雖殊靡不欽重於斯才也)。】

「　見機堪撫掌」


【(其或事有未萌
理有未達忽發先見之智誠謂天生之才盡在彀中了然任意仲𡰱所謂君子見機而作
不俟終日)。】

「　取捨盡由心」


【(或利及於萬世或專美於一身用之則行捨之
則藏用捨二道盡由於心矣然則聖人利物唯擇其善者而從之)。】

「　緲邈情何遠」


【(前觀千古旁敻八紘縱至道寂寂以何求任塵寰渺渺而無極我若體理如觀
掌中道雖無方情亦自遠)。】

「　清歡力不任」


【(恬淡之道清淨為心達性之理
自覺情歡守道了性所得之樂不同雖任其智非運力乎)。】

「　風前與月下」



【(凉我懷者清風入吾牖者明月內樂逍遙之性外持造化之權美景良辰以道而樂)。】

「　玄



【(先王治世之機本性任緣之理幽微玄妙造次何知唯大聖智
明而行矣自餘莫測何啻海深)。】


「　何湏山下住」


【(倦朝市好雲水者見喧靜有殊取捨不等而大隱之士閙
中自靜濁處能清)。】

「　道性有閑位」


【(君子樂於道小人笑於道不能勤而
行之豈同居要若閑臨事而緩)。】

「　園裏群飛鳥」


【(苑圓當春鸞凰畢集樂遲
遲之淑景戲灼灼之韶光俾一物無失其所也)。】

「　花開向有香」


【(青帝垂同
百花𥪰秀吐芬芳之艶發散漫之香眾目咸窺群心共羡)。】

「　幽深情自樂」



【(見於未萌察於未兆而樂在其中熙怡自若而汪洋之量每陶陶然)。】

「　老大減踈狂」


【(言年長而知損益之道識可否之機縱逸自止聲色不惑)。】

「　寂滅安居靜」


【(無聲曰寂可證稱滅唯猒苦斷集之土處閴寂無塵之地安閑自樂任淳和之性)。】


「　齊天地久長」


【(天無私覆而長地無私載而久唯聖人之道與二儀同休
三才一貫)。】


「　老冝飡薄粥」


【(髮邁桑榆年侵蒲柳非羹粥而不食非湯藥而不飲)。】

「　省睡一甌茶」


【(良夜慵眠白晝廢寢每孜孜於問道常矻矻於觀書非茶無以滌其
昏昧)。】

「　風味全然減」


【(其或神思澹泞情興闌珊觸目倦為庶事停罷但樂
熙怡之性)。】

「　花開任眾誇」


【(桃李芬芳馨香散澷時容賞歎無阻矜誇詠發
生之力)。】

「　慵來多冷澹」


【(務煩則憚事併則慵旨趣寂寥况味蕭索)。】

「　意緒足交加」


【(正属冥搜又思致理雖勞神用無廢彛倫)。】

「　拄杖閑庭立」


【(引從容之步資舒坦之情故緩帶以立秋庭笻而對殘照)。】

「　怡情笑月華」


【(嘉景良辰清風朗月足可怡情適思滌慮開襟疑望煙霄以資吟笑)。】


「　老漸惟慵墮」


【(歲華相遷漸增慵墮屈伸時減取捨䪺踈遂性任真唯慕

幽靜道德經云大巧若拙大辯若訥斯之謂歟)。】

「　園林罷去遊」


【(是處林泉
非能遊翫是處臺榭豈欲登躋妙淨精通真境湛寂道德經云湛然常存)。】

「　驕矜除以定」


【(不恣驕矜自登仁壽品彙服淳和之性生靈去逸樂之心)。】

「　常為世情憂」


【(方輿之內民庶之閒勞苦之所未均貧富之所未化憂民之思其惟此乎)。】


「　隙地多荒草」


【(閑隙之地不耕則繁草寔多𦘺濃之田弃捨則穀麦不足)。】


「　祥煙落照樓」


【(煙鏁長空霞明遠道蓋非煙非霧嘉瑞之應也)。】

「　自娛閑自樂」


【(萬機之外無事之時或快思以幽吟或怊懷而及物清風朗月轉適
道情)。】

「　細水小波流」


【(泛若波流潤成天澤道德經云上善若水水利萬物
而況水在天為霧露在地為泉源流注無窮而潤下矣)。】


「　不言行教化」


【(聖人處無為之事行不言之教萬物作焉不言而信焉不
化而行焉自然澤及無疆化之廣矣)。】

「　有位豈胡顏」


【(居經濟之位執王佐
之權若不輸莭奉忠助帝王行無為之化者豈不胡顏哉)。】

「　雲靜眀開閉」



【(雲靜則虛豁性靜則清閑閉其銃紛開其閻閨守真抱一得無眀乎道經云善閉無関鍵
而不可聞)。】

「　人忙我自閑」


【(世人不了大道冲靜其體淵深而被情欲所拘
得無忙忙矣是以聖人知其雄守其雌謙其德樂其道得不自閑而無事乎)。】

「　畏繁非市井」


【(鍾鼓纔動輪蹄交蹤碌碌世情忩忩市井貪名𥪰利得不畏繁也)。】


「　就靜覔深山」


【(喧雜之境既猒念山之思盈懷就靜怡神自出塵寰之土
居山𢍆道端然物表之人可不逸樂也)。】

「　道士與和尚」


【(道以虛无為本
釋以寂靜為宗然而辤親慕道雖同長生悟證全別)。】

「　杳𡨋天地間」


【(杳兮
冥兮道宗巳出言教之表天覆地載釋門永離欲界之緣徵事引情得不暢於道乎)。】


「　三春無意賞」


【(青陽始至綠野鋪煙繁華妍麗孰不動心以道遣懷機緣

內息恬和養神則自安於內清虛棲心則不誘於外所以追賞情罷巳登老氏之春臺游
翫心無自𢍆仲𡰱之木鐸)。】

「　老大畏看花」


【(淑景融和百花鬪艶引心神之
㸦亂生情欲於流年知而誡之是以湏畏道經云不見可欲使心不亂以是聖人清目而
不視靜耳而不聽弃心而不慮弃智而全真遣情而接物自然境絕攀緣遊心䪺息)。】

「　雲重思時雨」


【(雲比於德德立而道生雨比於恩恩行而物潤道生則無為之
化翛矣物潤則豊年之瑞足焉雲行雨施德立道存矣)。】

「　心憐落照霞」



【(喻君子之道長小人之道消道長則無為而民自化道消則無事而民自富察物緣情覩
時徵事落暉斜照散作餘霞杲日初昇聚為叚錦光舒景象極目心憐)。】

「　傷情離別恨」


【(送子南浦執別東門春草碧色春水淥波會散之情既切別離之恨
空深世務之心感情如此然而以道為念何傷情之有哉)。】

「　壯志少年誇」



【(幼而多才少而多學自稟生知之性早有老成之風起翦之徒韓彭之士方堅志氣自矜
自誇之謂也)。】

「　短褐厚綿襖」


【(披褐懷玉任真養神寒利短褐飢利糲食
晏如忻樂而忘憂也)。】

「　神清澹薄茶」


【(道既清神茶仍爽思恬然淡薄其德
可大其或內定則神府不亂外密則形情不擾以此清神自洽澹然之性者)。】


「　意遣隨冝樂」


【(謂人心不能變通自凝滯於物理嗜慾紛㧝未聞省巳喜
怒交并但見尤人苟能以心釋疑以意遣惑隨冝致用不滯物情則百慮不能侵万化不
相悖周易所謂消息盈虛蓋言天地陰陽運乎四時春生夏長秋獮冬狩各隨其冝然後
告成故知聖人智周造化德合照臨辯文物之冝觀適用之理天下不得不樂推矣)。】

「　寂然無是非」


【(夫人有七情自相攻伐喜則彼我皆是怒則文物成非蓋不知
其宴安稽乎用捨苟能以心意內遣性識外眀隨物我之冝順語默之致自然各住其性
是非不興)。】

「　不欺終降福」


【(論語云君子有三畏畏天命畏大人畏聖人之
