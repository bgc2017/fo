










御製秘藏詮卷第八　　　　　　　車


「　純實合僧祇」


【(念念無差深窮實際歷僧祇劫功行何殊法華經云自我
得佛來所經諸劫數無量百千萬億載阿僧祇)。】

「　無令一行虧」


【(戒德瑩淨
行解無虧慧見圓眀萬法清淨法華經云威儀無缺淨如寶珠)。】

「　法身非所着」


【(凝寂無相非所著故智無所得法亦通流華嚴經云入深智處無所着)。】

「　言下𢍆心時」


【(頓了無生𢍆心絕念圓成正覺便𢍆無生華嚴經云了達無時
法一念成正覺)。】


「　益身唯正法」


【(入道之人離諸邪見身定有益於法無餘維摩經云入持
正法救護一切)。】

「　如夢心邪慮」


【(妄境如夢色塵所緣為有邪見違背正法
華嚴經云如夢不真實)。】

「　幻化自紛紜」


【(虛假不真惣是幻化眾生顛倒
塵慮紛紜維摩經云是身如幻從顛倒起)。】

「　慈雲乘救護」


【(佛具悲心能為
救護慈雲彌布法雨普霑一切眾生無不受潤法華經云慈意妙大雲澍甘露法雨)。】


「　和氣皆真正」


【(和氣者謂心性柔順內外無差故得惣合真源同歸正道
法華經云柔和質直者則皆見我身)。】

「　金僊妙覺依」


【(金僊即佛能修智人
妙覺依者覺性圓妙所依理也圓覺經云一切如來本起因地皆依圓照清淨覺相)。】

「　剎心平善樂」


【(剎即剎𨚗心乃意識平善樂者觀法無二順理輕安𢍆果之因
也法華經云善修如是安樂心故)。】

「　方𢍆法王機」


【(法王者說法自在眾
生依止故為平等利樂為心淨名經云法王法力超群生常以法財施一切)。】


「　迷識獨何為」


【(癡心曰迷了別為識識隨妄轉迷逐邪生不𢍆真機如何
證悟楞嚴經云從迷積迷以歷塵劫)。】

「　無心般若依」


【(般若智慧本為無心
法海融通慈航廣度大般若經云依般若波羅蜜多故)。】

「　能於言下悟」




【(真機極妙悟即忘言法貫大千借事顯理維摩經云說是語時五百天子得法眼淨)。】


「　攝受透深機」


【(能攝法要透頂全真悟在臨機不生別念法華經云
佛知彼心行故為說大乘)。】


「　語要通津法」


【(語合佛旨法自通流不假智舟自達彼岸法華經云通達
大智到於彼岸)。】

「　迷情見轉癡」


【(不執迷情則達真見為癡所縛不悟妙明
法華經云眾生諸根鈍著樂癡所盲)。】

「　自將顛倒意」


【(妄心顛倒真智
亦迷意馬奔馳淨途漸遠楞嚴經云迷失真性顛倒行事)。】

「　沉滯向阿鼻」


【(妄源流轉情執循環不悟真空何由解脫業緣漂沒即入苦趣楞嚴
經云墮阿鼻獄)。】


「　故驗修行人」


【(故驗者標舉揀擇之詞依法順習謂之修行善解任持故
稱人也金光明經云勤心修習六波羅蜜)。】

「　朝聞及薄暮」


【(上士趣道
朝暮精勤聞自耳根發明心識遺教經云晝則勤心修習善法初夜後夜亦勿有廢)。】


「　觀音妙智力」


【(觀音者因觀聲得号以無作妙智力開示普門從聞思修
而行果無不證楞嚴經云無作妙智力)。】

「　如電善分布」


【(大悟豁然纖毫
無礙譬如雲間電布照道路以皆明遺教經云譬如夜見電光即得見道)。】


「　塵內勿嫌疑」


【(塵即煩惱障礙菩提不離於塵嫌疑轉盛維摩經云行少
欲知足而不捨世法)。】

「　其中定是非」


【(妄塵擾擾迷海轉深若了真心断諸
迷執維摩經云諸煩惱則眾生為之塵勞諸佛以此而作佛事)。】

「　因緣無障礙」


【(初起為因展轉為緣為因緣故而生障礙若達實際了無生滅淨名經云於因緣果
報無違無諍)。】

「　祖印合真機」


【(迦葉初傳如來法印不隨情見自合真機
如印印泥無先無後楞嚴經云如來印我成阿羅漢)。】


「　垢淨情何著」


【(情忘境泯垢淨體空一理皎然二法清淨維摩經云

垢淨為二見垢實性則無淨相)。】

「　還同糞上衣」


【(棄本逐末二見顛倒唯淨
與穢一法何殊服糞掃衣昇如來座法華經云着如來衣坐如來座)。】

「　心中能自照」


【(從心起智不照本源非智非心萬法清淨圓覺經云心照清淨)。】

「　何物欲相違」


【(智通妙理全物我以皆真見徹靈源融違順而齊致楞嚴經云發真
歸元此十方空皆悉銷殞)。】


「　往事不堪追」


【(過去名往營辦曰事謂定心淨妙俯視塵區念彼迷徒豈
堪追憶楞嚴經云却來觀世間猶如夢中事)。】

「　夤緣沒了期」


【(夤緣者聯
綿相續沒了期者歎于不歇謂妄情無際流轉何窮金光明經云生沒無際輪轉不息)。】


「　頓拋有漏相」


【(忽契道理乃曰頓拋未到無生惣名有漏相則擾擾之狀
金光眀經云遠離一切諸𢙣等)。】

「　寂默達宗師」


【(寂默者攝念達即了徹歸
真得旨故曰宗師謂見性還源心印相契法華經云又見菩薩寂然宴默)。】


「　顯證無非慮」


【(顯者淨智纔起證乃本覺明圓永断於邪思故云無非慮
圓覺經云當依正智慧便得超邪見)。】

「　聖賢相會遇」


【(聖則十地菩薩賢乃
三賢位人行願相昭智德同契故云會遇維摩經云諸佛賢聖共相稱歎)。】

「　說法在靈山」


【(如來隨世示樂幽居山靜境閑堪為說法法說廣大利益尤深故法
華經云常在靈鷲山)。】

「　皆是真空喻」


【(海藏法門理同言別故此惣指皆是
真空喻則譬喻金光眀經云無量餘經巳廣說空又云本性空寂)。】


「　來即心無事」


【(悲心三界為物而來善契一真翛然無事維摩經云不來
相而來)。】

「　去還却有為」


【(法本無生性亦不滅情拘三界還着有為金剛
經云無所從來亦無所去)。】

「　妙聞真實相」


【(正智悟理名曰妙聞達本歸真
實相斯顯維摩經云其法者無聞無得)。】

「　凡解不通知」


【(凡愚不了智解
寧通妙理廣深安窮淵極淨名經云雖信解深法而取相分別)。】



「　人見他心相」


【(維摩經云先當入定觀彼人心見謂真俗二智俗智照他
人心真智證自實相)。】

「　忻然性不迷」


【(唯識論云生如來家住極善地忻然
謂得無漏正解發生既證二空安然寂用)。】

「　一真求問處」


【(一真未悟問處
湏求了達現前俱無所得金剛頌云空生初請問善逝應機詶)。】

「　觸目是菩提」


【(四智洞明萬法本寂逍遙無滯觸目皆真智度論云菩提菩提断俱名為菩提)。】


「　種類分難辯」


【(色心差別種類難分解𢍆歸真逈超心識法界觀云一多
緣起之無邊)。】

「　其如理要齊」


【(一真妙理凡聖皆齊方便千途會歸一寂
維摩經云一切法皆如也至於彌勒亦如也)。】

「　不於真正法」


【(真如圓寂
正法菩提理智俱融自常安靜唯識論云安樂解脫身)。】

「　㸦亂起狂迷」



【(遍計起執諸見纏心既悟圓成而生正解維摩經云菩薩於諸見而不動)。】


「　攝心歸實中」


【(真心離緣攝歸實際既無二相恒在理中金剛經云信心
清淨則生實相)。】

「　眾生悉普濟」


【(眾生所化之境諸佛普大慈悲既悟二空
平等與樂維摩經云行安樂慈令得佛樂)。】

「　不能障宿緣」


【(本謂緣識障於
菩提能了真空自除宿業彌勒頌云先身有報障今日受持經)。】

「　終知福與慧」


【(業殃既泯福慧二嚴過去未來悉令知覺維摩經云順佛智慧不隨他教)。】


「　東注滄溟海」


【(智源洋溢覺海生光量等無邊如波東注圓覺經云能入
清淨覺海)。】

「　法心理不虧」


【(法歸實相心悟菩提證悟無虧能成妙用維摩
經云一念知一切法)。】

「　能無增減相」


【(智有淺深法無增減能體真相定證
涅槃大般若經云在聖不增處凡不減)。】

「　歷劫道可疑」


【(佛法廣妙窮刧
難量於道無疑能成正覺淨名經云於無數億劫得入於佛道)。】


「　西域傳經梵」


【(佛化西土經梵東流極無邊方咸能悟解維摩經云廣流
布於閻浮提)。】

「　年流日月飛」


【(歎出遷流刧成刧壞浮生瞬息石大電光

日月輪廻若生滅法我佛妙用誰能覺知維摩經云或現刧盡燒照令知無常)。】

「　愚營身外事」


【(外相營營曾無暫息內照寂寂智用疑然彌勒頌云愚人看似
妄智者見唯真)。】

「　豈曉法中機」


【(既拘情想豈悟玄機達正覺之宗乘證無
生之法忍維摩經云除彼不肖人癡𡨋無智者)。】


「　智種因中得」


【(四智現果起自因中五位修真聞熏習得真諦論云下佛
種子於眾生因)。】

「　能祛妄想為」


【(妄謂客塵想謂心垢無漏智火能燒惑薪
染法既除真性清淨維摩經云無妄想是淨)。】

「　忻逢隨意釋」


【(忻逢法教
頂受無猒了悟真空隨意開釋投之未悟令獲本真淨名經云應大忻慶頂受此法)。】

「　魔境暗消疑」


【(未了真覺魔境惑心若證二空疑暗消遣華嚴經云十千諸魔
發菩提心)。】


「　受福力無窮」


【(智悟圓成名為受福真理難屈乃力無窮維摩經云我今
略說不可思議解脫之力窮劫無盡)。】

「　如來皆實記」


【(童子戲沙如來授記
行因雖少感果彌多報應之徵故言皆實維摩經云世尊授仁者記)。】

「　果從天上捐」


【(果謂佛果即護明菩薩捐天上快樂下降人間菩提樹下成無上覺維摩經云得甘
露滅覺道成)。】

「　五蘊無遮閇」


【(既成佛果五蘊皆空周遍十方故無遮閇
維摩經云洞達五受陰空故)。】


「　愚能守一善」


【(一謂純精無雜善謂順理益生捨偽求真證寂滅樂法華
經云純一無雜)。】

「　動靜合玄機」


【(動謂彌綸法界靜謂內𢍆一真雖現和光
玄機自𢍆維摩經云常自靜故)。】

「　六賊皆自散」


【(六塵擾乱迷妄相仍動靜
皆空幻塵自散金剛經頌云因塵皆是幻)。】

「　真緣好仗依」


【(真常佛性信心
所緣三身菩提人天依仗起信論云常念真如法故)。】


「　舒展三千界」


【(法性無跡舒展而俱遍三千實際無邊智用而體周沙界

華嚴經云法性遍在一切處)。】

「　真空不強為」


【(真空法身體非強弱自然
正覺無法不包維摩經云我觀如來無強無弱)。】

「　蓮花生水底」


【(蓮華喻於
佛性水喻法也佛性清淨若蓮出淤泥法演大千如水能潤下淨名經云卑濕於泥乃生
此花)。】

「　智慧達人知」


【(諸佛智慧妙盡宗源非凡情之所知乃達人之可入
法華經云善入佛慧通達大智)。】


「　結盡千花子」


【(花表輪王千子曩結願因於此賢劫之中盡當作佛維摩
經云其王千子即賢劫中千佛是也)。】

「　如來救度知」


【(如來大悲知根熟者
度觀有苦者救普益眾生平等與樂淨名經云稽首能度諸世間)。】

「　黃金銀世界」


【(黃金者表佛土純淨可寶重故銀世界者彰行願無瑕皆無漏故寶積經云無
量功德寶莊嚴土)。】

「　佛法道隆時」


【(諸佛出世法教流行三寶欝興道隆
永刧歸依者皆蒙果證見聞者盡發菩提淨名經云紹隆三寶能使不絕)。】


「　化度眾生業」


【(益物興悲利樂為業慈流大化令度苦源遺教經云所應
度者皆巳度訖)。】

「　法流無所滯」


【(法澤普潤流布十方動植霑恩故無所滯
唯識論云法駃流中任運轉故)。】

「　壯心展臂時」


【(未悟真常久縈三界壯心
如展剎𨚗頓圓彌勒經云譬如壯士屈伸臂頃)。】

「　去偽歸真諦」


【(心猿既伏
意馬自調去除虛偽之情真諦安然可見淨名經云諦是道塲無虛偽故)。】


「　法諭流三界」


【(法諭者佛至教也能悟真法出離三界佛垂慈諭普及含
生維摩經云三轉法輪於大千)。】

「　圓眀照十方」


【(理生智解洞照十方性相
俱融圓明無礙法華經云光明照十方)。】

「　觀音常自在」


【(觀音菩薩悲智
自在救諸苦惱應病與藥功無不利慧無不用楞嚴經云觀音自在力)。】

「　善種福無疆」


【(善能利物為世導師福慧無彊含生普益法華經云福聚海無量)。】


「　旨妙因緣境」


【(因緣境寂妙旨圓通善了唯心真常自現維摩經云以因

緣故諸法生)。】

「　威儀大化楊」


【(世尊演法四威儀中闡大慈悲敷演聖化
維摩經云諸佛威儀而作佛事)。】

「　如來真密諭」


【(如來密語於道幽深證妙
圓通詮上乘義淨名經云諸佛秘藏)。】

「　殊勝演為祥」


【(法皆殊勝道𢍆吉祥
言教無窮咸利群有佛地經云名稱吉祥及尊貴)。】


「　玄微平海岳」


【(法性玄微量超海岳智心平等遍滿真如華嚴經云證平
等法心無高下)。】

「　性相實諸方」


【(性相本真無明起惑能證實義廣遍十方
華嚴經云唯願證知諸法實性)。】

「　應現無差別」


【(應物降生興慈雲之普覆
現身說法何怨親之差殊法華經云常為一切平等說法)。】

「　隨機化道塲」



【(化䪺漸機設實權教蘊積眾善高坐道場法華經云我始坐道場觀樹亦經行)。】


「　俗諦門庭事」


【(凡夫違真常遊俗諦事依心起假号門庭法華經云譬如
長者有一大宅又云唯有一門)。】

「　迷情終日醉」


【(心迷塵境終日纏綿情著
愛貪長時昬醉唯識論云昬醉纏心曾無醒覺)。】

「　分眀至理言」


【(至極言論
理甚分眀小智聞之迷惑不悟維摩經云眀宣無常苦空無我寂滅)。】

「　大道玄幽細」


【(佛道無上号曰玄幽八萬威儀大乘細行淨名經云欲行大道其示小侄)。】


「　甚深依妙理」


【(二空妙理實曰甚深萬法所依出生諸善故華嚴經云微
妙定所依)。】

「　智慧福延昌」


【(智慧萬行之體福為眾德之源若乃行願齊修
是即延昌福智華嚴經云智慧誘誨是菩薩福德)。】

「　利益眾生苦」


【(聖人
興悲念眾生苦捨淨居穢利益遊心法華經云眾生沒在苦引之令得出)。】

「　菩提願久長」


【(因中四誓果極而三願巳圓菩提智中利生久長而何盡寶積經云
我為智者說菩提應說一切法法量)。】


「　大善勝諸𢙣」


【(善謂無為無漏善也體深用廣稱之為大超諸有漏是勝
諸𢙣梵網經云教化是𢙣人令生大乘善法)。】

「　施張徧八荒」


【(王之有道

恩及遠荒佛住無為法施沙界華嚴經云廣大周遍)。】

「　去邪歸正道」


【(正道
乃佛之正法也設聞外道種種虛妄說法不生邪見正道自歸楞嚴經云諸善男子堅凝
正心魔不得便)。】

「　時敘自然康」


【(時敘謂時之次苐既住正道於一切時邪
魔不侵自然康樂佛地論云坦然安坐妙菩提座)。】


「　信解真如教」


【(信謂心淨除疑解謂印境决定俱達教旨同了真如唯識
論云於三寶真淨德中深信忍故)。】

「　威雄振法王」


【(居尊化物示現威雄
破諸魔軍神通遠振辯說自在稱為法王法華經云大雄猛世尊諸釋之法王)。】

「　深窮清淨理」


【(智断具足号曰深窮理自恒真名為清淨寶積經云見一切法自性
清淨)。】

「　聖語遠昭彰」


【(詞法義音乃為聖語理事明著号曰昭彰千古遠流
誠可依信法華經云我今分明宣說汝等)。】


「　報現兩般心」


【(不悟一真強生二見了斯幽旨兩心自亡金剛經云修一
切善法則得菩提維摩經云殺生是殺生報)。】

「　斯言終失墜」


【(斯言之意
指前二心固執情堅誠終失墜淨名經云解脫者無所言說故)。】

「　千經與萬論」


【(千經萬論諸聖真談極則玄機佛理自顯淨名經云天帝白佛言我從佛聞百千經
典)。】

「　一悟皆玄𢍆」


【(一悟了達萬法皆空玄𢍆真常名言都泯維摩經云
達諸法相無𦊱礙故)。】


「　智海深無極」


【(法華經云諸佛智慧甚深無量海即深奧之義為佛果中
無漏智起證法身時無有邊極也)。】

「　真空理要長」


【(真即真正謂佛断盡
煩惱所知二障顯證二空圓常之理一得永得更無窮盡法華經云世間相常住)。】

「　分眀歸有實」


【(分毫證盡明解真源即佛果位中涅槃寶所有實體性法華經
云唯此一事實)。】

「　餘外更無方」


【(法華論云除一切智智更無餘事為唯有
法身理智一大乘外其餘有空莫能比方)。】



「　我見眾生性」


【(法華經云我以佛眼觀見六道眾生為佛果中起後得智
觀一切眾生悉有佛性)。】

「　迷途認法王」


【(迷者墜墮生死名曰迷途為眾
生未悟大乘不能信受法華經云迷惑不信受妄認法王)。】

「　若能依體用」



【(法華經云知此眾生種相體性若依一乘理行六度行作用修行必成佛道)。】

「　妙覺可稱揚」


【(妙者妙理覺行圓滿法中稱揚大乘正法法華經云今復轉㝡妙無上大
法輪)。】


「　真語非常意」


【(金剛經云如來是真語者實語者佛後得智中說三妙觀
得三身果因果决定非常意也)。】

「　冥心坐道塲」


【(冥者默義即佛大悲心中
起現化身示現王宮生降伏眾魔菩提樹下坐金剛座因成果滿名坐道塲法華經云我
始坐道塲)。】

「　無為歸解悟」


【(無為法身湏歸悟解起根本智證二空理法華
經云悟佛知見出現於世)。】

「　上聖轉高昌」


【(佛為聖人證一乘法断二種生
死道德遐昌更無過上法華經云於諸世間為無有上)。】


「　應用千途遙」


【(依理起用應機說法法華經云為求菩薩者說應六波羅
蜜巧化千途萬般逕路)。】

「　善心無上逝」


【(如來善心更無過上法華經云
無上士調御丈夫逝者往也)。】

「　福來一念時」


【(為佛修百福因來至果位
相好圓滿法華經云百福身莊嚴始從一念發心時也)。】

「　利樂先施慧」



【(佛依玄理起於智慧觀眾生性隨機教化說法饒益拔苦與樂法華經云多所饒益安樂
眾生)。】


「　平等恒沙際」


【(大乘法性無有高下名曰平等法華經云大乘平等法即
諭如恒河沙[婁*殳]了無邊際)。】

「　圓中法異常」


【(圓成妙理真實無為即佛法身
凝然常在體離生滅法華經云常在此不滅)。】

「　化超千萬億」


【(法華經云
超出成正覺意言佛現世間出過千萬億一切眾生也)。】

「　慈眼大開張」




【(慈悲之眼觀視眾生廣宣教法輪大法輪法華經云慈眼視眾生福聚海無量)。】


「　說法無窮盡」


【(說者演也法即教法聲名句文宣暢不息名無窮盡法華
經云無量億劫說不能盡)。】

「　如來順世情」


【(如來者佛之通号善應玄機名
順世情法華經云隨順世間若干種性)。】

「　靈山高會上」


【(靈山高峻表一
乘出過二乘佛常在此山宣說妙法法華經云常在靈鷲山)。】

「　語話甚分明」


【(語話者談說詮文顯所詮理大乘法相分明示現法華經云分明說諸法)。】


「　理性深何遠」


【(理性者即法界性迷之則一一差異悟之乃本性真常法
華經云知法常無性)。】

「　玄源意轉精」


【(玄謂玄妙源謂真源悟之意轉精通
迷之心多惑亂金剛經云欲達無生路應當識本源)。】

「　圓通諸果證」


【(圓者
圓妙通謂通達楞嚴經云佛遣二十五聖證入圓通皆獲果位)。】

「　覺海湛然清」


【(覺謂妙覺海即性海不動不搖絕塵絕染楞嚴經云覺海性澄圓圓澄覺元妙)。】


「　成道在今辰」


【(佛示現出世為救眾生耶今生成道證無為也法華經云
佛得道甚近)。】

「　高天接厚地」


【(如來在人天之上名高㝡尊勝故佛号為
天法華經云㝡後天中天号曰然燈佛佛接引眾生離深厚生死之地也)。】

「　無以報佛恩」


【(佛居淨土非聖非凡隨機教化實為恩深無以報也法華經云世尊大恩無量
億刧誰能報者)。】

「　慈護龍天衛」


【(慈謂慈悲護念眾生常無疲倦法華經云
諸天晝夜常為法故而衛護之)。】


「　詮法傳真諦」


【(佛佛相傳印印相𢍆如來妙旨化諭多方金剛經頌云有
形終不大無相乃為真)。】

「　寬容徧八紘」


【(法身妙淨徧滿十方妙善通容
咸令悟解金剛經頌云圓明法界內無處不寬容)。】

「　大慈垂惣要」


【(唯佛
慈心常垂方便萬法之要絕離一源法華經云以大慈悲力度苦惱眾生)。】

「　道果達無生」


【(佛具無漏智悉證無為理體顯然離諸生滅法華經云今者見世尊安隱

成佛道)。】


「　求道無心是」


【(求謂忻求道謂佛道無心者断執著緣無取無捨唯心澄
湛是真道矣法華經云求道過七日得諸佛智)。】

「　冥然合上清」


【(真心寂默
妙理通流𡨋𢍆上乘萬法清淨法華論云進取上上清淨義故)。】

「　真空非語默」


【(妙空無相語默皆真理體離言非語非默金剛頌云去來終莫見語默永無方)。】

「　妙用智縱橫」


【(法華經云是乘微妙清淨苐一即顯真智縱橫證法界理也)。】


「　解悟開諸暗」


【(佛證法身於法解悟断盡煩惱非明暗相法華經云無明
暗蔽永盡無餘)。】

「　舒張自作程」


【(圓成妙理遍周法界卷舒無礙體理圓成
佛道法華經云自作若使人皆以成佛道)。】

「　因緣垂一問」


【(因佛放光大眾
生疑彌勒發問文殊為荅法華論云問於一人多人欲聞生希有心)。】

「　萬法坦然平」


【(真如法性無為萬德即觀此理無有高下金剛頌云常能作此觀真妄坦然平)。】


「　就本論其聖」


【(始從凡位終至菩提皆在真空法性海中維摩經云無住
為本)。】

「　光眀普照世」


【(如來放光普照世界約理談妙事理寂然寂而恒照
楞嚴經云從肉髻間涌出光明遍照法界)。】

「　唯心道覺緣」


【(心含萬法法妙
一真至道無私非緣非覺仁王經云十善菩薩發大心長別三界苦輪海)。】

「　智以生慈慧」


【(以智為因能生慈悲拔苦與樂慧濟眾生法華經云慈悲於一切度
脫苦眾生)。】


「　證果生千刧」


【(證謂證入果謂道果法華經云自證無上道即佛具修萬
行歷生千刧方證佛果)。】

「　安居善化成」


【(即佛安住大乘善化群品眾生
本性咸能悟觧法華經云善能教化眾生之類)。】

「　周知無所礙」


【(謂無罣礙
周遍了知法性皆空無所不利法華經云如來觀知一切諸法通達無礙)。】

「　大法救眾生」


【(一切有情稟教依行断𢙣修善捨凡成聖法華經云說大乘經名妙

法蓮華)。】


「　志欲超三界」


【(棄彼六塵超昇三界欲界色界無色界法華經云如來巳
離三界火宅)。】

「　緣來有重輕」


【(法皆平等緣非輕重為執因緣而分輕重
能祛執妄三界寂然法華經云眾生垢重)。】

「　非心終是悟」


【(因位妄執未證
真常靜慮澄思終成佛道法華論云悟者巳發菩提心入佛法故)。】

「　豈獨化無情」


【(慈悲功行普利含生情與無情化現開悟法華經云憐愍教化利益我等)。】


「　果喻恒沙數」


【(法華經云如是大果報種種性相義諭者譬諭如恒河沙
數無有限量佛果亦然)。】

「　昬衢執有情」


【(有情妄執若履昬衢生死涅槃
不能出離寶積經云隨執著造業由於煩惱故)。】

「　寂然無上智」


【(無上智即
佛轉因位苐八識得果中大圓鑒智本體寂然法華經云得無上智慧)。】

「　萬法自圓成」


【(法身萬德受用圓滿化喻群生智遍成就法華經云度不可計眾成
就一切智)。】


「　觀曉如來教」


【(觀謂觀察曉悟真常栖心典誥大乘教法一味無差法華
經云觀諸法性無有二相)。】

「　慈悲度一切」


【(興慈運悲廣度諸苦法華經云
大慈悲為室柔和忍辱衣度謂度脫眾生無量故名一切)。】

「　山河時暫間」



【(山河大地噐世間也成住壞空不能長久故彰時暫楞嚴經云如彼山河擊則成炎鎔則
成水非久遠矣)。】

「　至道還𡨋𢍆」


【(法華經云我今成佛道還冥法界𢍆合真
理故)。】


「　因地行脩證」


【(佛性為因地依七地四菩薩而脩大行證者佛道圓滿法
華經云證於無上道)。】

「　無差釋子名」


【(法華經云佛昔從釋種子者佛之弟
子即一切出家人常脩梵行惣名釋子)。】

「　攝心歸正念」


【(攝諸妄心不逐
外境即歸於真正之念楞嚴經云攝念未久身心忽空即歸正念)。】

「　真理未



【(真如法性體無生滅名未嘗傾楞嚴經云而如來藏隨為色空相傾相𡙸
而彼性未曾動)。】


「　義達僧祇劫」


【(凡所修行經三大阿僧祇劫方成正覺法華經云過阿僧
祇刧乃成等正覺)。】

「　還隨福業亭」


【(福即六度業謂行業即萬行隨因中
福慧亭修方成佛道法華經中種種修福慧皆巳成佛道)。】

「　能求真妙法」



【(去妄成真達玄通妙凝然正法即微妙理也法華經云甚深微妙法)。】

「　佛事敢言輕」


【(有四智菩提即佛一大事深奧功德豈敢言輕法華經云諸佛以一大事
因緣故出現於世)。】


「　法教依玄理」


【(法華論云教菩薩法者為教化隨器法成就故顯依玄妙
法界理流出)。】

「　還將諸會同」


【(說法隨方無問雙林鹿苑教宣一味根逐
淺深華嚴經云同會一法道塲中)。】

「　金人來入夢」


【(法華經云又見諸佛
身相金色後漢明帝夢金人身長丈六佛法之始)。】

「　聖教遠流通」


【(迦葉
摩騰及笁法蘭遠賷聖言流傳東土法華經云閻浮提內廣令流布)。】


「　見善照諸緣」


【(見悟善性當體即真照諸境緣無所起滅也楞嚴經云緣
所緣盡即真性顯現)。】

「　麤針求縫細」


【(理事皆空攝相歸性冥合真空非麤
非細華嚴經云麤者無所有微細亦復然)。】

「　一塵障大海」


【(海即佛智海也
自體凝皎靜相無動華嚴經云不動如山智如海)。】

「　妄起無邊際」


【(唯識
論云如天忽雲如鑑忽塵若息妄旋真當處寂湛若隨緣興變即無邊際)。】
