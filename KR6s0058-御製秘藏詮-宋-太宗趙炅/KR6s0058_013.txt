










御製秘藏詮卷第十三　　　　　　駕


「　理道相緣習」


【(理道則往種習相緣在聖不增處凢匪減斯則聖人起智
断惑遊履之通津不真空論云道遠乎哉觸事而真)。】

「　經文亦大同」


【(聖言無漏
詮表幽宗令諸有情究義窮文乃尋蹄而獲兔苟或廢筌明旨者則殊途而同歸華嚴經
云文隨於義義隨文)。】

「　如來慈悲智」


【(佛地論云如來則果稱也以慈能與
樂悲而拔苦智乃後得涉俗利生遊刃萬機斷除疑障)。】

「　徧布薩訶中」



【(彌布法雲徧從物欲薩訶者梵語此云堪忍即指處也前以仗佛為緣後乃言音廣徧)。】


「　詮演千經論」


【(佛之所說名曰𢍆經以詮表勝義貫而復攝菩薩所造目
之為論以研窮性相㭊理精微以經以論文之不一故曰千也)。】

「　真乘無不通」


【(唯識論云真者離妄得名乘乃運載為功以文以義詮表真乘令悟達幽宗理而通
貫乃造無為也)。】

「　眾生得佛意」


【(佛地論云眾生者即五蘊和合假而不一
故曰眾生佛即宣揚秘賾使味代有情得意而忘言也)。】

「　覺性自惺𢛌」



【(覺者智也性者體也以覺悟智體神虛性朗運禪郍於慧苑遊常樂之淨都是謂惺𢛌)。】


「　求佛不求法」


【(佛即覺者法則真理瑜伽論云謂三賢位菩薩聖智未生
執有菩提可求眾生可度執著佛求不明於法也)。】

「　佛法亦容通」


【(佛法
巳見上文以地前菩薩解心若起𢍆會真如垢障斯盡當成佛也華嚴經云佛即法法即
佛是容通也)。】

「　求法不求佛」


【(既若證法𢍆會真如不分能所無能證智
亦無所證如故不求佛金剛經頌云有身非覺體無相乃真形)。】

「　佛法自昬曚」


【(求法者迷於佛求佛者迷於法又不知佛即法法即佛也既自相迷是曰昬曚)。】


「　昬衢人似醉」


【(昬衢不曉人我空高無智慧光增煩惱暗不遇良友似醉
如癡法華經云醉酒而臥不覺內衣裏有無價寶珠)。】

「　見用何堪述」


【(非識

所思非言所議如蓮花清淨出於游泥淨名經云塵勞之儔皆如來種)。】

「　若了明中暗」


【(暗則煩惱明乃真心正智生時断除暗障攝論云眀來暗謝智起惑忘)。】

「　十方開慧日」


【(既自覺了悲他未悟開發智心猶如慧日朗而且徧故曰十方
普賢行經云眾罪如霜靈慧日能銷除)。】


「　解法不知佛」


【(無心趣物始契真如智合理𡨋心將慮絕尚非思議云何知佛
楞嚴經云我以海印發暉汝暫舉心塵勞先起)。】

「　佛心大化從」


【(佛心者無心也
以無得之心化如空之眾是為大矣金剛三昧經云欲化眾生無生於化不生無化其化
大焉)。】

「　諸邪非正法」


【(群邪謬解計執誠多佛智深觀皆非正法百法論云
邪徒乱轍也)。】

「　佛豈恕凢庸」


【(佛慈廣大利物為功憎愛殊無冤親等視
華嚴經云譬如日光無私而現亦為生盲人作利益事)。】


「　證悟皆因果」


【(證悟者印𢍆也謂印𢍆佛心了真空而體寂精研慧思達
因果之深根法界觀云迷時三界有悟乃十方空)。】

「　非因豈異同」


【(真際
非因誰云同異魚既獲而筌亦忘方明佛旨病既差而藥亦棄始曰通人十地論云非同
非異)。】

「　圓成無𦊱礙」


【(圓成真實理絕名詮不揀自他湛然清淨離諸障礙
種習俱除金剛經頌云智入圓成理)。】

「　視現不爭功」


【(心鑑洞明萬象影現
無私而照是不爭功光境無根虛通絕謂華嚴經云猶如海印現無前後)。】


「　志散求還聚」


【(志散者動也還聚者靜也謂心若散乱攝之令聚其若聚
散同根則動靜一䡄放光般若經云法無去來無動轉也)。】

「　聚散本來空」



【(緣會而生名之曰聚緣離而滅目之為散聚散緣虛真空獨淨中觀論云因緣所生法我
說即是空)。】

「　欲知何諦當」


【(想念若生情思倐構當深諦察絕謂冲虛不真
空論云聖遠乎哉體之則神也)。】

「　萬法眾心同」


【(情由識起萬法是生照達
性真群心一致味其幽旨良在觀心起信論云心生種種法生)。】



「　蒙昧若羅縠」


【(以十地聖人每一地中有一障二愚蒙蔽暗昧障其淨心
不明了故瑜伽論云似隔羅縠見彼遠象)。】

「　翻拘十地律」


【(性本圓明不能
照了翻拘律縛礙彼智心華嚴經云若有菩薩聞此不思議法門不信不入不名真實菩
薩)。】

「　階梯來往用」


【(往來者三世也謂三世諸聖皆依此道階級而備苟
或得旨忘詮者楞伽經云不從一地至一地)。】

「　欲界見何物」


【(欲界以樂
欲心多種種差別若乃即物而真則當何所見不遷論云近而不可知者其唯物性乎)。】


「　果亦因何有」


【(果不自立必假因成因不自名仗果方顯既因果之同根
亦相名而俱離圓覺經云生死涅槃猶如昨夢)。】

「　果成因亦同」


【(真本無名
果成示表窮乎至趣因亦同然華嚴經云眾生心中有佛念念成等正覺)。】

「　還如因證果」


【(因同實際證果還如無相理明始終不異華嚴經云應以佛身而作
眾生身)。】

「　無事可相宗」


【(華嚴經云因同果海果徹因源靜乱俱忘湛猶
停海則何事理而可相宗也)。】


「　因地從於果」


【(未造玄門猶從因地無心荷物豈憚勞勤益濟自他方期
證果華嚴經云虛空界盡我願乃盡)。】

「　果圓宗自通」


【(果達圓成宗歸至趣
旨明詮廢性乃自通唯識論云菩薩成佛時根根塵塵周遍法界)。】

「　無因復無果」


【(情虛體露說彼無因覺智初成強名曰果其若神居物表因果何存金剛
經頌云非因亦非果)。】

「　因果亦歸空」


【(混融性相泯絕果因淨若大空量含
真際淨名經云畢然空寂舍)。】


「　義切方求法」


【(言前了義方曰切求句下尋名誠非得法諒唯神會豈是
心知法華經云是法非思量分別之所能解)。】

「　韜光迷自通」


【(𠕀象玄珠
徒窺赤水韜光息慮迷性自通金剛經頌云澄神生覺性息慮滅迷情)。】

「　不論兼與悟」


【(妄生智立黃葉止啼病起藥興空花為喻迷既不有悟亦何存楞嚴經云

言妄顯諸真妄真同二妄)。】

「　雨施潤溟濛」


【(大施門開無論得失真宗玄𢍆
法雨溟濛法華經云雨大法雨)。】


「　香剎極無邊」


【(香城粉骨洒血求真雪嶺捐軀心聆半偈塵沙莫數
浩劫無邊華嚴經云我於無量無邊劫發大菩提心)。】

「　慈音甚秘密」


【(常啼
大士誓募慈音秘密琅函𡨋心開啓了無所得實相方明辯中邊論云說智反智處)。】

「　法門性自然」


【(心法無形門通法界不因推度性本自然楞嚴經云本自其然
無然非自)。】

「　不出如何入」


【(迷於事境似有攀緣悟達真宗了無出入楞伽
經云離言自證法不出亦不入)。】


「　法宗何以故」


【(法持真䡄宗本離言表示不生如何啓悟華嚴經云佛法
無人說雖慧莫能了)。】

「　境相利緣通」


【(境生相立利慧方除見旨悟心緣空
照絕正理論云現現別轉利貫通緣)。】

「　寂滅離諸相」


【(寂然不動滅𢍆無生
諸相都無真心獨鑑淨名經云寂滅具菩提滅諸相故)。】

「　修行聖化中」


【(精修
練行順益自他利叶玄規剋彰聖化華嚴經云善財童子入普賢菩薩一毛孔中修行問
法經不可說不可說劫)。】


「　旨傳千億刧」


【(離言宗旨唯自心傳一念相應悟通多刧華嚴經云無量
無邊劫知之即一念)。】

「　顛倒理難窮」


【(真心迷暗虛妄難窮覺了緣生理無
顛倒楞伽經云非智慧者見有顛倒有智慧見不生分別)。】

「　法雨施迴向」


【(法雨
遍滋利該空界大施趣向深廣無涯大乘密嚴經云愛樂佛菩提精專以迴向)。】

「　菩提義種通」


【(菩提大義含識皆同悟證方通速成種智淨名經云一切眾生即菩
提相)。】


「　玄邈杳難窮」


【(玄宗絕相杳邈難窮寥廓太虛言思不及華嚴經云如來
真境界其量等虛空)。】

「　真元造化同」


【(真元者真如體也造化者有為萬法

也以真如與萬法㸦作依持故云同也起信論云真如隨緣流轉三趣)。】

「　了然還自悟」


【(真性了然還原自悟冲幽寂靜悟澹怡神楞嚴經云圓明了知不因心念
如是修持斯為第一)。】

「　諸聖證其功」


【(登地諸聖親證真如行盡果成功超
千地大乘密嚴經云聖人自智境依此入諸地自利及利他功業悉成就)。】


「　風搖萬樹葉」


【(無量壽經云微風吹動諸寶行樹及寶羅網出微妙音一
切眾生聞是音者皆自然生念佛法僧得入律行)。】

「　還取自真實」


【(霜殞
負存自唯真實還源見旨識末知根法華經云我今此眾無復枝葉純有真實)。】

「　從此作何觀」


【(作何觀者淨名經云觀身實相觀佛亦然我觀如來前際不來後際
不去今亦無住斯乃正觀也)。】

「　終朝無了日」


【(晨鍾夕磬世相而似有遷
流悟遠緣空真理乃本無生滅華嚴經云如來得菩提實不計於日)。】


「　深邃行方便」


【(幽邃冲深良開方便月因指見心得智明法華經云第一
方便甚深微妙)。】

「　其如理要宗」


【(要旨玄宗如神禦理出思慮之表非可度
量所以善吉無言淨名杜口法華經云是法不可示言詞相寂滅)。】

「　一言道意實」


【(意無思審實際可歸萬有皆真一言合道文殊菩薩偈云一念淨心等正覺)。】


「　益顯自周通」


【(淨照虛通靈明益顯蘊法界於心極廓宇宙以無疆華嚴
經云智慧自在圓通法界)。】


「　大海波瀾遠」


【(注洋性海實際無涯心識波瀾虛舟恒濟心通至土彼岸
何遙起信論云無量功德藏法性真如海)。】

「　門開悉會同」


【(智慧門開會滄
溟於㳙滴真空月照印心水以皆同華嚴經云如來智慧月普應諸心水)。】

「　混然歸一味」


【(湛然性相光影虛含本末交㕘浪同一味楞嚴經云無漏真淨云何於中
更容他物)。】

「　萬派盡朝宗」


【(殊源異𣲖遠近朝宗九流謂乎淵府眾聖以之
𡨋會仁王經云佛法眾海三寶藏無量功德於中攝)。】



「　信意無拘束」


【(信運縱撗靜乱無非佛事意無拘束順逆皆是道場圓覺
經云智慧愚癡通為般若)。】

「　隨緣任所從」


【(隨緣得趣任物皆真非下士所
知則上流境界華嚴經云菩薩成就深心力普於諸法得自在)。】

「　歡喜與煩惱」


【(歡喜則神怡性恱煩惱則智昧心憂悟憂喜以同無乃慧明而獨照寶積經云煩惱
界性即佛界性菩薩聞時心生歡喜)。】

「　何處不容通」


【(淨慧生時以日輪而
高鑒大光普照何事相而不通華嚴經云其心廣大猶如虛空)。】


「　中道得安心」


【(深達有空乃為中道了知妄念是曰安心苟或妙悟緣生
則有何中畔仁王經云了達此無二真入第一義)。】

「　忙中減妄逸」


【(不修
靜慮是曰忙中染著心塵謂之妄逸其若迴光寂照則虛妄元無楞嚴經云狂性自歇歇
即菩提)。】

「　如香積施飰」


【(淨名經云四海有竭此飯無盡以戒定慧解脫
知見香之所熏修乃至廣作佛事)。】

「　福種因無失」


【(淨心福種誠為無漏
之因益濟利深克處智身之土淨名經云欲淨其土當淨其心隨其心淨即佛土淨)。】


「　皆願過諸巳」


【(謂佛智願超諸十地出過二乘巳證解脫瑜伽論云如實
安立清淨妙智為利有情起此願故)。】

「　其如見不同」


【(唯識論云其如見道
謂觀現前苦等四諦各有二心一忍二智開說有八合說為一)。】

「　無情與愛境」


【(非相之境乃達無情至理之源方空妙用緣起經云若由欲愛造諸福行)。】

「　達識悟真空」


【(謂於見道悟彼能取所取是無故名真空唯識論云通達唯識真勝
義性)。】


「　我鄙眾先見」


【(謂觀前因不趣佛果後發心心志願成就無上菩提法華
經云成就大志自鄙先心)。】

「　苦心定有功」


【(苦心不倦謂佛修行功利群生
妙圓萬德法華經云戰有功者心亦歡喜賜以禪定)。】

「　常如病施藥」


【(佛是
醫三法名良藥療眾生煩惱病令得安樂維摩經云如大醫王善療眾病)。】

「　慧解自



【(唯識論云慧能决断勝解印持心王心所自在無礙故名流通)。】


「　真如唯一法」


【(真如者諸法之性勝義之理唯識論云此諸法性義亦即是真
如於一切位體本凝然更不分別故唯一法)。】

「　何法不通風」


【(法華經云
以何法念以何法思以何法修謂諸如來清淨法界本離動搖體性真實名不通風矣)。】


「　無心無事法」


【(無心者唯識論云心及心所田無始來虛妄分別都無實
體無事法者幻事假法亦無所有)。】

「　心法遞相宗」


【(百法論云心王色法
更相為依㸦顯其理心藉法而為宗法藉心而明了故云遞相宗也)。】


「　天雨如花會」


【(謂佛於靈山會說一乘法先乃放光雨花動地集諸大眾
皆至佛所法華經云天雨曼陁花等)。】

「　大仙依法律」


【(佛是大仙從初發心
依於法則秉持律儀得成道果法華經云如三世諸佛說法之儀式)。】

「　有情俗諦門」


【(若有情執不了真門世俗皆假勝義諦真唯識論云四勝義勝義謂一真法界)。】


「　亦可攀緣得」


【(唯法清淨不可攀緣若著有邊不断迷執維摩經云若無
所得則無攀緣)。】


「　慧遠曾行步」


【(不獨智見常以慧照智慧證寂行於道塲觀所緣論云遠
行及獨行無身𥧌於窟)。】

「　無疑三際同」


【(疑心失照深入妄源三際融通
幽隱佛性金剛經云過去未來現在之心皆不可得)。】

「　稻麻兼竹葦」


【(外道
迷執[婁*殳]如稻麻小乘謬解似同竹葦豈可測於佛之智也法華經云如稻麻竹葦充滿十
方𠛴)。】

「　至理一真空」


【(智達至理解悟真空妄計分別都無所有大般若經
云有為空無為空)。】


「　說法周知理」


【(借事顯理故云說法因理喻性性實妄言金剛經云無法可說
是名說法)。】

「　空歸實相中」


【(法喻實祖趣向菩提空有二邊俱無所立雖歸
中道亦無所得維摩經云不在內外不在中間)。】

「　但求前與後」


【(自心明了

不起妄塵菩提本空何拘前後非求非證乃達本源維摩經云不著佛求不著法求)。】

「　妙道更無窮」


【(佛之妙道玄理難窮深遊不二之門大運無邊之法法華經云
一乘一妙道)。】


「　理洞如來法」


【(真理寬廓性本實常洞達圓成含容法界如來法者金剛經云
無所從來亦無所去故名如來)。】

「　廣施慧難窮」


【(廣者唯識論云無上菩提
廣大甚深施慧者謂財法無畏生法二空施亦廣多慧濟無邊故云難窮)。】

「　誰能柱兔角」


【(誰能者法華經云誰能為我說大乘者柱兔角者唯識論云若二分
是遍計所執應如兔角體非有故)。】

「　終有是非同」


【(終有其理還無作用
非是同者悟了則是同不解即非同法華經云終歸於空)。】


「　精進無高下」


【(攝大乘論云謂無怯弱無退轉加行勇猛不捨善軛名為
精進更無高下平等而轉)。】

「　湏依戒定律」


【(唯識論云戒有三種一律儀戒
二攝善法戒三饒益有情戒依此三種防護身心戒𢙣修善安住靜慮得成道果)。】

「　勿差一念間」


【(瑜伽論云一念一𠛴郍得證百三摩地以淨天眼見諸佛土更
無分別)。】

「　眼見光眀日」


【(般若經云佛眼如千日照異體還同若菩薩之
修行不住法之布施)。】


「　詮秘真如藏」


【(包含眾德開示妙門令彼群生達如來藏見佛真性頓徹
法源法華經䟽云𢍆秘𧷤之微詮)。】

「　聖語普皆同」


【(過現未來三世諸佛
一音始震万法皆如法華經云皆說一乘法化無量眾生)。】

「　信心若相門」



【(如來諦信功利恒沙隨佛問訊了無所得般若經頌云信心生一念諸佛盡能知)。】

「　莫作耳邊風」


【(百法論云耳識能了於聲境謂佛說法令人解悟無智疑惑則
不信受如耳邊風)。】


「　求之不可得」


【(唯識論云求住唯識真性義性一無數劫三種鍊磨真實

義性猶不可得)。】

「　俄然有自逢」


【(一𠛴那中頓悟至理不除煩惱自證菩提
維摩經云一切煩惱皆是佛種)。】

「　迷情與實相」


【(迷妄之情去道遠矣真實
之法一味圓融唯識論云謂於二空有迷謬者生正解故)。】

「　蓮開業水中」



【(蓮生淤泥性淨無染真性出於煩惱妄緣起於性中業累所昬如水浸閏無量義䟽云亦
名妙法蓮華)。】


「　解即祛煩惱」


【(解即妙解祛即遣除煩惱如明來暗謝智起惑亡唯識論
云由断煩惱障證真解脫)。】

「　視之似若空」


【(理本無形覽之若空性非好醜
視之不見唯識論云起如實智印二取空由帶相故)。】

「　有情俱是妄」


【(謂流
轉五趣執法執人不求菩提妄生分別金剛頌云有情俱是妄無執乃名真)。】

「　妙用智西東」


【(謂佛後得智生大悲心興起妙用化利有情法華經云東西馳走䟽
云東喻善趣西喻𢙣趣勝劣殊故)。】


「　嚴持香與花」


【(真性不昧實理現前嚴潔香花來詣佛所法華經云香風
吹萎花更雨新好者)。】

「　普供人天日」


【(以心喻香以覺喻花心覺清淨不生
分別達法本際普供人天當是日也法華經云慧日大聖尊)。】

「　迦葉信心時」


【(佛說妙法各生信心深知密意佛與授記法華經云我此弟子摩訶迦葉)。】

「　如來說法畢」


【(因機說法令生信解迦葉親聞說法巳竟法華經云如我所說法當
生大信力世尊法久後要當說真實)。】


「　證真知染性」


【(證解菩提體性明淨故無染污唯一真如唯識論云即證
真如智也)。】

「　薄伽梵說通」


【(慈恩䟽云薄伽聲也梵謂具德若有為此薄伽
梵聲必具六德能破四魔體用貫通說成十法)。】

「　分明無盖覆」


【(分明指喻
謂法界性本離盖覆亦無障礙法華經云皆於四衢道中露地而坐無復障礙)。】

「　不達羅籠」


【(凡夫未悟不了真如處生死網入煩惱籠不求出離三啓經云譬如

蚕作蠒吐絲還自縛)。】


「　志匪因心作」


【(志通妙法不以心求自悟真空不由造作瑜伽論云三界
心心所故從因緣生)。】

「　妄將心識同」


【(不能滌妄趣真秖恣緣心求識若無
求證定了涅盤唯識論云識性識相皆不離心心所心王以識為主)。】

「　眾生緣業重」


【(眾生業重不達真理託因仗緣尋佛覔祖認賊為子夙殃不除法華經云
無明緣行行緣識等)。】

「　達者陟高峯」


【(達人先見如陟妙峯上土忘機
真理自會唯識論云入見道位住唯識性尋𢙣位極名之為頂)。】


「　果若因修證」


【(果者謂菩提涅槃二轉依果若由如也因修證者唯識論
云說能證得因位攝故)。】

「　浿知證果通」


【(湏者要也知謂覺知證果通者
因果二位皆能通達唯識論云惑似先因後果隨轉)。】

「　𢙣人難化喻」


【(𢙣人
者不具大乘種性佛以方便巧說譬喻終不迴心法華經云五千之輩聞說一乘拂席而
去故難化喻)。】

「　聾聽五聲中」


【(五聲中者聲雖五種不離一音維摩經云
佛以一音演說法眾生各各隨所解若有罪根深重聽之不聞癡如聾耳)。】


「　陰暗人中業」


【(未究佛乘不明陰入迷雲所覆業累牽纏金剛經云如人
入闇則無所見)。】

「　無明逢善吉」


【(無明作相癡濁所蔽若悟無生之理自超
言象之外涅槃經云我等無明暗猶如日出時)。】

「　當初譬喻時」


【(如來說法
種種譬喻令知本未洞達一真法華經云諸佛有智者以譬喻得解)。】

「　真實言如一」


【(因言顯理理實全真諸法一如更無差別涅槃經云真實之法一切如也)。】


「　因理諸方外」


【(法妙一乘理非內外當機合轍殊途同歸維摩經云
法不屬因不在緣故)。】

「　心邪妄想中」


【(謂邪見故於法顛倒淪溺智識
恣縱妄心若悟真常諸念寂滅顯楊論云心邪見者所執皆倒)。】

「　不言功德力」


【(性相俱泯功德普圓妙力無邊不言自化維摩經云欲依如來功德力者當住度脫

一切眾生)。】

「　聖事自然通」


【(如來聖事化用無方妙力難思隨緣任運法華
經云自然智無師智)。】


「　義取煩河水」


【(煩惱深重如河流注能於妙義而達真源維摩經云菩薩
隨所化眾生而取佛土)。】

「　眾生䧟業中」


【(謂諸有情受輪迴苦不達福業
淪墜三塗唯佛大悲能為救護法華經云世尊如我心拔邪說涅槃)。】

「　卷舒能自在」


【(萬法無礙真性圓通自在之明卷舒得所法華經䟽云能卷大入小舒小令大
以一芥子納於湏彌)。】

「　信諦盡成空」


【(諸佛言教真實諦信一聖成道萬法
皆空華嚴經云發言誠諦皆可信受)。】


「　法教妙難測」


【(如來法教慈誨眾生妙慧難思深入覺海唯識論云微妙
甚深自內證故非諸世間喻所喻故)。】

「　如來三昧時」


【(維摩經云如來者前
際不來後際不去今則不住法華經云入於無量義處三昧身心不動)。】

「　一真常寂默」


【(真空秘密不可言宣我佛慈音真常寂默唯識論云大覺世尊成就無上
寂默法故)。】

「　六識欲何之」


【(謂法身功德離色離相湛然不動不可測量六
識狂迷自成顛倒不能開悟刧刧漂淪唯識論云如濤波依水意識現起)。】


「　陽化上天堂」


【(陽謂發生萬物與法摽持開謝歸空窮理盡性化度群有
福證天堂法華經云天上諸宮殿及在妙法堂)。】

「　迷情皆出沒」


【(此陰沒而
彼陰生喻浮漚之不久迷情晦而覺性現悟水月以非真維摩經云身如浮雲湏臾變滅)。】


「　人心有罪福」


【(迷悟影現罪福感招境不現心心能滅境罪福不作果報
不生維摩經云無大福無小福不為益不為損)。】

「　多少較量得」


【(不執罪福
誰能較量二法皆空絲毫不立唯識論云其業多少住一境界)。】


「　旨意明中說」


【(旨為所詮意者第六識明即慧義有簡擇之功敷揚妙教
故名為說法華經云說大乘經名無量義)。】

「　真空豈易知」


【(人法具空即是

真空入初地門方證此法非下位能知即法華經云不能知佛智)。】

「　根塵俱不動」


【(根即六根塵即六塵若悟無相根境俱無首楞嚴經云入流忘所所入既寂動
寂二相了不可得)。】

「　境界入湏彌」


【(境為外境山河大地界即六界地水
火風空識唯識所變大乘果位純無漏智現不思議故維摩經云以湏彌內芥子中)。】


「　玄言禪入定」


【(玄即妙理塵沙萬德菩提涅槃由能詮教趣證修學於定
自在三僧祇耶行六度行即法華經云安禪合掌以千萬偈讚諸法王)。】

「　逆順有相違」


【(逆生死流順真如海言陳善𢙣自有相違即法華經云東西馳走東喻善
趣西喻𢙣趣是故相違)。】

「　根性等何物」


【(三乘五性即根性也如來教法
一味由若天雨悟解不同即藥草喻根莖枝葉卉木藂林)。】

「　古今多是非」



【(自古及今凢夫多迷真性起煩惱業自是非他方便品云不自見其過)。】


「　幽深道所為」


【(道者佛果理智體遍用寬德俻法廣由因中發四大願行
二利行方能克證法華經云深固幽遠)。】

「　日用故難知」


【(真性平等智用無殊
明慧方達愚迷未曉不離四三業盡𢍆無為維摩經云舉足下足當知皆從道場來)。】


「　大聖能消息」


【(大聖即如來簡餘小聖隨眾生病應機設藥淨名經云眾
生病愈菩薩亦愈是能消息)。】

「　語話不思議」


【(若言若論究暢真宗即文
殊以言遣言居士用默遣言淨名經云乃至無有文字語言名不思議)。】


「　俗世起迷情」


【(俗者對真得名世者有漏所攝由生妄念莫了依圓故是
迷情還同遍計楞嚴經云不實如空華)。】

「　真心智不勇」


【(真心即智體用
雙彰本不動搖性圓周遍維摩經云前際不來後際不去)。】

「　山河尚改變」



【(山河噐世閒有成住之功安免推遷尚猶改變未若出離穢方常居淨剎法華經云大火
所燒時我此土安隱)。】

「　精進無能用」


【(本起經云釋迦因中遇底沙佛忘下
一足七日七夜妙偈讚歎超於九劫精進圓滿三祇果位無月無功豈勞精進)。】



「　大信勿輕非」


【(智度論云信為能入智為能度十信之初故名為大仁三
經云十善菩薩發大心長別三界苦輪海)。】

「　如來好信依」


【(念佛三昧經云
十方佛四方佛皆因念佛觀相得道如來者三身實相四智洪源悲濟十方慈觀三界般
若大䟽云為令眾生不信者信)。】

「　經文菩薩用」


【(六波羅蜜諸妙法門依之
修學證不退位般若頌云度河湏用筏)。】

「　妙行入玄微」


【(教理行果惣名
為妙悟摩訶般若證解脫法身唯識論云微妙甚深自內證故)。】


「　信智明宗習」


【(夫信者淨心為性了覺智之明可崇可上達元生本
有悉藉新薰唯識論云皆識為性由薰習力)。】

「　緣中故不疑」


【(假增上緣
體教中意除疑離障示果談因法華經云若有疑悔者佛當為除断)。】

「　朝凢及暮聖」


【(未發菩提心二障所覆無所覺知即是朝凢善根成熟頓悟修行了知真性便
同暮聖彌勒頌云信根生一念諸佛盡能知)。】

「　不達又何之」


【(審知流轉三世
無窮念念遷流𠛴郍生滅真心聖道此外何求般若論云故說無我以為出離)。】


「　智妙捿靈境」


【(直往大乘歷一僧祇𠛴郍之中入真見道得妙平智若棲
靈境彌勒頌云創居歡喜地常樂遂忘憂)。】

「　逍遙自在時」


【(到極喜地登大
牛車法華經云乘此寶乘直至道場是故逍遙長時自在)。】

「　迷情皆妄執」



【(返觀因位不遇良緣起四倒情迷於妄想對二邊執不悟涅槃唯識論云又去來世非現
非常應似空花非實有性)。】

「　真聖坥平為」


【(自他無異離高下心諸相
坦平無為蕩蕩楞嚴經云十方如來同一道故)。】


「　教法不思議」


【(教則文字般若法則令生慧解祛煩惱業示真如性法華
經云開示教人令住涅槃又云億億萬刧至不可議)。】

「　求之情不動」


【(瑜伽
論云有人來求菩薩共為煞盜等菩薩化初不逆意乍捨自百千身不惱彼意性海情田
湛寂無動)。】

「　性中是本元」


【(性即體也自性清淨本來而有𢍆元達本唯識

論云無始時來性雖本有而由染淨新所薰發)。】

「　妄想何堪重」


【(妄者分別
想為由心縱境馳流能招苦果楞嚴經云妄想因緣而自纏遶)。】


「　我見中人性」


【(上智之人達無我理中智之人猶生我見故如來隨根接
物顯示權方法華經云亦各巳離我見及有無見)。】

「　強名說是非」


【(由迷
巳見不了執心故有是非莫分耶正離名絕相妄執皆除般若頌云無非亦無是無下亦
無高)。】

「　四天拘不定」


【(四王天下即四大洲金輪所化而况真境無邊性非
拘礙楞嚴經云圓滿十方寧有方所)。】

「　了達悟詮機」


【(若能達此即是玄機
知詮旨之門見隨時之用法華經云隨冝方便示又云悟佛知見)。】


「　真要何窮極」


【(要謂要妙真謂真實體徧十方用周三界密嚴經云
法身無有盡是佛之境界)。】

「　達來性上知」


【(果德因行體解是難一念俻知
炳然齊現皆無自體同一法性楞伽經云我了諸法唯心所現)。】

「　坦然三界內」



【(無彼無此絕幻忘憂雖居三界之中不礙隨緣之理恒時自在故是坦然密嚴經云出世
而安住其心轉清淨)。】

「　運轉暫無時」


【(悲及含識拯救四生運用方冝
無時暫息密嚴經云雖於一切處現為明智者而在密嚴中寂然無動轉)。】
