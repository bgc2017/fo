










御製秘藏詮幽隱律詩卷第二十三　𦘺


「　皆救眼前急」


【(目前之事於理非深淺識微流貪求急速豈思仁義不務
歲寒)。】

「　錐刀蘊利名」


【(仲𡰱曰見小利則大事不成其或𥪰朝市之名爭錐
刀之利但隨自見蘊積無休)。】

「　愚心違善道」


【(善因施作順理益人行既
清純福恒隨逐愚癡漏識不悟苦空違善道於今生感沉淪於他世)。】

「　慧眼自分明」


【(智慧照了能盡根源簡擇是非無幽不燭鑒而常靜朗然分明)。】

「　大海波浪沒」


【(四海澄清永無鯨浪觀驚波之巳息知聖化之大孚万邦一家朝宗歸海所謂滄海
無驚放也)。】

「　深山棟木成」


【(巨材貞榦生長深山採而用之在乎良匠亦猶
賢人藝士隱逸山林聖朝搜徵同致康濟)。】

「　旁求存古意」


【(旁求儒雅徵訪
賢能俾子育於蒼生盡師承於古道淳素之化二五堪同者[栽-木+(万-一)])。】

「　後學法先生」


【(初始習學觸目未閑欲蘊於大噐才佛倣斆於先賢聖哲不謬其事立身自然)。】


「　天地𢙣諸盈」


【(天覆地載萬物生成故易云天道下濟而光明地道卑而
上行天道虧盈而益謙地道變盈而流謙鬼神害盈而福慧人道𢙣盈而好謙儻能保守
謙和誡其盈滿則何往不矣)。】

「　良賢蘊盛名」


【(忠孝良善正直賢明有經
天緯地之諜抱富國安民之術方能蘊藏名器垂於不杇也)。】

「　折旋知俯仰」



【(折旋之道湏察高位隔品致恭乃明貴賤亦猶軒鑑在懸妍[山/虫]不能進其影也)。】

「　進退有嘉聲」


【(君子之智湏察進退休咎儻能知命忘機安貧守道美譽嘉聲而自
遠矣)。】

「　自料皆冝靜」


【(恒自審思常冝寂黯安詳守靜而能發明全身之本
矣)。】

「　非言事莫輕」


【(天之無言四時行焉萬物生焉是以聖人終日言而
無言終日作而無作蓋慮功用著而民尚其跡)。】

「　輸他依法度」


【(佛制戒律

儒述仁義但能依而行之謹而守之必獲益而無盡)。】

「　寡譽亦無成」


【(德行
素寡聲譽不振既不握靈蛇之珠又安得為瑚璉之器空勞白首何所益焉)。】


「　宣𡰱明有說」


【(夫子垂[簐-欠+凡]訓道昭彰刪詩述易之功制禮作樂之說載其
仁義演以溫良恒為千代之揩謨永無差忒矣)。】

「　德行㝡為先」


【(易曰不言
而信存乎德行論語曰君子之德風小人之德草孝經曰行滿上下而無怨𢙣此皆言餝
身之道先以德行也)。】

「　純厚皆垂慶」


【(純和為其心養浩然之氣謙厚為其
性有不器之能自然慶及子孫昌盛於宗嗣也)。】

「　芳名信古傳」


【(名實相副
德行芳馨事君盡其忠貞處眾彰乎問望遂致清風遠播美稱恒存卓木不群揚名於後
代也)。】

「　在家思孝義」


【(君子在家常以孝道為務處朋友以義讓為先故有
笋生魚躍之祥共被易衣之羙矣)。】

「　食祿斆英賢」


【(食君之祿志在清通
湏學往哲前賢而畏四知三戒留牘懸魚之嘉譽褰惟露冕之芳猷常在書紳尤冝倣斆
也)。】

「　具載光經史」


【(或直書其文或俻載其事實增輝於史冊永為後代
之箴規矣)。】

「　仁風樂舜絃」


【(還淳㱕朴謹身節用積德累功上行下斆薰風
永扇同舜五絃謌化也)。】


「　知淺復知深」


【(違情背理即愚見遠知深可謂賢達)。】

「　先賢不尚爭」


【(先賢即古聖不爭即坦然此盖大道淳淳古今不易道經云不尚賢使民
不爭)。】

「　誡貪興有利」


【(外捐難得之貨內守淳和之理弃有作之務合無為
之性善利物情而無慾念)。】

「　無事保長生」


【(無事即無為無為乃合道既符
自然之道可以保守長生道經云聖人常有欲而無欲常有事而無事)。】

「　物性三才逈」


【(物謂世塵情物性即大道虛無雖運動不離於三才且取捨逈分於高下也)。】


「　幽通萬象清」


【(虛無之性潛通萬靈若皎月之流光方圓順器同白蓮之

在水泥滓不沾)。】

「　但教依法度」


【(守靜訥言修身奇德既行藏而無爽居祿
利以何虞孝經云非法不言非道不行)。】

「　積善響佳名」


【(順理益物名之
為善慈和仁恕與物無違才行進身忠貞奉國自然佳名著矣)。】


「　壯年勤且樂」


【(壯年乃成人之歲勤樂則顯彼苦心既能稟氣二儀湏務
立身世上故孔子云學而時習之不亦樂乎此明果行育德者也)。】

「　暮齒免荒凉」


【(暮齒乃摽其衰杇荒凉即顯彼無學此明材不可觀行無所取故語云幼
而不遷悌長而無述焉若少而積學老免荒虛者矣)。】

「　有路撗霄漢」


【(此明
幼而學問長而有知乃習之以禮儀雖以成文藻則世人榮之既遂榮貴若昇霄漢之路
也)。】

「　英風自蘊香」


【(英則顯其明敏風乃彰彼神儀既學海方深乃詞鋒
獨利則可內蘊德業外播馨香者歟)。】

「　湖中明印月」


【(人負德行世不可藏
若月印長霄影澄秋水此乃舉况以顯於人事者也)。】

「　星碎亦光芒」


【(月既
見朗星亦光芒煥若懸珠瑩如玉集以况乎處上位而明敏居下位以清通若陳仲弓之
處世時乃号為少微星也)。】

「　誡懼知盈溢」


【(誡乃懲勸復即若驚天道𢙣盈
而明進退易曰君子以裒多益寡稱物平施者乃終吉矣)。】

「　神靈禍不祥」



【(神靈即聡明正直其功不測以福賞善以禍不祥善惡賞罰當無不均乃明乎天道無親
善𢙣不爽也)。】


「　庶事情多感」


【(萬機日用唯道是從而眾事彌多莫不情感于中也)。】

「　光陰自古長」


【(聖人體於妙道混同萬物曲濟無遺深妙之宗舌古而常一則
四時雖行運轉日月推移終古巳來純和而一致者也)。】

「　貪癡何意遣」



【(於物有所著謂之貪於心有所滯謂之癡大智之人用周萬物忘功體道經濟蒼生夫如
是則於物無著於心無所留執滯之懷何莫由斯而遣矣)。】

「　愚軰為誰忙」




【(於物既執亦滯於心則徒然駈馳不能任於物性苟無滯執則於道玄通萬化各適其冝
斯不遠而明矣)。】

「　紅𦰧隨時變」


【(春則萬物生焉秋乃萬物衰焉春秋
肅成則萬物隨時而各得其所也)。】

「　白頭㝡易傷」


【(二毛相待皆情所共
傷自非識達幽深洞了玄微則情傷易往也)。】

「　世途能及物」


【(愛󵠪矜孤
憐貧恤物有益之道豈徒然哉)。】

「　善德遠馨香」


【(恩惠及物德之大者
故聖人布一善政人神以和出一嘉言士庶咸恱斯之善德若秋菊春蘭自邇及遐氛氳
而皆聞矣)。】


「　終日窮經史」


【(經謂九經史則三史終日窮覽必通其幽而洞其微則日
益其道故經云為學日益)。】

「　思惟事轉深」


【(經義淵博學而思之必達
深玄乃日新其德)。】

「　公忠懷士意」


【(公忠者恭儉無私也聖人行於恭儉
以和百姓之心使天下無私咸歸淳素而行上古之風莊子云必服恭儉拔出公忠之属
而無阿私也)。】

「　是理好相尋」


【(理者化治之道也夫天地無心而萬物各
適其冝隨時運轉聖人法於天地亦同無心而萬化行為各適其所冝也)。】

「　聦惠玄高鑒」


【(聖人用公忠恭儉治於天下百姓所以然者其欝聖人聦明惠鑒能順時而治
使移風易俗以除矯害之心也)。】

「　回邪抂用心」


【(回邪者矯偽之心也夫聖
德御宇不獨親其親使天下皆親人心共一故去搖蕩矯偽抂用之心使淳素而歸乎一也)。】


「　從容閑歲月」


【(聖人妙用虛無為道德而一以貫之使天下自然而不知
其所以然日出而作日入而息皷腹擊壤夫如是則歲月位以閑居延容而無所施為者
也)。】

「　學古便知今」


【(古之常道博學而通之以古明今然後進道修德今
古一以行焉)。】


「　一逢又一別」


【(邂逅長途相求也傾之以蓋鞠躬受命唘行也贈之以言不期

而會以纔忻瞻望弗及以還慘世態如此誠感嘉謀)。】

「　來往幾時休」


【(聖人
在上四海一家來無弃繻之艱去勿焚舟之念故所以東西南北行伇勿休)。】

「　處世皆名利」


【(夫名器倣利君子小人各從所好也然則立身揚名急急於祿先哲猶此
况與人乎)。】

「　貪生去不留」


【(固巳自弊命曰貪生不尚道以修德豈誡貪而
去情是以嘆息逝川悲嗟隟駒而巳)。】

「　善緣看似夢」


【(立身行道善莫先於
五常此仲𡰱之善也抱一守素善莫大於二要此伯陽之善也即境唯心善莫大於一乘
此瞿曇之善也言似夢者示見義不為之流也)。】

「　𢙣事勿臨頭」


【(且夫為𢙣
者懲勸不聽而作禍患弗覺而至是以誡之在未萌之前俾知過而必悔也)。】

「　如病心常怕」


【(病者不可不懼過者不可不悔不懼則身殞不悔則禍積也)。】

「　生前樂自由」


【(知善為去𢙣之本則神府不乱達𢙣為生善之根則形骸不優)。】


「　爵受帝恩榮」


【(爵者量也量其才而授其軄故爵之所設也或三等五等
二十五等故白虎通曰三等法三光五等法五行是知我有好爵與爾縻之非縈何謂也)。】


「　英才一德清」


【(英才俊造之士適時之器也時不我與則良匠弃之愚者
鄙之今君臣道合雲龍符契一德之清也)。】

「　昂藏真體㒵」


【(聖人之有天下
也羣賢必萃奇㒵必臻莫不映玉連珠彰堂堂之風彩凝脂點漆輔落落之神姿)。】

「　忠孝福蒼生」


【(軒皇六相虞舜五臣為雨為霖為柱為石盡忠盡孝有始有卒
熙熙生民咸受其賜非福何謂)。】

「　武士干戈偃」


【(夫兵者聖人不得巳而用
之是以庠序文教倒載干戈昇平之業既成耕鑿之民必暢華山㱕馬桃林休牛今見之
矣)。】

「　文儒𨦣氣橫」


【(祖述堯舜張皇禮樂游心於六藝之外留情於五常
之中勁健辭鋒雄布筆陣此即文銳之氣橫也)。】

「　盈朝多有位」


【(形儀眾彥
[施-也+圭]飾殊勳禪翰紫宸草撽黃閤三才九位靡不軄官詩云濟濟多士文王以寧)。】

「　各願



【(上之所行下之所好故聖人執大象以治天下行之以中孚茂之以
盛德君臣諧穆上下咸熙是以聖人感人心而天下和平其斯謂歟)。】


「　謙遜明然得」


【(仁以立身謙德之柄卑以自牧遊而履順抱其智而不自
言智蘊其賢而不自言賢在易著謙爻之象於書有受益之文所以明然道自心得)。】

「　滋章道理踈」


【(法令滋章刑之煩也盜賊多有莫不由斯与道相違誠為踈矣)。】


「　榮枯皆定分」


【(春榮冬悴類人生定分皆有貴賤矣釋教也顯驗因果儒
典也克明壽數理亦昭然矣)。】

「　巧拙亦何如」


【(陰符經云性有巧拙可以
伏藏謂人之巧拙不可顯露慎言擇行無使患生)。】

「　撫俗從仁感」


【(撫之以道
育之以仁唯德是修惟刑是恤五教在寬不嚴而治民感化矣)。】

「　臨民用教敷」



【(覩民設教容人畜眾寬猛相濟德惟善政是以庶績咸熙彛倫攸敘)。】

「　高明歸輔德」


【(皇天無親惟德是輔盖以體天垂蔭並日舒光有屈皆伸無幽不燭玄穹垂祐歸輔
明德也)。】

「　建立紀功書」


【(至德鴻名人君能事子育兆庶雍熙朝政功惟
建立自有國書)。】


「　爽思開清眼」


【(心契冲寂道符閑澹對清境寓其目臻玄默爽其思當靜
慮求理致無為景化矣)。】

「　看書見古風」


【(仄席求賢觀書乙夜遐探遂古
慱考前編酌三代淳風明百王令則非聖賢之道詎歸其意)。】

「　靜來入深道」


【(靜者人生而靜天之性道者虛極之妙用是以心境俱靜一無所有心与道合入無
間矣)。】

「　廓落意何窮」


【(大道既坦運意寬廣居惟恬淡自得清平廓尓何窮
優游靡凝矣)。】

「　荏苒郍堪久」


【(人有違道迷而不信荏苒塵容郍堪久矣
不了虛極由居世網)。】

「　愚癡治不通」


【(執言滯教愚癡難治欲求了悟其可
得乎德經曰下士聞道大笑之)。】

「　含生皆是性」


【(含生者釋教云異生性也

造異類業起差別見懷含善𢙣皆是其性矣)。】

「　氣槩壓群雄」


【(人之氣槩
各負凌雲其或理蘊通明識包擁變寧知契無競之筌守必勝之理)。】


「　慵修勤苦志」


【(夫學以唘志道以處身且畫寢之情既篤時習之義殊乖
飽食終日無所用心難矣哉苟無心於修尚又怠墯於進業非慵修而何也)。】

「　將甚立於身」


【(而又學也入乎耳著於心布乎四支動靜皆法如此則性服忠信身
行仁義餝禮樂選人倫上以忠於主下以撫於民若非學何以立身乎)。】

「　經史何窮極」


【(九經三史諸子百家文章之淵源禮樂之盛治若研精覃思窮理盡性然
後知學之不足秉筆之士刻苦之人仰之彌高鑽之彌堅又何窮極耶)。】

「　文儒慕好親」


【(君子進德修業景慕文儒良有以也戴仁而行抱義而處尊賢而容眾毀方而
瓦合不以金玉為寶以忠信為寶苟非其人比肩不親)。】

「　緣深觀在掌」


【(君子
自強不息切磋琢磨學以聚之問以辯之既大業之富有又威德之日新徵一事引一辝
依然在目豈非指諸掌乎)。】

「　不奈世情人」


【(仲𡰱云中人以下不可以語上
者盖上智與下愚不移也處乎塵世隨彼時情愚智天生無以教導也)。】

「　鑒誡無休息」


【(夫聖人立事設教無不䡄範將來儀形多士使昧道者知學而後祿勸在位者
明忠以事君一則順素王鑒誡之言一則振芳聲無休息之譽凢諸庶品得不勉歟)。】

「　泥蛙問海津」


【(且性有巧拙學有淺深所以聖人不求俻於人不巳長隔物盖
愚不足謀智淺不足測深亦如泥蛙不可語之以東海之廣大也)。】


「　清霄白鶴唳」


【(夫言生於象象之也必幽象發於言言之也必隱况乎天
高氣爽景靜風沉[婁*殳]聲之遼鶴唳空入極之清秋入望是時也徵吟取象焉能捨諸)。】

「　移步向前看」


【(乘興褰裳臨軒送目語玄微則妙將神契論雅正則穎逐天生)。】
