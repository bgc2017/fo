










御製秘藏詮卷第十九　　　　　　駕


「　說悟有情解」


【(說所悟法令生物解解謂得法契神悟則見性是佛華嚴
經云佛法無人說雖慧不能解)。】

「　心明離業緣」


【(知諸法即心得佛智光明
故曰心明楞嚴經云從真妙圓重發真妙一切妄想滅盡無餘)。】

「　若人知道路」


【(見佛性者名為道諦此是諸佛一路涅槃華嚴經云以法無性故無有能了知如是
分別知此人達深義)。】

「　無染達無邊」


【(一法道場住無染地洞達無邊法界
清淨楞嚴經云塵消覺圓淨淨極光通達寂照含虛空)。】


「　理不通玄機」


【(妙真性如來藏心體本自知不因境起幽靈絕待謂之玄
識佛頂經云如來藏中性識明知妙明真識妙覺湛然徧周法界)。】

「　靈臺業障牽」


【(靈臺心也瞥起妄心一迷萬惑情生智隔業障滋多佛頂經云知見立知
則無明本)。】

「　心迷空色相」


【(前巳迷本此復迷末佛頂經云汝等遺失圓妙
明心寶明妙性認悟中迷晦昧為空空晦暗中結暗為色)。】

「　更欲妄求詮」



【(知文字性離方能說解脫性本法不明更執末教是不可也)。】


「　解注如來藏」


【(如來藏者即是九聖之本源萬法之正體發明知見日用不覺
楞伽經云寂滅者名為一心一心者名如來藏)。】

「　十方梵語詮」


【(十方薄伽
梵語皆詮藏性華嚴經云知語境界不思議此是說法三昧力)。】

「　𠛴聲隨眼見」


【(𠛴動有聲因聲有色故隨眼見不了目心非幻成幻楞嚴經云因明發性性妄見生
是故世界因動有聲因聲有色)。】

「　真諦本無牽」


【(萬化紜紜一真本寂群生
擾擾實際凝然楞嚴經云舜若多性可消亡爍迦羅心無動轉)。】


「　求法觀身智」


【(了陰本來性空空故即見實相於自身中開佛知見名觀
身智原茲本致功在求法法正則栖心有地華嚴經云性空即是佛不可得思量)。】

「　非



【(非是乖於大心而馳逐外求失其自然法尓之道盖志存利物而巳)。】


「　令他離苦惱」


【(求法之功務在利物苦則一切苦受惱則一切煩惱法華
經云普欲令眾生如我等無異)。】

「　福相等無邊」


【(自利利人功報所顯然福
相通於世間出世間上上等自因至果皆名福相盡於東際故無邊也)。】


「　解亦非真智」


【(解謂達法之解雖假正解存解為障圓覺經云勞慮永息
得法界淨即彼淨解為自障礙)。】

「　邪魔妄自牽」


【(心存法愛魔得其便說差
別法壞乱其心楞嚴經云修行止觀魔境現前汝不能識洗心非正落於邪見)。】

「　善根還不立」


【(失其方便善還不立楞嚴經云心中不眀認賊為子如無聞比丘妄言證
聖還墮阿鼻)。】

「　迴向莫知緣」


【(差之毫𨤲不可牽復莫知迴向之緣楞嚴經云
落於魔道雖欲除妄倍加虛偽汝妄自造非菩提咎)。】


「　證道成何法」


【(虗心體極曰證法界無礙日道心法清淨道本寂滅寂滅
性中知成何法)。】

「　修知解脫緣」


【(造修方便湏知解脫之緣知一切法皆同無性
心無所著名解脫緣圓覺經云於諸妄心亦不息滅住妄境界不加了知於無了知不辦
真實)。】

「　六塵離懈怠」


【(六塵無體諸染不生觸緣對境妙冥心印應無所住
而生其心圓覺經云得念失念無非解脫)。】

「　方了祖宗禪」


【(心為萬法之祖
性本常定故号曰禪文殊經云有佛無佛性相常住)。】


「　志認玄中旨」


【(真如以照明為性其旨玄邈辯認至難不得不一其志有
之則乖其性無之則傷其軀彌綸萬化靡所不在而獨邁於有無之表得非玄言邪)。】

「　堅牢一切緣」


【(得諸法性觸境對緣一性常住道遠乎哉觸事而真法華經云資生
業等與實相不相違背)。】

「　非常心得達」


【(非是專注一境常一其心而巳
但隨所施作不乖正性佛頂經云發明了知見聞知覺皆如來藏)。】

「　利益大悲前」


【(逮得巳利庶品咸益運大智之悲起度二死之表法華經云大慈大悲常無懈倦)。】



「　削𩬇壞形體」


【(削𩬇為除心垢而淨智頂被忍辱服以御法身法華經云
剃除[肆-聿+((彰-章)/(泳-永+頁))][肆-聿+((彰-章)/犬)]而被法服)。】

「　為僧号佛子」


【(梵云僧伽此云眾和合為義事理融
和混然無礙華嚴經云統理大眾一切無礙從佛口生得名佛子)。】

「　善求解脫緣」


【(心性本淨名性淨解脫知心妄動名離垢淨解脫性本不淨不能離垢垢不能
離不顯性淨三昧經云迴神住空窟解脫魔所縛)。】

「　莫䧟業中水」


【(見性
不了不護戒囊漂流業水生死無窮十住經云一切業海從妄想生)。】


「　果招深信士」


【(菩提大果從無始來常熏眾生令生大信法華經云如是
大果報種種性相義除諸菩薩眾信力堅固者)。】

「　境界了他緣」


【(群生未悟取心
外法境界名他其或頓悟全同寶王法華經云今此三界皆是我有)。】

「　方便門中事」


【(一切諸法不可思議含靈得之疲勞諸佛得之妙用法華經云開方便門示真實相)。】


「　超諸聖境前」


【(一切障礙即大圓覺寂滅真如有何次位法華經云不退
諸菩薩亦復不能知)。】


「　因論陳諸善」


【(有因議論陳諸善利無謀而應不撓神機華嚴經云法性
不在於言論無說離說恒寂滅)。】

「　縱橫自坦然」


【(萬慮常寂而用自縱橫心
跡繁興而坦然無礙華嚴經云於諸世間法不生分別見)。】

「　勿將機巧事」


【(事之
機微智謀曲巧徇情逾理勞而無功事機巧而失方便淨名經云有慧方便解無慧方便
縛)。】

「　動止返情牽」


【(若不與正智相應雖靜而動若與正智相應雖動而
靜返情則曠刼凢夫順智則一念大覺楞嚴經云認悟中迷粘湛發見)。】


「　義現娑婆界」


【(大千塵𠛴宣無盡之圓音百億彌盧演一味之法雨雖奧
義常現而大機方會華嚴經云一即是多多即一文隨於義義隨文)。】

「　精通解悟詮」


【(該羅萬法皆為能詮隨義精通誠堪悟解苟得法契神何必在乎言說華嚴經
云𠛴說一切法)。】

「　祖師知佛道」


【(人能秉法法在人傳故祖祖相師師師授

手法華經云於道場知巳導師方便說)。】

「　攝受向人天」


【(向人天中而攝群品
法華經云餘諸天眾請我轉法輪)。】


「　法門長不断」


【(一句法門無盡如海大法性綿綿無間如華嚴經云盧
舍那於一塵中示現十佛世界轉大法輪)。】

「　豈用妄分張」


【(不稱本法分大
分小若徧若圓權實性相不得一味曲順根器皆為妄分法華經云但不盡能受於一佛
乘分別說三)。】

「　如解證真理」


【(若能清淨心垢從聞思修入三摩地栖神
法界名證真理金光明經云遊於無量甚深法性)。】

「　詮談徧十方」


【(若能悟入
法海但有詮談言滿法界口輪清淨法華經云若欲以妙音徧滿三千界)。】


「　指引千經論」


【(不貴宗通唯尚言說廣引經論心不入法何所利哉楞伽經云
說通宗不通如物被雲籠)。】

「　還緣一味長」


【(自性清淨心名為一味頓悟自
心疾成正覺如飲甘露是無上味華嚴經云如來一味大悲之水無有分別)。】

「　六根都不染」


【(根境皆緣生緣生故性空空故清淨清淨故無染餘根亦如是楞嚴經云
一根既歸源餘根成解脫)。】

「　萬劫自清涼」


【(六根不染萬劫無累永居清涼
法性大海楞伽經云不生分別隨順清涼)。】


「　玄化無窮信」


【(化復作化化化無窮於無窮境起無窮信智論云佛法大
海以信能入)。】

「　本來法義長」


【(華嚴經云法海漩澓處一切差別義種種
方便門演說無窮盡本來如是是無上乘故云長也)。】

「　三乘如是故」


【(三乘
本來即一乘故自行則三一融泯即化則三一被機同性經云皆從毗盧之所流出也)。】


「　豈可妄稱量」


【(三乘境界本不思議蠢尓君類可稱量哉法華經云及佛
諸餘法無能測量者)。】


「　幽奇心廣大」


【(如來藏同法界性謂之幽奇心同三世諸佛境界故曰廣大華
嚴經云究盡法界不思議廣大深心巳清淨)。】

「　生滅順陰陽」


【(本滅不生本生

不滅同塵順俗示有生滅法天法地開物成務華嚴經云晝夜日月及年劫此時數智方
便力)。】

「　釋意無邊際」


【(生滅巳滅常光現前放意天外無所拘累杳不可極
華嚴經云隨意自在所得無㝵)。】

「　圓眀自久長」


【(圓明妙性不逐物遷萬化
諠然一真常住華嚴經云無等法如是體性常不動)。】


「　大慧何為是」


【(無礙大慧佛之知見即佛為大慧之尊何為賢明所重如
是耶法華經云平等大慧)。】

「　賢明重法王」


【(佛為法王於法自在為法中王
即是賢明所重之尊維摩經云我今稽首三界尊大聖法王眾所歸)。】

「　覺華開萬葉」


【(大覺開悟如妙蓮華敷榮萬葉傳付無窮如梵網經說一花百億國一國一釋
迦)。】

「　世代作津梁」


【(正法像法此土他方歷代羣生所歸有地無明大海
為作津梁華嚴經云廣度一切猶如槗梁)。】


「　調御丈夫儀」


【(心源不妄了了見性能成妙覺十号昭然涅槃經云見佛
性者名曰丈夫)。】

「　清虛皆得理」


【(得法正性廓然無累混融性海湛湛無涯
華嚴經云佛身清淨恒寂滅)。】

「　寶壇香積世」


【(寶壇淨居香積世界重重
無盡曾何間然心冥中道佛土自嚴維摩經云有佛國土名眾香)。】

「　逢善心懽喜」


【(了性則處處道場悟善則心心法喜華嚴經云心得清淨生大懽喜)。】


「　信及凢愚性」


【(正因佛性何間凢愚豈一闡提軰而獨無耶智論云火是
熱性水是濕性眾生是佛性信矣)。】

「　眾生智業長」


【(本覺智業其用難思
起信論云而自然有不思議業常熏故長)。】

「　福因無盡藏」


【(諸佛功德無盡
藏海全同法性一味常熏以之為因故曰福因起信論云法性真如海無量功德藏)。】

「　應相現真常」


【(不增不減經云法身即眾生眾生即法身法身與眾生義一而
名異摩訶衍論云諸法不動無去來是知世間相常住明矣)。】


「　智不覿真智」


【(住相世智如人入闇則無所見不能見無相真智金剛經

云不住相布施其福德不可思量)。】

「　身財布施長」


【(身財為法財以法財
布施慧用無窮以世財執見不忘不能出離法華經云是名真法供養如來)。】


「　慈悲常自在」


【(楞嚴經云上合十方諸佛本妙覺心與佛如來同一慈力
下合十方一切六道眾生與諸眾生同一悲仰)。】

「　性認法中王」


【(一性為諸
法中主能性諸法名為法性認得此性名為法王維摩經云巳於諸法得自在是故稽首
此法王)。】


「　我󲊛隨情忍」


【(住法界性不見順違運同體悲冤親一際隨情彼此起忍
辱心智度論云不忍之人聖人所輕)。】

「　經文有大綱」


【(安心法界成大忍力調謙
自他不令轉動華嚴經云自能安住於佛法中亦令眾生同得此法)。】

「　明通精進法」


【(無童行海㝡猛利者唯精進法為行中王華嚴經云伹為知一切法界故而行
精進)。】

「　福業照三光」


【(無等精進普徧諸行㝡無能勝故照三光華嚴經云亦如
日出時照曜於一切佛福田亦然滅除諸黑暗星月亦尓故)。】


「　真見他心外」


【(真見根本智證真離所見相體圓常寂而超境界之外他心不
能緣故萬行經云見猶離見見不能及)。】

「　增暉智性光」


【(性源不異真見
洞明法界圓通慧根朗耀楞嚴經云如一見根見周沙界)。】

「　業歸沙界重」



【(如來妙行剎剎相含菩薩慧心塵塵無礙法華經云人天所戴仰)。】

「　修習惠他方」


【(若在因位六度萬行修習二嚴多是利他之行廣徧十方佛頂經云願今得果
成寶王還度如是恒沙眾)。】


「　慧達千途逕」


【(慧即智慧有照真達俗之功真俗雙明即聖道大通而無
礙滯决定藏論云智慧靡不通於淨更無垢)。】

「　如來大教張」


【(如來張大
教綱以化導有情逗彼根機彰乎聲教華嚴經云張大教細漉人天魚也)。】

「　精修還彼物」


【(因中發四大誓至果中三願巳周唯度生之願未滿今既度生故乃還於彼物

法華經云以是本因緣今說法華經)。】

「　願即口談揚」


【(初發願即語業㝡章
故云願即口談揚今既酬因故當轉於法輪法華云大聖轉法輪顯示諸法相)。】


「　說理因遷遠」


【(諸佛經三大劫方證圓真之理在因位遙長綿歷五十二
位故云說理因遷遠金剛般若云得值八百四千萬億郍由他諸佛)。】

「　無窮萬法𠬧」


【(久居因地廣修六度故云無窮萬法今至果中収攝唯有一乘法華經云唯有
一乘法無二亦無三)。】

「　顯然俱是智」


【(謂智體徧周法界如云虛空大地咸
是妙明真心中物楞嚴經云若能轉物即同如來)。】

「　不曉幻浮漚」


【(眾生不曉
幻妄不證聖流往還三界出沒無恒楞嚴經云如湛巨海流一浮漚)。】


「　理道若疑惑」


【(理即所證之理妙解幽玄道則通理之元妄情自泯不生
猶豫直截利根法華經云迷惑不能了)。】

「　迷情悟到頭」


【(凢情未曉迷妄
日生實相顯然真空自照楞嚴經云認悟中迷)。】

「　六根終未淨」


【(以妄為真
六根昏昧強生知見長養無明楞嚴經云一根既歸源六根成解脫)。】

「　撈月悞猿猴」


【(猿猴探水月對境迷真背覺合塵無有是處金剛般若論云猿猴探水月)。】


「　詮法人難解」


【(未證性者而滯能詮之教即難起入法之智楞嚴經云根
無始來一向多聞未全道力)。】

「　真純悟即周」


【(真純則無雜為名謂理智同體
若迷即觸處千差若悟即頓周何滯楞嚴經云理即䪺悟承悟併銷)。】

「　更無殊勝相」


【(如來三無[婁*殳]刧所修萬行今既滿足故獲殊勝之相金剛經云凡所有相皆是
虛妄)。】

「　覺海盡波流」


【(覺即所證之理海即廣大無邊雖逐波流湏登彼岸
肇論云海湧千波千波即海)。】


「　經戒如來法」
