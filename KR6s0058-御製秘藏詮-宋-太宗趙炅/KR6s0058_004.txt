










御製秘藏詮卷第四　　　　　　　車


「　疑心深似海」


【(唯識論云謂於諦理猶豫為性能障不疑善品為業如海深廣)。】


「　大藏不虛詞」


【(如來大藏本為出纏法法所宣因非虛妄法華經云佛語
實不虛)。】

「　一向能如受」


【(本實半偈而捨全身何况如來所宣一句一偈寶積
經云我昔為求一句法棄捨身命為菩提)。】

「　慈悲喻化時」


【(得意忘言頓悟
玄旨誠非化喻自證真詮維摩經云行大悲慈導以大乘故)。】


「　禪門何境致」


【(禪宗虛寂應用無方萬法皆真本來非有維摩經云諸法
究竟無所有是空義)。】

「　猶豫疑深旨」


【(無念實相即是深旨妄想虛情猶豫
不決維摩經云一者所未聞深經聞之驚怖生疑不能隨順)。】

「　對面妄虛名」



【(謂迷心境翻著妄緣能究假名於法何執大品經云色是假名乃至涅槃亦假名一切諸
法皆假名)。】

「　寒冰終是水」


【(冰本是水水能成冰不離假名却成迷妄超凡
入聖本無所拘楞嚴經云如水成冰冰還為水)。】


「　興法不相違」


【(所化緣熟佛現世間能宣妙法順符機理觀根受道而無虛設
佛地論云能貫能攝名為經法)。】

「　人天法聖威」


【(能演教法無漏聖慧潛通密照
有大威神開決心眼令悟真理維摩經云諸佛威神之所建立)。】

「　演文兼證義」



【(詮演真文善解妙義能通性相諸法皆空維摩經云了達諸法深妙之義)。】

「　常握涅槃機」


【(涅槃圓寂即是玄機若證菩提皆通妙用維摩經云導眾以寂故稽首)。】


「　念念經千劫」


【(有為之法前滅後生𠛴𨚗不住經彼多劫維摩經云是身如電
念念不住)。】

「　三途滅惡時」


【(三途地獄餓鬼畜生正受苦報能解無心火變涼池
維摩經云至于地獄無諸罪垢)。】

「　不言眀取捨」


【(取捨之心皆是妄情分別計有
實慧照了即是一相取捨非有維摩經云知一切法不取不捨入一相門)。】

「　無礙有



【(性無障礙有真如理一切諸法通漏無漏智法句經云畢竟空中熾燃建立)。】


「　但見如常法」


【(唯獨觀達不變真如遠離諸相空寂深法唯識論云顯此徧常
體非虛謬)。】

「　聲聞化導時」


【(聞聲悟道名曰聲聞只將四諦染淨因果化導
有緣維摩經云以聲聞法化眾生故我為聲聞)。】

「　色空眀與暗」


【(質礙名色無障
曰空光顯稱明𡨋昧為暗妄計為有論體永無唯識論云此徧計所執自性無所有)。】

「　分別竟奚為」


【(分別有異終究何為但觀實相不見有珠分別亦空法華經云
而觀諸法如實相亦不行不分別)。】


「　釋別欲何為」


【(但起執情便生分別無埶無妄本自圓成維摩經云若於
佛法出家有所分別為不如法)。】

「　忘心終未是」


【(情昧本末不欲忘心心不思惟
自證實際維摩經云此比丘久發大乘心中忘此意)。】

「　不著妄情言」


【(虛妄
情計名言是有了妄本空即是不著中邊論云虛妄分別有於此二都無)。】

「　福善天懽喜」


【(妄執既無自然𢍆理不起福報善入無為非偶人天而生懽喜法華經云得所
未曾有心懷大懽喜)。】


「　習性非言定」


【(習性未斷故惱相纏不究本原熏習所染唯識論云為不定性
獨覺聲聞)。】

「　真空不可思」


【(真實無相空離妄執心緣罔及尋思路絕宣說
不能名言道斷維摩經云有解脫名不可思議)。】

「　經文說不盡」


【(經文言說分限
有偏理徧周圓故說不盡維摩經云若廣說者窮刧不盡)。】

「　頓悟暫時疑」



【(若能頓覺即心是佛疑心覺然即便除遣圓覺經云唯除頓覺人并法不隨順)。】


「　除疑為舉唱」


【(疑謂猶豫得聞深法信順趣向疑悔即除法華經云聞如
是法音疑悔悉巳除)。】

「　動靜十方知」


【(真心常照周徧十方所有動靜無不皆知
楞嚴經云譬如人靜居十方俱擊鼓十處一時聞此則圓真實)。】

「　從今兼亘古」


【(從今法會通達往古迷真妄有終無解脫圓覺經云若諸眾生本不修行令妄想心

云何解脫)。】

「　悟來一剎時」


【(忽然洞達萬法皆空一念之頃便登正覺華嚴經云
無量無數劫解之即一念)。】


「　想情何是物」


【(妄想情計我人之相萬法皆擁執為何物唯識論云由彼
彼徧計徧計種種物)。】

「　妙行不相疑」


【(深修善法無相妙行志心進趣不得
生疑楞嚴經云我今巳悟成佛法門是中修行得無疑惑)。】

「　有病來求藥」



【(三障煩惱發業招感生滅苦迫身心欲差此病湏求法藥法華經云懽喜問訊求索治病)。】


「　如來示法醫」


【(果海如來說三藏法誨示有情應病與藥維摩經云為大
醫王善療眾病)。】


「　意定法何為」


【(意常湛然寂靜分眀動念散亂安法何為維摩經云無亂
是菩提常自靜故)。】

「　真心存道理」


【(真心實行審察至道修習眾善順向
正理起信論云一者直心正念真如法故)。】

「　因從是本根」


【(因依空性本來
之理出生眾法內身外器楞嚴經云有漏微塵國皆徙空所生)。】

「　欲界詮源止」


【(由欲而生名為欲界源謂法本息慮稱止了達定乱空性一如名詮湏止維摩經云
在欲而行禪希有亦女是)。】


「　譬猶慧日照」


【(謂佛智慧三覺圓眀光極無際徧照法界喻如杲日昇天
朗其萬像華嚴經云慧日照十方能除幽闇處)。】

「　於法無親踈」


【(覺性平等於親
踈境無有憎愛金剛經頌云法中無彼此理上豈親踈)。】

「　通透真如理」


【(謂通
透法界真實理事楞伽經云體常不變目為真如所為心性常無念故名為不變無真妄
故名曰真如)。】

「　經音一梵書」


【(修多羅教如標月指故以標月之指喻於言教
謂見月湏籍其指悟心湏假其教證理豈在言詮)。】


「　喻若恒沙水」


【(起世經中說有恒河金沙混流沙細如麵內有金沙以喻真性
表於貪愛河中有其菩提妙果)。】

「　清心溺濁渠」


【(圓覺經云雖現塵勞心恒

清淨現染之中而不染也如良王投泥王性自淨)。】

「　解理如冰釋」


【(悟解真理
如釋春冰水喻真性水喻妄想解屬智慧所以心迷執我假悟心銷情執俱盡照體獨立
圓覺經云如湯銷冰無別有冰)。】

「　一貫道虛無」


【(道體無邊廓周沙界沖虛
真境妙有玄樞肈論云大包天地細入無間謂之一貫也)。】


「　圓證通無塞」


【(楞嚴經云十方俱擊皷十處一時間此即圓通真實圓者滿足
證謂所證即真理也故以圓通妙觀寂照虛空理事俱融慧門無壅)。】

「　真同色相俱」


【(大般若經云真即真空是其真諦色相即俗諦真俗二諦不即不離即是俱也)。】

「　更無出入路」


【(凡情未悟潤惑循環真智湛然實無往復故維摩經云不出不入一
切言語道斷)。】

「　寂見法王都」


【(如來正智能斷諸惑妙出世間所依真境
故金剛經頌云如能雖諸相定入法王家)。】


「　不悟生煩惱」


【(十地論云謂眾生本來未曾覺悟故生煩惱隨迷悟之緣造業
受報遂名眾生修道證真遂名諸佛)。】

「　真如無比擬」


【(真謂真實顯非虛妄如謂
如常表無變易有大方便智除滅無眀有不思議種種之用故唯識論云非諸世間喻所
喻故)。】

「　智情若有差」


【(密嚴經云眾生若有迷惑顛倒則有差別之義心既
無差同一覺故)。】

「　窮劫常存矣」


【(謂真淨心源應用塵沙盡未來際常住法
界感而即通故云常存也)。】


「　滿足多從願」


【(法華經云我本立誓願欲令一切眾如我等無異故以諸佛因
地湏發四心依此願修方成正覺)。】

「　敷揚極太虛」


【(華嚴經云剎說眾生說十
方三世一時說又云一切國土中恒轉如是輪故以至理敷揚量等大虛)。】

「　亦無關鍵閇」


【(法華經云故佛開示皆令悟入佛之知見真實妙門無有關鍵)。】

「　何不證空如」


【(智度論云但與相應煩惱自無所有無所有處即顯空如)。】


「　功滿行空虛」


【(功謂功德行謂行業徧滿空虛周圓無際寶積經云不思議業

用徧一切處隨眾生見聞利益)。】

「　中心白[慘-(彰-章)+小]舒」


【(寶性論云但以中心不住一切
無執無著覺諸相空心自無念念起即覺覺之即無故以修行妙門唯自慘舒)。】

「　無情能解脫」


【(無情者即無為真解脫之性離縛義也淨名經云解脫相者則諸法也謂
名生於法法生於名名既解脫諸法同解脫)。】

「　終不鎮常愚」


【(瑜伽論云覺知
前念起𢙣能止後念令其不起覺察妄念終不常愚)。】


「　德量天然性」


【(德量無邊皆同一法楞伽經云謂此真實常如甚性故曰天然
也)。】

「　大施恩惠敷」


【(有大恩惠普濟無遺敷祐四生垂休億劫故寶性論云隨所示
現利樂有情)。】

「　不能歸正道」


【(華嚴經云不能了自心云何知正道彼由顛倒
慧增長一切𢙣故以悟了方知正道)。】

「　聖教離凡夫」


【(謂佛言教能令眾生遠離
諸惑故對法論云依教起行違甚深理精進長時不滯二邊證大聖果)。】


「　論識卒無休」


【(論者言論識謂心識謂有無窮之義故曰無休唯識論云一切
諸法唯依妄識而有差別之義若離妄識則與覺慧相應)。】

「　迷情流似水」


【(蓋由
迷自法身執此四大為我所以種種造業長溺愛流故金剛經頌云若到波羅岸入理出
迷情)。】

「　覺他不妄時」


【(楞嚴經云自覺巳圓能覺他者願度一切末世眾生故以
除滅妄緣成聖性故)。】

「　真境靈臺是」


【(般若論云真境者即真諦也是般若
所緣之境靈臺即是智也能了其惑性)。】


「　利樂人天法」


【(謂佛演說妙法利樂人天令得菩提皆成種智故維摩經云以
斯妙法濟群生一受不退常寂然)。】

「　宗門作楷摸」


【(宗門者圓頓一乘之教楷
摸邪正入聖津梁直示圓成俱為要妙法華經云如三世諸佛說法之儀式)。】

「　若能廻向智」


【(寶積經云恒以善心如是廻向為令一切眾生成就㝡勝證無上道)。】

「　慈海道無踰」


【(興慈進道積德累功如海無窮更無過越故華嚴經云廣集眾善猶
如大海為眾善所依之處)。】



「　益眾他緣利」


【(謂利他妙行益眾良緣開示遂徒令歸正路故智度論云
智慧自在常為利益一切眾生而修善法)。】

「　三千行不麤」


【(虛心泯相行願不麤
密用潛興威儀微細故楞嚴經云乃至三千威儀八万微細身心寂滅)。】

「　妙門終自有」


【(密嚴經云妙觀門中無思念慮任運成事終自圓明本來妙有)。】

「　解者越毗盧」


【(毗盧者此云徧寂即法身也妙智現前方成正覺妄心無處即越毗盧故華嚴
經云毗盧遮郍佛願力周法界)。】


「　癡心不可見」


【(癡心者謂於理事迷闇為性寶積經云癡從分別生亦非有癡
性與佛性平等無差別不了癡迷惑故云不可見)。】

「　舉止自相拘」


【(身心動靜
妄執自他故成拘礙我空法寂何所不通般若論云泊然永寂靡執靡拘)。】

「　多為迷塵障」


【(為迷妄想執取繫著塵障淨心未能開悟圓覺經云皆是六塵妄想緣氣)。】

「　菩提道法孤」


【(智度論云菩提者此云覺義法為淨妙道曰真空假因果以圓成證
菩提之極品斯之謂欺)。】


「　悉說甚深法」


【(華嚴經云如是一切法皆悉甚深又云諸佛無有法佛於
何有說但隨其自心謂說如是法)。】

「　因中妙相起」


【(仁王經云十住菩薩謂地
前因位而現八相成道即是因中起諸妙相)。】

「　身空無舍宅」


【(舍宅者即五廕
也身空者維摩經云四大合故假名為身四大無主身亦無我故云空也又華嚴經云是
不著一切窟宅者之所住處)。】

「　解即心聞喜」


【(寶積經云悟解如是㝡上法門
如實不顛倒希有自在法能生歡喜心)。】


「　顯法無高下」


【(謂諸法本來亦無彼此常自相寂無所動轉維摩經云法無高
下法常住不動)。】

「　真空性本如」


【(智度論云以顯法體空無虛妄淨妙滿足常如
其性般若頌云始終無變異性相本來如)。】

「　眾生無處問」


【(塵經未出寶藏猶埋
秖由迷自真心所以無處問覓楞嚴經云勝淨妙明不從人得)。】

「　波底隱靈



【(識浪波中心珠自隱情塵障裏覺性常存華嚴經云譬如大海有無量寶珠)。】


「　攝義歸真海」


【(所有無量義攝歸則唯是一法一法之外更無可得起信
論云無量功德藏法性真如海)。】

「　縱橫理不拘」


【(百法䟽云能緣智寂所緣境空
理事不拘縱橫無礙)。】

「　能眀一切法」


【(華嚴經云能解身如實眀達一切法知法
悉虛妄其心無所染世間所有法一切悉虛妄)。】

「　諦信自懸殊」


【(諦謂審諦
信乃誠信了真諦之法本性空寂俗諦之法似有即空真俗之法信自懸殊華嚴經云諦
了是身者於法無所著)。】


「　受想多無定」


【(受想二法和合為義緣就即生緣散即滅生滅之間故云無定
大般若經云受想行識亦復如是)。】

「　眀人智却愚」


【(謂慧解俱勝智斷愚癡故
名明人肈論云悟其所由則智心自眀智心既眀則道心自成又云眀達正理無所分別
故嘆以為愚)。】

「　燒金兼鍊礦」


【(礦喻煩惱金喻覺性鍊出潔淨即有晶光密嚴
經云碎末於金礦礦中不見金智者巧鎔鍊真金方乃顯)。】

「　大匠顯真如」



【(大匠者即大智慧也又密嚴經云能除垢染以顯真如)。】


「　住則證他因」


【(謂修證地位他因自果於其相中而智慧不住不著華嚴經云
住於無所住普徧而不動我觀一切法皆悉得明了)。】

「　根源佛種子」


【(唯識論云
本性住種性習所成性謂持佛種性得妙法身心住菩提集眾福田華嚴亦云為不斷如
來種性故)。】

「　懷疑障自生」


【(疑者於諸諦理猶豫為性疑自不能入理起信
