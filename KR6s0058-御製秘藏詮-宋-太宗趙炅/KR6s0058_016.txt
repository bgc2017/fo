










御製秘藏詮卷第十六　　　　　　駕


「　幽冥如是說」


【(幽冥即三𢙣道之名法華經云作如是說幽𡨋之處日月
威光所不能照)。】

「　善道信心同」


【(人天二趣名善道以人信十善因定歸善
趣故修習三業而行善行唯識頌云善謂信慙愧無貪等三根)。】

「　地獄非遙遠」


【(地下有獄名曰地獄瑜伽論云此洲下過三萬二千由旬有等活地獄下過四千由
旬有諸地獄聖教指陳目前可驗名非遙遠)。】

「　凡愚智不通」


【(三世因果
善惡分眀上士猛浪而知凡愚疑惑不達法華經云肓𡨋無所見不求大勢佛)。】


「　伏忍嗔心住」


【(論云断物命根故名曰嗔忍受眾苦制伏三毒攝心一境
住無為本仁王經云初伏忍位起習種性)。】

「　受持語自默」


【(領受真心持之
不散萬緣俱默一法頓清道絕語言妄依何起大般若經云若取文字相即非實相)。】

「　能思一念時」


【(一念者時之極少能思者運智之端運智一念便悟無生法華
經云於剎𨚗頃發菩提心便成正覺)。】

「　眼覩天花出」


【(一法通真色心俱幻
悟妙理凝然不變達妄法眼見空花護國經云無量色心皆如幻故)。】


「　不同彼有慾」


【(愛慾覆心昬乱重濁法性澄潔不同慾染圓覺經云煩惱
斷盡便證實相)。】

「　聽受廣無窮」


【(聽受者領納義智納真如廣無涯畔則安
樂解脫永處無生華嚴經云一切法門無盡海一言演說盡無餘)。】

「　種福非緣捨」


【(捨施之福但屬色相終有邊際𡨋心內達道契無生而無有竭能断經云
而此福德勝前福德)。】

「　心魔淨界中」


【(若著塵境乃是心魔觀空趣真方為
究極誠謂淨界華嚴經云彼有顛倒惠增長一切𢙣)。】


「　信緣心不逆」


【(信符大道一志眀真心遇妄緣豈能違信華嚴經云心不
違思思心不違境自然入覺海)。】

「　安樂自然通」


【(情除妄幻離縛逍遙危怖

不拘自然安樂法華經云菩薩善修如是安樂心)。】

「　善法眾生得」


【(真常
之理名為善法性順益故眾生曉解同入法源華嚴經云令一切眾生證一切功德智普
皆清淨)。】

「　如歸指掌中」


【(浩渺真如徧周沙界悟之如觀掌內迷之曠劫
難尋法華經云如來眀見無有錯謬)。】


「　智了何分別」


【(無相妙智中外皆空纖塵不立觀門寂滅都無分別大般
若經云內空外空故)。】

「　周知幻化中」


【(包括曰周通真曰知聖人以無知之
智達一切法皆如幻華嚴經云如響亦如夢如幻如變化)。】

「　若能觀自在」



【(有無二相都不縛情居塵離塵觀心自在華嚴經云無有造作及作者觀心於此得自在
者矣)。】

「　如法滿虗空」


【(如謂真如有生解義名法此理廣博巨周沙界控馭
真空十方皆遍華嚴經云諸法寂滅非寂滅遠離此二無分別)。】


「　蓬若夾蘭生」


【(蓬喻庸鄙蘭喻聖人庸人近聖必滅塵勞蓬夾蘭生漸資
香味涅槃經云猶如伊蘭花出於栴檀香)。】

「　薰蕕悟染俗」


【(薰蕕令生覺悟
知染俗因捨染歸淨同會本源彌陁經䟽云慕善芳蘭襲慶朋惡鮑肆薰風)。】

「　非同一氣類」


【(賢愚類別稟氣非同根行雖殊法性無二華嚴經云伹隨眾生根噐
所冝略說如來少分境界)。】

「　寒暑相催促」


【(陰極曰寒陽熾曰暑三界催煎
未逃遷變豈若無為妙理永寂囂塵對法論云夜半日沒中日出四洲等)。】


「　我信眾生苦」


【(我者勝大之義信謂忍可之辤眾生苦即三界無樂故法
華經云貧窮困苦愛別離苦)。】

「　來時去日同」


【(往來人間去生天上徒經
刧數不悟正因心心不達處處皆同華嚴經云是故諸凡夫輪廻生死獄)。】

「　善根無惡趣」


【(地獄餓鬼畜生号之𢙣趣觀空智起名曰善根念念無生有何惡趣
金剛經云以於百千萬億佛所而種善根)。】

「　降伏妄心通」


【(身行調而利濟
心語順而道圓妄識斯除心通奧旨華嚴經云知妄本自真見佛則清淨)。】



「　真諦更無說」


【(佛說一代時教二諦攝盡惟真諦一名獨稱法性名言路
絕體徧用寬華嚴經云法性寂滅離言說)。】

「　迷情事更重」


【(速情不了強立
名句依言計著妄幻遂生塵事加增不能曉達華嚴經云譬如目有翳不見淨妙色)。】

「　能求知識見」


【(鑑達是非名曰知識揀擇善惡稱之曰見今倣習彼名之能求
華嚴經云親近善知識愛樂白淨業)。】

「　境外自相逢」


【(不著六塵制心一定
運智於妄境之外相逢於真如之海華嚴經云無取亦無見性空即是佛)。】


「　慧意重宣說」


【(憫群生未悟遂起化身應彼大根重宣要旨故名慧意重
宣說華嚴經云法慧先巳說如來真實性)。】

「　如來大信通」


【(大信者佛性也
通者傳易義演易如來妙理普使聞知賢愚共有令修果證華嚴經云於如來所起淨信
以信得成不動智)。】

「　平心三界內」


【(一道觀真更無虗假雖居三界之內
而性逝塵勞之表華嚴經云三界一切空是則諸佛見)。】

「　法體自優容」



【(空法之體本自優容聖性了然逍遙廓寂華嚴經云其來無所從去亦無所至)。】


「　蒿萊同賤質」


【(蒿負陰陽之氣豈識榮枯人依真如之理不知求悟同彼
蒿萊法華經云父知其子志意下劣)。】

「　通理喻盈縮」


【(空性普遍淨猶虗空
盖緣機器不同遂致盈縮有異盈縮凡聖兩別華嚴經云諸佛於一微塵中而常轉大法
輪)。】

「　若變世間心」


【(緣事之心顛倒分別名世間心革除妄想攝念無動
陶鍊聖智世心自變楞嚴經云若能轉物即同如來)。】

「　願求真眼目」


【(真即
法身理眼即報身智願求真智證於妙理法華經云願得如世尊慧眼苐一淨)。】


「　說經眀聽受」


【(世尊說經空生聽受佛非所說迦葉親聞真理凝然離言
離意金剛經云無法可說是名說法)。】

「　妄相盡為空」


【(妄相不實審達皆空
真界洞然都無所得唯識論云若時於所緣智都無所得)。】

「　障礙無形影」



【(障礙不了妄生立相如形似影欝起交蘆淨照虗明量無等比法華論云無障無礙故無

形影)。】

「　詮源在處通」


【(詮釋法源顯現真相體用俱徧於法皆空金剛經云
若此經典所在之處即為有佛)。】


「　理勝皆無相」


【(理者真理也勝謂超過一切有為法故名無相法華經云
諸法寂滅相)。】

「　除疑任所從」


【(除疑者唯識論云疑唯分別入見即除悟
理證真廓然自在名任所從)。】

「　經文論慧福」


【(一切經文多眀福慧福趣
無生之法慧修無漏之因福慧二嚴法性微妙法華經云說佛智慧故諸佛出於世)。】

「　惡草善因風」


【(真妄交叅善惡相濟不息二見迷起根塵若了一真善芽增長
法華經云譬如大風吹小樹技以是方便令法久住)。】


「　詮意心中妙」


【(詮量聖意莫若了義經文微妙法源俱現一心之內故法
華經云甚深微妙法我今巳具得)。】

「　五真六化眀」


【(五真者唯識論云四
智心品及清淨法界六化者即六通化現一一無邊明示眾生起智断惑悟證真理)。】

「　金剛堅固力」


【(金剛者喻智也智力堅強能摧煩惱故法華經云智慧力堅固)。】


「　世法自蹤撗」


【(世法者即世俗法也證真達俗障礙俱遣任自蹤撗法華
經云通達無礙)。】


「　昏法難眀照」


【(無眀煩惱覆蔽真心惑障所昏智難眀照瑜伽論云真義
心當生常能為障礙)。】

「　𡨋心達妙覺」


【(𡨋𢍆本心通達實性微妙真覺一念
相應法華經云通達大智到於彼岸)。】

「　何曾有二三」


【(一乘理體本自凝然
性絕自他何有名數法華經云無二亦無三除佛方便說)。】

「　理性盡通學」



【(理即對事彰名性乃就相立号聖賢俱證故盡通學金剛經云一切賢聖皆以無為法而
有差別)。】


「　求梵藂林譯」


【(慈力能仁演金文於西土悲心大士求梵偈以東來譯士
如林共翻大義西域傳云群賢共譯)。】

「　高僧論亦通」


【(高僧者即今古之德

也造論製䟽悟性眀心具述果因通論慧福法華經云定慧力㽵嚴)。】

「　凡情知正覺」


【(凡情仗其言教了義皆通自知本心有菩提性故言知正覺法華經
云自知當作佛)。】

「　心印了然空」


【(心心相印祖祖相傳諸去本空了然洞達
華嚴經云海印發揮現無前後故)。】


「　解譯中天語」


【(印土中天真梵之語群賢解譯華夏流通如梵語素怛囕
譯就華言名之為經)。】

「　因緣法相中」


【(理事俱彰故名法相因緣道理具在
其中法華經云如是因如是緣如是相)。】

「　彌勒菩薩頌」


【(四十九頌彌勒
親談讚彼經文流行不墜金剛經頌云佛教如是著萬代古今傳)。】

「　如來一信同」


【(諸佛設教一味皆同證人無為湏憑實信法華經云如來說法一相一味)。】


「　證果西方土」


【(權身起化證果西方淨穢何殊生滅不異生滅不起淨穢
阿居方了真心寂然無證淨名經云心淨即佛土淨)。】

「　分眀善事同」


【(分眀
者曉了之義善事同者即此方他界理體無別也金剛經頌云無別有圓成)。】

「　一真無委曲」


【(一真本體性離百非心境都無何有委曲法華經云正直捨方便)。】


「　高士解心通」


【(解心既起迷執俱除故云高士智性真常通含萬像法華
經云隨義解佛語)。】


「　善譯經文聖」


【(經文了義善譯玄音惑障遣除悟聖言理金剛經云一切
諸佛皆從此經出)。】

「　法門喻的礭」


【(化門應現指喻無窮法體絕言了然
的礭法華經云諸法實相義巳為汝等說)。】

「　若言諸世情」


【(唯求世法未測
真機性相交叅迷情妄執法華經云一切眾生類無能知佛者)。】

「　男子女人惡」


【(劣心未遣善𢙣立名執自執他情計斯起唯識論云善不善俱非)。】


「　志願慈悲力」


【(四大誓願稱性無邊慈悲力增暜及法界一切群品盡獲
善利法華經云以大慈悲力度苦惱眾生)。】

「　光含諒彼同」


【(光含萬像共轍

味途法界凝然體無變異信知本性一切皆同法華經云我智力如是慧光照無量)。】

「　無因無著相」


【(若見諸相計執紛紜情想不生復何所著淨名經云不著相求)。】


「　消息自然通」


【(既除執性息慮澄神消遣迷心自然通達金剛經頌云息
慮滅迷情)。】


「　果差難證覺」


【(一乘果體無差五性隨心有異執著小果不求大乘攝論
云二乘作意障障大菩提故)。】

「　大戒一微中」


【(大戒體空本無持犯外同
真際內處一微非表所明離諸塵累華嚴經云在於一塵中一切塵亦然)。】

「　寂淨起諸界」


【(體淨真常凝然湛寂愍諸情界起應權身法華經云十方諸國土無
剎不現身)。】

「　因緣唯識通」


【(因緣之理唯識所宗識性若通諸緣自寂華嚴
經云諸法所生唯心所現)。】


「　因依無上果」


【(因中求果六度為依萬行妙因克成果證法華經云今得
無漏無上大果)。】

「　聖道雪山峯」


【(法性真身本無方所化體示現雪嶺修心
苦行六年巳圓聖道法華經云而起大悲心我始坐道塲)。】

「　真實慈悲力」



【(真如實智廣運慈悲法雨一霑三草皆潤金剛頌云菩薩懷深智何曾不帶悲)。】

「　如來一境空」


【(如來體性用徧河沙一境了然空無所有法華經云如虗空無所有性)。】


「　香水灑壇塲」


【(嚴潔壇塲起內心淨慧灑持香水表五分齊臻法華經云
以香水灑地)。】

「　心通諸事實」


【(悟彼妄緣心通妙道弃其虗假理事皆實
性本無差真空義顯法華經云引導諸眾生通達無量義)。】

「　欲知無上道」



【(天親論云菩提涅槃二轉依果智無等故無過上故名無上道法華經云持此一心福願
求無上道)。】

「　妄想忙終日」


【(妄想既生心忙終日若達無上諸緣不生法華
經云我等終日盡夜每自剋責)。】


「　義從心識變」


【(如來義海平等無差從彼識心生解有異法華經云上中

下性各有差別)。】

「　决利愛河中」


【(法雨既施群情獲利决定免愛河之苦當
來證常樂之門法華經云利益一切而生三界)。】

「　始未知無益」


【(無始巳來
未聞真教知身無益徒伇輪廻法華經云不覺不知不求出離)。】

「　清凉世界空」


【(一念了空便入清凉之境真心若起方知世界之虗法華經云如清凉也能滿一切
諸渴乏者)。】


「　法智修行妙」


【(精進修真去疑断惑以絕學智妙證菩提法華經云盡行
諸佛無量道法)。】

「　慈音徧九霄」


【(慈音一震普利十方妙達離微極無邊際
法華經云佛聲深遠甚微妙)。】

「　唯心即是佛」


【(心者緣慮義百法論云一
切㝡勝故造善造惡五趣輪廻乃至成佛皆由心作)。】

「　禪定善根超」


【(禪定
者四禪八定三眀六通諸功德法離根塵也法華經云禪定解脫等不可思議法)。】


「　旨演真宗要」


【(如來秘旨演暢真宗善開方便之門能證無為之性法華
經云此經是諸佛秘要之藏)。】

「　心逢意馬調」


【(正真之要必能觀察不起
妄念唯寂本源意馬如閑咸歸聖道大般若經云如調慧馬)。】

「　攀緣情猛利」


【(攀緣諸境惑障拘牽猛利真心能出緣慮法華經云深着於五欲癡愛故生惱)。】

「　忍辱自逍遙」


【(忍辱自在無所冤親道本逍遙自然清淨法華經云柔和忍辱
衣諸法空為座)。】


「　風惡吹舡舫」


【(惡業風生吹飄苦海眾生性量日用不知法華經云假使
黑風吹其舡舫)。】

「　縱撗不可測」


【(五趣縱撗往來不定如於大海不測波濤
法華經云一切眾生類無能測量者)。】

「　迷心閑日月」


【(迷情不悟黑闇蔽心
雖然佛日長眀業力自生障礙淨名經云是盲者過非日月咎)。】

「　秘教本來密」
