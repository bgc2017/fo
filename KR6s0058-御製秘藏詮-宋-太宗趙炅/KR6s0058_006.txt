










御製秘藏詮卷第六　　　　　　　車


「　蓬轉緣何事」


【(秋蓬小草飄轉逐風可喻人天業識流浪究緣何事妄念
為因法華經云成處人天是小藥草)。】

「　非能比樹高」


【(心能明了取諭菩提
似諸叢林還有等降不執根性豈論高仾法華經云如是菩提名為大樹)。】

「　無窮不解義」


【(法義無窮等塵沙而莫盡小智量狹不解達彼云微起信論云菩薩
以深信真如理故少分而見)。】

「　束手豈能操」


【(欲濟大事不假施功靜慮
澄思非求操執圓成勝道於理如然楞嚴經云若不修行如人說食終不能飽)。】


「　去住有無形」


【(過去是無現住是有有無形對俱是假名華嚴經云無去
無來亦無住)。】

「　禪僧傳六祖」


【(梵語禪郍華言靜慮悟心即佛了法全真
一切現前清淨三昧僧者傳心印人自達摩傳可可傳璨璨傳信信傳忍忍傳能如上六
祖皆付法傳衣也)。】

「　真誠是道塲」


【(無上覺道寂滅之塲要是真誠無諸
妄慮動靜如一取捨元空楞嚴經云汝坐道塲消落諸念)。】

「　一切能堅固」


【(一切
者含攝義佛位證真一切諸法悉能堅固莊嚴論云捨無常色獲得常色)。】


「　蓬萊凢草質」


【(芝蘭諭苾蒭蓬蒿類𢙣草甘雨雖潤臰性不移楞嚴經云
草葉縷結詰其根元咸有體性)。】

「　林木莫知高」


【(法喻林木長育善苗悟達
真源如植勝地情與無情之類皆含妙慧之宗楞嚴經云皆由不知常住真心)。】

「　自是本情物」


【(真如妙覺不逐物情一性本真誰強分別法華經云知此眾生種相
體性)。】

「　愚懷一等褒」


【(不矜智慧不執庸愚能自覺知何勞褒貶法華
經云我觀一切普皆平等)。】


「　昬暗人知覺」


【(無明諸惑昬暗真途行人覺知起智照燭障惱永断勝道
斯圓唯識論云菩薩金剛喻定三界頓断)。】

「　緣來氣味香」


【(假修進為緣趣

當來佛果滋解脫味成五分香氣象清真克圓妙證淨名云以香飯氣味而為佛事)。】

「　頓超無上果」


【(𠛴郍獲果超證僧祇䪺悟真乘達無上果佛頂經云忽然超越
世出世間)。】

「　戒定慧昭彰」


【(楞嚴經云所謂攝心為戒因戒生定因定發慧
是則名為三無漏學欲圓大果此三為基佛語昭彰依之修學)。】


「　智益甚深理」


【(諸佛之智妙用甚深順益有情其理叵測唯識論云四智
心品妙用無方皆順益故)。】

「　三途不可量」


【(瑜伽論說三途名三𢙣趣亦名
三𢙣道謂地獄餓鬼傍生造非福業之所招感多分苦受於深妙理不可測量)。】

「　豈能窮道性」


【(窮究道性唯在人天若彼三途豈能分別楞嚴經云不遇如來
不聞正法於塵勞中法尒輪轉)。】

「　暗想願無方」


【(𢙣趣無慧同彼暗中發願
想心希諸善利今約別報願力無方楞嚴經云有諸鬼等發願護持修菩提者)。】


「　住法不思量」


【(悟諸法不生寂而安住了自心元湛誰作思量性靜神怡
廓然自在寶積經云能知出世法如是平等住)。】

「　圓光世所覩」


【(性光圓發三世
洞明證解現前達無所覩一切動用語默皆真華嚴經云汝獲智光明法界悉充遍)。】

「　端嚴智慧門」


【(萬行圓修福智雙俻獲塵沙勝法依妙慧為門華嚴經云若獲
妙福端嚴身)。】

「　救度無常苦」


【(起無緣慈救眾生苦度煩惱深海遠諸法
無常般若經云度一切苦厄)。】


「　香界如山岳」


【(淨名經云其界一切皆以香作樓閣經行如是諸佛國土
山岳園林盡為香事又云身諸毛孔皆出妙香)。】

「　圓周大吉祥」


【(一切包攝
名号圓周殊妙應奇惣真法界稱大吉祥華嚴經云諸吉祥中㝡無上)。】

「　三天示法信」


【(三天者欲天色天無色天也佛出三界示化群生說法運通令生大信起信論
云為欲令眾生起大乘正信)。】

「　濁世有清涼」


【(染惑有情處濁世而熱惱
明真大士即此界以清涼楞嚴經云若能轉物則同如來)。】



「　風起心中火」


【(外境如風恚生似火中心發起違境作緣楞嚴經云心熱發火
鑄氣為金)。】

「　四天欲簸揚」


【(四天者欲界四洲天有情也起煩惱火隨境簸
掦熾盛飛騰焚燒善法法華經云其舍𢙢怖變狀如是)。】

「　三光無變相」



【(三光者日月星三也以光喻性無變異相染惑既滅性光自圓楞嚴經云常光現前)。】

「　花卉自芬芳」


【(真界清淨道樹芬芳覺卉花敷自性香潔維摩經云無漏法林
樹覺意淨妙花)。】


「　中外妄謌掦」


【(迷情計執心居中外二邊既立妄有謌掦中道理非宛成
滯礙楞嚴經云在外非義當在中間無有是處)。】

「　無人識義方」


【(無智慧人識心
昬昧不解法義迷方便門起自執情必違妙義楞嚴經云種種顛倒錯乱修習)。】

「　凢夫說我見」


【(末起聖道名曰凡夫諍論𥪰興說為我見分別惑起正解不生瑜伽論云
薩迦耶見唯依分別)。】

「　終不解施張」


【(不解正理迷倒自分妄有施張終成
異見因斯流浪失本真靈華嚴經云不隨正道入於險道)。】


「　慈妙大悲心」


【(佛位大慈得名為妙拔苦無量名曰大悲運此二心普濟含識
華嚴經云慈悲憐慜遍一切)。】

「　惠施如法雨」


【(惠施一門惣包諸度三施之首
法施為初佛所轉法輪猶如大法雨法華經云普雨大法雨度無量眾生)。】

「　迷情含垢根」


【(迷倒有情識含垢染六識起惑發自六根由此輪迴諸趣不息楞嚴經云生滅
輪廻亦汝六根)。】

「　功德難勝舉」


【(無明覆境結惑尤增進道力微功德斯減
華嚴經云凢夫愚癡煩嚚貪著流轉無窮)。】


「　天人常快樂」


【(光潔自在神用名天修諸禪定常受快樂善戒經云但以
人天善根而成熟之)。】

「　福慧等無疆」


【(深忻靜慮福慧無疆雖獲殊緣果唯
有漏攝大乘論云布施持戒福生天上)。】

「　不獲摩𡰱寶」


【(摩𡰱寶即如意
寶也喻眾生佛性本自圓明而彼天人不悟菩提心因唯修有漏故云不獲楞嚴經云迷

悟由心如藏大寶)。】

「　還憂失妙光」


【(淨慧未生恒居快樂五衰相現還失妙光
佛頂經云天福將盡光明乃失)。】


「　精懃無懈惓」


【(用精進心求無上道懃筞三業俻修眾善即無懈惓也法
華經云情存妙法故身心無懈惓)。】

「　不憚度炎凉」


【(憚由懼也發心行道何懼
炎凉將歷三祇俻修萬行佛本行經云發心求於道俻受諸炎苦)。】

「　若達菩提境」


【(菩提若達境絕攀緣色相俱空一邊不立維摩經云順是菩提了境相故)。】


「　清心是道塲」


【(淨名經云隨說法淨即智慧淨隨智慧淨即其心淨其心
淨故即佛土淨穢淨平等俱為道場)。】


「　嚴飾威儀像」


【(莊嚴聖像示現威儀妙相可觀離諸見故瑜伽論云真相
非相即如來相)。】

「　龍天百寶裝」


【(龍梵天人福因修感雖獲隨身百寶常以
裝嚴佛事華嚴經云諸佛經行處龍天常供養)。】

「　不能光妙用」


【(發明妙慧謂之
光起依真智謂之用不能體斯玄旨雖日用而不知法華經云甚深微妙法難見難可了)。】


「　無色亦無香」


【(如來不可以色相見不可得以音聲求故無色無香無受
無觸般若經云無色聲香味觸法)。】


「　悲感諸天龍」


【(悲能拔苦感應物機故龍王見影皈依天人蒙光獲利華
嚴經云隨緣赴感靡不周)。】

「　信行調御主」


【(信行為十地之階調御則三界
之主富有萬德号兩足尊法華經云調御丈夫天人師)。】

「　常持般若心」


【(常持大智
般若之心教化利樂一切有情同昇彼岸維摩經云即隨所應而為說法)。】

「　大救六情苦」


【(運廣大心悲救諸苦皆令解脫以證無生法華經云為說涅槃盡諸苦際)。】


「　蘊界設壇塲」


【(蘊者聚義界者因義此即眾生五蘊十八界也壇塲即是設法
之塲華嚴經云諸佛於一切眾生身中轉大法輪)。】

「　重宣大法張」


【(華嚴經云
眾生本有佛性自不覺知因佛說法方能曉悟故云重宣大法即無上菩提也)。】

「　明珠



【(華嚴經云摩𡰱寶珠与天帝珠網一摩𡰱中盡含珠網眾珠網裏揔見摩
𡰱一攝一切一切攝一故云涉入)。】

「　交㸦映靈光」


【(靈光者喻眾生真性与佛
無異若寶網与摩𡰱交㸦相映)。】


「　陽烏出海面」


【(陽烏則太陽之精喻佛後得智出無明海慧日麗天昏蒙
皆照法華經云慧日大聖尊久乃說是法)。】

「　湏彌照耀光」


【(湏彌眾山之王四寶
所成日光先照喻頓根菩薩㝡初聞法機緣相契照耀皆光華嚴經云如日初出先照高
山)。】

「　普見十方界」


【(佛日無私慈雲普覆十方橫遍三際竪窮華嚴經云
譬如淨日輪普照十方國)。】

「　真空示法王」


【(一真法性湛然常寂真空顯示應彼
群機自在無畏故号法王維摩經云法王法力超群生)。】


「　俗阜民知善」


【(金輪王治四天下三教鼎峙俗阜民安皈仰三乘精持十
善不墮地獄自證天堂功行成熟必契真如華嚴經云念念增長諸善根)。】

「　修行業不生」


【(修六度萬行趣求佛果断諸煩惱業即不生維摩經云又於諸法如說修行)。】


「　能窮寂滅法」


【(𡨋證真如能窮寂滅則法法皆如乃空空無礙法華經云
諸法從本來常自寂滅相)。】

「　靜慮泯浮名」


【(人法本空性相非實安住靜慮
故泯浮名維摩經云假名是菩提名字空故)。】


「　修道求根慧」


【(修無上道求根本智證佛妙慧利諸有情法華經云修習
一切無上之慧)。】

「　乖為成妄想」


【(不修無漏實智而求生滅之法凢有所作
皆成妄想也法華經云但離虛妄名為解脫其實未得一切解脫)。】

「　欲知無上真」


【(無上真理欲樂求知盖諸佛之本源乃眾生之自性不断不常非生非滅法華經云
涅槃真實法)。】

「　還同明鑑像」


【(對真顯妄因妄談真真既不存妄由何立
明鑑影像病眼空花維摩經云如智者見鑑中像)。】


「　諦信如來果」


【(悲興四攝化被三身果海無邊誠堪諦信法華經云道場

所得果我巳悉知見)。】

「　精嚴妙解成」


【(精進嚴持求佛勝道心無退轉妙解
圓成華嚴經云勤行精進於一念頃得百千億三昧)。】

「　當仁能體用」


【(當仁
即是學佛之人也諦觀諸法三際平等一念現前體用無差湛然常住法華經云觀諸法
性無有二相)。】

「　真相廓然清」


【(真如實相本自凝然虛妄既除廓尓清淨
華嚴經云真實平等常清淨)。】


「　語解因緣聖」


【(因言悟解了諸法之緣生不離見聞達諸佛之聖智維摩
經云不離眾生界便入佛界)。】

「　眾生用不平」


【(眾生妄起見三界之不平
本覺明時了一真而常寂華嚴經云諸佛境界亦復然體性平等不增減)。】

「　萬緣徒坦蕩」


【(我執情分法生一異真源坦蕩境絕萬緣其若淨照雙流咸歸一致華嚴經云
遠離妄想及諸取令心所向皆無礙)。】

「　一性自圓明」


【(六情迷執分別自他
一性圓明湛然清淨楞嚴經云妙見圓澄更無塵象)。】


「　言用精窮當」


【(麤言細語皆契真理精窮妙義故曰當機維摩經云言說
文字皆解脫相)。】

「　詮源道理深」


【(詮演真源一乘妙法道理無極故曰甚深
法華經云深知諸法之門)。】

「　功成無障礙」


【(功滿三祇二障永断通達自在
故云無礙華嚴經云成就十力四無畏具足智慧無所礙)。】

「　種果一般心」


【(種果
修因寧圓萬行菩提妙理唯一真心非二非三更無有異法華經云發心等皆一般趣果之
心也)。】


「　道合萬緣心」


【(至道虛凝如空周徧隨冝攝化皆合萬緣華嚴經云佛智
廣大同虛空普遍一切眾生心)。】

「　五天皆信仰」


【(大聖興化始自五天機緣
感應咸皆信仰法華經云天人所戴仰龍神咸恭信)。】

「　無如我世尊」


【(外道迷執
小乘偏見降伏魔冤成等正覺無如世尊也佛本行經云世間所有我盡見一切無有如
佛者)。】

「　法妙玄中象」


【(法謂妙法玄即幽玄微旨難窮故曰玄中之象華嚴

經云如大明鑑內現眾象)。】


「　純性自通靈」


【(純一之性無諸垢染与法相應故曰通靈涅槃經云陶冶
塵滓如鍊真金萬累都盡靈覺獨存)。】

「　澄源不暫停」


【(澄源真性常自湛然
出生諸法故不暫停仁王經云口常說法非無義心智寂滅無緣照)。】

「　欲知迷悟事」


【(迷即六趣輪廻違背正道悟即一念成佛了妄皆真楞嚴經云迷悟生滅了無所得)。】


「　萬象本無形」


【(諸法自性皆空遍計強分名相若了依他本無形相圓覺
經云幻從諸覺生幻滅覺圓滿)。】


「　和順如來教」


【(佛以一音演說諸法眾生隨類各得解和順教理咸歸一乘法
華經云我此九部法隨順眾生說入大乘為本)。】

「　敷揚妙義成」


【(一切諸法
本性咸真若善敷揚皆成妙義法華經云以微妙義和顏而說)。】

「　通流無𦊱礙」


【(如來妙法體即一真益物流通故無𦊱礙華嚴經云入一切諸佛妙法無所障礙)。】


「　何用巧經營」


【(真如法性本來湛寂理絕巧拙何用經營仁王經云住於
無相無生忍方便善巧悉平等)。】


「　迷妄如冤賊」


【(迷妄纏心動違真理喻如冤賊劫善法財楞嚴經云認賊
為子自劫家寶)。】

「　貪饕意不平」


【(貪愛染著堅執不捨情有憎愛意乃不平
法華經云知諸眾生有種種欲深心所著)。】

「　三祇無出日」


【(煩惱所覆迷惑

「　煩惱出無明」
