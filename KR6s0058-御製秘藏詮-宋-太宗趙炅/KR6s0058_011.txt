










御製秘藏詮卷第十一　　　　　　駕


「　志願親三寶」


【(夫佛法僧者人天師範貴賤福田故發願心志求親近華
嚴經云稽首一真三寶海)。】

「　威嚴布薩訶」


【(威神嚴肅恩被無邊今且指薩
訶略標化境故法華經云世間所歸趣救護於一切)。】

「　圓眀無礙法」


【(圓眀
能證智無礙所證理含褁十方指掌萬象更無纖毫能為障礙楞嚴經云坐微塵裏轉大
法輪)。】

「　西域禮彌陁」


【(極樂國土彌陁所居即無量壽佛其或心眀華藏道𢍆笁
皇如楞嚴經云不動道塲徧十方界向言往禮盖方便之談也)。】


「　果經千萬刧」


【(千萬刧即三大刧四願無邊六度俱備方圓極果号天人
師法華經云無量百千萬億載阿僧祇)。】

「　茂草一娑婆」


【(欲塵煩惱造作
遷流生滅循環寧窮數量亦猶繁茂之草徧一娑婆界尓)。】

「　真心非染污」



【(楞嚴經云寶覺真心各各圓滿故知妙智雖處惑中不為染污又圓覺經云圓覺清淨本
無修習及修習者)。】

「　妙藥愈沉痾」


【(妙藥則無上法味沉痾則七漏十纏
擾惱有情汩沒諸趣醫王出世方能差之遺教經云我如良醫知病說藥)。】


「　因會緣來合」


【(因辦果體緣助成功便能𢍆會真心筞勤善法經歷塵劫
求趣佛乘故肈論云一切諸法緣會而生)。】

「　能休溺愛河」


【(無眀發業貪愛
潤生種現相資漂溺群有唯無漏智起頓能斷除楞嚴經云愛河乾枯令汝解脫)。】

「　琉璃涵寶月」


【(法界之內真性混融故法華經云如淨琉璃中內現真金像然光輝
支暎表裏洞分實不思議之境界也)。】

「　皎潔印禪波」


【(禪波即禪定之心眀
印萬象皎潔無差月燈三昧經云如淨虛空月影現於清池)。】


「　和會恒沙眾」


【(恒沙則恒河中沙彼河方圓四十里喻眾生界[婁*殳]量無窮
於中種性差殊習業各異和而會集其道可知楞嚴經云隨眾生心應所知量)。】

「　真空



【(涅槃經云一切眾生悉皆有心即真空心若能轉物即妄眀真菩提
之階立可登矣)。】

「　但拋猶豫心」


【(唯識論云疑者猶豫為性若觸類無滯則疑心
自拋法華經云若有疑悔者佛當為除斷)。】

「　信解不能及」


【(信則十信解謂
勝解將趣極果假信為先華嚴經云信為道元功德母故知難及也)。】


「　義徹三天外」


【(真義非界繫所攝且欲色等三天皆是有相業唯無漏智
方能超越華嚴經云如來智慧無邊際一切世間莫能測)。】

「　非真不可過」



【(涅槃經云若有一法過涅槃者我說如幻如化故知真法何止出三界超萬有)。】

「　破邪成正覺」


【(邪謂心境俱非錯行亂學既破斯見正覺遂成法華經云我悉除邪見於
空法得證)。】

「　九十六冤魔」


【(外道所計本唯六宗師資順從各有十五執心
執法事火事龍正見既虧冤魔無別法華經云破魔兵眾而擊法鼓)。】


「　法界人天喜」


【(法界之內唯天與人能發大心遠求佛果將入初地故生
喜心楞嚴經云覺通如來盡佛境界名歡喜地)。】

「　真心是道塲」


【(願行堅固
真智相應積集莊嚴蘊聚功德念念無捨直趣菩提法華經云乘此寶乘直至道塲)。】

「　三身圓四智」


【(登金剛座獲寶王心四智三身六通十号具足妙果名正徧知唯識
論云又智殊勝具攝三身)。】

「　萬劫引流長」


【(起大悲心演深妙法盡未來際
利樂有情法華經云普欲令眾生亦同得此道)。】


「　旨達無眀要」


【(無眀之要貪愛為本楞嚴經云應當審詳煩惱根本今此
玄旨決定妙義破彼昬闇除諸惑染也)。】

「　菩提大吉祥」


【(障患既忘真源
可挹圓明之智廓尓現前參羅色塵皆為妙用菩提之果勝莫大焉故佛地論頌云名稱
吉祥及尊貴)。】

「　三乘皆聖道」


【(修心者根有利鈍悟有先後雖殊途暫異
而一轍終歸法華經云尚無二乘何況有三故知皆聖道也)。】

「　法始自稱揚」



【(敷宣大法讚揚一乘令棄羬軒使登擾駕法華經云稱歎所行道)。】



「　妄水翻波浪」


【(妄愛之水漂溺長時欲浪未寧情波安息唯識云說愛如
水能沷潤故)。】

「　崑崙徹底石」


【(我人山高借山立諭眾生以一靈性而迷
悟崑崙以一拳石以成立山雖有石未嘗覺知人各秉性不能悟解豈徹底窮理也維摩
經云起於我見如湏彌山)。】

「　端嚴袪妄執」


【(世覩端嚴而求果證若祛妄執
遂獲真見金剛經云若以色見我等)。】

「　一法終成立」


【(真般若者非有非無難思
難議名之一法亦為強名法句經云一亦不為一為欲破諸數)。】


「　玄言資智識」


【(三藏殊文統之者一致即玄言也天王般若經云利根性
人說文知義良以古教照心其心自明淨名經云多聞增智慧)。】

「　妙覺解空王」


【(因教詮理眀此覺性馳耳目於聲色之外空神心於塵象之中法體既如圓眀頓證
寶積經云是故虛空即是如來)。】

「　四眾常圍繞」


【(佛地論云無量天龍四眾
八部常所翊從蓋欽承妙旨仰矚慈顏)。】

「　金容化十方」


【(紫金光聚威容炳曜
道被千刧而彌固功流萬世而常存法華經云十方諸眾生普皆蒙饒益)。】


「　幽暗分眀理」


【(楞嚴經云雖明暗等種種差別而見無差別且如閇智塞聦豈
見精明性肯隨物而幽暗)。】

「　圓隨日月光」


【(大般若經云日月等光所不照處為
作光眀應學般若波羅蜜多即知妙智能破重昬度諸群品)。】

「　去邪歸正說」


【(真說者摧邪慧解開正知見唯識論云為於二空起迷謬者生正解故蓋破執有滯
空也法華經云拔邪說涅槃)。】

「　清淨達西方」


【(華嚴經云嚴淨不可思議
𠛴供養一切諸如來此乃智通無外心澄妄銷豈以西方而為遠哉)。】


「　大信流天下」


【(宇宙無極唯信㝡大所以至仁說一法立一事皆同䡄千聖合
轍萬化楞伽經云若諸法虛妄何故取解脫)。】

「　靈臺自樂康」


【(神者形主
心者物靈若去勞慮絕取捨樂康之道在於我矣法華經云能生淨妙第一之樂)。】

「　更談深妙理」


【(妙理無言假言詮理故摩竭掩室毗耶杜口用示無說以顯勝義寶

積經云是義無文字而長說文字)。】

「　果助法中王」


【(聖人者法法同證印印相
受蓋佛不並出餘皆晦跡師事以助揚化法華經云護助宣正法)。】


「　津岸合諸流」


【(四暴流河奔注淪溺唯六度智舟能截而渡般若經云於
一切法無所著能於此岸到於彼岸)。】

「　澄清不渾雜」


【(覺性澄湛煩惱不能
染唯識論云雖有客染而本性淨又如來藏經云煩惱身中有如來藏)。】

「　終湏歸大海」


【(華嚴經云令修諸行海疾入大智海即是具四願經三劫功成道圓歸泛
如來大寂滅海)。】

「　㳙滴亦波涉」


【(法界無際真源浩渺然一塵若礙則萬象
勃起亦如㳙滴異沠遂波涉殊途行方便經云不見一切相名見如來)。】


「　信慧無遮隔」


【(華嚴經云一切佛興皆從信起況慧目眀徹照覽無遮法
華經云如風於空中一切無障礙)。】

「　恒沙大化揚」


【(恒沙非[婁*殳]量能知歷如是
佛𠛴闡揚化道華嚴經云廣大悲雲徧一切捨身無量等塵𠛴)。】

「　龍天常擁護」


【(三界天人十方龍眾舉此二類揔攝八部常來衛護冀聞妙說法華經云天人龍神
所共供養)。】

「　聖境得清涼」


【(佛境清涼純淨無漏然鶖子見同堆阜梵王覩
若天宮取捨不生淨穢斯絕法華經云我此土安隱天人常充滿)。】


「　智遠文殊聖」


【(文殊無漏智海為諸佛宗師出贊釋迦名妙吉祥菩薩久
成正覺号歡喜藏摩𡰱寶積經云文殊大智人深遠法源底)。】

「　談非善逝章」


【(妙往菩提故稱善逝且文殊所談真理無不符𢍆佛言法華經云諸所說法隨其義
趣皆與實相不相違背)。】

「　我當常稽首」


【(我即能歸依之者既欽妙道故
發勤奉法華經云是故咸稽首歸命無上尊)。】

「　念佛一爐香」


【(念則明記為性
香以表信為功法華經云以深心念佛又云眾寶妙香爐燒無價之香)。】


「　我信三乘法」


【(華嚴經云若能信奉於尊法則聞佛法無猒足且三乘之
法理唯一味證解有殊名相致別然皆能運載到於彼岸)。】

「　周圓不可量」




【(如來所說皆為了義自非大智豈得較量而知唯識論云周圓無際量等虛空)。】

「　如來垂密旨」


【(諸佛出世皆為開佛知見其或寶所未歸化城權設此乃密旨誘進二乘
法華經云有所言說意趣難知)。】

「　像教法根長」


【(像謂取像佛時金容玉撿
流布塵沙劫石雖窮法根長固法華經云正法住倍壽像法復倍是)。】


「　塵垢昬身業」


【(五塵六垢昬障身心固煩惱根作生滅本圓覺經云於虛
妄體重執虛妄)。】

「　浮生常急急」


【(淨名經云是身如幻從顛倒起蓋四相疾
如於迸火五衰馳若於奔流遺教經云世實危脆無牢強者)。】

「　西來皆有意」


【(大教東被祖印西來或以心傳心或以教傳教化道雖異接誘是同涅槃經云一切
諸法中悉有安樂性)。】

「　猛破迷情執」


【(聖人悲極為心眾生迷情為病談中
道破空有妄執示無言顯心法本如楞嚴經云菩提心生生滅心滅)。】


「　真實非虛假」


【(涅槃經云猶如閻浮金無能說其過即喻真實之法體非
虛假證之則三界長辭背之則千生自咎)。】

「　愚徒信不長」


【(愚者暗於理事
既無深信徒廣見聞法華經云淺識聞之迷惑不解)。】

「　一心歸正法」


【(三乘正法
一心巳歸既實資粮仍堅願行優游聖道何往不臧阿差末經云常正其心不事餘學)。】


「　六趣不驚忙」


【(本心未悟妙智不眀六趣四生實堪怖畏今既情忘五欲道𢍆
一真塵垢頓銷驚忙自歇華嚴經云能壞諸見堅固牢獄)。】


「　慧徧娑婆界」


【(諸佛慈慧等視怨親閻浮界人咸承恩力法華經云我常
在此娑婆世界說法教化)。】

「　恒脩願十方」


【(華嚴經云了相十方無所有為
世廣開清淨道即是常具大願悲濟群生又法華經云佛名聞十方廣饒益眾生)。】

「　花鬘周淨土」


【(佛土真淨大地山河皆為妙用故眾花結鬘周徧嚴飾法華經
云金花諸瓔處處垂下)。】

「　寶網自生光」


【(修行位極頓證法界於帝網觀
門現種種勝事身身相攝法法㸦入故楞嚴經云交光相羅如寶絲網)。】



「　說法𦒿闍崛」


【(西域有山名𦒿闍崛勢接王舍福鎮中印佛多住彼演說
妙法法華經云汝往詣耆闍崛山釋迦牟𡰱佛所)。】

「　瑜伽道自成」


【(瑜伽是梵
語此云相應所謂心與境相應因與果相應等教若斯從聖功自滿瓔珞經云行從心得
心淨道成)。】

「　願窮真諦理」


【(履踐真諦窮究妙理願既不虛道無不剋天王
般若經云真心趣道真實思惟)。】

「　海印一時平」


【(海印即真如本覺心澄道顯萬
象皆真若海清凝無所不現起信論云無量功德藏法性真如海)。】


「　垢染真心佛」


【(如來藏經云若佛出世若不出世如來之藏常住不變然垢染
未除真心莫顯故圓覺經云彼圓覺性即同流轉)。】

「　幽眀何妄執」


【(不了色空
妄執眀暗見既成幻覺遂生勞一失妙眀復真安及楞嚴經云所妄既立生汝妄能)。】

「　塵埃頻拂拭」


【(塵銷則本體唯淨翳差則空花不生無為真如性自眀露圓覺經云
譬如磨鑑垢盡明現)。】

「　果願百靈集」


【(若事理障巳永斷滅即入如來微妙
圓覺滿足菩提願既果從道成靈集法華經云百福相莊嚴)。】


「　理達非長短」


【(真色無形真理無名故非長短之所及也寶積經云無方
無圓無短無長)。】

「　無緣及有情」


【(諸聖以無緣之慈度無緣之眾廣為佛事
也仁王經云無緣大悲常化眾生)。】

「　當空輪日月」


【(智慧光眀朗如日月超然
廣運邈矣無窮華嚴經云妙智清淨日大悲圓滿輪妙智清淨月大慈圓滿輪)。】

「　海水印分眀」


【(華嚴經所謂海印三昧也海水澄清物象皆見三昧心淨萬法自如起信
論云參羅及萬象一法之所印)。】


「　詮理堪消息」


【(雖超蹄象湏假言詮妙道故絕於聽聞至理真為於消息般若
論云言雖不能言非言無以傳)。】

「　紅黃碧綠青」


【(妄見青黃碧綠蓋是執情
正智會真寧分眾色圓覺經云諸愚癡者見彼摩𡰱實有五色)。】

「　強名俱是執」



【(肈論云以名求物物無當名之實以物求名名無得物之功成具經云是法無所有強為

其名)。】

「　達可定禪扄」


【(禪即淨慮之稱扄乃智慧之門悟達之心可定禪那
之要妙圓圓覺經云此方便者名為禪那)。】


「　求法虔心用」


【(維摩經云夫求法者不貪軀命如雪山童子為求半偈乃捨全
身此是求法虔心之用)。】

「　諸賢大聖名」


【(諸賢者地前三賢大聖者地上
十聖仁王經等說地前名三賢地上名十聖也)。】

「　𠛴那如發願」


【(𠛴那者時之極
少也發願者起菩提心少時發願便成正覺法華經云於𠛴那湏發菩提心又云便成正
覺)。】

「　法用智縱橫」


【(縱橫者自在無礙之義事無不融理無不適法界觀
云繁興大用舉必全真)。】


「　往往成煩惱」


【(百法論說煩惱有二十六煩者擾也惱者亂也擾亂有情
恒處生滅貪瞋癡等往往成就矣)。】

「　沉吟難測度」


【(小果沉思聖智凡夫

「　諸煩亂著緣」


【(迷執之過也由執我故起諸煩惱惑亂身心逢境遇緣便生愛著法華經

【(肈論云達病所應智慧意也以智慧

「　勿令顛倒見」


【(勿令者勸尓之

【(若於不二門中

「　達得始無心」


【(未了之時心常有相達得妙理始乃無心肈論云聖人無

心於萬物)。】


「　旨聖如來說」


【(如來所說旨趣甚深不可取著故名為聖金剛經云如來所說

「　靜思寬境界」


【(寂靜
思惟諸佛境界寬徧十方華嚴經云諸佛境界悉周徧故)。】

「　海月印波心」



【(海月昇空千波印影將喻佛身現諸世間㝡勝王經云佛真法身猶若虛空應物現形如

「　覿面無慚愧」



【(智見法性名為覿面纖塵不染具正徧知道越人天何所羞耻故無慚愧法華經云故佛

「　我佛周旋施」

「　眾生業自深」

「　照耀空非色」

「　理與真空合」


【(真空智起與理妙苻萬象混然故名為合維摩經云不一

「　圓融理自興」

「　三眀兼四智」

「　佛會善相稱」
