










御製秘藏詮懷感一百韻詩卷第二十六(卷上)　𦘺


「　一統𠃵坤順太初」


【(方今之御極也六合無外萬邦克寧以無心而為心
不治而為治恩霑草木信及豚魚玄同太始之風妙盡無為之化故邈𠃵坤而一統也)。】


「　照臨無外越唐虞」


【(窮日之所照極月之所臨咸一乎心率歸大化
故云無外也且堯有所不俻舜有所𣏋行我朝兼而有之不越而何)。】

「　樂推大定聲聞遠」


【(不舉干戈而天下大定凡聲教所被孰不樂推必使乎千載之下無
名而名不德而德故聲聞自遠也豈不然哉)。】

「　𦗟斷無疑自卷舒」



【(握機創歷執天下之柄以𦗟以斷可卷可舒則上帝之心授蒼生之福高懸睿鑑固無疑
也)。】

「　化導隨時彰覆載」


【(天以不言而覆而四時行焉地以不言而
載而萬物生焉今御天下者以沉潛剛克高明柔克化之導之隨時之義遠矣哉所以彰
天地覆載之德也)。】

「　周通利物混車書」


【(物即萬彙之通稱也周無
不俻通無不該因民之利以利之則及物之心曾無有量矣故書䡄以之而混同也)。】

「　唯慙寡昧齊堯世」


【(惟皇作極而民用康式驗帝書寔同堯代故云齊也
又以其大而不言其大故云寡昧矣)。】

「　深愧微言頌帝居」


【(百代之下
世出世間聖人之言通謂之微也多銘於帝座之側時或諷之必造其極然以生知自慶
崇重前脩故持言之愧矣)。】

「　端拱未知何所措」


【(垂衣端拱負扆凝旒
常慮政之未敷化之未洽而智用明略尚謂無所措焉)。】

「　招賢軫念慮関子」


【(國家懸科待士修仁來遠豊爵扶以禮忠良峻刑賞而旌懲勸並出宸衷矣)。】


「　常持警誡綏仁惠」


【(每念蒼生過於赤子而兢兢在位惕惕臨朝如
馭杇索以莅天下)。】

「　運偶承平免負芻」


【(代属昌期運當泰民安

佫阜國富家肥聞道路之歌並京坵之詠)。】

「　區別動來敷教令」


【(發號
施令黜幽陟明並革故鼎新移風易俗中興禮樂載造典章)。】

「　庶幾閑暇少縈紆」


【(萬機暇日庶政餘時或命以琴碁或[門@允]之翰墨樂無為之化暢有道之風)。】


「　千峯路險非三島」


【(千峯鏁翠萬仞盤空雖非三島之游頗盡十洲
之美幽人至止不亦遠哉)。】

「　一𦰧輕舟泛五湖」


【(懷靈之士養素之流
高乘一𦰧之舟遠泛五湖之上)。】

「　分𣲖長流丕祚永」


【(長流天潢也滔
滔遠注邈無津涯散施即為天澤分𣲖即為長河聖朝継緒垂慶弃葉流芳實猶分𣲖靈
源丕顯永祚無彊之美方諸洪流)。】

「　寰瀛相踵別親踈」


【(父慈子孝
兄友弟恭民之要道政化既洽仁惠巳敷區宇之人自相踵継和睦親踈㱕乎孝道)。】

「　姦欺巳塞根源盡」


【(寰海宣穆清之化朝𨓍任鯁亮之臣剛憲有章賞黜
司秩則姦宄之軰欺詐之徒不能恣彼兇狂肆其驕踞)。】

「　暴虐潛消蔓草除」


【(教化之所不及則侮𢢔自貽暴虐是作農工之所不勤則蔓草滋延敗乎
稼穡今無為道著可大功周強暴之人不相侵凌如蔓草之於農工不能恣其滋蔓矣)。】


「　才業所高歸汗渙」


【(抱經天之才懷佐理之器若皐戞稷卨入
凱八元俾帝王明齊日月恩同雨露施之則為乎渥澤汗渙於天地之內霑濡於動植之
中者矣)。】

「　賞功何恡借吹噓」


【(褒善貶𢙣載在春秋有功者加位准
爵無功者削地降官是知春之來昆䖝逐吹噓而生秋之至品彙隨肅殺而謝不肖則肅
殺是冝若賢則吹噓何惜也)。】

「　狂心屏跡皆依度」


【(任賢勿貳賞功
無私則猖狂愚鄙之徒復何恣貪殘之心縱猾亂之志動虞赴溺畏甚投荒遵我典章慎
茲制度)。】

「　率土咸寬整下愚」


【(普天之下莫非王土率土之濱莫非
王臣是知萬國之風繫一人之化聖朝德澤既廣仁惠亦孚惸嫠耆艾孰不願整愚蒙稟

我王度踰越之法復何聞哉)。】

「　憲法逈然摧狡蠹」


【(憲章有則法令
有䂓狂悖之徒狡猾之軰自絕姦詐俱奉條章如蠹䖝在木無所施為也)。】

「　公平俄見辯精麤」


【(公平之義精麤之門事有所歸理無不當辯別既符於政化公
清俄見於昭蘇委之軄官咸得其理)。】

「　淳和巳變澆漓態」


【(求衣未明
孜孜致理欲使澆漓之薄俗咸歸淳朴之古風守智服愚皆得其態)。】

「　奔競還知禮讓趍」


【(遁跡避傜之軰逢肆赦而盡復家鄉頑嚚愚鈍之夫遇文明而
咸知禮樂)。】

「　庶󵠪漸營鄉巷陌」


【(堡里之間巷陌之內秋豊歲稔俗阜
民康樂哉庶老之人美矣昇平之化得不營募耕耘皆遂其所矣)。】

「　守臣無事訟𨓍虛」


【(巍巍聖化囹圄空而軄掌清閑蕩蕩皇風詞訟息而公私舒泰)。】

「　願聞唘沃同韶護」


【(韶護正聲之樂唘沃直諫之詞聞忠讜之言若得韶謙之曲)。】


「　至理忠貞𢙣謟諛」


【(想先王之選士譴謫謟䛕念古聖之擢賢忠良
錄用將今倣古忠貞進而狐媚自退矣)。】

「　純直但能酬雨露」


【(為臣之道
純直是先恩既霑於雨露酬合効於忠勤)。】

「　矜持必可離差殊」


【(矜持大道
純一之理者則必超乎異端息乎紛糺自然離差殊矣)。】

「　升高目下存卑小」


【(升高者皆自下而舉理含存其卑下流其謙光不可居其安而忘其危處其貴而忘
其賤)。】

「　捨短從長入奧區」


【(捨其所短從其所長者所謂捐之又損可以入
大道淵奧之區宇也)。】

「　赫弈明然欣世俗」


【(赫弈明然則帝道如日月之明
照臨天下而無私也所以民受其賜而欣樂也)。】

「　希夷道泰握𠃵符」



【(視之不見名曰希𦗟之不聞名曰夷則聖人無為之道泰然也夫如是可以握𠃵符而升
大實矣)。】

「　優容儼雅周通濟」


【(優容則王道蕩蕩儼肅和雅周極有
祗通濟萬方遠安邇肅)。】

「　鄙陋恢張作教敷」


【(鄙陋之俗聖人皆視

如赤子所以恢張至道作教敷演而化導之)。】

「　等級遞遷崇厚德」



【(凡百之官所以等級遞遷者蓋以德而升也故有位者競修其德矣)。】

「　豈遑安處用相拘」


【(為君之難為臣不易且居上者猶虛心以御物豈居下者不盡
心以事君必能夙興夜寐居不遑寧則寵辱之憂無以拘也)。】

「　端莊盡善延鴻福」


【(端莊乃士之常也猶可以資其大福况以天下為懷務其兼濟而天
不再祐乎)。】

「　返朴終期立楷模」


【(夫帝道之行也久矣猶懷返朴之心
終遂必成之志盖所以欲立楷模於千載之下也)。】

「　成敗誰知容易失」



【(得喪之間其猶返掌耳必湏以其難而圖其易誠不可以其易而從其難也故多失矣)。】


「　安危美譽甚難乎」


【(安危之道美譽之名究理而言甚為難也)。】


「　焦勞不避千年日」


【(千年之運出三代而崇規待旦之勤與萬方而
作則故雖焦勞而不避也)。】

「　救苦如烹萬億爐」


【(天地陰陽所以
謂之爐者盖取其能成熟於萬物也今拯人之苦惠人之樂論其爐也孜孜汲汲寧止乎
一二將隣乎萬億矣)。】

「　去就若恒持耿介」


【(人之事君如子之事父是
以折旋進超宛約謙遜有倜儻之冲襟蘊豁達之大度心堅匪石性比貞松是持耿介矣)。】


「　遠傾竭節是良[圖-囗]」


【(石雲向化航海來王皆竭志於彤𨓍盡臣節於
丹禁捨小事大去危就安是良[圖-囗]也)。】

「　但憑直道來干祿」


【(國家以懇
爵待士仄席求賢有不覊之才抱正直之道濟民利國干祿養親者無不得意遂志者也)。】


「　莫遣回邪用巧謨」


【(事君不得以讒侫在口矯詐為心造其偽而弃
其真狐其面而狸其德情多貪競言不由裏是以有機智之巧君子耻而不為也)。】

「　訓誥恬然更不法」


【(聖人所以崇德廣業是乃外訓農桒內興文教扇淳
風而禮和樂暢闡無為而天成地平更往朝奢侈之䂓改累代繁難之制矣)。】

「　品提



【(藻繪皇猷發煇鳴祚恤物而物無弃物用人而人無弃人
是以山木澗松咸入輪轅之用樓丘飲谷皆陪鴛鷺之行矣)。】

「　昌朝畏愛懲貪吏」


【(臨下有赫御眾以𡩖同堯日之照臨有畏有愛比舜絃之教化無黨無偏邇肅
遐安刑清俗阜矣)。】

「　我代銓衡委大夫」


【(光宅寰區奉天明命司銓
土士絕僥倖之門持衡之臣蘊輔弼之德也)。】

「　主聖當因思獵渭」



【(獵渭者昔周文王當有天下時呂望避紂之亂隱釣於渭文王夢得賢臣於是出獵渭濱
之際遇而舉之文王乃知其賢而用果為良弼後立大功斯謂聖德文明應符玄感故賢
手輔翊輻湊天階者也)。】

「　臣驕未必擊珊瑚」


【(為臣之道善在謙恭
苟縱驕矜曷保終吉如石崇恃富擊王愷之珊瑚必不免矣)。】

「　神光夜燭崑山玉」


【(崔山積於瓊玉瑩然燭其光溫潤無瑕盖比之於德且聖人之所化洽異寶奇
珍靡不燭其神光者)。】

「　璧月霄分合浦珠」


【(月彩分輝洞顯無私之照
珠還合浦爰彰化德之功盖燭物無私致物無不復者矣)。】

「　縹緲恒沙搖碧落」


【(縹緲者高遠之㒵麗景芬芳帶于碧落之上高而且遠足以觀之適于道情者也)。】


「　霏微雲雨潤紅蕖」


【(紅蕖則芙蓉也霏微雲雨時以潤之滋以光輝
堪為景玩也)。】

「　花飛起樹驚幽鳥」


【(春滋百卉花木燦然加以
幽禽麗于景致暢道自樂何樂如之)。】

「　水壓清天看躍魚」


【(閑步遊賞
縱翫林泉觀魚戲躍冷浸清天且曰道德之餘無為無事足以任性逍遙堪觀者矣)。】

「　北闕閱兵精騎射」


【(人君負扆親撫龍韜秣馬勵兵長轡遠馭閱武挍戰
制動安危非耀武功常佐王道雄師百萬今古寧如也)。】

「　南方下水伇征途」



【(江鄉之地水田所務負鋤灌水播殖耕耘每念征途遐矜力穡也)。】

「　德音湯網開三面」


【(仁深覆載恩極照臨類成湯解三靣之網則翾飛被其澤惠極好生人推

宥遇亦其矜憐之德也)。】

「　拯溺𥟦甿樂四隅」


【(拯救𥠖甿皆知利病任從
所欲各親其業康哉入極之間樂矣四隅之內)。】

「　稟奉朝儀修軄貢」



【(朝章俻舉禮容咸修諸侯皆稟於朝儀萬國盡修於軄貢書曰禹別九州任土作貢)。】

「　觀瞻景象駕乘桴」


【(四海無波乘桴入貢八蠻𣫍袵納賮來𨓍仰奉天朝
觀光象魏乃皇家之盛事盖王道之一畔也)。】

「　秦川菓熟思鸚鵡」



【(鸚鵡出於秦隴之間翠毛紅觜能効人言異於常禽故見殊貴每至川原果熟因乃思而
詠之)。】

「　澤國春殘詠鷓鴣」


【(江國出於鷓鴣羽翼如錦入詩人騷雅之
詠復以梅殘楚甸春晚江皐舉翼平蕪相鳴遠樹賦成隹句邈尓怡情矣)。】

「　域外縱情閑隱逸」


【(域外者世表也高尚之士放情逸意遊乎萬景藏名嵒谷暢道
煙霞盡出林泉乃合聖道)。】

「　村中皷腹唱樵漁」


【(堯舜之代化行于國
國人樂其化雖田壤之間自然皷腹而歌以樵以漁各遂其性)。】

「　陶鎔鼎鼐窮玄奧」


【(造化鑪開陶鎔萬物和其鼎鼐妙盡玄功動合無私潛符至理書云啇高
宗命傳說為塩梅以和鼎味人代天二取其象也玄奧之理於斯在焉)。】

「　軓範人倫絕覬覦」


【(天垂其象聖人則而行之故成䂓範鑒于下土人倫者等差也
覬覦者倖望也盖人倫仰乎天鑒倖望之心自止)。】

「　爽氣調勻身力健」



【(爽氣者秋之節令也聖人調元至于勻和又易云天行使人身俻載天地時令既行人身
亦得行焉)。】

「　陰晴蕩颺鴈相呼」


【(雲𠬧煙斂虛豁晴空秋氣肅清
景物蕩然遂有賓鴈離塞成行而來應乎節𠋫月令云後五日鴻鴈來𡨋矣)。】

「　千尋樹影光煙彩」


【(建木千尋煙籠光彩頻扷群木獨聳長空是顯至人聡邁
群儒之上矣)。】

「　一片江山入盡圖」


【(今四海為家遂得江山言乎一片孔
子云仁者樂山智者樂水故使天下江山秀麗盡入畫圖)。】

「　野客信緣離島



【(君子藏器於身俟時而動今明天子以御世正南靣而垂衣梯山入貢襁負而歸可
以觀國之光取乎豹變則野無遺賢皇澤無不及矣)。】

「　經商㸦往汎荊吴」


【(璿璣既正太階自平禮振樂張君唱臣和兆庶樂業玉帛充𨓍商汎荊吴各得其所)。】


「　辤家勿慊盈襟淚」


【(道無艱阻離人樂遠別之懷客有行謌執袂免
分襟之淚天下蕩蕩四鄙無虞)。】

「　銜命兼騎內廄駒」


【(遠銜帝命馳馹
是行使于四方以敷君惠行如綸之令宣內廄之駒以慰勤勞冝慎行矣)。】

「　萬里迢遙程迆邐」


【(萬里之程奉皇華而鴻漸九霄之澤宣帝道以風行故迢遙之
途自邇而遠也)。】

「　九疑高嶮路崎嶇」


【(人無高嶮登九疑而自平心奉
至尊履崎嶇而亦坦苟懷忠真之志又何患乎艱苦者[栽-木+(万-一)])。】

「　解將省已應難得」


【(事君者將至卑之軀事至尊之主尊卑道合忠節無虧每自省巳恒愧無功是知為
臣之不易也)。】

「　多是群迷䧟薄徒」


【(處天地之間為臣子之道承恩
匪報忠節不全尸祿素飡臨難苟免者自䧟薄徒也)。】

「　遠占松蘿森澗谷」


【(松即雪中分翠蘿乃澗底承雲彰學道之人棲身巖谷者也)。】

「　道成鶴骨瘦肌鬳」


【(得道之士逈異塵俗或玉液神漿持顏永駐或飡霞服餌保氣延齡

【(遐荒遠塞辮髮

【(此明種植之人鑿以泉井架以轆轤畎水

【(城壘即指彼遠塞澷

【(関防乃言

【(書云
