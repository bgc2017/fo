










御製秘藏詮卷第十二　　　　　　駕


「　解即無邊智」


【(悟解實相其量無邊本智證時廓然平等唯識論云智與
真如平等平等也)。】

「　清流不見冰」


【(法性無垢有若清流結業本空如不
見氷法華經云諸有智者以譬喻得解)。】

「　因言而立意」


【(言者言教意即意趣
佛意深遠非言莫曉是知因言而立意者在得意而亡言法華經云乃至於一偈皆成佛
無疑)。】

「　佛戒剃頭僧」


【(戒者教戒僧者比丘別名毀形奉法佛令借言會旨
戒勿著言金剛經云汝等比丘知我說法如筏喻者)。】


「　愚迷能解悟」


【(智對愚迷權稱解悟法身清淨理絕中邊能契於斯是真
能矣法華經云悟佛知見)。】

「　報應可登陟」


【(報者報身應即他化一身也法
身既悟報應非遙日用不迷故可登陟法華經云一稱南無佛皆巳成佛道)。】

「　行有善因蹤」


【(居無邪念行有善因継諸佛之前蹤紹菩薩之正行維摩經云履踐如來
所行之跡)。】

「　往來皆得力」


【(由積眾善不忘菩提天上人閒往來得力法華
經云現世安隱後生善處)。】


「　證不眀於果」


【(證者能證智果者所證法無相真智正證理時不分能所
圓覺經云於此證中無能無所)。】

「　果因復不興」


【(至真果漏無果復興至覺因圓
無因更立莊嚴論云如來功德如瓶巳滿更不受添)。】

「　欲求清淨理」


【(清淨
理者所求法即無垢真如也唯識論云根本無分別智親證二空所顯真理無境相故)。】


「　方丈問高僧」


【(問荅密義應用無窮不二法門智者能入經云時維摩詰
默然無言此即表示如理無分別智也)。】


「　志遂還他暗」


【(心合至理妙智發輝眀相若來暗相何去華嚴經云成就
慧光明除滅一切暗)。】

「　三衣粥飰僧」


【(內冥一相外服三衣雖粥飲以充虛

實禪恱而為味淨名經云雖服飲食而以禪恱為味)。】

「　坐禪勤論道」


【(坐禪
者習定也論道者求慧也有定無慧必致冥心無定有慧還資應用維摩經云從止觀生)。】


「　免墮業緣燈」


【(由前心靜免墯惡業惡緣以彼多聞常燃慧燈慧炬華嚴
經云是則照世智慧燈)。】


「　果證來多刧」


【(如來果證莫刧寔多破塵𠛴而為空量筭壽而莫盡法華經云
我成佛已來復過於此)。】

「　還緣有愛憎」


【(根熟者悟調御出興緣背者空
如來示滅佛實常住曾無去來法華經云以方便力故現有滅不滅)。】

「　無憎兼不愛」


【(機有憎愛佛無二心無緣大慈何憎何愛維摩經云於善不善等以慈)。】

「　非橫豈相凌」


【(能仁淨土刧壞常安非染內緣銷除巳久無常外寂豈有相凌法華經云
眾生見刧盡大火所燒時我此土安隱)。】


「　舒慘豈關情」


【(順情名舒逆情云慘執有我者生礙達無為者何關維摩
經云毀譽不動如湏彌)。】

「　悲[栽-木+(万-一)]空歎息」


【(歎不悟法憫不解空歎息眾生
業苦樂等法華經云而眾見燒盡憂怖諸苦惱)。】

「　貪心不解脫」


【(心雖本淨貪所
繫縛未了貪空故不解脫薩遮𡰱乹子經云輪廻三界內恩愛不解脫)。】

「　法用無窮極」


【(法者二空真如法也三世諸佛共所𢍆證隨生所欲用無窮極法華經云以我
此物周給一國猶尚不匱)。】


「　因刧何曾減」


【(因即十二因緣刧即時分也因緣流轉時分相隨因既未
終刧何曾減維摩經云觀十二因緣無有盡相)。】

「　非因刧不昇」


【(刧依因有
因非刧生因緣既非刧即不起法華經云無明滅則行滅乃至生滅則老死等滅)。】

「　但隨因刧理」


【(因刧兩亡空性常住心隨此理諸佛道同涅槃經云十二因緣即是
佛性)。】

「　真實事為憑」


【(真常既悟解行相苻堅誓菩提去華取實維摩經云
依於義不依語乃至依於法不依人也)。】



「　義舉千千億」


【(隨機舉義教何止於一途應物無方法乃足於千億法華
經云無數方便引導眾生)。】

「　眾生勿妄興」


【(隨詮妄解逐事乖宗勸捨權冝
令歸真實故云勿妄興也法華經云雖示種種道其實為一乘)。】

「　勸知休作用」


【(勸知一相諸佛優遊休念二乘窮子作用法華經云今所應作唯佛智慧)。】

「　大海百川稱」


【(真如大海綰百法之宏川法喻相應見諸經之廣歎法華經云譬如一切
川流江河諸水之中海為第一)。】


「　法求無二說」


【(求即希求無二說者維摩經云一相無相為二若知一相即是
無相亦不取無相入於平等者即無二說也)。】

「　意默自然通」


【(淨名經云
若知意性於法不貪不恚不癡是名寂滅安住其中者是意默自然通也)。】

「　心不著空相」


【(心不著相諸法皆空若有所觀於空亦泯淨名經云若空無相無作則無心意
識即心不著空相也)。】

「　菩提道果同」


【(佛佛道同法法平等淨名經云佛即
是法法即是眾是三寶皆無為相與虛空等者即道果同也)。】


「　卷舒不繫念」


【(卷舒者瑜伽論云諸佛神變揔有十八卷舒二也謂促𠛴
為塵延念為刧等隨意無礙故云不繫念也)。】

「　何是孰難測」


【(誰於如是
大乘神變難可測耶舍利弗云我覩其為不可思議非意所圖非度所測)。】

「　達則𠛴那間」


【(智人於𠛴那間悟彼變化自性本空楞嚴經云不歷僧祇獲法身又復依空起
念三刧修行精進之心如𠛴那湏證得用也)。】

「　眾生多幻惑」


【(不悟真空
多迷幻有身心惑亂神變莫知維摩經云而四天王忉利諸天不覺不知)。】


「　旨真袪妄想」


【(達旨真之道祛妄想之心則一切塵勞咸為佛種金剛經
云若見諸相非相即見如來)。】

「　達了盡成空」


【(真本不立對妄名真妄體
既無真亦非有仁王經云通達此無二真入第一義)。】

「　種性皆為定」


【(一心無乱
動靜皆禪種性差殊無非佛事楞伽經云婆伽常在定無有不定時)。】

「　愛河渡



【(金剛經云知我說法如筏喻者法尚應捨何況非法此乃眾生結愛成河如來
說法為筏一念不生即升彼岸)。】


「　玄入虛无境」


【(以玄微智入虛无境不見一法而為障礙華嚴經云了達諸世
間假名無有實)。】

「　善緣有業風」


【(法華經云善惡業緣受報好醜由心所造
隨業飄然又大灌頂經說善念方起惡念便生)。】

「　不惟高與下」


【(金剛經云
是法平等無有高下謂高即妙境善即業風佛理智中悉皆平等)。】

「　佛理智圓通」


【(佛者覺義因果齊收理即真理純無漏法智謂四智十智及一切智由佛妙智
證於真理理智無二圓通自在唯識論云根根塵塵徧周沙界)。】


「　幽寂乍濛濛」


【(幽謂幽微證理本也寂謂寂靜𡨋真智也乍昇覺苑頓息
塵機濛濛然得常樂也楞嚴經云生滅既滅寂滅現前)。】

「　圓眀了自通」



【(境無不真曰圓智無不照曰眀境智一如了自通達又楞嚴經云圓明了知不因心念)。】


「　更無施巧拙」


【(純一無雜巧者無以詃其功不二之境拙者無以造其極
古德云本自圓成不勞雕琢)。】

「　坦蕩化玄穹」


【(坦然驀然化之至也玄穹
即諸天也天且弗違即聖人之道昭矣金剛三昧經云不化而化其化大焉)。】


「　疑慮深無底」


【(疑者不決慮者不達深廣無底猶如大海法華經云疑惑
不信故墯於三惡道)。】

「　空心印二月」


【(心體唯一空則心相㸦分月本無二印則
月體交映一心空則無有二心一月真則無有二月楞嚴經二但一月真於中更無是月
非月)。】

「　昬蒙幽顯理」


【(佛性論云理為迷悟之依心為迷悟之本由昬蒙性
昧幽顯理理無幽顯因迷而生)。】

「　虛妄何分別」


【(虛妄之心云何分別分別
若無不真何待大中論云虛誑妄取者是中何所有)。】


「　大戒如來子」


【(大戒者體周法界即佛性遮二戒也如來子者奉持正法即僧
寶也大乘梵網經云眾生受佛戒即入諸佛位位同大覺巳真是諸佛子)。】

「　修行



【(三劫之中具修萬行華嚴經云於一行中修一切行)。】

「　智求心上得」


【(妙智上求於無所求中而求真心證得於無所得中而得大般若經云以
無所得故而得)。】

「　三界大師雄」


【(瑞應經云菩薩示生即行七步一手指天一手
指地天上天下唯我獨尊)。】


「　信不窮深奧」


【(不窮深者由淺識之強知不窮奧者由寡聞之無取如是
學者信而有之)。】

「　山河與地同」


【(聳之則山成五岳濬之則河導百川雖三者之
異名而四大之同致楞嚴經云四大和合發明世間種種變化)。】

「　不唯阿頼識」



【(梵云阿頼耶華言執藏迷則無眀所蔽悟則大智現前楞伽經云如來清淨藏世間阿頼
耶)。】

「　隱顯杳冥中」


【(寶藏論云佛性普徧無問凡夫隱則奔𥪰於四流顯
則圓成於萬德惟斯隱顯不離杳冥)。】


「　智欲周沙界」


【(聖人之智欲無不周應現垂形徧於沙界入法界品云以
無二智知諸境界普見一切世間諸趣徧往一切諸佛國土智)。】

「　慈光導世風」


【(大慈之光圓眀普照導以世俗之風令歸真實之法法華經云放一大光照無量國
又云佛放一光現希有事)。】

「　遐荒無不被」


【(玄化之道應及遐荒妙極圓眀
無幽不燭華嚴經云譬如淨日輪普照十方國)。】

「　威德顯穹崇」


【(聖威神德
蕩蕩巍巍顯彼穹崇高不可仰法華經云威德熾盛光眀照曜)。】


「　悟在𠛴那間」


【(眀悟真心一言合道機感緣會應在𠛴那攝大乘論云處
夢謂多年悟了湏臾頃)。】

「　迷中何濟拔」


【(迷者之心隨物生解執權滯實
濟拔尤難華嚴經云世間顛倒執常樂我淨想故難拔也)。】

「　望陽入有無」



【(渴鹿奔馳求愛陽熖望之則有入之則無淨名經云是身如熖從渴愛生)。】

「　覺後無狂渴」


【(覺知非水狂渴頓息悟本空性真妄齊袪佛地經云如人夢渡河覺後都無事)。】


「　我將譬喻說」


【(聖人說法為顯一真方便隨機多以譬喻楞嚴經云一切

智者要以譬喻而得開解)。】

「　窮劫淺深通」


【(淺即權教化以三乘深即實教
眀其一理說者聽者悉得通達維摩經云佛以一音演說法眾生各各隨所解)。】

「　唯有真空理」


【(真空理者即一真佛性也一性五性二乘三乘約化則殊會本無異法華
經云唯了此事更無餘事)。】

「　圓眀六合中」


【(圓者普周眀者不昧廣大周徧現六
合中楞嚴經云性色真空性空真色清淨本然周徧法界)。】


「　真空不可見」


【(真空者先佛妙源諸祖真性眾生本體萬法實相非色象
故名不可見金剛三昧經云金剛王寶覺五眼不能見)。】

「　萬劫杳冥中」



【(達真空定經多劫識性昬昧在杳冥法華經云從冥入於冥永不聞佛名)。】

「　日日知盈縮」


【(日月垂照盈縮可知瑜伽論說日月在迷盧半右繞而行以分晝夜故天
親菩薩頌云近日自影覆故見月輪缺)。】

「　冝然幻化同」


【(真理隨緣亦同幻化
楞嚴經云汝以色空相傾相𡙸於如來藏而如來藏隨為色空)。】


「　慧甚何分別」


【(瑜伽論云慧者揀擇為性以深慧證法即分別性空華嚴
經云善離分別者亦不見分別)。】

「　阿那陁識中」


【(梵音阿那陁識華言無垢
識即淨性也凡聖俱有因果齊彰深密經云阿那陁識甚深細)。】

「　深藏諸種子」


【(本識微細小智不測名深其體廣大包攝一切名藏諸種子者一切法因也深密經
云一切種子如瀑流)。】

「　瀑水各西東」


【(性海騰波名瀑水善惡二趣名東西
楞嚴經云那陁微細識習氣成瀑流又楞伽經云海風緣浪起識違順亦然)。】


「　知本還其末」


【(本謂一真末謂萬法達本還源唯真心尓法華經云如是
本未究盡等)。】

「　真心方了達」


【(達法之心真實一致豈容本末於其間哉
華嚴經云了達諸世間假名無有實)。】

「　兼能行布施」


【(布施為萬法之先基
真心為一如之極致智度論云菩薩安立性空中故能行是布施等善法)。】

「　山上望明月」


【(山即鷲峯乃說法之境月謂指月為顯教之由見妙居高必然先覺窮理盡性

何事不通圓覺經云修多羅教如摽指月)。】


「　欲說是非同」


【(夫說法者如法而說是非二相同一真源法華經云隨其所說
皆與實相不相違背)。】

「　如常理亦通」


【(如常妙理悉皆通達以一切法無不
真故如來藏經云我為隨眾生方便說正法)。】

「　六根湛清淨」


【(念𢍆真六根
本淨楞嚴經云六湛圓眀本所功德又云彼六知根一時清淨)。】

「　五陰本來空」


【(五陰幻有體本自空金光明經云五陰舍宅皆悉空寂)。】


「　理近凡心遠」


【(理謂真理近為觸目皆如凡心測量去道懸遠法華經云令顛
倒眾生雖近而不見)。】

「　道存理不通」


【(若存於道則心有滯執於其真理故
不通達楞嚴經云存心乃虛妄云何獲圓通)。】

「　深窮真諦事」


【(離心分別
故曰深窮智𢍆如如名真諦事淨名經云深窮實相)。】

「　寂靜信無窮」


【(寂者
真寂靜者純靜信淨為性其理無第淨名經云觀於寂滅亦不永滅)。】


「　詮言要的實」


【(真法無名藉言詮顯要在的實以導群生法華經云汝等
當信是如來誠諦之語)。】

「　立意喻言通」


【(立如來之密意假諸法之喻言
得意忘言通達正理法華經云諸有智者以譬喻得解)。】

「　若解非情見」



【(若解正真即非情見華嚴經云除滅諸邪見隨應化眾生悉令得安住一切種智地)。】

「　真緣義理融」


【(以真實心緣於妙法義理平等一念融通楞嚴經云我以不生不滅
合如來藏而如來藏唯妙覺明圓照法界)。】


「　非本色皆空」


【(真色無形真空無名非本色性悉皆空寂維摩經云色性
自空非色滅空)。】

「　幽微眀始末」


【(欲顯幽微之理湏眀始末之原名雖萬差
理唯一貫涅槃經云世尊知其本末因緣)。】

「　體真無妄動」


【(若能體真心安
如海無有妄動一切皆如華嚴經要義云繁興大用舉必全真)。】

「　塵垢身清潔」


【(六塵妄境垢染其身若能對境無心此身自然清潔華嚴經云離垢清淨身如空滿

十方)。】


「　覺聖無為法」


【(覺者覺悟聖即如來無為法者佛所證真理智度論云無
所得故名無為)。】

「　修心一念中」


【(修平等心住於一念即法法皆如唯真境尓華
嚴經云一念悉了知三世一切法)。】

「　但求無事去」


【(本自圓成非勞修趣無學
妙果湛然圓滿文殊般若經云若知無法即知無境界)。】

「　習性徧虛空」



【(修習種性周徧虛空如華嚴經說善財童子發等虛空心又云諸佛微妙智清淨如虛空
明了一切行其心無所著)。】


「　行與解何同」


【(解謂悟解行即正行由解達真由行證果法界觀云非解
所到是為行境)。】

「　由來一義中」


【(解行雖殊元由一義仁王般若經云通達此無
二真入弟一義)。】

「　若無分別相」


【(住法空觀解行都融分別相無唯一真理
維摩經云若無所分別即是法觀)。】

「　真實自含容」


【(分別既無動皆真實
於一切法悉得含容故勝思惟梵天謂文殊師利法王子言仁者所說皆是真實)。】


「　證善何人解」


【(證者證得善者善常謂妙證善常之法唯佛能解唯識論云此
即無漏界不思議善常)。】

「　終成一義空」


【(法華經云無有二相猶如虛空
謂一登妙果了法皆空第一義中無所有故仁王經云又一切法悉皆空故)。】

「　輪廻無了日」


【(圓覺經云念念相續循環往復種種取捨皆是輪廻即善惡往來如輪廻
轉從生至刧無有了期)。】

「　惡業善心同」


【(華嚴經云捨離諸惡趣清淨諸
善道超諸世間者示我解脫門若達真空善惡都寂混然妙理故云同也)。】


「　江河與大地」


【(楞嚴經云山河大地諸有為相次弟遷流終而復始即器世間
相也)。】

「　智慧無纖默」


【(一真理中本無所有世智聦辯妄生推度無纖毫沉
默也法華經云慧智眀了多聞強識)。】

「　根本是煩惱」


【(體狂智之根本皆煩
惱之所生究煩惱之深源悟菩提之妙道楞嚴經云決定惑為色心之外)。】

「　寂然中



【(本妙覺明無有生滅萬法煩惱不離真心楞嚴經云咸是妙明真心中物)。】


「　志蓄人儀表」


【(志隨心寂妄念不生外顯莊嚴內泯色相瑜伽論云由布
施故得大福德果由持戒故得大端正果)。】

「　縱橫不可衝」


【(妙智縱橫隨機
應用間無橫竪法有淺深維摩經云善惡之業亦不妄如影隨形耳)。】

「　更能歸正見」


【(涅槃經云寧毀壞尸羅不毀正見尸羅生善道正見得涅槃是知賢達離色取
真)。】

「　萬刧自從容」


【(得法圓寂妙用無窮因行成時聖果圓滿唯識論云
一得永得盡未來際受用無窮廣大法樂更無生滅足致從容)。】


「　果敢亦奇士」


【(酬因曰果求法名敢修行精進猛利進趍報恩經云若有智慧
人能善了因果斷三際惑敢求當果)。】

「　不唯賢與庸」


【(華嚴經云地前菩薩
四十心配上中下三賢位故庸者愚也但存報應賢愚有歸)。】

「　若然知道覺」


【(能覺道元超凡入聖夙殃消滅正法寬容法華經云心無所著求無上道運心造境
明而不昧)。】

「　業力自銷鎔」


【(瓔珞經云業有四種一順現二順生三順後四
順不定此有漂流沉溺之功名力若或心歸覺道注想真乘觀罪性空永自銷滅)。】


「　因機興妙用」


【(宗記云因託也機謂三乘五性之機冝興起也法華䟽云謂諸
如來託所被機起三業化名為妙用也)。】

「　勝事會遭逢」


【(法華經說大乘
妙寶名為勝事根熟遇緣任運際會故曰遭逢也)。】

「　世福有深厚」


【(佛說
三歸五戒等行雖感世間有限福樂便能以此廻向菩提即成深厚也法華經云若人有
福曾供養佛)。】

「　當來善必從」


【(當來者眀三世也唯識論云此世他世俱
為順益善必從者揀惡進善漸求佛果也)。】


「　海納百川水」


【(海者天池也百川之水盡納天池故華嚴經云譬如巨海眾流
所依覺海亦尓差別義理皆歸覺海)。】

「　一非終是一」


【(二非者一情二相有此情
相非𢍆真理故維摩經云以情相為二一真法界辦情相故)。】

「　眾知等妙



【(華嚴經云三祇刧滿將成佛時入頂寂大定起等覺智又入理盡三昧起妙覺智因
此二智永出三界)。】

「　惣攝戒兼律」


【(戒能引善律能止惡故大般若經云
淨戒由智攝乃得名戒彼岸以戒律不離等妙覺故)。】


「　義可貫眀日」


【(眀誠猶能貫日妙義可以證真況如來之法教金剛經云
如人有目日光眀照見種種色)。】

「　巍峩善事雄」


【(巍峩者顯三身之妙用也
天親頌云不思議善常理智巍峩雄勝之極也)。】

「　更眀真實理」


【(般若經云
涅槃名實理此理無邊輙說不盡今更明之)。】

「　萬象隱其中」


【(楞嚴
經云森羅及萬象一法之所印一法是心心依真理心隱理中故)。】


「　法滿真空界」


【(法演玄微徧真空界眾生悟解妙理流通情與非情還達
本性法華經云十方世界中唯有一乘法)。】

「　如來大覺中」


【(勝王經云一切如來
有三種身一法身二報身三化身覺慧也報身大悟真性是智所知境也)。】

「　眾生是佛智」


【(一切眾生皆有佛性無明障蔽不證圓明涅槃經云一切眾生悉皆有心
心佛智種也)。】

「　密教豈通風」


【(佛說三時教第一第二時說理未盡名密
第三時雖顯了說義旨玄奧名密勝鬘經云若不知佛此密意失壞正道)。】


「　旨玄行大信」


【(玄旨幽微如何證悟佛示大信善誘眾生智度論云信為入法
之初基智為究極之玄術)。】

「　信立示愚蒙」


【(華嚴經云十信第六名信不退
方得信立又說十信菩薩八相成佛化利群生能示愚蒙故)。】

「　不會信中信」


【(信中之信唯一真如聖人達之凡小不測顯揚論云信現觀者初地證入)。】

「　聞音耳却聾」


【(大辯若訥大智若愚洞達信源法相空寂設逢聲境耳不貪著塵不
能累維摩經云法非見聞覺知故)。】


「　降魔大眾力」


【(瑜伽論云如言王來必有臣從智斷惑時定有二十二法
與智相應同断煩惱魔故)。】

「　萬法從心入」


【(萬法所緣之境一切善惡之階心生

法生心滅法滅唯識論云此能變唯王謂異熟思量及了境識)。】

「　生滅化無情」


【(情與無情不生不滅為無出入相而證真淨理法華經云我觀一切普皆平等)。】

「　少求多亦得」


【(能悟善果必顯菩提因之與緣故無多少瑜伽論云少施等善感
無量果)。】


「　玄默非相應」


【(理幽不測名玄智證冥合為默心境都寂色空混然不與惑俱
即非相應唯識論云無得不思議是出世間智)。】

「　寂然天地中」


【(寂然者瑜伽論
云清淨法身永盡諸惑然隨事相其量無邊譬如虛空徧一切處)。】

「　凡思兼與聖」


【(能證涅槃非間凡聖是法平等妙理皆如法華經云其法一味解脫涅槃)。】

「　微妙悉皆通」


【(微妙真理無所不通上至諸佛下及含識悉皆平等維摩經云一切
眾生亦如也眾聖賢亦如也)。】


「　幽深行至道」


【(至道無私真源秘奧不以識識不以智知法華經云諸佛
智慧甚深無量)。】

「　小異大家同」


【(殊途同歸其實一體於法無礙豈有異同
法華經云一味之法隨力修行)。】

「　無礙兼心智」


【(法華䟽云無礙有四一法
二義三詞四辯兼者與也心王與智同緣法義等四了達無滯名無礙兼心智也)。】

「　凝然湛寂中」


【(妙絕思議故曰凝然體即真理也一心不亂名為湛寂法華經云深
入諸禪定身心寂不動)。】


「　大哉深信士」


【(大哉則歎美之詞深信則不退之意華嚴經云菩薩修行入如
如海證不壞信得逈行真如)。】

「　一語便音風」


【(一言道合理𢍆真如華嚴經云
如來於一語言中演說無邊𢍆經海)。】

「　纖芥心中得」


【(色塵質礙纖芥不通一理
觀空真實不露心中眀了自得圓成法華經云觀一切法空無所有)。】

「　善隨惡業空」


【(惡業纏綿四生不斷妙智纔起諸業皆空維摩經云如日光出時眾暝皆除)。】


「　伏性若寒灰」


【(心若寒灰於法眀淨不起他見真理顯然法華經云常處空閑

深修禪定)。】

「　任從枯栲質」


【(道本無心諸法无㝵於緣頓息心境都寂正理
論云瘦悴枯羸形如骨𤨏)。】

「　若能養聖胎」


【(聖胎即四智種也瑜伽論云無
漏智種名為聖胎依法界腹生如來家證徧行真如住極喜地真如與智為所依名之胎
也)。】

「　去二還歸一」


【(色心未泯二相當情般若興時洞然歸一華嚴經云菩薩
悉知入不二門西歸一道)。】


「　信及四天下」


【(法華䟽云由信故為入大乘之基地上菩薩四不退中以信為
初對法論云即四大洲也信及真理觸處皆同)。】

「　寂然一貫中」


【(法性清淨寂然
不動真理無朕萬法皆空智度論云諸佛圓寂中眾法皆一味)。】

「　誰能覿妄想」


【(攀緣妄想良由未悟之心滅境觀空故在達人之智法華經云但離虛妄名為
解脫)。】

「　真性隱虛空」


【(真性妙理隱諸法中性相有殊喧寂不二維摩經云
是故當知一切煩惱為如來種)。】


「　智慧有增減」


【(智能決斷慧善揀擇未逃四相遷流有增有減豈似一真
法界無去無來法華經云是菩薩以若干智慧)。】

「　心貪惣是空」


【(心者緣慮
為名貪即染著為義妄心染欲於法不真有類是空無所趣向法華經云貪著於五欲癡
愛故生惱)。】

「　未知施設用」


【(於一切法不能施設處三界中而善妙用華嚴經云
一切眾生平等清淨)。】

「　權變類真風」


【(法本無心善行遷變物情無礙妙旨
通真法華經云如風於空中一切無障礙)。】


「　我知山聳翠」


【(妙峯聳翠法日高眀普照十方無幽不燭對法論云蘇󲊛
盧四寶出水皆八萬上智了知聳翠矣)。】

「　劫壞地平同」


【(劫有三種一火
災二水災三風災七日現時山石鎔盡仁王經云刧火洞然大千俱壞湏彌巨海磨滅無
餘)。】

「　實理無高下」


【(法無高下智有淺深一悟全真根塵寂滅涅槃經云外器
與有情涅槃性平等)。】

「　歸空運化中」


【(有為妄法決定歸空妙理圓成常居

真寂華嚴經云如是所作徧周法界)。】


「　蓬長類麻生」


【(蓬類妄緣麻方執見三草二木甘苦自知真妄同源薰蕕
不別攝論云菩薩依於善友如蓬在麻)。】

「　薰蕕相㒵直」


【(薰蕕狎習曲直
相扶轉迷成真猶愚近智維摩經云直心是道塲無虛假故)。】

「　溟濛法雨滋」


【(佛之慈慧如雨涵濡穢淨普滋悉皆潤澤法華經云以一味雨潤於人華)。】

「　可漸性中得」


【(性即真性凝然而證古眀今迷即不迷悟乃心融妙法法華經云雖小欲
懈怠漸當令作佛)。】


「　真不與妄合」


【(真性常通妄情恒著非真真寂非妄妄祛真妄交叅妄真
非合華嚴經云譬如淨日輪不與昬夜合)。】

「　恒沙妙相同」


【(佛地論云無相不相
名為實相具多功德諸佛同證名恒沙妙用)。】

「　諸大與聖眾」


【(對法論說
即三界二十八天有清淨光絜智慧強咸聖眾則五位一十三住惣名諸聖也)。】

「　達識大乘空」


【(攝論云乘之大性名為大乘即真如是天與聖眾共證此理華嚴經云湛
然如虛空一切無有生)。】


「　慧本因緣得」


【(慧謂妙慧非假因緣四大本空念念生滅法華經云以此
妙慧求無上道)。】

「　十方所在通」


【(佛法玄妙無所不通觸目菩提無所質礙
華嚴經云普至十方諸世界而於一切無所礙)。】

「　欲眀真諦理」


【(真諦者妙理也
三乘妙道清淨一真根性有殊玄理無別法華經云尓乃集大眾為說真實法)。】

「　人法兩相宗」


【(人法二執權假相宗若達真源自然俱遣金剛經頌云人空法亦空二相
本來同)。】


「　說經兼說論」


【(研竅性相於法皆空貫理精微於經非說華嚴經云佛法
無人說雖慧不能了)。】

「　經論信心同」


【(信心無著處處流通於法無情佛佛印可
華嚴經云一切無礙人一道出生死)。】

「　不說及常說」


【(常說者相續義上傳

佛旨下化有情恒無斷絕之時不作說法之相華嚴經云雖說量法句而心恒住)。】

「　因緣幻化空」


【(有為之法仗因託緣說法之時觀自他相了一切法幻化不堅仁王
經云如影如毛輪日緣故幻有)。】


「　蒿產於凡草」


【(蒿夹蘭生香臰必異法從心起凡聖同途悟則皆空達無
邊際法華經云而諸草木各有差別)。】

「　榮枯無了日」


【(謂逐榮枯不了生滅
能了生滅定絕榮枯苦惱纏綿終無了日唯識論云前身速滅後有重生)。】

「　勿生分別心」


【(彼我心存障蔽真智於法無念分別何宗法華經云亦不行不分別)。】

「　各自本緣出」


【(愚智不同盖由根性聖凡類別修證本殊各逐業緣豈寧一揆法華經云
以是本因緣今說法華經令汝入佛道)。】
