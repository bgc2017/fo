










御製秘藏詮卷第五　　　　　　　車


「　違道如山積」


【(違者背義道者聖道謂染習煩惱相因生起諸業煩惱積
集如山故云如山積也)。】

「　迷情去復來」


【(諸有情類躭迷五欲愛取潤緣
生死往來巡環不絕法華經云相續苦不断)。】

「　貪財無布施」


【(貪吝財寶不起
施心既闕行檀永墮貧苦法華經云見六道眾生貧窮無福慧)。】

「　罪即犯靈臺」


【(靈臺者心識之異名犯者違犯之義為諸眾生無始巳來起惑造罪損壞淨心
故犯靈臺花嚴經云清淨無垢智無壞)。】


「　順則菩提行」


【(隨順眾生是菩提行不居淨土處此穢方隨逐凢夫恩深
難報本行經云自兜率天來降閻浮)。】

「　慈悲道眼開」


【(常開道眼指喻群生
興慈運悲三界𢪺苦寶篋經云如來雖離三界救於三界)。】

「　眾生業力重」



【(如來大悲不捨有情罪業障閇自不見佛淨名經云眾生罪故不見如來)。】

「　損益自心裁」


【(眾生造業受諸苦惱則為損也依佛言教起行修因當證樂果則為
益也華嚴經云應觀法界性一切唯心造)。】


「　知真不起妄」


【(既生真智不起妄心如日出時昬蒙自遣淨名經云日光
出時不与𡨋合)。】

「　魔眾可哀哉」


【(梵音魔羅此云殺者能断惠命不達一真
並為魔眾汩沒苦海甚可哀哉淨名經云一切天魔外道皆吾弟子)。】

「　世界光明裏」


【(如來放光照東方万八千世界俱在光明之內法華經云佛放眉間光現
諸希有事)。】

「　天花七寶臺」


【(世尊說法天雨四花散於寶座法華經云天雨
曼陁華)。】


「　情悟脫迷津」


【(有情悟解乃出迷津永離漂沉證於常樂唯識論云
常自受用廣大法樂)。】

「　祖宗何伴侶」


【(達祖宗體煩惱不拘理本出纏何分

伴侶大莊嚴經云離諸攀緣無所行處)。】

「　真心妄不同」


【(真智朗然妄心迷昧
明暗自別真妄不同金剛頌云有心俱是妄無相乃為真)。】

「　求百還千與」



【(初求小果入於化城但證生空真理㱕一一則一切何止大千證道謌云一法徧含一切
法也)。】


「　疑惑失真趣」


【(不断盖纏失於真趣疑惑未遣理事難明楞嚴經云而不
自覺枉入諸趣)。】

「　分明聖慧開」


【(煩惱若盡理體分明聖惠開時昬迷䪺遣
金剛頌云慧燈如朗日)。】

「　一音從法界」


【(法音一震沙界普聞機性雖殊
深淺獲益既蒙大利永出迷津依法奉行必登彼岸法華經云隨上中下各有所受)。】

「　龍象影中來」


【(放光召眾來佛影中學法求真如大龍象楞嚴經云在佛影中
應令暫顯又云學佛法者如大龍象)。】


「　興心詮像教」


【(欲修萬行先要興心詮顯一乘湏依像教法華經云我以
種種方便廣演言教)。】

「　佛理卒難裁」


【(佛根本智自內所證真如理體離相
湛然聖智群情經劫難測伽耶山頂經云不可憶念離攀緣故)。】

「　萬卷真如旨」


【(萬卷經文為真如奧旨一乘秘典乃凢聖同㱕法華經云若不信此法無有是處)。】


「　冥然性相開」


【(理體𡨋然了達在於真智分明性相開顯藉於教文法華
經云種種性相義開闡一乘法)。】


「　念徹無窮義」


【(無窮義海難究其源一念了空俱無實體精舍經云義者
無體也)。】

「　玄関秘不開」


【(玄関幽遠秘密不開心莫能緣智不可側法華
經云深固幽遠無人能到)。】

「　一真非鏁鑰」


【(常樂一門非関鏁鑰真空本性
豈染塵勞光明藏經云於真俗門洞達無礙故)。】

「　萬法眼前來」


【(朗日當空
冥暗自謝眼前觀境色相分明如起本智四事俱捨二空齊證萬德河沙諸功德法悉圓
證故唯識論云如大圓照現眾色像)。】



「　欲性本來差」


【(眾生情欲本自不同因果差殊名異生性法華經云若生
天上及在人間)。】

「　冬寒如夏暑」


【(日月運行陰陽所變妄境則暑寒逈異達
真則俱是一如六英經云譬如雲風雷電和會作雨離緣無體故)。】

「　迷途不是緣」


【(若言冬夏無體四時不實聖智可以了知迷情不能緣慮法華經云疑惑
不能了)。】

「　報應非言語」


【(天地功深育群生而絕望真如湛寂含万法以
無言文殊問菩提心經云無覺無知亦無所得故)。】


「　恒式持金偈」


【(三慧熏修以聞為首恒持金偈聖種轉增金剛頌云熏成
無漏種方号不思議)。】

「　真風入座來」


【(奉持真教眾聖加威必感花雨晴空
香風入座法華經云香風吹萎花更雨新好者)。】

「　玄機娛自解」


【(玄微機要
契合一乘悟解法門自心娛樂法華經云心懷大歡喜)。】

「　理性洞然開」



【(依教開悟洞達真空万法一如湛然清淨法華經云開佛知見使得清淨故)。】


「　習善保康哉」


【(修習善道種植良因遠離百殃必保康奉法華經云現世
安隱後生善處)。】

「　迷徒甚可哀」


【(迷徒不悟妄執六塵火灾煎煩甚可哀慜
法華經云我為眾生之父應拔其苦難)。】

「　慈光燭法界」


【(如來妙定慈光
不絕照燭身土恒徧法界無量壽經云放無量光照十方國)。】

「　七寶湧樓臺」


【(報佛國中不見穢境七寶樓臺㽵嚴間餝法華經云七寶臺觀充滿其中)。】


「　除濁歸清世」


【(欲歸清世濁染湏除離眾冤煩永居淨域法華經云我淨
土不毀天人常充滿)。】

「　業隨䧟刼摧」


【(造業有情自生摧壞隨䧟三刼難免
焚漂法華經云眾生見刼盡大火所燒時)。】

「　琉璃光瑩淨」


【(報土之中琉璃
為地光明瑩淨佛常住世法華經云如淨琉璃中內現真金像)。】

「　勿染六塵埃」


【(六塵者謂色聲香味觸法六境能坌污眾生身中功德名為六塵起智断除即不染
塵埃法華經云汝等勿貪麤弊色聲香味觸法)。】



「　求慧入幽微」


【(先求智慧諦察善因趣入幽微當祈解脫菩提心經云求
方便慧為本)。】

「　疑緣多執處」


【(疑若未除諸惑不息妄情多執三界為緣
法華經云眾生處處着引之令得出)。】

「　慈悲四眾間」


【(常於四眾諸有情類
起慈悲心饒益救拔淨名經云饒益眾生是菩薩行)。】

「　何用經營取」


【(但行
利樂普濟群生何用經營別生求趣淨名經云菩提者不可以身得不可以心得)。】


「　想絕是非跡」


【(窮情極思想絕是非断煩惱之稠林去我人之蹤跡法華經云
不說他人好𢙣長短)。】

「　輪迴妙相開」


【(自然輪廻永息妙相開通一真之理
體分明六趣之業緣䪺謝文殊問經云開於正路離諸𢙣業)。】

「　眾生迷不見」


【(眾生迷執妄失宗源悞入邪山不求出離法華經云深著虛妄法堅受不可捨)。】

「　終日路獃獃」


【(獃獃者癡愚不明之狀謂諸眾生無量刼來終日晝夜於三界路上
愚癡蔽心不明本性楞嚴經云雖終日行而不自覺)。】


「　𥙿態多昬翳」


【(世俗情態多是昬蒙染翳淨心不能明顯法華經云以貪
愛自蔽育𡨋無所見)。】

「　亦能障法身」


【(煩惱障礙妙用難分不唯癡暗自迷
亦覆法身真性華嚴經云離顛倒見知如實理)。】

「　光明照八極」


【(神光洞照黑暗
皆除八極俱明群生獲益淨名經云或有光明而為佛事)。】

「　物象盡成真」



【(湛然本性悟時而物象皆真惑遣理彰了達而色空無異華嚴經云知一切法皆無分別)。】


「　諦信成三寶」


【(如來成道佛寶也初度陳如僧寶轉正法輪是法寶志意
信受定出迷津淨名經云三寶於是現世間)。】

「　冝依作道隣」


【(諦信三寶
冝作道隣捨妄趣真俱登彼岸華嚴經云具足功德成佛道)。】

「　梵天興教日」


【(佛具六德稱梵謂自在熾盛端嚴名稱吉祥尊貴普覆一切名天興教破暗如日法
華經云慧日大聖尊久乃說是法)。】

「　覺樹鎮長春」


【(覺樹道成如春煦育
長刼化利覆廕群生華嚴經云或於樹王下自然成正覺)。】



「　願為十方說」


【(佛有大願為度眾生化利十方廣宣正法法華經云無數
方便力演說諸法相)。】

「　論千及萬與」


【(聖居無垢明門高啟於一千滌眾塵
機教海湏揚於八萬法華經云以萬億方便隨冝而說法)。】

「　日烏將月兔」



【(日烏晝照明徧四洲月兔夜輝光生八趣法華經云普明照世間)。】

「　照破無眀處」


【(兩曜騰輝破世間之黑闇慧燈明朗照煩惱之昬衢法華經云大光普照
徧滿十方)。】


「　語默皆為道」


【(九有言說皆𢍆真空泯相離言俱為妙道仁王經云通達
此無二真入第一義)。】

「　忘機絕妄塵」


【(心機不起便絕妄塵去執除疑自獲
安樂解深密經云但自淨心妄塵不起)。】

「　有無依妙理」


【(我心執有迷情著空
有空不拘方依妙理唯識論云若離有空妙符中道)。】

「　誰作解空人」


【(空有
雙遣二障皆除更無緣慮之心方作解空之士仁王經云一切清淨悉皆是空)。】


「　言妄迷情障」


【(迷情內起外發妄言雖則心法無形但以語言表示攝論
云外發身語業表內心所思)。】

「　非空不是真」


【(唯空是真為萬法之所依
妄想皆虛乃徧計之所執文殊問經云無語言道是法皆空)。】

「　色心成相法」



【(色心緣生則有為事相攝相歸性乃俱是圓成楞嚴經云色心諸緣及心所使皆是妙明
真心所現)。】

「　根種離埃塵」


【(楞嚴經云一根既復源六根皆解脫本智從種
既生煩惱塵埃遠離)。】


「　純熟如來智」


【(精修萬行純熟眾生如來妙智自然圓滿菩提心經云緣
於眾生真智為本)。】

「　多聞似不聞」


【(悟解必假於多聞證理全無於視聽
楞嚴經云無聲号無聞非實聞無性)。】

「　無心即是道」


【(解脫之處無相無心
智與理𡨋即為真道楞嚴經云空所空滅無心之道)。】

「　象教紀經文」


【(佛垂
象教綱紀經文為塵刼之良䂓作三祇之刊定法華經云如三世諸佛說法之儀式我亦

復如是說無分別法)。】


「　安置断非心」


【(理本無言方便安立設三乘之權教断虛妄之非心攝論
云轉四諦法輪断三界虛妄)。】

「　垢塵流土宇」


【(惑業如土則能招因行五
蘊如宇則所感果體為五蘊身中煩惱塵垢流轉不絕瑜伽論云五蘊區宇惑業潤生)。】


「　但能日用真」


【(但能了達真性日用淨心當得無漏智生永断纏綿惑種
攝論云真無漏道能断隨眠也)。】

「　不必更疑慮」


【(依教起行断障除染必證
真常更何疑慮法華經云我聞是法音疑悔悉巳除)。】


「　和合歸真理」


【(心能融境即和合相事与神𢍆妙理歸真密嚴經云若了
所知無能知即非有)。】

「　緣流性自純」


【(真通自性但任緣流心既歸真不為
妄惑楞嚴經云歸源性無二)。】

「　菩提清淨境」


【(清淨覺智照物無方諸佛
眾生非一非異淨名經云一切眾生即菩提相)。】

「　經劫不生塵」


【(悟真之功
既顯出塵之量永存虛妄不生自𢍆斯旨祖師云本來無一物何處有塵埃)。】


「　迷背人難解」


【(性有迷背理本圓通我執堅牢故名難解圓覺經云使諸
眾生迷悶不入)。】

「　愛河彼岸津」


【(了煩惱即菩提悟無明即法性愛河自竭
彼岸通津涅槃經云若離於愛何憂何怖)。】

「　未嘗興一念」


【(於曠大劫值遇如來
真念纔興妄念永息大智度論云諸法入佛心中唯一寂靜三昧)。】

「　色相擬為隣」


【(悟即色是空非滅色見空故任為隣般若經云色無邊故般若無邊)。】


「　彼岸梁為搆」


【(能達彼岸巳搆津梁復念眾生沒溺苦海起大悲心設種
種法未渡与沒應用分明華嚴經云廣度一切猶如槗梁)。】

「　清波濁水津」



【(清波喻法性濁水喻煩惱愚者見二智者唯一華嚴義海云清水濁水波濕無殊)。】

「　非心為筏喻」


【(如來淨智名為非心能度眾生強名筏喻金剛經云如筏喻者
法尚應捨何况非法)。】

「　宿世有良因」


【(正信難立正法難聞得非久殖善根

獲勝妙利般若經云宿種善根惣持不忘)。】


「　樂道信非常」


【(樂道者謂心遊大道理𢍆非常體用昭彰非言象之所及
華嚴經云信是道元功德母故曰非常)。】

「　善心無猛虎」


【(達色見心何𢙣
不善強暴之緣頓息怖畏之境即忘金剛三昧經云善不善法從心化生)。】

「　受持不可思」


【(信解受持於法不昧妄緣俱捨心觀豁明若不然者於教何益華嚴經云如貧
子[婁*殳]他寶自無半錢分)。】

「　枉用虛勞苦」


【(不興正見妄起邪心去道愈遙
徒言勞苦圓覺經云即知此生性自勞慮)。】


「　曩刧何迷障」


【(悲歎曩劫受報輪迴諸惑覆心不能了悟為迷所障何猶
覺知楞嚴經云迷晦無明也)。】

「　多成不解人」


【(於法迷惑不能解空眾生
妄業所纏世尊乃興悲歎楞嚴經云我信是人終不能解)。】

「　宗乘能了達」



【(佛之宗乘開發迷者一悟永悟了達上乘涅槃經云了了見佛性猶如文殊等)。】

「　三界好相親」


【(華嚴經云三界所有法唯是一心作既了一心之旨寧偱三界之輪
法華經云不如三界見於三界)。】


「　垢膩昬諸意」


【(不能離垢趣真為染無明妄意不悟本性向外馳求大乘
攝論云通達唯是意)。】

「　如何見法身」


【(如來法身隨機應現不能了悟全昧
真常華嚴經云譬如暗中寶非燈不能照佛法無人說雖慧不能了)。】

「　為開方便故」


【(開張大道直露玄機示方便門咸令悟解圓覺經云開發未悟)。】

「　此理卒難陳」


【(至理甚深法身玄邈非事非理於法難伸法華經云是法不可示
辤相寂滅)。】


「　抂用成顛倒」


【(依不了義生妄想心智用狂迷識量顛倒用名質礙不了
