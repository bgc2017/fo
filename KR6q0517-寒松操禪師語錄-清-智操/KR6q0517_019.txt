










寒松操禪師九峰艸(原版藏吳江今收入全錄卷第十九)
吳江　顧有孝茂倫　王載咸平　評定
同學諸子參閱

鳳凰十詠(已下俱和韻)

三星閣


「　臨流傑閣自天題半展丹青與日齊不向家園供嘯傲惟同雲月老山谿」


(顧茂倫曰三星閣出自御學不向家園供嘯傲深得徐文
貞晚年心事)。

鐵漢亭


「　客去留亭想鐵公竹窗似立畫圖中年來筆下無窮思盡付山前一笛風」


(徐介白曰東海風流如見)。

且止園


「　簇簇雲留洞口春殘碁半局且陶真山花野鳥情無盡何必桃源去問津」


(趙之曰興致悠然)。

錦谿橋


「　虹枕蒼波勢欲飛水聲西下浣苔衣花開兩岸搖紅影鎖住春光不放歸」


(王咸平曰自是道人本色語莫錯會)。

山月軒


「　軒前山色滿闌干軒裏清光照影寒相識只餘山與來同客細盤桓」


(顧茂倫曰山月俞先生為方正學門人保方氏孤初冒險
姓後更為余萬脣時橋延筠督學南幾疏請復方姓山月其讀書齋也弔古之什玅在不實寫其事)。

芙蓉莊


「　露染紅英出短牆迎秋又欲占秋光窗開十二煙霞徑別是人間天一方」


(王雙白曰大為拒霜花生色)。

南村宅


「　半山半水半村居野色空濛霧捲舒閉戶不聞塵世事消閒且閱古人書」


(殳山夫曰即埋甕樹下摘樹葉書之滿甕更見他甕所謂
輟耕錄處詩有遠神使人尋味不到)。

來儀堂


「　窗懸一榻早安排茅屋交遊柱石才道是鳳凰棲隱



(包明成曰詠鳳凰而獨舉明月匪夷所思)。

梅花樓


「　方伯欣栽冷澹神繞樓花放淨無塵不勞寒策孤山蹇自有香飄滿室春」


(王咸平曰方伯謂莫公中江也人品書法俱能一筆寫出)。

摩霄壁


「　青山江干第一峰劈開半壁勢重重雲來拜石橫難度倒臥長松欲化龍」


(姚文初曰壁下有拜石堂堂去面壁存讀起一句想見振
衣千仞之)。

厙公八詠

採藥徑


「　林幽複道亂雲飛石子高低繞翠微採藥厙公隨月轉清歌一曲踏香歸」


(倪天章曰單公如在呼之欲出)。

藏書函


「　書藏年遠待誰還雨老蒼苔點石斑分付山靈深寶惜莫將一字落人間」


(葛瑞五曰只恐寶惜不住奈何)。

覽德坡


「　山前客到想高流剩得清輝月一鉤為問當年誰覽德知心只有鳳來遊」


(顧樵水曰只舉明月鳳自來遊此意甚深旨)。

鼓琴石


「　一爐沉水一張琴留跡空山歲月深滿塢松風鳴舊曲卻無鍾子孰知音」


(王咸平曰感慨)。

釣漁磯


「　一片巉巖臥碧谿水痕濃澹浪高低苔深不見林鴻跡時有沙鷗月下棲」


(金孝章曰苦拈漁釣未免失卻磯字此為得之)。

洗鶴灘


「　谿老幽情古渡邊誰將素練浴胎仙潮來兩岸雲千片猶報風流勝昔年」


(殳山夫曰鶴為仙人之騏驥方發得洗鶴意透士衡河橋
之歎正未夢見在)。

鄭公莊


「　曲徑深存五柳風小橋斜架石玲瓏不應輕放閒名



(顧茂倫曰道著多少名人病處悔恨處真是度世慈雲)。

陸寶村


「　高風陸子露華清藏寶誰留去後名過客欲詢棲隱事枝頭啼鳥一聲聲」


(周安節曰吳時陸氏藏寶處曰陸寶山後人爭發其寶山
遂夷為平壤藏寶誰留去後名一語冷極)。

佘峰十詠

徵君故廬


「　山水多年皆舊識文章六代起東南徵車不與辭庭外門掩春風共笑談」


(程石門曰老杜哀壑無光留戶庭不及此起句之深遠)。

道人舊址


「　多年玉麈寂雲屏尚有人來訪二青踏遍殘霞何處問關關啼鳥悟空靈」


(顧茂倫曰大青小青聰道人所留二虎名客來不訪道人
面訪二青卻似當頭一棒)。

講寺竹林


「　上方瀟灑報平安一徑扶疏列萬竿山靜竹風涼似水老僧六月尚餘寒」


(安大巳曰此之者道人六月賣松風句更勝一籌)。

香谿石徑


「　藤花壓水滯輕舠剩有谿眠一徑高人在畫圖行欲盡不知身已醉谿桃」


(徐嚮芝曰別有悟入處)。

沐堂曉日


「　日出先蒸谷口霞光搖殿角半窗斜客來獨把琅玕拍一簇金流滿地沙」


(丘嶼雪曰昔有人上抱犢山見日出散髮大叫宿疑頓盡
亦是此意)。

塔院午鐘


「　庵居谿畔似山農愛打棲頭日午鐘不問人間天早晚聲聲帶翠落危峰」


(王咸平曰晚鐘晚鐘詩多午鐘詩少此作居然獨步)。

層霞高閣


「　一閣高懸碧漢閒霞光滿貯醉人顏重重門掩重重水面面窗開面面山」


(陳長發曰如身立赤城之巔與五百應真相往來無一字
經人道過)。


白雲曲沼


「　雲水鄉沾彭澤居臨流洗墨繞遊魚橋邊活計三株柳石上生涯一卷書」


(張九臨曰唐子百山靜似太古日長如小年不知誰為第)。


梅花書屋


「　深入長廊曲徑幽交加冷豔壓羅浮書邊一枕春風老疑是和身臥雪舟」


(領樵水曰曾同山夫雨七醉臥其處讀此詩情境俱真何
必向羅浮遊也)。

東山艸堂


「　一室深幽抱石蹊道人多半借雲棲有時直向東山去那許尋芳信杖藜」


(徐禎起曰有空山無人水流花留之致)。

細林十詠

崇真曉鐘


「　巖高金闕白雲封別是蓬萊訪舊蹤為惜下方清夜永一聲夢破五更鐘」


(周其凝曰他處曉鐘移易不得)。

金沙夕照


「　霞迴斷靄影初沉風響仙巖雜梵吟不必滿山舖碎錦直教遍地是黃金」


(湯兩七曰與前作風格相同)。

丹井靈源


「　玲瓏三尺老山根傳是雷開冷石垣不信丹成仙去後誰遺海眼貯乾坤」


(濮澹軒曰眼界大放淡令香嚴海性不止為仙家點綴)。

西潭夜月


「　澂潭另是一江河每對澂潭發浩歌人月雙融寒不溼水天一色淨無波」


(錢礎日曰千偈漏觸不如一句道破)。

素翁仙塚


「　鶴去峰前曲徑幽仙蹤多為白雲留白雲也道仙蹤好一枕青山萬壑秋」


(殳山夫曰彭素雲元時人洪武初遣中使召之忽化去其
仙跡甚多于嘗有山中人導觀仙塚長句頗為同人嗟賞讀此自覺形穢矣)。

義士古碑


「　誰識深山表義碑芳流千古賑時饑斯文寄與煙霞



(殳山夫曰此元夏義士樁也碑臥荒艸中山中俟菴欲
策亭覆之當鐫此詩子亭中亦一勝事)。

洞口春雲


「　空明一寶巧成紋綠樹參差散遠芬徑外似聞雞犬吠仙人知有臥深雲」


(俞無殊曰宵然仙境)。

晚香遺址


「　山川夢老錦千堆風月猶餘土一抔艸色似知憐故舊年年帶雨上高臺」


(趙砥之日滴紙寒香)。

甘白澂泉


「　石罅流來色愈鮮甘澂不讓漢宮泉誰將靈液移於此放出東山一線天」


(華漁石曰色聲香味俱足)。

五爻奇石


「　攜杖搜奇入翠巒雲根早得附金蘭乘閒共坐谿邊月彷彿論交話歲寒」


(包朗威日米顛呼石為丈後人繪圖傳之此詩亦可畫也)。

玉屏八詠

紫芝巖


「　天開瑞靄繞層阿三秀光含紫氣多春到巖前花滿地猶驚人唱採芝歌」


(王咸平曰偏寫得不寂寞)。

玉蓮池


「　水鏡深沉照玉顏花光一色繞池閒含情似欲人同語擬是瓊仙月下還」


(黃敘九曰水光花色是一是二會心處正不在遠)。

仙人床


「　閬苑仙人笑語溫白雲一榻臥黃昏年來想是飛昇去枕簟猶餘石上痕」


(徐介白曰字字飲飛)。

學士亭


「　窗開曲岸抱清谿雲作珠簾捲戶西視艸不須人秉燭一亭明月代青藜」


(顧茂倫曰亭為諸乾一所染二沈學士翰墨風流千載如
新詩中有圖分明一幅學士亭圖)。

興雲嶺


「　高疊空濛映遠岑天將時雨便成陰悠悠不欲從龍



(安大己曰便有最陵高臥之思)。

苦節碑


「　風摩斷碣倚雲隈自分冰心老不灰素節若非堅白玉清名誰肯勒蒼苔」


(戴耘野曰苦節先生二沈學士之父也碑為風雨剝蝕得
此詩如珠光劍氣不復可)。

宜晚堂


「　興老林泉白髮新滿堂秋色一閒人賓朋揖別香山後尚有黃花作四鄰」


(陳長發曰林下何曾見一人曹時中自堪不朽詩足配之)。

景華橋


「　玉屏山隔水西疇勢展蒼龍枕碧流無那題橋人去後至今華月鎖谿頭」


(徐禎起曰唐人風調)。

大陸八詠

丹霞嶺


「　丹巖一帶淨如春目斷雲霞憶古人誰解登高生隱思不隨青夢走紅塵」


(周安節曰喚醒多少)。

赤鹿坡


「　艸眠赤鹿幾榮枯鳴入斜陽半有無想是尋芝回嶺去石邊遺跡月同孤」


(王咸平曰只落落寫大意自好實發赤鹿便俗)。

平原村


「　一灣流水小橋過煙柳人家枕綠坡啼鳥聲聲渾似昨聽來疑有客吟哦」


(歸元功曰士衡十年閉戶讀書即此地也俯仰之間便成
今古詩能引人著勝也)。

寶樹菴


「　煙堆怪石翠森森蘭若初開玉樹深青磬敲殘幽夢冷梵音原是舊書音」


(陸孝山曰以音聲作佛事)。

真珠浦


「　潮回浦口浪初移白鷺隨風展釣絲每欲尋珠來水面輕帆一片逐雲馳」


(金孝章曰從白鷺入想幽極異極通首輕雋回風舞雪不
足喻其萬一)。

大陸亭



「　玄圃光華冠世雄亭留石上正思公知心端是雲間鶴又自巡簷唳曉風」


(程石門曰弔古興哀覽宋沈遼詩尚淺)。

醉眠石


「　不羨功名不羨仙杯斟竹葉飲青天笑看醉裏山河小一枕清風抱月眠」



白雪灘


「　寒深玉骨暗飄香疏影平鋪傲眾芳寄語青山嘗作主莫教空老白雲鄉」



橫雲十詠

連雲峰


「　縹緲層巒嬾不飛散成波浪奪天機風來陣陣吹難斷似待蛟龍雨未歸」


(戴耘野曰山石固奇詩亦筆飛墨舞)。

麗秋壁


「　秋入山城影欲然驚人滿眼醉風煙不知霜老樹中樹惟見雲酣天外天」


(周安節曰橫雲山多奇石見于山夫贈山石序又復見之
寒和尚詩諸篇奇玅獨此更稱奇絕)。

碧崇巖


「　空臨一嶂碧雲涵似有苔錢障瘦巖我欲乘閒來拾翠侵人寒色手慵芟」


(王咸平曰正復不慵)。

雙泠澗


「　山分素練小亭西燕尾雙承石浪低玉漱雲根寒枕枕誰遺琴瑟一清谿」


(倪天章曰溫潤中饒有異光熊熊)。

太原碑


「　為憐斷碣半山斜石蘚年年點落花有客頻來敲不去只留名字老煙霞」



來谷潭


「　練入深雲洗嶽嵐擬將新月釣寒潭源流欲問來何處自有漁人為指南」


(吳海序曰一往破的)。

清風院


「　樹樹花開不自刪瓊林幻出老禪關一聲鐘落山光



(徐介白曰列子御風行後重見此詩)。

忠孝祠


「　深開畫棟列英豪名節雙懸日月高血灑不妨山亦醉猶存松柏見貞操」


(沈偶僧曰謾謾如松下風)。

謫仙菴


「　竹裏仙居翠欲分風敲石戶掩山雲當年獨唱清平調漫散天香處處聞」


(李膚公曰似謫仙人句)。

白龍洞


「　傳說飛騰風雨遊雙谿深鎖石門幽山靈昨報重歸壑滿洞閒雲澹不收」


(顧茂倫曰每于無可著想處得最異之想大奇大奇)。

天馬十詠

二俊堂


「　英譽曾魁翰墨場滿軒詞賦頌聯芳名高山斗千秋重茅屋還存俎豆香」


(孫無言曰弔二陸者多貶剌之辭此作不失忠厚和平之
遺平原兄弟有靈當復空山夜哭)。

三高塚


「　壟上花開連理枝四圍春色護穹碑高人去後憑誰弔日落空山鳥語悲」


(顧茂倫曰楊廉夫錢思復為寓公陸宅之乃郡人太守林
公慶同葬之于山人奇事奇三百年後又得此詩嘻何多奇也)。

雙松臺


「　森森臺畔立雙松待鶴心堅不受封幾度春秋無異色影搖千尺看飛龍」


(畢西臨曰詩思堅挺如勁艸之受疾風)。

一柱石


「　臨虛峭似筍抽條風雨揩磨渾不凋只道近天剛尺五那知身已在雲霄」


(金亦陶曰氣格高渾)。

濯月泉


「　石出靈源照眼明悠悠浮玉映山城落花不與春流去深浸寒光轉二更」


(顧樵水曰指月拈花無處無玅悟)。

看劍亭


「　干將亭下隱孤巒付與春風帶露看冷焰不隨天地



(王雙白曰是說劍不是說劍憑人築著築不著莫負此老
婆心)。

八仙坡


「　石坐群仙列紫苔想從渡海入山來松醪共飲三更月一曲清歌跨鶴回」


(李膚公曰峰頭笙鶴儼然未散)。

半珠菴


「　誰剖珠函挂碧層煙光欲露翠猶凝曉來池上分清影半屬驪龍半屬僧」


(葛瑞五曰千錘萬煉絕不露半分斧鑿)。

留雲壁


「　高崖花落蘚紛紛峻峭何妨宿斷雲顧影不嫌山寂寞疏鐘猶帶扣斜曛」



揖秀軒


「　春入朱欄夢眼醒笑談面面海山青簾前一樹山梧好時有清聲到耳聽」


(王咸平曰詩以秀出以秀入)。

崑崗十詠

婉孌堂

(殳山夫曰婉孌堂為陳徵君初隱之地後移居東佘白石

神虎穴


「　曾聞聚石講山頭伏虎還應似狎鷗何日好風生穴外一聲長嘯滿林秋」


(徐電發曰後二句用事化用意沉)。

紅菱渡


「　菱葉菱花傍水開倒牽山色浸樓臺一聲款乃微風起滿櫂清歌兩岸來」


(陳長發曰竹枝詞無此清麗采蓮歌無此靜遠)。

楊柳橋


「　楊柳橋邊楊柳風楊花點破碧玲瓏不辭帆落谿風裏只看人梯水鏡中」


(顧庶其曰絕句推青蓮龍標第一此作彷彿似之)。

乞花場
