御製蓮華心輪廻文偈頌卷第一(并序)　富

「朕聞。」
【(朕者我也高大尊勝之德唯一人之稱也雖非目擊金容而乃耳傳寶偈故曰朕聞也)。】

「如來妙法理深也。」
【(承真如實道來成正覺故曰如來妙法者能詮教也理深
者所顯義也摽能詮之妙法開中道之玄微喻顯蓮華法明離染理果俱出水教行並開敷也)。】
「
不可
輕易而銓量。」
【(妙法高深秘要難測万象含於一字千訓備於一言似於優曇難逢
希有非聲聞小智容易而銓量也)。】
「
真諦奧玄智廣也。」
【(菩提涅槃理智二果
也真諦者真實理佛性也智廣者無漏根本智也奧女者心欲緣而慮絕口欲談而詞喪不可思議者哉)。】

「豈非邪見而測度。」
【(因中菩薩聲聞尚未窮究佛之深理豈是凡夫九十六種
邪見之徒而能測度)。】
「
諒慈悲而攝受諸相。」
【(慈能與人天之樂悲能拔
三途之苦攝受者瞿師羅長者覩三尺而發心五百婆羅門見灰身而起信隨其諸相攝受舍生也)。】
「
匪
宏遠而導達有情。」
【(匪者非也宏者大也若非大智慈悲誰能導引群品
令出輪迴)。】
「
救度群生胎𡖉濕化。」
【(現三類之化身救長夜之業苦普使四生
九類同修十善五常)。】
「
開大利益示現傍通。」
【(三變土田八方嚴淨開示要道
大化群生故曰開大利益現鹿馬之形提接同類居尊高之位誘化人天故曰示現傍通矣)。】
「
喻說
分眀。」
【(經云諸有智者以譬喻得解故云喻說分眀也)。】
「
遠離於昬濁顛倒。」

【(遠離者不著也昬者昬衢濁謂五濁顛倒者三顛倒也若能修行遠離顛倒自然出於昬濁也)。】
「
人我
執著。」
【(居凡夫地有人執法執二執不斷預聖無由也)。】
「
可超於智慧菩
提。」
【(可超者可證也若菩薩遇法華之會入開示悟門何智慧而不超乃菩提而必證也)。】
「
發自
予懷。」
【(發者舉眀也予懷者宸衷也將欲舉眀宸衷覆燾之心湏與方便是著心輪偈也)。】
「
而

成斯偈。」
【(而者語助之辝成者巳圓之義斯偈者即心輪頌也)。】
「
破煩𢙉障。」

【(眾生無始輪迴不得出離盖為煩𢙉覆障是以佛說一乘七喻誘生智慧令破煩𢙉覆障漸求大智)。】
「
入
修行門。」
【(入謂趣入眾生自無始時來往復諸趣不見正道如來方便開諸教門謗勸進修使
之令入貧子勤作斯喻顯矣)。】
「
達彼岸而精進無邊。」
【(達者到也菩薩
欲出生死此岸求達菩提彼岸而乃興慈運悲得勞忍苦歷無數刼方得成佛故云達彼岸而精進無邊)。】

「渡苦海而逍遙自在。」
【(渡者越也苦者眾苦也因眾苦無邊乃喻於海
依法華事理修證必能越渡苦海方可名為出三界人淪溺既超故逍遙自在也)。】
「
將其小
善。」
【(將者猒也其小善者且以君之務化本在無為合於法華會三之理今言小善是謙諭也)。】
「
普
施眾生。」
【(普者遍也施者與也眾生者黎元也經云一雨所滋三草並茂乃普施之義也)。】
「
且
夫萬仞山高。」
【(四寶峯巍七金磅礡匪常情而可測唯聖智以方知妙高山四寶所
成黃金白銀水精翡翠瑠璃七金即香水海內周圍七帀)。】
「
劫盡而尚為灰燼。」

【(曰歷三灾俄消萬仞蓋洪波而浩浩矧熾煏以炎炎是致喬巒俄成煨燼經云劫火洞燃大千俱壞湏彌
巨海磨滅無餘)。】
「
五慾業重緣牽而海變桒田。」
【(六塵由是紛飛
三業以之而掉悔此表心不孤起仗境方生如居於劫外地前必是乎境強識劣更若以寒來暑往他季
而谷變陵遷烏兔如飛光陰似箭深合教理密契筌蹄五欲即色聲香味觸三業謂身口意等三業也矣)。】

「悟三乘者不戀愛河。」
【(上中下根名曰三乘聲聞喻於羊車緣覺喻於鹿車
菩薩喻於牛車謂之三乘巳越暴流尋達彼岸)。】
「
明四智者常居淨土。」
【(四智
乃菩提之本稱也諸佛轉八識而成四智所謂大圓照智平等性智妙觀察智成所作智昧之者暴流難
越了之者佛國定生非聖智以難窮匪宸衷而莫究)。】
「
曉浮生於幻夢覺塵
世若浮泡。」
【(迅速流季真同幻夢瞥然浮世何異浮泡盖未證於真常致難於朽宅苟非

洞達莫曉斯文故道德經云愚者暗於成事智者見於未萌也)。】
「
每行化俗之方。」
【(屬力
黎民之類常懷惻隱之心施惠愛以臨之布德澤而潤之不唯表君聖臣賢抑亦使父慈子孝)。】
「
庶
招真空之理得而自樂退想忘情。」
【(然後住想空門留心
釋氏貴以雙空之理潛袪八側之情即封執自除疑滯斯遣也)。】
「
朕機務之餘留
心釋典。」
【(天下文軌混同帝萬國洪纖機務躬自宸衷旰食宵衣觀書闕禮叶文武之道固
慈憸之心而復景慕如來睠懷真典)。】
「
乃搆廻文之偈精求玄妙之
源。」
【(澄神靜慮冥合真如搜龍藏之琅函採貫花之妙旨窮覆至理囊括群言即聖人起思製作之
意也)。】
「
起因一章終成千首。」
【(起興也首自心字冠在諸篇起例廻文成二百
首心字拘觀字交㸦羅文為八百首通心字計成心偈一千首矣)。】
「
舒展狀蓮開
一朵。」
【(蓮者生於淥波開敷菡蓞根深葉茂蓊鬱馨香如心偈者文句出於睿思理義喻於蓮華開則
通遍於三乘合則同歸於五字極疋機之妙並造化之功)。】
「
聯綴似月彩初圓。」

【(偈成千首卷分二帙蟬聯珠綴玉清生旨一心該乎萬法若月滿於河漢如彩散照曜群生
光流遐注)。】
「
立名曰蓮華迴文偈。」
【(心偈取象是以蓮華之号貴彰妙法
之深迴周旋粲然義顯)。】
「
深非帝王之能事媿辤以
縱橫。」
【(底定四方子育天下民居域化治太平聖人之能事也若乃旋心玄默吟詠性情化群
有之幻心傳諸佛之唱道則聖人之餘暇也獨以心偈大情濬時睿旨縱橫製作非常委順辤正亮流
布復諭言)。】
「
隨分可觀甚為魯質。」
【(魯質之議之辤以所製
心偈庶幾真理引導修存臣下如觀可悟佛性)。】
「
兼詔高僧注解稍究根
源。」
【(蓮華心頌織出神機羅文而寶網交光務其接物中旨而醍醐上味意在滋生而又丹詔飛來
降自凢天之上令臣注解寧詳萬乘之心且臣等智謝笙融學慙安遠本乏超人之譽何當出永之名負

荷不吠憂兢過甚今則釋承聖製懾測律注𡨋搜三藏之真文永賢一人之金偈矣)。】
「
大振於
寶鐸金文。」
【(寶輪迴文實法憧之高樹集文成注利並鐸以泠音盖所附之出應使聲含於
雅韻)。】
「
貫穿於玄言妙旨。」
【(貫穿者攝持之理也玄言者能詮之教也妙旨者所詮之
理也謂教能攝理舉冀生機也故慈恩云攝列集斯妙理貫又御彼庸生令畢離苦津終登覺岸)。】
「
永
作津梁而濟沉溺。」
【(迴文至教傳於未來寔苦海之舟乃泥犂之梯如世尊言
張大教網流眾生魚至彼岸矣)。】
「
常為慧炬以破昬迷。」
【(巳二障之昬黑覆蔽
慧心使諸趣之迷喻長時不忠留茲心偈永照昬徧者也)。】
「
俾使信心咸知朕意。」

【(今皇帝務令眾生發菩提心修出離行趣無上果終致成佛即其意也)。】
「
今雕成圖像
注并偈頌共二十五卷具列于後。」

「　心會解垂要」


【(如來之心離於名相指蓮華而為喻出水功高演妙法而稱奇會解
無為方便垂訓深要之理也)。】

「　理真詮羨非」


【(佛意真理不可以語議論量人天仰
羨行利他之教敷有道之文並曰流輝依天運轉淺近非而能知也)。】

【(如來甚深之意包含大千喻教五常問於四海誰能濁曉尒)。】

【(賢愚雖異皆羨真詮智若非深豈能達解)。】

觀業苦之逼迫甚可憫嗟垂利益之慈悲便為依怙)。】


「　運孝齊嗔協」


【(運之於德孝之於心齊之於賢協之於順明彰四行誡君子之嗔心高
運六通顯聖文之勸善也)。】

「　健古尋寬在」


【(具勇健智尋古佛心寬隆自在之功必到
圓滿之地)。】

「　訓照迷循業」


【(訓騰今古照貫去來因循沒溺於迷中業緣繫絆勤確誘
化於出離惑障永無)。】

「　願苦侵觀靄」


【(佛興悲願觀眾生諸苦所侵法闡慈雲靄群品
俱時解脫)。】


「　孝齊嗔協健」


【(混元始分孝道齊彰於初首真如才證佛智協健以揚詮闡斯利門
調化嗔毒)。】

「　古尋寬在訓」


【(尋赫胥之道曰古證如來之智稱寬發心在訓警之言成
道居利生之路)。】

「　照迷循業願」


【(願聞大惠照示迷情循根識之有差隨業緣而化度)。】


「　苦侵觀靄運」


【(眾生遭惑苦之吞侵靄霑三界諸佛運觀智而救拔令離六塵
各駕法舟速登佛路)。】


「　齊嗔協健古」


【(和贊王道名協自他俱濟為齊發精進勇健之心断貪欲嗔恚之火
令遵古典依法修行)。】

「　尋寬在訓照」


【(披尋至典寬究根源知三教之指蹤但言訓照
了五乘之玄妙獨在經文)。】

「　迷循業願苦」


【(荏苒循環於三界蓋為迷心輪迴墜於

四生都緣業苦佛持大願度此有情)。】

「　侵觀靄運孝」


【(有精微而孝生有智證而
觀起慈雲高靄運動勞生法雨遍侵潤茲物品)。】


「　嗔協健古尋」


【(細窮至理謂之尋聡惠不群謂之健仰法水以來霑滅情田之嗔火
咸言上古盡協佛心)。】

「　寬在訓照迷」


【(道含万象謂之大教流百億謂之寬智懸照於
迷淪意唯在於提訓)。】

「　循業願苦侵」


【(憫循環而不定被業苦以相侵願開揚化之門
廣演玄溦之法)。】

「　觀靄運孝齊」


【(悲智齊運忠孝兼行布靄靄之慈雲降霏霏之法雨
咸觀佛道孰不進修)。】


「　協健古尋寬」


【(躍行古道謂之健情含法藏謂之寬靜閱真詮覬智心之漸朗細尋
奧旨協大覺之指蹤)。】

「　在訓照迷循」


【(像教流通軌式恒在訓示沉迷之眾照開昬暗之心
不使因循虛忙險道)。】

「　業願苦侵觀」


【(佛願若海化質如雲觀五苦之含生未崇白業唱一
宗之妙典驚倒黑山既無邪黨以來侵必有證真之方便)。】

「　靄運孝齊嗔」


【(靄運神通
大崇孝行放光說法與古佛以道齊止貪離嗔作今來之誡制)。】


「　健古尋寬在」


【(求法勇𨦣謂之健也古今時異佛旨體同躍躍在心正合寬仁之念
明明尋善必躋寂滅之門)。】

「　訓照迷循業」


【(訓照迷謬因循業縈立名文為導利之門
顯光明作出生之路)。】

「　願苦侵觀靄」


【(行苦相侵刼石銷而未息靄願深廣智觀
鑒以不停)。】

「　運孝齊嗔協」


【(運動情樞伏除嗔慾心宗孝行志協事親不惟賢聖以咸
遵抑亦古今之齊詠)。】


「　古尋寬在訓」


【(師古尋真寬仁獎善潛想諸佛之行跡削除眾識之妄緣欲訓示於

來在先修於殊事)。】

「　照迷循業願」


【(高懸智曰謂之照不歸佛道謂之迷循察深機
廣開化行令除業苦願作津梁)。】

「　苦侵觀靄運」


【(死生逼迫謂之苦善惡相吞謂之侵
運靄靄之定通令修觀智懸明明之法印願作指歸)。】

健貪嗔俱断謂之齊協大覺之玄風行孝悌之善道)。】
