










御製蓮華心輪廻文偈頌卷第二十五　富　


「　曉其逆順」


【(曉者明解也即三乘五性之人能緣了境之智隨機領悟達解
分眀謂之曉也逆順者根法相修謂之逆根法相符謂之順且無性者只樂人天遂被之
以三歸五戒定性者唯希趣寂乃教之以四諦緣生進退取捨之流三時漸誘䪺發大乘
之者直示根源各稱本心俱生歡喜即是順也又大乘菩薩作用難思或現四眾之形或
處三界之內如維摩詰入諸酒肆能立其志入諸婬舍示欲之過乃至廣說如經本文然
則不樂二乘如怖雜毒觀彼持戒即是破戒故佛喻云如貧人食是輪王毒此即小
法不契大根刀非順也又如不定性人雖則身含大智然且本性未熟預聆記證之音彼
即深懷不信故指喻云譬如以寶示之愚人愚人便笑何況無性定性能順大乘者哉斯
則大法不契小機亦非順者也)。】

「　如草隨風」


【(但法法投機根根領悟根法相順則
如草隨風故夫子云君子之德風小人之德草草上之風必偃又如法華經云譬如大風
吹小樹枝)。】

「　惣摽三喻」


【(惣者苞羅綰攝之理摽者舉指顯示之詞三喻者若
指羊鹿牛之三車為喻即顯三乘囙行力用有差故法華經云羊車鹿車大牛之車今在
門外若指兔馬象之三獸渡河即顯三乘得果證理不同如優婆塞戒經云三獸渡河得淺
深別若指小中大之三草為喻即顯五乘稟教滋長各異故法華經云一雨普潤三草二木生
長不同也)。】

「　器用周通」


【(言器者容受之理隨根應情聞法能受謂之噐用謂大用
斯則運用有勝劣渡水有淺深等也周通者即徧被也如來大悲願力不捨一一眾生各稱

本機示之妙理故無垢稱經云佛以一音演說法眾生隨類各得解即噐用周通之理也)。】

「　境外取捨」


【(境謂五塵即五識所緣之外境也五識見分變相緣之則取捨隨生)。】


「　萬類皆空」


【(五塵本質則第八相分五識見分則託彼為質各變相分為自
所緣攝相歸見實唯有心乃至大地山河星辰日月万象當情則皆由心變若離能變之
心實無一法可得以其有漏識心諸所變相千差万類一切皆空故法華經云觀一切法
皆空亦如般若經云一切有為法如夢幻泡影如露亦如電應作如是觀故云万類皆
空也)。】

「　譬由蟬蛻」


【(若了妄境為外空達真心為內實者誠由蟬蛻也尓則高
棲綠樹鳴達九霄此則永脫樊籠清昇彼岸矣)。】

「　夢𥧌不同」


【(外若取著於聲
色內則弃捨於靈源知見既盲寔由夢𥧌故世尊云未得真覺恒處夢中之謂)。】

「　但去麤𢙣」


【(外弃六塵之麤𡚁內𡨋四德之妙真故法華經云勿貪麤𡚁色
聲香味觸也若貪著生愛則為所燒等也)。】

「　宇宙齊功」


【(廣兮昊天能覆万物
謂之宇也古往今來謂之宙也若能了境歸心內𡨋真性者非圓盖之大功無白取象非
古今之深致曷以求齊也)。】

「　𡨋心著像」


【(意云欲令專守一致不起他緣觀像類
之分明向研搜而無倦也)。】

「　坐臥勿慵」


【(意使𢝰斆甯戚忘喰之篤志皆鄙宰
予晝寢之餘功故也)。】

「　敦兮若朴」


【(敦者寬厚也朴者淳素也此則務令去華
取實還淳返朴之意者也)。】

「　絕學優隆」


【(若學富而無倫非登仕而不止故積
雪聚螢之士終致身榮懸頭㓨股之流盡諧優寵也)。】

「　開[門@(恙-心+皿)]戶牖」


【(天門開[門@(恙-心+皿)]
能為雌乎天門者歷[婁*殳]之所從也開[門@(恙-心+皿)]者謂治乱也即聖哲應期受命能守雌靜也古者

陶穴以為室亦開戶牖故)。】

「　恍惚其中」


【(道之為物也唯恍唯惚此明先聖所從之
道不有不無沖用難名故云恍惚其中也)。】

「　知白守黑」


【(知其白守其黑為天下式
常德不忒復歸於無極也能守柔靜常德不離德雖明白當如暗昧如此則為天下
法式常德應用曾不差忒德用不窮故復歸於無極)。】

「　闇昧和融」


【(闇昧者昬
迷之相也和謂和柔即陰陽含孕中氣調和然後万物阜成也融者明也五常在身道德
斯俻則鑒達分眀矣故德經云用其光復歸其明注云還守內明則無與身為殃咎者如
此是謂密用真常之道也又如佛說智起惑亡明來暗謝)。】

「　自矜者溢」


【(道經
云自矜者不長意云矜衒行能人所鄙薄故不長也斯則聖人誡其薄德之者令勿自矜
使同君子之人滿而不溢)。】

「　鮮𠑽有終」


【(鮮者少也𠑽者得也故道經云少則得意
云抱一不離則無失是以聖人抱守淳一故可以為天下法式也又鮮𠑽者即不自矜伐
之名也故道經云不自伐故有功不自矜故長也終者成滿也即顯道德俻矣)。】

「　蠢尓生植」


【(且如春風一扇則眾卉齊萌亦猶往聖順陽氣而養万物故使民皆抱信信既
植而仁義禮智自然生矣)。】

「　覆載清崇」


【(無私覆載之深功方齊養育之厚德
者矣)。】

「　盈虧莫究」


【(日中則昗月盈則虧天地盈虧與時消息然則梭飛眀滅
難窮剖判之初絲吐光陰莫究始終之極故也)。】

「　上叩玄穹」


【(既覩盈虧之難
究但仰高天而莫窮亦如受大蔭之無私唯荷深恩而𠕀盡也)。】

「　寒熱增忍」


【(德經
云躁勝寒靜勝熱清淨為天下正意云於躁勝者則寒寒乃薄也於靜勝者則熱熱乃和
也故若守靜安神清淨為真也)。】

「　鑒照盲聾」


【(無信之人聲教絕被盲聾

之理其在茲乎然則聖哲處世若智鑑之臨臺寔幽微之洞照故道經云知人者智自知
者明智者役困以歸物明者融照以鑒溦智則有所不知明則無所不照)。】

「　賤而不貴」


【(貴以賤為本高以下為基且侯王貴高兆民賤下然其國者以人為本基
當勞謙以聚之令樂其愷悌之化勿使離散也)。】

「　惠必英聡」


【(夫聡晤明敏辯解英
捷者皆由於惠之力故云惠必英聡)。】

「　狐狸邪患」


【(狐狸者猶小人之姧邪也若乃
龜龍應圖賢人處世則狐狸邪患之類各自藏形矣)。】

「　蜂蠆怔忪」


【(蜂蠆者類
小人之含毒也怔忪者惶悚怖怕之義也若乃鳴鳳在樹騏麟應時則狼貪蜂蠆之流分
頭避迹回賢參孝之士盛茂於時也)。】

「　寢食無暇」


【(且以西伯之聖姬公之才
猶有日待旦之勞也女文王自明至日不暇食用修其德以安兆人也周公亦夜坐
待曉以憂政事)。】

「　飢改秊豐」


【(由是一人有慶則兆民賴之故淂飢改秊豐人
無顇色者也)。】

「　海晏安靜」


【(覩赤子則盡無愁聲觀滄海乃悉無驚浪亦如
金輪御世大海減而七寶興瑞花生而聖人出)。】

「　勇猛從容」


【(勇謂勇捍即銳利也
猛謂威猛即豪俠也從容者閑雅也即寬猛處中之理也)。】

「　讒滅盡去」


【(小人
在側而君子信讒盜言如甘而巧言如簧昔費無極者楚之讒人也屏楚王之耳目
使不聡眀意使智者則除讒以自安故云讒滅盡去)。】

「　比屋可封」


【(人無讒搆
之痕戶有可封之德當堯舜處位之世亦比屋可封故也)。】

「　囹圄寬宥」


【(囹圄者周
之獄名也今觀其教不肅而成覩其政不嚴而理遐布從輕之德廣宣赦重之恩故
得囹圄冤宥也)。】

「　桎梏摧兇」


【(桎梏者杻械之類也即自古摧兇之具今則人

修十善弊𢙣皆除囹圄既空桎梏斯廢也)。】

「　處薄持儉」


【(仲𡰱德而處薄
顏回賢而持儉況乎君子以進德修業為寶豈斆小人以垂權竊行為利)。】

「　𣗥變花叢」


【(夫荊𣗥者取象於小人也故古人云地卑只解生荊𣗥芝草名花者盖比
於君子也故禮云王者慈仁則芝草生國穰時豐則異花秀我皇帝無為而治宇內也四
海晏清天下蕩蕩則小人易慮盡懷君子之心忠孝賢良方滿太平之世斯則𣗥變
花藂之謂也)。】
御製蓮華心輪廻文偈頌卷第二十五(正文一百九十字注二千一百六十字)

甲辰歲高麗國大藏都監奉
