周注西剌史北旋贈別


「　般若緣深一見真，暖風和日對相親。此歸憑豁金剛眼，去看嵩華翠色新。」




「　技盡懸河義未該，入門一喝笑盈腮。潼泉也合遭人怪，錯與周公說破來。」



東吳延皋孝廉三絕


「　小雨溼松花，披衣巖下坐。起身謝白雲，不覺莓苔破。荳葉開新畝，藤花爛舊棚。庭分朝樹色，溪轉夜泉聲。終朝看瀑流，夜復坐空閣。明月上高峰，幽人不寂寞。」



贈卞令之廉憲


「　一會靈山法道親，利生權現宰官身。三千里外擎香飯，正使金牛笑轉新。」




「　補袞調羹信自優，文情天宇至風流。楚江寬闊行相訂，為與空王啟壯猷。」



贈孔月恒居士


「　頻來問祖意，雪夕與花陰。寡過無他伎，修名但了心。意消忘覿面，疑失笑披襟。松下時烹茗，濡毫作孝箴。」



贈孔魏公居士


「　籃輿重至意翩翩，滿口平傾不涉詮。況是鴻儒能慕道，方期振起兩宗傳。」




「　夫子達磨元一貫，多門謬解豈其然。人窮返本方堪語，試看昌黎問大顛。」



贈陳曇侯鄭公常二居士

壬寅元日示眾

客普潤菴鐙巖文吏部同公郎允信右獻過聚


「　垂楊近冰綠千條，最喜幽人冷挂瓢。乞食江千容懶拙，放參湖上問漁樵。深山有社堪招炳，今日相逢宛似陶。風雨匡廬登未得，好將真意論逍遙。」



示諸行務


「　山居家風，別無絡索。廚庫僧堂，尋常矩矱。放下笊篱，提起木杓。飯是米炊，麵是麥作。茄看花開，瓠連蒂落。逐日生涯，大好不惡。」



悼解菴張新愚居士


「　沙路環菴瑣舊苔，高山何夢不君來。軍持帶有寒泉水，忍向風前奠一杯。」

鄂州夏伏不雨寄湛智水


「　極暑難為畫打眠，江流稍引到池蓮。何人不愛山中寺，日午時聞樹裏泉。」



贈一菴居士游夏公則菴居士游賡臣兩兄弟

客嘉魚有懷檗菴法兄

宿普度菴逢任仙孟詞林



「　水落江天淨，山城一水連。正思過別浦，相識有奇緣。宿望名垂遠，斯文道借傳。好懷言不盡，且共佛燈眠。」



與蘇菴豁菴大心分韻書懷得笻字


「　作客多時賓似主，竹爐炊茗解疏慵。是非不上閒名牒，去住難羈了事笻。落日鳴蟬增旅況，平湖秋水照衰容。疊來清坐知何意，午飯前頭到夜鍾。」



宿李氏書館


「　東嶺連嘉渚，書聲載草堂。遠山藍潑几，近水綠迴觴。里靜門無鑰，沙寬月似霜。我來勤下榻，恰好值新涼。」



客夜別友


「　風雨留人又一宵，別君燒燭度溪橋。百錢買得舟人醉，明日雖晴莫放橈。」



陶溪過訪戴小宋文學同廣生兄


「　山深春淺共驅車，攻苦人真似仲舒。側理舊藏宣德製，花時再到倩君書。」



再過小宋郊居


「　澹圃疏籬菊正扶，秋香掠徑好朋俱。新詩半在黃梅裏，收盡奇峰入畫圖。」

「　乘漲輕來一夜船，快移行李就林邊。選官選佛名雜異，觀海觀河見不遷。幾處弟兄團此日，閒窗風雨憶當年。攤書問我如何說，為指秋空月正圓。」



過訪王報菴解元


「　廿年師友夢相關，鶴澤今看喜破顏。曾憶截江風雨句，手緘曾寄玉泉山。」



壽王報菴


「　南國人文合遜尊，都來鶴澤訊寒暄。疏經日暖花成巷，愛客荷香水到門。藜擁多男森樹玉，霜清兩鬢健加餐。長齡不借安期棗，一粒真丹內久存。」



題龍門寺壁



「　澗道盤蜒上，霜根結斷霞。鼓鍾開佛地，雞犬靜人家。竹榻橙香雨，松鐺石火茶。別山秋到葉，輕重落平沙。」



柬黃安植聖澹老師


「　竹簾茅屋一身閒，只為閒多復閉關。怪底白雲風不問，仍來覆我蘚苔斑。」



寄懷秦茂林居士


「　光黃奇勝昔曾探，植聖松門至再三。最愛清溪秦處士，好山遍捨作精藍。」



壽郭些菴


「　春日花香透水流，補山晴色上眉稠。七旬四海稱元老，一衲深林法比丘。潔淨躬行誰與等，虛閒心境復何求。古今多少存疑史，檢定精神日見優。」



示死心道人


「　夜行不許投明到，正是出身轉位時。更把吹毛揮向上，空輪無跡刃無虧。」



付睿首座楚林法偈


「　天山法道巍寰宇，奕葉芬芳祖氣鍾。十載求人今得子，直承大統啟綱宗。」



坐潤慈開士郭北靜室


「　放住閒心自不群，石城精舍鎖重雲。一聲午磬高寒裏，雪路何人駐馬聞。」



付祚西堂慈裔法偈


「　信知汝力能擔荷，是我尋常慣用伊。會向深峰深處卓，抽條成蔭見施為。」



留別禪喜非異


「　依依楊柳采茶天，又欲之吳問曉船。七十二峰青正好，歸期多半在來年。」



憶匡山伊菴禪師


「　玉門擎出滿壺冰，向日開窗榻遠朋。笑指鄱湖千頃月，當余蓮社一枝燈。」




與卞令之廉憲

尹蘭友嵩山甸方屏山覃谷育有郭茂行至山


「　微雨灑高林，同人枉屐尋。有懷勞夢寐，得聚感秋陰。居僻容逃禮，來稀苦索吟。果然怡此道，一宿可明心。」



送友往天台

與孔則山居士

寄五峰怡然師


「　木落空山萬慮忘，病兼風雨度重陽。寶林秋色隹無恙，記許霜柑欲就嘗。」

夏田寺作示宛然


「　夏田江上寺，紺殿聳雲峰。靜夜濤聲落，長途客夢重。高林巢老鶴，古岸護神龍。安得年豐稔，常鳴說法鍾。」

贈恰菴居士劉是玉

夢歸竟陵

寄懷雲水居士毛子霞舊儀部

寄懷素巖傅兄

贈耿二戶部公府

柬耿遠情文學

智泓尼日誦華嚴偈以示之


「　華藏海底穿針孔，繡出鴛鴦看有誰。五十三人沒意智，大家賺箇善財兒。」



買石得雲饒


「　不是華山山頂骨，亦非鬼斧削鴻濛。一錢博得千峰色，喜見龍吟出袖中。」



夜讀魯論至顏回章


「　桐華月影疏，愧讀古人書。孰以回為死，誰從子所居。簞瓢流歲月，風雨壞門廬。師弟存知巳，斯文未喪子。」



柬嵩杲


「　古寺宵寒發素琴，高梧露滴答空音。懷人海國鴻無影，作客天涯歲漸深。杖策擬尋芳草徑，披雲更選綠蘿陰。出門幾度猶回首，為爾長年共此心。」



賦沖雲鷂


「　眼底秋聲驚到樹，好風不借翼偏牢。凌虛始見孤標格，在野何殊眾羽毛。豈為牽群常落影，肯隨輕舉尚回翱。兒童那解騫飛意，拍手惟論燕雀高。」



基隆山禮無念有禪師遺像


「　塔豎別峰影在茲，楸杉秋滿碧參差。苔荒古砌蟲啼冷，煙鎖閒龕月上遲。機裏無言全殺活，罵中有藥喪情癡。比來此道多文藻，拜起愴然長憶師。」



題五峰寺壁


「　我悅山中住，栽成五嶺松。種田秋博飯，說法午鳴鐘。遶屋流清澗，長吟醒睡龍。舉頭雲影外，明月上高峰。」



趙公遠夏言讀書紫雲山


「　秋光寸寸寫幽奇，別業深雲護作籬。白日山中天地老，清宵月下弟兄宜。圖書代主先留客，花木因人亦笑時。近得好峰堪置我，免從風雨起遐思。」




懷國清曇兄能仁無兄二和尚


「　昨書云其丹丘隱，又轉剡溪走一回。好鏡新機酬唱盡，明春定到楚山來。」



贈桐巖靜主


「　三十餘年住此山，渠儂意氣果能閒。支笻且趁月明夜，種得青松耐老看。」



喜秦公蘧文學至山


「　天臺卓出萬山中，君到山頭秋正紅。為問十年得力句，夜深清露滴枯桐。」



示湛維那


「　草菴忽卸琉璃瓦，萬丈深寒徹底窮。獨有梅華知此意，一枝斜放短牆東。」



示學副寺


「　大統綱宗須識句，多年古灶冷重搘。家風切莫嫌疏澹，想像楊岐乍住時。」



歲朝示眾


「　年頭佛法無多子，拄杖抬頭又見春。莫謂青山能放嬾，青山慣肯笑閒人。」



懷寶安去兄和尚


「　錫山華里寺，別忽逾三年。默坐枯腸響，還思第一泉。」



示喻君華

山居雜詠


「　百卉青青二月春，梨華綻處只如銀。雨晴隨意經行入，萬樹寒香擁一身。」




「　不嫌眼冷眉撐月，且放籬荒水過林。新竹纔成低戶影，曉雲慣作覆巖陰。」




「　一日天晴兩日雨，藤華未摘巳春闌。枝頭小鳥聲如渴，報我西谿欲放完。」




「　芳草離離覆石根，無端終日雨煙屯。時人那解飛龍處，輕作尋常埜水論。」




「　翠壁如屏啟戶前，落華曉砌石幢邊。缽盂有口平吞盡，折腳鐺中栗棘圓。」




「　天涯身寄綠蘿菴，雨過群峰似潑藍。幾點客雲歸遠岫，杜鵑聲裏憶江南。」

「　扇子芭蕉自剪為，幽居是事獲便宜。秋心不在梧桐上，葉落空庭曉未知。」

「　苔徑斜暉侵錫影，石床華雨放閒身。敲空作響翻成拙，道在何方此日貧。」




「　都知傀儡倒棚休，宇宙勞勞事更悠。洞口縱無黃葉出，靈源端在古溪頭。」




「　三更還自煮雲芽，挂壁青燈照影斜。手接南星忘把盞，為伊一伴惜年華。」




「　地爐煙穩火猶紅，壞衲情深肯伴窮。幾處寒聲驚落木，故人千里自同風。」




「　老屋三椽僅蓋頭，春泉脈脈遶床流。白雲不放青山嬾，出入松門意未休。」
