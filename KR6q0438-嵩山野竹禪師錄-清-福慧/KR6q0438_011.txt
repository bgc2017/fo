










嵩山野竹禪師語錄卷十一

鄂　州參仝門人宗宏　錄
鄂　州龍光門人宗上　潞南州兜率門人宗堅　仝編

雜偈

示愈書記


「　初日當囪曉色青山滿岫晴光悄然快活松下何必丹丘石旁」



示祥監院


「　早夜勤勤不倦金陵老子風規有時摸著鼻孔與汝一根竹篦」



示和侍者


「　曠大劫來一物朗然偪塞虛空此即相傳的事原原委委何窮」



示敏侍者


「　槲笠艸衣幾載芒鞋竹杖平生如何更問消息別指籬前菊英」



示玨侍者


「　不變不遷不改何思何慮何憂庭前百尺松子每向其中臥遊」



示學侍者


「　山中學得無為嬾把殘經對月千七百則葛藤何為成佛之訣」



示牧之禪人


「　溈山一頭水羖隨處拖犁抴杷莫道無人收管牧之也解倒跨」



示近山禪人


「　諧去諧來十載腳跟下事浮沈室中問我宗旨且聽樹頭鳥音」



示不仝禪人



「　荊棘林中不住翻身卻到慈雲來時步步思忖何者是吾真聞」



示湛涪禪人


「　巾缾缽杖三衣四海五湖任往榮辱自然不干何須別問安養」



示悅也禪人


「　欲醒吾家法眼棒頭別有根源若將學解為勝依然三世佛冤」



示石雲禪人


「　深山寂寂禪室何事心心未閒不若通身放下自然萬慮俱刪」



示指南禪人


「　落日霞生萬壑山靈列幕環屏回頭石上東顧山色朧朧杳冥」



示佛旨禪人


「　止說偷身學嬾那知學嬾非間嬾閒兩字無取馮他日上松關」



示會一禪人


「　曠劫汪汪愛水今朝一瀝傾乾只諧明月來往一切塵緣不干」



示空碧禪人


「　山僧不解談禪亦不與人講究晚來散步長廊不覺依然漏逗」



寄白馬講主


「　白馬峰前翻經青蓮華裏悟理高原鑿井之喻舉世何曾惺此」



寄古問禪人


「　江頭一片明月白露千珠艸頭借問廣寒何兔擣藥聞之滿臼」



示覺一禪人



「　欲識死生覺性瑩然覺一恢恢此理若知真實原非槁木寒灰」



示大一禪人


「　月來原自渡口雲去無非峰頭雲月往來不盡時時醒我雙眸」



示仲安禪人


「　闍黎發足金陵四海名山過目今日相逢益州挂錫三間茅屋」



示不悉禪人


「　為僧獨貴清真莫學勞勞度日動靜常常中規自然大道安逸」



示不易禪人


「　不明生死云何須究西來大意只到佛魔兩忘神珠一顆自至」



示岫雲禪人


「　色色空空世界瀟瀟灑灑人家且過眼前日子任他說落天花」



示三目禪人


「　摩醯三目難鑒衲子相逢若何古人別有方便打鼓且聽笙歌」



示明旨禪人


「　欲知明暗之旨曉夜何常兩邊明暗拋來莫管回頭蹋破三玄」



示淨月禪人


「　月中桂子垂朵一夜風來送香鼻孔依俙識得開眉便見清光」



示尼宗一


「　至道祇須一言自然體露金風且徵何者是道只可返尋己躬」



示尼安然



「　三尺竹篦悟得。任他東土西天。慈雲不是多口。祇要女心了然。」



示廣國宋夫人持法華


「　禮誦蓮經竟日。言言字字深香。有時念到寶塔。便是儼然法王。」



示賀居士宗福


「　為仙為佛高志。衹在當人一決。若是因循日子。依然錯過時節。」



示雷居士遠


「　地僻山幽可靜。谿清石怪堪情。看他鳥弄春日。支上一聲兩聲。」



示戴居士璞


「　住著山中幾載。日高猶自貪眠。為僧得到如此。何必羡他大賢。」



示閻居士禧


「　宿昔閭丘居士。相尋拾得寒山。遺來一片明月。至今猶在林間。」



示張居士印


「　西來大意何旨。白日朝來水東。剔起眉毛便是。莫教錯過此中。」



示白居士恩


「　山頭錦鯉如雪。海底金烏若雞。今日為君說破。明朝各自東西。」



示徐居士太原


「　君家住在何處。身上猶帶落霞。早把機心放下。馮他碓嘴生華。」




「　秤錘蟲蠹久矣。不見不聞隨他。前三後三省得。即是吾家叉牙。」



示何居士常


「　迷悟兩塗未判。端居看是何物。分明一段天真。不是




示王居士幻


「　十類眾生是夢。三世如來夢人。山僧為你直道。爾我亦是浮塵。」



示楊居士顯


「　野馬如無玉絡。奔狂之性何調。學人棄此繩墨。未免伴奐自憍。」



示湯居士昶


「　萬尺寒潭把釣。細鱗無意吞鉤。可憐襏襫漁叟。香餌翛然別投。」



示眾


「　今朝白禮諸人。事務不煩問我。有求佛法因緣。但指庭花幾朵。」




「　山僧雖是村翁。領眾何常廢禮。只是尋常鬻飯。不曾別有臧否。」



示陳居士萬


「　多年失卻寶所。今日相尋道人。話到無生奧義。何勞拋棄紅塵。」



示丁居士輔


「　祇知鑿井高原。不識水源就下。縱是施工百生。何曾有滴如瀉。」



示易居士生


「　早晨洗面穿衣。下晚上床脫襪。申手去摸枕頭。不知打著心月。」



示龍居士智


「　一艇閒游性海。蘭橈打倒須彌。此是大人竟界。非仝緣木之龜。」



示龍居士乾


「　無說無傳聖道。有為有作猿心。一豪不去分別。當下全消古今。」




示趙居士本


「　佛心無住為本。道者無為是根。無住無為不二。鐙籠露柱何言。」



示王居士朗


「　山門佛殿相對。土地金剛鬥拳。既是旁人不識。隨他明月前川。」



坐象巖示昆明諸子


「　象嶺巖前獨坐。朝昏滿目煙雲。回觀隔岸松子。依舊落他見聞。」




「　葉上欲書半偈。枯腸括盡無能。不如石上高臥。此事全超一乘。」



示許居士三山


「　推落匡床枕子。驚起檐樹鳴鳩。安南城裏收賊。塞北山前失牛。」



示劉居士應文


「　才見提刀出塞。又來撿點家珍。兩重公案無別。可謂全詠法身。」



示陳居士宗性


「　道人識甚家風。手裏把箇木勺。驚起甑箄搖眉。鐙籠築破雙腳。」



示石居士廷瑾


「　槿籬靄靄雲合。石澗潺潺野水。入路分明不差。幾曾輪轉生死。」



示李居士可植


「　松直棘曲之理。鶴白烏玄之言。喚作歸家時節。何常悟得心源。」



示許居士云盛


「　誨人不是不切。回頭恐落虛名。偶上東山散步。月明一塢松聲。」



示崔居士天壽



「　般若深談四相。法華喻說三車。究來都是閒事。且坐室中喫茶。」



示徐居士伯遷


「　月照前峰石色。風吹上澗波聲。誰知見者難識。仍教孤他此情。」



示杜居士三元


「　水水山山畫圖。風風雨雨世界。從頭柬點將來。智者如何蒂芥。」



示徐居士化淳


「　騎牛覓牛痴矣。上筏舍筏愚甚。一切是非莫存。自然華屋高枕。」



示祝居士良傑


「　傑出中山獨步。虎頭燕頷稱奇。指耑亦有明月。返照必然得之。」



示徐居士大成


「　大事因緣不別一揮。智劍心寒成仙成佛。多事閒看前峰月團。」



示張居士希舜


「　消閒無過山家。萬嶺清風素霞。閒到峰前一曲。樹頭驚起飛雅。」



示郭居士成功


「　道學無非方寸。逃禪何用求心。自然應物如鏡。一切魔王敢侵。」



示平居士不二


「　喫飯穿衣不二。提缾把碗偏仝。鳥啼花落巖下。悟得從來法空。」



示王居士世統


「　七尺軀中性靈。無耑落在風月。今朝落脫些兒。便上蓮華一筏。」



示陳居士棟



「　萬木森森巖石。風吹瑟瑟清音。禪房有此幽韻。任他世上浮沈。」



示張居士自起


「　一根既得返元六處。自能休伏二十五人。圓通當時名起天篤。」



示曹居士成功


「　石鞏一弓廿載。三平方露胸襟。當陽兩扣絃子。聖箭猶傳到今。」



示張居士登高


「　碧雞午夜啼寒。金馬日中嚼雪。分明好箇風頭。只是當人不決。」



示羅居士守庫


「　當年濟北三玄。今日嵩山一喝。莫謂古今兩塗。總然者點生活。」



示張居士天祥


「　瑞像道人不取。道人只貴心光。問余何謂心光。依舊龜毛尺長。」



示史居士得功


「　空華陽燄潭月。三箇本來不真。若謂竹僧錯判。何妨緘口殘春。」



示薛居士聯


「　帶月連雲艸坐。石頭為枕高情。了然松韻徐響。不勝笙歌幾聲。」



示程居士雲鳳


「　相傳大道多載。只是無人承當。所以慈雲一默。管他東土西方。」



示張居士朝玉


「　佛殿苔生艸。合然鐙過去。慈尊門前沒箇消息。佛法有誰一論。」



示陳居士揀



「　至道惟嫌柬擇。洞然明白何詮。眼光爍破天下。南北東西見前。」



示王居士尚勇


「　道人一箇蒲團。四海任他游戲。若問如何是道。倦來放下憨睡。」



示王居士應顯


「　青峰陰裏高臥。白水谿邊放船。此是維摩玅著。馮他天下相傳。」



示馬居士顯祥


「　者箇渥洼駿骨。有時齧膝千里。非仝款段騰蹋。所以名高野史。」



示劉居士文聖


「　井市山林不二。春華秋月相期。嗟乎者段三昧。須是達人乃知。」



示曹居士應桀


「　良驥從來貴德。蟾蜍亦重精明。余聞先佛心印。不是虛頭假名。」



示王居士良友


「　嵩少安心可師。慈雲勤道良友。莫謂古今不仝。蝦跳何曾出斗。」



示趙居士良珮


「　囀囀支頭鳥語。閒閒林下人家。夜來一輪明月。客至兩桮閔茶。」



示殷居士自常


「　舉世無人好道。滿堂盡是閒僧。雖然有箇居士。何曾離得愛憎。」



示呂居士友才


「　我輩相逢話舊。芝蘭气味相投。落花流水游戲。誰管南州北州。」



示金居士成功



「　金剛撝散魔軍。慧日光開暗室。瑞艸靈芽不珍。端然領取真實。」



示李居士國珍


「　曲說直說兩路。宗通說通一門。分明隻字無得。便是歸家之言。」



示郭居士朝鳳


「　松鶴巢來古樹。鐘聲散入平原。此中即是宗旨。更要阿誰一言。」



示張居士弘名


「　心佛眾生名字。泥豬癩狗生涯。明明如何不會。驢鳴犬吠從他。」



示張居士計美


「　世人百計生涯。嬾漢只消一事。日日蒲團石上。閒來便是濃睡。」



示王居士文柱


「　古鏡生光似雪。自然方良潛蹤。騎牛忽過松嶺。管甚春風淡濃。」



示肖居士二卯


「　豎起一條鐵脊。打開半箇疑團。了然當下成佛。何必重重細看。」



示湯居士國印


「　先佛一枚海印。一印萬象森羅。誰知落在君手。何妨來往娑婆。」



示溫居士泗


「　生死關頭識破。端然成佛多時。坐消昆海明月。踢脫凡情聖知。」



示張居士權


「　參禪本自求悟。斷妄何須猒迷。只把吹毛利劍。聖凡情見排擠。」



示張居士教



「　菩提煩惱如燄。生死涅槃似漚。若是安心選佛。秋高月上芳洲。」



示黃居士宗


「　數日不打口鼓。安閒自在煙村。剛才搖動檀板。笑破諸人面門。」



示王居士道


「　利門名路元假。山鳥谿煙可尋。若問道人樂處。翻身又上幽岑。」



示金居士豁


「　如玉如冰如水。若華若鳥若晴。道人祇此坐臥。佛來祖來嬾迎。」



示楊居士鶴紀


「　少室安心斷臂。兩重公案紛紜。不如言外求之。免墮從前見聞。」



示馮居士文魁


「　道骨千生練就。衣珠一日何明。逢煙如可休去。到底逗遛半程。」



示保居士佛手


「　小往大來名泰。輕清重濁兩間。見聞不住聲色。是謂佛祖之關。」



示楊居士中立


「　密雲密布三千。一雨普沾艸木。喚作如來大慈。亦是指馬為鹿。」



示余居士不一


「　勞勞終日持佛。只是恐防錯認。若謂佛居淨邦。目前安得隨順。」



示羅居士郁


「　鬱鬱黃花非外。青青翠竹本真。總是當人佛性。何須猒棄芳塵。」



示楊居士茂



「　眾生諸佛眼孔。只在喫飯穿衣。直指出來不會。枉然此箇真機。」



示耿居士迥


「　祕魔擎叉奇怪。溈山笑裏戈矛。慈雲不是無語。卻有一條棒頭。」



示俞居士永


「　秋日蘆華華下。釣魚舟上坐歌。分明一箇船子。元是華亭頭陀。」



示熊居士昌


「　殺佛殺祖雲門。奪人奪竟濟北。正好一坑活薶。免得是非兩賊。」



示李居士繼


「　學道當如埽地。只須埽到無塵。了然一片乾淨。莫管誰為假真。」



示劉居士時


「　室中請益禪學。善惡兩關莫取。父母未生面目。分明祇者言語。」



示劉居士齡瑞


「　維摩病中說法。嵩山埽除藥病。拈起拂子便打。少室親行此令。」



示雷居士奭


「　大雄峰頂明月。光含萬象森羅。世人祇是懵董。不見目前奈何。」



示蔣居士嘉


「　金馬碧雞坊下。太華昆水之間。悟得眼前景物。誰為生死兩關。」



示王居士晶


「　趙州一箇繩床。野竹一間艸屋。受用雖則不仝。從教天下相噱。」



示舒居士章



「　鼻祖西來直指。虛虛豁豁性天。孃生面目全在。劫火從他洞然。」



示楊居士智


「　山上鯉魚破浪。海中塵土坌天。指示當人不信。生生錯過目前。」



示孟道人奇


「　奇哉鼻舌身意。三世如來安宅。更問如何是道。趙州有箇庭柏。」



示陳道人瑞


「　為人為仙為佛。祇要吾徒志堅。一點明明會得。便知父母生前。」



示朱道人惺


「　勤行德業為首。道眼處處圓明。信手摸著鼻孔。元來不自孃生。」



示張道人映


「　心佛眾生不二。眾生心佛何殊。會來三無差別。青青翠竹真如。」



示毛居士安


「　地盧撥火談道。了沒一言可陳。若問那邊理事。本來無妄無真。」



示張居士寅


「　菴外石牛穩睡。火中蓮子飄香。西來大意如此。義气全章法堂。」



示沈居士鼇


「　不二門中何物。梅花帳裏真機。堪憐一段消息。落得人前是非。」



示熊居士易


「　不種梧桐免鳳。多栽綠竹引賢。道人雖沒文行。卻有祖師別傳。」



示聞居士聖



「　無泥內外中閒。豈云聞見非塵。雖然不是真智。了得從前之因。」



示張居士實


「　惟此一事真實。本來何有三乘。千頭萬緒歸一。曠劫無明可冰。」



示謝居士有


「　一段西來玅機。許多大士難識。今時縱有些兒。究竟何曾得力。」
