










布水臺集卷第二十
住明州天童寺匡廬黃巖沙門道忞著

贊三


「　年在垂髫早知瞿僊之可學衣方脫白立詢祖道之玄津得非出家行腳總是正因者乎繼而親師盡瘁荷眾忘勞則較諸同壇尊證上中下座更復鐵中錚錚石中而粼粼者矣一朝龍天推出高踞玲瓏彼耽耽虎視之輩謾謂名器可倖而得之動致譙讓而狺狺及乎退遜將來熱炙奉上則禍不旋踵災反逮乎其身而後循環往復物返青氈益驗出處光明去留正大而不挂人之齒脣焉獨是浩歸湖海恨無東山門下濟濟多士之如鳳若麟耳然而食我桑黮懷我清音調師絃于西吳東越之會乘霧昇霞鼎成僊去震雷吼于北齊南楚之濱豈遂謂無法寧克勤諸子盡屬法門之罪人哉」




「　胸中墨漆黑口裏亂胡柴人天遭其揶揄佛祖受伊詼諧本是石敢當回車而避螳螂方怒之臂元非鄧林木獨力欲支萬仞將穨之崖喫他官酒便臥官街如斯丹青玅手邈者薄福形骸是誰冷眼相看只合卷向屋角落頭千秋萬歲塵封而土薶」




「　窅忘管帶廓絕瀟灑立地無卓錐蓋頭無片瓦偶蹋風泉雲壑之間坐茲翠壁丹嵒之下水將落而含秋楓欲黃而蛻夏說甚麼天地一指萬物一馬費韶石之雲翻勞德山之雨打直得萬象稽顙聽指呼森羅合掌供攄寫無示而丕彰要玄不說而深譚般若咄若將恁麼會渠儂負爾何多不將恁麼會渠儂負吾不寡好語無人下得賴有在傍侍者(止水明侍者請)」




「　嬾搖普化鈴猒把龍牙杓養家雖是一般道路剛行別著反張螮蝀以為弓左搓電光以作繳浮天空翠弋雲霄箭筈弦離金鳳落」





「　懷中佳慈樂育只為口觜好傷人大小渾家都不睦峭巍巍處碧嶙峋錦簇簇時華簇簇一語排空來萬人遭氣築超然物外彷古高僧慣作白拈名真賊禿無端撞著上大人乙己丘門下幾箇依仁游藝底陳上舍張學錄又道是苾芻龍門緇衣山谷阿呵呵天外罕人共舉頭潑藍徒見千蠃矗」




「　咄者山農有年無德少吉多凶派出笑嵒不援曹谿正脈燈傳太白常非濟北綱宗黑子玩千邦思浮幢以跨步電光拂賢聖擬空劫以拄笻天孫兮嫌未巧造物兮恨未工欲何工磨瑩大地成秋水染就冰華作絳紅(絳雪瑩侍者請)」




「　生機劈破玉團圞齒濺冰霜六月寒太白山前龍蛻骨燄摩天上鼓驚湍」




「　秋滿鬚髮春盎肺腸呼山入座喝月回光瞥瞋瞥喜乍陰乍陽風來震澤雨過瀟湘」




「　宗門九鼎一絲縣誰拄神州半壁天敢道樗材能著力仔肩強自弼衰年」




「　出語洪爐烹不透摛詞雲漢共昭回千尋意樹空刀斫萬仞我山一默摧戀仰生七眾之心有以也夫敬畏起三吳之士豈徒焉哉」




「　蘇浙人爭圜炤本道躬自揣甚微寒金閶未出吳興怨冤結多生脫信難」




「　指柳罵槐慣得便疾雷忽震二陽天一千七百橫該抹祇許滹沱得一玄是甚玄猛虎舌頭書梵字定華版上打鞦韆」




「　者毛驢最無狀願大力綿脧心高意豪曠波傾活眼滴涓泉鼓作須彌千尺浪當處阬人只活薶那顧死來依火葬如斯滅裂行藏底作宗門師匠將知此地無硃砂固宜赤土也為上」




「　尊性乖乖崖崖寶襟灑灑落落調御諸方龍象不許





「　操活落鉗鎚扇無風爐鞴黃金遇而色增藥汞遭之立敗鼻孔摩觸太虛髑髏常干世界大千無地問生緣劫火洞然渠不壞」




「　眼生三角頭峭五嶽埽蕩列祖徽猷解去人天黏縛果然好肉終不剜瘡任是耆年更加艾灼變甘露作蒺藜之園化醍醐成蠱毒之藥錯不錯誰知正法眼藏被者瞎驢滅卻」




「　驗人不用設三關一句當天百句刪華擘虛空成畔岸桃源從此隔塵寰」




「　熨斗用煎茶黃河合眼跳自刊法印誰問汾陽十智同真一味胡柴那分臨濟先用後炤秋水橫按兮半提可滅寒木在握兮全機可笑煒煒煌煌煥乎其有文章獦獦獠獠兀爾全無孔竅閒名未謝於當世將比春雷之啟蟄而非倫長語相次落人間擬同孤鶴之唳空而或肖微雲澹澹綴晴霄素月娟娟生海嶠(月嶠閒侍者請)」




「　者漢過犯彌天筆難稱述口生白醭不譚如來之禪衲衣空閒嬾奉優波之律有眼別龍蛇無心問祖佛一言相應始開關三問遲疑立趁出晉鋒八搏擬何憑初三十一中九下七尾羽井臣雲角日(唯善常直歲請)」




「　古錐千七供生按那論人間鼬與鼨萬物為銅香海炭洪爐一點雪消融(爐雪慧副寺請)」




「　匡廬薙染太白開堂八陳法會七坐名坊金錯落玉鏗鏘峨嵋山色侵雲直巫峽灘聲入夜長(西蜀天虞衍知客請)」




「　翰苑英姿瑤臺僊骨弢彼有輝光事我空王佛出頭





「　胸有垢藏之壑誰分涇濁渭清家無白澤之圖安問妖差怪別漢陽江上樹鸚鵡洲前月等閒拓出古菱華歷掌清機明倍雪」




「　響靈柯飛神鏃射天狼折若木禿帚橫埽碧落天倒瀉明河珠百斛何似綺霞雲錦簇咄咄咄春山青秋水綠洞心目」




「　囊半肩而肯來歡騰四眾衣一拂而仚舉慘動群情業風未息驛馬常征安得千江月印如此炤之萬有千幅或坐或行色身對現而猒滿乎吳之儂閩之囝楚之傖也」




「　磨拭秋光肝膽徹嶽色芳鮮牙齒齧無蔕空華隨手折靈鍼倒砭青天血古宿何人推往哲周金剛去龍驤絕不見所思心蘊結」




「　十載煙簑海上來把竿曾上任公臺釣絲攪水星河動穿盡魚龍腦後腮而今老大興闌珊宿蘆灘一天風月自盤桓」




「　金梯驟玉馬井底種林檎戶外蠃峰秀階前洞壑深寥寥天地誰同調坐倚孤笻自嘯吟」




「　傲倪萬物糠秕群倫悲山川之寡德薄天地之未仁洗劍尚憐青海水看華尤惜錦江春是句亦鏟非句亦鏟今時盡卻空劫俱泯不與諸緣作對豈與萬法為鄰哉」




「　夜眠一宿日食三餐逆之則怒順之則歡識甚龍師火帝人皇鳥官敢道冰華熱曜靈寒半擲鼇鉤香水海倒翻筋斗萬尋竿」




「　生平無定力長隨業風播適脂穆陵車旋買勾章舸成甚麼亂剪齊州數十峰小摘海雲三萬朵」




「　欲歸匡阜仍環轍待隱中山隱未成賣狗縣羊知甚





「　賦性天真曠劫前不彰文彩況雕鐫棱棱玉骨橫嵩壁皎皎冰心飲楚天雅道有時行上國清風終是播林泉山眉誰把松煙埽繪事知難碧落詮(水墨)」




「　無心落䰂事空王何意天書下玉堂不貴九重稱本色那驚時輩口雌黃明河手抉從來事濁世風揚此日糠清淨光明衣等著從他濃抹任他裝(水墨)」




「　帶雪絲絲餘鶴髮逐雲片片碎田衣饒他佛眼尤窺半割水吹光任贊誹(半身)」




「　仲尼稱勇過調達遜頑兇驀地翻雲覆雨倏爾霽月光風身罹法網刑司殘曾不以之為纇道契佛心天子聖靡豪髮而加崇入虎穴處魔宮浮幢剎視菴羅果眼底何曾有大雄」




「　如雷性發迸青天一霎雲收片月圓冥行世多難耐見又因唾罵動瞋緣所以願天常生好人降聖產賢願人常行好事無黨無偏存誠履道裕後光前名全節全福全慧全大家齊樂太平年」




「　功成名遂合知機身退何勞遠采薇黃鶴倚笻閒就養風規且學老天衣(皙長老請藏佛日寺)」




「　舉手攀南斗回身倚北辰世人空悵望妒殺幾生瞋休生瞋松長棘曲誰爭得君自瀟湘我自秦(瑤實[王*(甬-用+(圉-土))]知客請)」




「　單明向上一事間習文身句身誰云見超佛祖況乃學富天人使並駕于前哲兮或竭蹙兮余臻際時流之憊瘵兮僅得路而先遵」




「　袈裟不展罩千峰似畫屏開列鑑中未到平原山盡處焉知渠別有家風」




「　打鳳羅龍藝自高鷽鳩誰羨弋蓬蒿恢張天網如空闊不是鯤鵬任脫逃」





「　燈傳太白席繼天童正令行乎吳越化雨沛及寰中踞皇宮而布薩陞帝座以弘宗王侯欽道德天子寫儀容蘭斯茂桂斯穠盡謂庸庸多厚福誰知後福更溶溶咦天藏發付名坊地秀水佳山遶萬重(具準發書記請)」




「　是即打殺老僧不是則燒卻真推出秦時方𨍏轢跛師辣手也難申留取著元禪和不道千年故紙好合藥吾宗到彼必定大行支那何故天之未喪斯文也匡人其如予何」




「　蒲團高閣黃金闕拄杖橫拖碧玉臺兜率何須升夢裏天宮親轉法輪來」




「　願為法王之輔戰退天魔願作中流之砥力障頹波願復崦嵫之佛日願揮返舍之魯戈願大濟北之流而漢接願扶雪庭之桂以雲摩惜乎髮如此其種種鬢如此其番番也閒名未謝徒動玉珂咦華雨晝飄龍座暖袈裟沐得御香多(正句遠維那請)」




「　歌殘玉樹又春葵仁義之中道豈違法付靈山恩四一珍珠又撒紫羅帷褦襶子謾相嗤一踢踢翻鸚鵡岸一拳拳倒老須彌畫堂燭燼東君睡你是何人我是誰(杖林機侍者請)」




「　鳥王金翅眼橫空蠃蚌何曾在意中一日七龍猶未猒回翔不住妙高峰咄是誰命盡遭災者阿呵呵一佛陀二達磨三元和其餘是甚碗躂丘休休不風流處也風流(曹谿宗長老請)」




「　切齒諸方肆罵訶央庠肯育混禪和孤芳素薄西山亮壯志長懷大集魔淚灑君王留不住苦憐獄戶唁相過為人劫劫忙終日罕捻輪珠靜剎那如此行持稱知識要求多陀阿伽度阿羅訶三藐三佛陀驢年馬月不為多」




「　月當斗杓之會身居摩蝎之宮與蘇韓而其命實毀





「　調御釋迦文人天中最上余忝伊後昆況復為宗匠家法祖靈山從來越格量羅雲等飲光眎孰偏而黨猶子尤夫人諸方擎置掌恩多反折齡福盡歸泉壤本闡寫余真請題留供養我真我贊題難度難摹倣憶初志學年生死結情想為道出塵樊一朝裂愛綱孑身天外遊一笠一瓢杖煙水歷百城爾乃身命放己靈曾不珍諸佛且回向七處九會場鞭笞盡龍象誰能為子寬垂手低窮相咦天童與麼告報莫是畏蹈諸方覆轍恐折汝齡消汝福添汝災生汝障麼要且不是者般道理然則畢竟如何不見道師絃奏而群音絕師子哮吼時而祖父母皆喪哉(師奏闡小師請)」




「　聖地靡加一簣功心塵不埽宇元空萬緣況落秋梧後山自西來水自東」




「　醜形寫出巧丹青口款那堪自著明罪惡己彰天下[番*支]決將東海洗難清者裏還有為渠滅跡消聲者麼捩轉南辰歸北陸天東七曜向西行」




「　生緣南國家在冀州佛性全無作佛是求咄者獦獠根性太利只合貶向獵人隊裏十六秋了十六秋教他長食肉邊菜終身出不得馬額并驢頭於是為之著贊曰除天蓋覆一人留驀地興戎口作尤白首長途空浪跡滿懷文翰待誰酬所以有般措大也恁稱道不肯低頭拾卿相又能落筆生雲煙據實而論要且渠儂不是者等流(嶺南興慈運侍者請)」




「　胸中富有山島腹內蕩無介鱗然往往法麟經懲逆惡嚴斧鉞剔莽榛口誅兼筆討三令更五申不辭好





「　乏煒煒煌煌之度無雍雍肅肅之容忝在法門耆宿謬承帝者師宗五鳳樓前聽殘玉漏須彌頂上擊斷金鐘老而不死闔國憂忪占盡江山尤自可兒孫遍地類頑兇(象輪覺侍者請)」




「　有藍田射石之能不住虎蛇之窟無五熱炙身之火直燒魔梵之宮志在驅邪以反正寧靳罵雨而訶風至若來慰雲霓之切望去當帝眷之方隆即不竣絕人而逃世亦未嘗淹三年而閏終豈與夫前固後通者並駕而偕𨍈彼六師波旬之輩又焉得藉為口實而大肆其諑訌哉」




「　波波劫劫浪走風塵有時石田布種或時碧嶂垂綸款乃一聲河滿子桃華落盡武陵春」




「　睦州版象骨枷常擔荷受波查累得通身皮肉盡依前蹋雪在天涯迢遞鄉關何日復長途豈為愁征逐大千收卷一伽黎帶雨兼雲和霧宿」




「　大統綱宗推濟北兒孫輩輩是英雄杜蠻無過遮傖父繼祖偏渠有父風物外獨騎千里象萬年松下擊金鐘真叵耐實難容」




「　佩覺王印為法壇將敕點飛龍奔波香象大海揚塵須彌鼓浪萬里雲潛青天喫棒咄你道是誰山翁無狀」




「　菩薩心肝夜叉頭首見義敢于勇為當仁那問誰某舉青龍以住能仁大剎固愛其道眼之精明也使慧山而紹養拙戶門直愍彼宗傳之乏後耳依約白雲幾翻蒼狗卻有道渠借華獻佛陀掠美市恩厚者矣





「　浪得虛名喧宇宙實無少法可流傳源憑甚續滹沱脈無位真人面目全」




「　執侍武原垂五稔巾缾太白更多秋百華雨足紅初放萬竹雲深翠欲流美著斐章信有由(斐然章禪人請)」




「　彊把瓊華石上栽看看碓觜碧蓮開大風吹到千年後遮莫仰山復再來(再來滿禪人請)」

「　直如瓠子彎彎曲似冬瓜儱侗諸佛未萌芽靈苗己[番*支]種每懷地藏利生之心不學長汀乾而打鬨一任簡禪持歸九子山中酌龍乳泉烹閩源茶燒楓香熟香餅香作楊岐第二十四代祖師之供(九華佛萌簡禪人請)」




「　毛吞大海鋸解稱鎚行塵㶿㶿玉屑霏霏剎那觀化電拂洞微城空芥子石盡僊衣」




「　隼視鷹瞻腦後腮太平怕見喜殃災如何也廁嵩山後親見密雲老漢來」




「　天童再住潦倒疏頑黃昏打睡日裏看山禪流有問西來意笑指白雲去又還」
