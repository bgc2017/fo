「　日月無私照」


【(懸象著明莫大乎日月運轉不息照臨無私以濟萬
物也)。】

「　人為似等閑」


【(日居月諸一寒一暑而常人不悟至理貪愛日
前如等閑尓)。】

「　忙催紅葉落」


【(發生之𠋫盛衰之理盖時使其然若
忙催之殞落矣)。】

「　靜想白雲間」


【(白雲無心悠然出岫靜思其慮堪養
聖胎)。】

「　善念皆知有」


【(念依心起情因意生儻盡去𢙣緣皆修善法道
豈遠哉)。】

「　非緣勿用攀」


【(理非合道事更違真緣習靡忘浮淺日益
也)。】

「　目前深邃理」


【(大道者不可得而親不可得而踈盖玄同和眾
常在目前儜慕學之流窮深曉邃矣)。】

「　郍是扣玄関」


【(扣寂寞之一造
希夷之域擇其流品寔難其人)。】


「　精心求一善」


【(精思鍊志到虗無心也一善者即得一冲妙之理也)。】


「　一善許相宗」


【(此一善者萬法妙用大開玄關從人之欲任物宗師
矣)。】

「　利物無窮盡」


【(善道之宗廣同覆載不唯一身獨善兼亦利物
無窮矣)。】

「　分明理道通」


【(宗善悟道益巳兼人道理顯然猶軒既之
辨妍醜矣)。】

「　正邪名恍惚」


【(對正名邪邪既不存正名何有是謂唯恍
唯惚妙用難名者也)。】

「　曉悟見真空」


【(曉其本元悟彼妙道以無見之
見悟真空之空也)。】

「　象外行持法」


【(達道高士土木形骸所行之行
所持之法無狀之象混合希夷謂之象外矣)。】

「　玄言顯聖功」


【(聖人
之功終日作而不作行而不行以玄言言之顯而導之者也)。】


「　百法還丹數」


【(神仙至人作事有法約言百數其䡄甚箋矣)。】

【(同體異名門門契道皆遵正本各達深淵)。】

【(真空有象象即虛無類杲日之無私同清光之燭物)。】

「　解脫本由心」


【(解諸縛著脫除凢質見真空境得不由心矣)。】

「　迷惑終無應」



【(先窮正道方履玄途豈冝攻彼異端迷於妙理者)。】

「　精修重古今」



【(倣先賢之繩凖遵今日之楷模必在精純湊乎靈域者矣)。】

「　循環豈蹔息」


【(子母㸦為於根本陰陽交致於生成功類循環輪轉無息也)。】

「　七聚寶高岑」


【(天地爐開神丹功事貴同七寶慱濟何窮矣)。】


「　剛氣縱撗合」


【(純剛則陽私邪不雜混然一氣契合縱撗矣)。】

「　無人到所思」


【(玄德深遠至理自然未見忖度所思而能扣寂)。】

「　浮生看即易」


【(滔滔塵世汩沒往來未了幽玄見如於易也)。】

「　至理杳難知」



【(冲玄至理眇邈無形安遣凢人而能轍扣矣)。】

「　返覆因興念」


【(念茲
在茲返覆攸致原其終始咸由憶念為因矣)。】

「　恒持入遠期」


【(夫士
之修道耿介其志恒持其心但慕進修遠期入聖矣)。】

「　孤雲霄漢外」



【(得理優游超九脫屣自地外漢逈若孤雲者哉)。】

「　花見晚霜悲」


【(霜華
萬物卉木其萎未悟於道雖榮而終喪也)。】


「　我說逍遙理」


【(聖人於物無滯觸類俱通廣演逍遙垂乎訓詰)。】

「　其如要妙深」


【(放曠之旨其道幽深妙達玄關造次何測)。】

「　悟來時頃刻」


【(曉達之者不假多求道在目前悟之頃刻)。】

「　迷者豈通尋」


【(失之毫釐謬以千里既迷於旨趣豈勞於研尋者哉)。】

「　達即隨情遣」


【(通達大道體會真淳即事隨情任緣而遣)。】

「　真緣不可侵」



【(真一之妙杳兮𡨋善萬象不能混其形六情不可縶其意者也)。】

「　從來皆是



【(森然景象邈尓沉思事事皆真從始無變矣)。】

「　巧拙在人心」


【(和粹
冲淡本無短長盖人心有於智愚致事理分於巧拙尓)。】


「　一法歸千法」


【(玄元妙旨博緫群經以一氣貫三才以一法歸眾法
故云聖人抱一為天下式)。】

「　真如即是真」


【(於諸法中體了真性行無
相行言無言教乃是真也)。】

「　迷情終不見」


【(迷情滯教不見真常達士
契真不害於教老子云我幅異於人)。】

「　達了悟知因」


【(悟道之人心照
清淨滌除玄賢能無疵乎)。】

「　海納分他境」


【(眾水朝宗群流藂萃雖分
異𣲖同納一源亦猶統道真乘萬化俱服也)。】

「　心平故可親」


【(了乎虛靜
之宗專務為治之本以万物利為利以百姓心為心得不平也)。】

「　但明修鍊路」


【(欲履大道精行為先必體真如群動咸歸也)。】

「　善業盡隨身」



【(谷神不死響應無方為善從之其理明矣)。】


「　真一生天地」


【(夫道出冲和妙氣含養萬物配陰陽為一二之基故
列子云一變為七七變為九九戄變為一一者形變之始也)。】

「　因依認宿緣」


【(天地孕靈冲氣調和因依生發皆契宿緣故陰符經云天至私用至公者矣)。】


「　智高通理性」


【(其或得意忘言智通心照悟空空之理析色色之談
玄解自深高仾殊隔)。】

「　凢下少周旋」


【(中士巳降溺於塵滓雖聞妙道
不能周旋也)。】

「　語默何奇絕」


【(夫不為可欲所乱令心境俱靜一無
所有則心與道合聖理奇絕者矣)。】

「　幽深隱聖賢」


【(智與聖冥識與
神會探𧷤索隱與賢聖而同轍矣)。】

「　誰人解我意」


【(逍遙懸解奧妙
冲虛內景湛然希夷自得矣)。】

「　大道寸心玄」


【(無名之道眾妙之門
理括恒沙達乎方寸)。】



「　天上與人閒」


【(清淨者生飛天嗜慾者生乎人自作所召非天命之有二
也)。】

「　賢愚不暫閑」


【(人之所以不遑暫寧者皆由不達於道也愚則
可尓賢何然哉斯乃世俗之所賢矣矜乎巧智背乎淳德故老子云不尚賢使民不
為爭)。】

「　精修歸至理」


【(精研道法胎養神氣造次於是絲毫不差則自
然歸於至理焉)。】

「　道大隱玄関」


【(域中有四大而道居其首焉迷之者
不能虛心而玄覽則隱於玄関矣)。】

「　聖境皆奇絕」


【(外則紫府丹臺
內則三田九竅道無不在智無不通是以聖人觀之皆為奇絕之境)。】

「　凢情豈易攀」


【(凢俗之情闕途成滯陟斯真境不亦難哉)。】

「　群仙寬坦蕩」


【(洞府神仙皆通至理寬而無外坦而無私蕩蕩焉不可得而名也)。】

「　却老盡童顏」


【(玄德內充神丹外助足以却老返少長生久視矣)。】


「　探玄何妙理」


【(太玄幽鍵妙一秘扄必欲探之以何至妙之理而可
及也此盖嘆道之深矣)。】

「　元化得均勻」


【(窮元化之源得亭毒之妙
生成品暈要在均勻也)。】

「　去住分明紀」


【(既得元化之妙顏同修養
之方故動靜思之分朋紀於深裏務盡勤行也)。】

「　終期遠大親」


【(遠者大者
唯聖人能知能行故終事躬親而為之所以作程於萬世也)。】

「　物華堪賞詠」


【(妙盡物事咸歸至理亦足以供於賞詠也)。】

「　離撓㝡名珍」


【(一真內照諸境外寂動無所著舉必皆通故無撓矣若此㝡名珍也)。】

「　碧落飜霞彩」


【(驂鸞馭鶴上昇碧落者必高謝塵蒙輕飜霞彩也)。】

「　千秋不老春」


【(歷千秋而不老斯可謂之長春矣神仙之壽皆若此焉)。】


「　誰人信我說」


【(希言道舉必忘筌旨趣常庸欽奉者寡老子云信言不
美)。】

「　我說亦非常」


【(契理誠言出乎與論窮淵至極實曰非常)。】

「　三才妙更深」


【(其高惟天其厚惟地其靈惟人同得於一

「　愚迷皆自縛」

「　解者不為非」


【(其言曲而中其

「　深信含玄理」


【(有為事中

「　幽通是可依」


【(幽深通達妙契希夷惟道可專誠堪依託)。】

「　達人皆論道」


【(道者一儀之母萬象之主其大也無外其小也無內悟達之士勤行無倦

亦論識其精微也)。】

「　狂酒敗稜威」


【(酒以成禮不至於淫也)。】

「　浩渺寰瀛內」


【(寰瀛之內包括八紘况乎滄溟浩渺無際)。】

「　寒暄隱聖機」


【(法天而行道者生成之仁則如春之暄號令之威則如秋之寒斯之妙用
皆伏藏於聖人之機也)。】


「　逍遙名不易」


【(夫物之生也稟靈受氣雖或萬殊至於品物大小各
當其分逍遙之理一也故逍遙之名曾無易矣)。】

「　一法貫真宗」


【(但以逍遙
之一法而貫之於真宗也)。】

「　入位誰分別」


【(始於長生之術也或分知

「　深玄隱大功」


【(既盡深玄之理唯達者能知光而不曜所以隱彼大
功也)。】

「　杳冥難測度」


【(杳杳冥冥中含至精誠非下士而能測之故云
難矣)。】

「　象外理皆同」


【(必能妙達玄中深窮象外則彼此之理無異也)。】

「　洞府群仙侶」


【(洞府群仙皆稟靈符授金簡羽駕電裳有道之士也所以嘯儔
命侶矣)。】

「　三天順九重」


【(三清之境謂之三天也且天有九重三天順而
次之寔無上清淨之靈居盖真仙之所御也)。】


「　靈無知者少」


【(虛極妙理希言自然達本知微世甚寡矣)。】

「　秘默在丹田」


【(精鍊內丹導引正氣凝注神府以潤丹田秘而默知信有神也)。】

「　鈆化金根樹」


【(夫採四神氣鍊五行精更為主客遞相含養遽令鈆化成金如
猶指鼎葉茂分華揚芳無際尓)。】

「　鼎光玉液泉」


【(鼎像於宅永為其主
名之玉液若泉不竭光彩藝物流潤其中)。】

「　相宗難且易」


【(難而後易必有
成功入宗道不離依制有準可不易哉)。】

「　絕妙理深玄」


【(陰陽不測靈變
罕窮非凢所知故曰深妙也)。】

「　真境猶來是」


【(明陰洞陽道成並既居惟

真境身超上仙皆自三丹感茲萬化尓)。】

「　依經斆昔賢」


【(依法丹經
効諸往哲順五金性明八石體修之靡忒何道不諧者哉)。】


「　得一便湏𠇲」


【(天清地寧神靈谷盈皆資妙用而得一者也學道之
士得真一之道者可以休息矣)。】

「　真空象外求」


【(象外無物真空強名
無求而求真空自證如求者也)。】

「　昭然寬世界」


【(道既融明心虛浩蕩
蓬岫崐丘之遠丹臺紫府之遙俯地拾芥耳)。】

「　幻化似浮漚」


【(既達清虛
之域却觀幻化之身不異浮漚倏無忽有矣)。】

「　秘隱玄中理」


【(玄中之理
眾妙之門恐執玄為滯寄又玄以遣玄秘之隱之者也)。】

「　幽深事莫投」



【(希夷之淵乃幽深之事忘視忘𦗟莫可投之矣)。】

「　天涯無畔岸」


【(天無涯際
六合至大巨海至廣道皆居之何所拘礙乎)。】

「　寶月順波流」


【(月在晴空
影沉波內道隱象外妙合身中寶而貴之隨流而已)。】
