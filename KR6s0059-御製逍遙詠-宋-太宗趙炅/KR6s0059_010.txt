










御製逍遙詠卷第十(七言)　輕


「　道妙懷真紀異常」


【(至道體虛用虛含妙鴻纖不弃綱紀恒存
故曰異常)。】

「　人間天上若存亡」


【(至體周通至性無礙存而下有亡所
不無道經云無狀之狀無物之象是謂惚恍)。】

「　潛機物類多中覷」


【(潛運玄機何往弗濟應乎萬化生成密觀)。】

「　納取虛無定久長」


【(體道治物功成不矜自然與天地同休能長且久也)。】

「　碧落遨遊歸洞府」


【(大丹既服仙界如歸恣羽駕之盤遊見玄功之不忒)。】

「　黃金不買自馨香」


【(河車正氣金砂妙丹本自天成豈勞力致馨香襲景
善不可加)。】

「　叅同契合諸經說」


【(達人著述妙不師心契往聖之
玄言作今時之洪範)。】

「　精魄𡙸來爛紫光」


【(得真一之氣精魄我
還類脫五行超絕五色乃強名乎紫光也)。】


「　無為道大入幽玄」


【(老子云域中有四大道大天大地大王亦大是
知無為妙道廣大無比盡入幽玄之理也)。】

「　幾許風光慕昔賢」



【(惟昔聖賢也儲精養神其門不一若黃石公之遐壽赤松子之長生風光靄然論情
幾許實堪傾慕也)。】

「　對境盡同閑見識」


【(體道之士動契真空
對境忘懷觸途無礙故盡同閑見識也)。】

「　忙中皆是急相煎」



【(冲虛一失萬慮忙然動必區區疾猶烈焰故不可也)。】

「　配匹直湏寬雅淡」


【(日精月華之為妙也配而匹之務盡玄微方明至益必湏寬縛于懷雅淡
乎性也)。】

「　會有逢時不偶然」


【(盡九轉之法度成五色之靈丹
會當逢遇之時故不偶然也)。】

「　修鍊但教堅志氣」


【(夫壽延金

石跡御風雲振物外之清摽顯域中之高蹈豈不彌堅志氣終契自然也)。】

「　叅詳穩審更周旋」


【(叅而同之詳而用之尤冝穩約審細則更甚周旋乃
長養之道盡矣)。】


「　含胎長養盡籠羅」


【(世日一陽迨乎秋望在胎十月含育
英華凡云長養之資盡在籠羅之內)。】

「　豈離玄談道理麼」


【(聖人
談其玄門後學募其師範雖云變萬化然豈離於妙道耶)。】

「　狐兔幾曾能乳馬」


【(鷰雀不生於鳳皇狐兔不乳於良馬得非氣貴純一小不育大
者乎)。】

「　麥田斷定不生禾」


【(蓬麻性異禾麥根珠且夫道尚真淳
日經誡雜是知一氣不可相乱也)。】

「　目前便是真佳境」


【(於北
守氣道豈遠乎丹田或合於冲虛目前便為於真境)。】

「　汞內消停自合和」


【(以四神之靈和二氣之妙汞內之消停得所玄中之寥廓自通裏外體和
長生可保)。】

「　認取門庭深見解」


【(開二八之門隨龍虎之𠋫依之
修鍊便契杳𡨋體認不虛是深見解矣)。】

「　紅砂奇異湧恒河」



【(十臺告曉金鼎成功赫弈紅砂顯彰奇異湧變無盡如崑丘之注恒水焉)。】


「　人心多是少相投」


【(人心不同猶若其面所以水火不可相濟
善惡不可同塗相投之心實為鮮矣)。】

「　非識塵中上品流」


【(大隱
居䣑真人由是混迹塵世或顯矣晦中下之流故不識辯也)。】

「　不見真宗歸物理」


【(不曉清簡無為之旨混合自然之玄以此冲虛同臻物理也)。】


「　難知妙道後先休」


【(非常道之玄豈名言而可及是以聖人後
其身而身先故難知也)。】

「　生前內鼎冝烹鍊」


【(人生瞬息冝以
修身養道精思鍊志吐濁納清矣)。】

「　殊勝華池好漸修」


【(神水

華池功玄理奧深知法妙好漸修之矣)。】

「　但見完全憑土德」


【(知之
修鍊仗五行以皃全故土有五德終始用之不可暫捨也)。】

「　能調火𠋫藥皆秋」


【(志士得日時昇降火𠋫進退秋後寸草秀實所以金丹此時功
畢矣)。】


「　玄之玄又見真空」


【(當欲而無欲謂之玄以玄遣執故云又玄
萬法斯了見真空矣)。】

「　巧拙臨時要妙中」


【(智者常守清淨愚者
迷於道源如能解心釋神返照正性乃要妙之本)。】

「　大智伏藏知遠近」


【(大智若愚故伏藏也洞達古今故知遠近)。】

「　纖微盡達了然通」


【(洞精鑑微無幽不燭達其道本了然而通)。】

「　周旋物性誰區別」


【(物任其往各得所冝大道區分故周旋也)。】

「　坥蕩由來是且同」


【(姒道虛極坦然寬大其來懸遠混而且同)。】

「　境致三才隨有位」


【(天能覆之地能載之人民定之且萬物苞含和柔守本隨
有位也)。】

「　神光照室瑩西東」


【(靈源妙性謂之神光洞曉內真
故云照室融輝瑩徹自西自東)。】


「　還丹未就且降魔」


【(夫欲勤𨦣湛陰勵操於粉面者猶慮藥中
狼虎貴且降魔矣)。】

「　二八從教細攪和」


【(鈆基汞子陰稟陽胎來從
二八之門起自一歿生處如膠與漆真契攪和故󵠪子云唯之與阿相去幾何)。】

「　离火催駈凡入聖」


【(士欲酣飲雲槳永服沆者必從凡而漸聖絳砂而
离火矣)。】

「　坎鈆功力甚僂儸」


【(金花凝液抱一含真功與造化爭權
妙在僂儸者矣)。】

「　紅塵自化真堪貴」


【(神室內秘凝永將成明𤗉者走
於紅塵烚下全𠬧於碧微)。】

「　白雪增盈不用多」


【(白雪功成增盈得理

妙符神用雅契玄津者矣)。】

「　九轉龍吟成虎嘯」


【(青龍若吟白虎必嘯
故易云雲從龍風從虎聖人作而萬物覩)。】

「　人非人是狀巍峨」


【(夫首
象蓬山旁分垣闕既顯巍峨之狀復彰尊貴之名既絕是非共生珍仰也)。】


「　難求的當問愚癡」


【(欲叩玄関湏親上智求諸孤陋惟益其疑
如問道於瞽者爾)。】

「　𨚗箇堪言好事知」


【(學者雖眾尤多泣歧
孰知其微堪可稱譽)。】

「　名利猛貪無有足」


【(名湏副實利要無苟
若不度德知命貪猥無厭是不知止足也)。】

「　多端動止沒䂓儀」



【(千途萬轍馳騁鋒鋩動靜咸非寧符教義作偽日拙斯之謂歟)。】

「　浮生不實將何用」


【(儒稱務本道貴去華巧偽日新終身何益玉虱無當何所
用哉)。】

「　欲把非常亂作為」


【(不恒其德或承之羞專用非常實為
紊亂)。】

「　謾語饒教能配類」


【(欺天罔俗言不由衷畫空成有轉白
為黑巧生品配其鋒蒠當)。】

「　一朝患害殢他誰」


【(罪既貫盈必速
其禍自貽伊戚不可尤人)。】


「　千般萬類不相逢」


【(道非聲色故寂寥虛靜不可以有象求縱
機巧千途奇詭萬狀豈可得而逢也)。】

「　天地尊卑豈易窮」


【(天尊
地卑𠃵坤定矣及賾其至微窮其幽秘其惟聖人乎)。】

「　海岳高深隨日轉」


【(日月不停由如磨蟻是使石爛南山塵飛北海高深之固尚尓危脆之物
豈得長存者哉)。】

「　神仙聚樂隱塵中」


【(世代之人為五色五音之
所惑乱故神沮氣濁不能輕舉適意令真仙之流窒嗜欲門處清淨域雖隱塵中亦
常聚樂盖心無可欲之所污染也)。】

「　鮮求名利知貪少」


【(老子
云名與身孰親身與貨孰多此盖昔人欲令去功名擲珠玉而全真養和今貪求既

鮮悔吝不生可幾於道也)。】

「　多事閑忙惣是空」


【(為學日益為道
日損便能增見聞悟道體令翻習不稽之事務非理之忙無實可憑緫成空也)。】

「　幾箇心賢平坦蕩」


【(內省不疚先執厥中心既賢明情惟坦蕩久而
不渝者幾人于斯)。】

「　十洲三島自從容」


【(海湧十洲鼇負三島
顏不耗於日月壽不促於寒暑今既塞九竅之邪戮三尸之鬼從容仙府何所阻焉)。】


「　含元得一本真宗」


【(含和元氣得一冲虛本妙道之祖宗皆𡨋
漢之真要)。】

「　悞者平論惣是空」


【(昧乎大理悞致論量詮萬類以全空
述冲元之非有背矣哉廢乎表裏也)。】

「　萬象都來皆俻[婁*殳]」



【(廣而不可量微而不可察混漠無際皆萬象之端俻在其閒矣)。】

「　千言譬諭了然通」


【(御物指真俻乎上士而中下者形乎諭况舉設言端方能
通解也)。】

「　才高俊彥優堪賞」


【(才高至理氣俊怡神寄塵境以
優游踐仙鄉而賞翫矣)。】

「　竊祿偷安有異同」


【(尸祿素飡氶顏
順旨噂沓背憎反道敗德與夫夔龍輔政房杜佐君一何異哉)。】

「　唯願他心如似我」


【(聖人無心盖從民所欲務彼清淨思復淳和使天下一同皆為善也)。】


「　不教䧟惡罪身中」


【(不䧟物於不義但教人以知方罪漸滌於
身中跡或升於象外矣)。】


「　𠃵関萬象審消停」


【(夫修金藥六丹神符凝液必先體高天之
物象審兩曜之盈虛消息變通凖繩造化)。】

「　子母相傳日月精」



【(積陽之魄為母積陰之魄為子母子相成成乎大藥)。】

「　產藥要教千日化」


【(法天修鍊妙極至玄金鼎三年藥成千日)。】

「　玄門全藉五行清」


【(大哉玄門神仙至教仗五行之精潔立修進之功夫)。】

「　靈胎脫



【(始託黃牙為母終成凝液之形故云脫落五行身登上品)。】


「　鈆汞梯媒不是名」


【(以鈆以汞為梯為媒少遵修鍊之功未是
至神之号)。】

「　金鼎就來從變質」


【(積功累德藥變鼎中非五色之
可求顯二儀之力大)。】

「　陽和陰氣兩俱生」


【(陽氣純和陰氣純靜
二氣交感大藥成矣)。】


「　九重天上帝仙居」


【(楚辤云圓則九重居天圓而九重也彼天
之上天帝列仙以居之盖得自然太真之道惣三清而無外凝一氣以渾同也)。】

「　愚昧求真道理踈」


【(愚茲妙境昧彼玄津繫風捕影之流却粒茹芝
之士求真失趣與道轉踈)。】

「　混沌初分長日月」


【(河圖括地象云
易有太極生兩儀兩儀未分其氣混沌
混沌既分伏者為天清也偃者為地濁也
在天成象故有日月之道行焉盖與入地而同𠇲也)。】

「　精魂聚散記丹書」


【(龍虎相對魂魄相求或聚或散具載丹經)。】

「　离男坎女皆相類」


【(二位相生配合日月氣類交感萬物化成)。】

「　壬子丙丁異境殊」


【(自子及午為陽位也一陽既生二元合體土王四序羅列終始
且火能生土寄治丙丁故云異境也)。】

「　歷刧因緣隨世在」


【(靈骨
挺生袖襟獨秀因從浩刧緣熟于今雖世世恒存也)。】

「　陰功及物㝡玄殊」


【(玄德動天陰功及物既優既遠且玄且殊)。】


「　神水華池妙更精」


【(神水華池者九轉之第五轉也有若息遊
慮於神水澄浩氣於華池道在茲乎精而復妙)。】

「　非鈆非汞妄于名」


【(真丹之體也不待神靈而生不待陰陽而形不待日月而明不待舟車而
行其道自然非聖不達若於鈆汞而求者是謂妄干其名焉)。】

「　河車本



【(千載紫河車者九轉之第七轉也斯乃功將漸就體變河
車未階化羽之期先示延齡之力)。】

「　戊子從教配類成」


【(戊為
土也土從火生子為水也水居陰位一佐一助一伏一飛配類五行成于至藥)。】

「　朱雀坎虵安在後」


【(南离為朱雀北坎為騰虵安置丹爐然後為得)。】


「　青龍白虎免交爭」


【(東之陽春青龍在矣西之肅殺白虎居焉
欲免咬侵冝各處其方位也)。】

「　若使真金全二八」


【(龍虎金丹之九
轉也自夘至酉十月之功若能體彼神玄全乎二八之𠋫者無不成金矣)。】

「　何憂住世不長生」


【(妙道有成神丹克就去則乘風御氣生則久視長生何往
弗藏無所憂矣)。】


「　還知受氣稟天然」


【(道與氣凝故道氣混然是以氣住神住神
住形存乃天然也)。】

「　百法方為事理全」


【(金丹之術百數其要
在神水華池配合混融方全事理)。】

「　假合都來歸至道」


【(天地
動用之玄陰陽運轉之妙五星四象惣合虛無也)。】

「　修持遠大更通玄」


【(爰從太易直至坎离二卦其間從無生有從有生無遞相含養乃更
通玄妙矣)。】

「　二儀交感成祥瑞」


【(日盈月滿㸦為相生陰陽數就
方成丹矣)。】

「　八卦剛柔被俗牽」


【(聖人託易象立卦𠋫發智者而
悟真英豈彼浮俗而能消息焉)。】

「　何謂此中生恍惚」


【(志修大藥
動契天機何在狐疑強生恍惚也)。】

「　直湏欵曲細精研」


【(靜而
思之精而鍊之直湏通幽洞微必證徹視證聽者也)。】


「　玄門至道故無涯」


【(深妙之用生乎萬物謂之玄門虛極妙性應用
無方目之至道應機設教故無涯也)。】

「　自是人心所見邪」


【(道本凝然

志當歸一學者迷真動而有變故所見邪也)。】

「　智識未能通大藥」


【(金丹大藥其旨深玄或智識淺微豈精通深妙也)。】

「　豈將容易擬榮家」


【(一失玄津動皆是妄既無所得何可榮家乎)。】

「　千年聖境何虛謬」


【(洞天聖境道在長生或住壽於千年或延齡於萬世
事豈虛設想無謬焉)。】

「　百刧曾修不用遮」


【(夫達其道遇其真或
百刧以修心或千生而鍊行福緣善慶形直影端矣)。】

「　龍虎欲成堪鍊鼎」


【(陰陽交感龍虎功成妙契真玄乃同鍊鼎矣)。】

「　勿為利路亂交加」


【(夫欲滌除玄覽象外求真鶴駕驂鸞龍軒濟物或布利而乱心
成交加矣)。】


「　論功湏是一陽初」


【(夫造萬物之基湏向一陽之始鴻漸於陸九層
之初矣)。】

「　二八調和作楷模」


【(結氣朱英金花凝液初二八之交配
畢子母以相生)。】

「　固濟但牢封密閉」


【(八卦為爐四神護衛猶荊
山之蘊玉若岱嶽以泥金)。】

「　莫教中道有差殊」


【(德冠五行爰從
真一青乃東方為首黃仍中道稱尊甚勿差殊極湏秘密)。】

「　依時消息專心記」


【(將欲神棲於靈寶消息於洞真者必精專為功節𠋫為念也)。】

「　在意停騰運火徐」


【(丹臺有名神室用意既徐徐於火𠋫恒惕惕於
爐前矣)。】

「　日月無虧眀節𠋫」


【(從一陽生造十月滿積功累德
自下升高將㱕沐浴之時可近臨冠之日)。】

「　從容願所不躊躇」



【(滌除玄覽情抱冲虛果夙願心何躊躇之有也)。】


「　還丹百數細湏尋」


【(黃牙白雪鶴頂月華如是異名[婁*殳]盈於百
考其功用妙在華也蹟奧探微理冝周細)。】

「　以驗從來道理深」




【(大丹真訣展轉幽玄若不研窮豈知深奧)。】

「　火𠋫混成如白屑」


【(离宮
有凖實鼎無差陰抱於陽混然成一如王之屑皓影無倫)。】

「　水中凝結是黃金」


【(陰中有陽混同一氣鍊之得妙結以成金陰陽相生自然之理)。】


「　循環復始周旋鼎」


【(鼎之設也法彼二儀通乎四序終而復始
循環不停)。】

「　造化駈馳要用心」


【(修養之功齊於造化區分擒縱
至險至難切在問防慎終如始)。】

「　消息玄門窮八卦」


【(大藥玄関
通乎八卦六㕛變動二氣感通其象昭然善自消息)。】

「　飛騰自在遠山岑」


【(神丹外就玄德內充功至行高自能輕舉丹丘紫府任性優游)。】


「　真空修鍊永長年」


【(靈臺內靜玄德外運修之於身則功齊造
化養之於壽則年等龜鶴也)。】

「　不識還丹在眼前」


【(道經云常
有欲以觀其撽盖性失於欲將觀妙本則遠在邊徼是知還丹之法內境虛明在外
物空寂若背此而求雖在目前莫之識矣)。】

「　一得清來光宇宙」



【(天得一以清乃至侯王得一以為天下正雖遠舉天地之清寧會㱕只在於侯王俾
令守雌州道求光宇宙也)。】

「　千朝湏是用烹煎」


【(且天行倢君子
以自強不息所以日慎一日惟一惟精庶修鍊精氣臻於妙道尓)。】

「　人間天上分眀錄」


【(業就功著世人攸仰道光德馨仙籍明錄盖由絕學無為而無
不為也)。】

「　汞裏花開豈偶然」


【(華池發潤金鼎流光伏剛
以柔制動以靜致花開汞面藥熟丹爐事匪偶然盖自真修之所感也)。】

「　達取但教依理路」


【(大道之路坥然夷平好逕之流自踈履踐今既或遵妙典
不爽格言無怠勤修何道不克)。】

「　勿貪世利悞周旋」


【(世利未息
寵辱且在故為患之本也既失前功行又悞彼周旋垂誡之言尤冝佩服也)。】



「　烘烘火色水中論」


【(純陽赫赫在乎上純陰𡨋𡨋處乎下陰陽混蒸
可論中矣)。】

「　至寶湏教鼎內存」


【(五行有緒四位無差列鼎乎造
化之中至寶存陰陽之內)。】

「　點化自然常在手」


【(夫點䥫為金化
凡入聖雖理極自然且常居掌握矣)。】

「　區分道理有玄門」


【(造化
雖廣物象有歸隨冝而容應用而遣然大理真漢自有玄門)。】

「　莫將容易誇言說」


【(變化之理道極神功妙在真歸談何容易)。】

「　盡入虛無是我尊」


【(萬類紛然皆歸無象無象之理乃是尊焉玄綱論云
此所謂返我鄉歸我常與道傴彊)。】

「　天動地神邪可畏」


【(𠃵坤
位正則鬼神不得肆其邪陰陽丈進則水火安能固其位當修鍊之時冝其慎乎)。】


「　丹能飛走似雲奔」


【(妙用通玄功成道至可以飛靈彩走神功
通幽洞微邈雲奔也)。】


「　人是人非意頗同」


【(人之持論㸦出異端其意頗同是非殊解)。】


「　較量此事盡歸空」


【(較彼是非品量見識未臻大道觸事皆空)。】


「　無為政化求真理」


【(體真理而合無為扣玄門而敷政化盡善
盡美功成不矜矣)。】

「　方表深仁大道雄」


【(不仁而仁仁膈有截
不德而德德被無彊大道深仁非雄何謂)。】

「　舉止但能恒有節」



【(夫聖人之動息猶天道之卷舒舉必有常止必有序靜專動直其在茲矣)。】

「　尋思何處不周通」


【(道體周虛道性無礙出處由此無幽不適)。】

「　寰瀛善惡知多少」


【(民之善惡皆由乎心以道治之物無不化)。】

「　莫忘生前積德功」


【(至功至德法地法天民惟樂推史無不載矣)。】


「　因緣業障遇真稀」


【(名未桂於丹臺品未登於仙籙故因

緣尚遠業障尢多所以遇真者稀也)。】

「　黃帝乘龍豈是非」



【(史記云黃帝鑄鼎於荊既成有龍垂髯而下黃帝因遂乘之上昇於仙又豈非也)。】


「　達識尚凝常恍惚」


【(有達識者常凝然於妙道故老氏云恍恍
惚惚其中有物也)。】

「　愚蒙學解故相違」


【(愚蔽蒙昧之者學寡
洞真解非愽物執以五金八石而為妙者故相違也)。】

「　華池秘密人天授」


【(華池者乃九轉中神水華池也人不虛授天不易求故云秘密也)。】

「　神室光揚日月輝」


【(𠃵坤交泰离坎相生契彼三乃應茲九地乃謂
之神室也以日精月華而為體故光揚顯其輝彩也)。】

「　智者無言皆口嘿」


【(達大方者到佊無言之津所以嘿也)。】

「　敢將容易論玄機」


【(天非愛道人亦能修故就無言之言以申無說之說諒非容易務盡玄機庶幾
乎名利仙籍者一議究焉)。】
御製逍遙詠卷第十

甲辰歲高麗國分司大藏都監奉
