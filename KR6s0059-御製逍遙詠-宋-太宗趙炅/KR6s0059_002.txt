










御製逍遙詠卷第二(五言)　輕


「　人救眼前急」


【(凡情處世以近事嬰心逐利眼前未知於道矣)。】

「　何曾利益心」


【(背彼至理沉滯常情既無向道之心豈有利他之行)。】

「　愚迷終淺見」


【(顓愚昧道於理不通縱令誘以義方終成浮淺矣)。】

「　達者智高深」


【(達士之智得用不群妙出希夷高超物表矣)。】

「　苦行湏知應」


【(夫聖人之道誡欲持行𡨋契玄穹所謂作善降之百祥作不善
降之百殃報應昭然若影響尓)。】

「　餘憂力不任」


【(非常之人必有非常
之事若非其流則憂不遠故易云力小而任重鮮不及矣)。】

「　經書無限意」


【(古先典籍百王之道其意萬殊志在乎導生育物者矣)。】

「　稽古便同今」


【(夫聖人者見真素抱淳朴乃稽古之道以御今之有也)。】


「　逍遙安且定」


【(放曠自得是曰逍遙方圓不違安而且定)。】

「　但信莫懷疑」


【(優游之旨信者方通但執之以心誠莫懷之以疑慮)。】

「　大海波中水」


【(滄溟浩汗渤澥渺瀰雖巨浪以滔天亦任真而自若)。】

「　狂風樹擺枝」


【(勁風皷怒大樹根搖仼動息以不常乃隨緣而無矣)。】

「　樂天分造化」


【(樂天知命秉正守中澄閑適之本源分造化之深力)。】

「　慧眼細觀之」


【(究升沉之要道達真假之本根慧眼明觀無細不鑒矣)。】

「　方寸無諸惡」


【(積善於心仁者之行是以方寸之內不苞藏於諸惡焉)。】


「　恒將利益持」


【(夫利於物者是聖賢之行故天道福善鬼神助順欲
久而且長必常利益於人則任持可久矣)。】


「　恍惚人難曉」


【(恍兮惚兮理越情外無象之象故曰難曉)。】

「　天仙



【(真人契理通幽洞微體語論以忘筌處救默而咸妙)。】

「　修心無道理」


【(夫欲修心息慮而背真常則於道彌遠)。】

「　終日自忙然」


【(依道川心者則終日必成專已師心者則寸陰虛擲也)。】

「　我命真中趣」


【(稟天之命依道之性旨趣咸真逍遙自若)。】

「　長生認宿緣」


【(躋夫
壽域以久以長若匪宿緣曷致能事)。】

「　機權饒使用」


【(夫遠道者至於
無為機權之智何所用哉)。】

「　聖事不虛傳」


【(動必全真開物成務聖人
之旨豈虛也哉)。】


「　清虛閑靜得」


【(妙本湛然寂寥冲寞視之不可見聽之不可聞斯得
道之真體故老氏曰孰能濁以靜之徐清也)。】

「　奉道見真歸」


【(道者
虛極之稱妙用無方故澄湛內心精潔儀軓必觀乎真妙尋乎道源)。】

「　契合皆堪用」


【(夫神明之本天地之源其大無外其微無內契會有方應用無竭者矣)。】


「　因緣事好依」


【(因緣生發原始反終依真淨而得契無為而成也)。】


「　精誠求諦信」


【(精思諦求神凝感徹真應功成可以左馭青龍右乘
白虎者也)。】

「　湏有是兼非」


【(自澆淳散朴大道不行是非競作聖人統
而化之和光而同塵也)。】

「　豁達幽深遠」


【(同流六虛傍遊無外斯可尋道
之源觀道之用也)。】

「　奧妙入玄機」


【(探𧷤索隱鈎深致遠窮眾妙之門入
香𡨋之際是謂玄之又玄)。】


「　清靜人安樂」


【(人之所以多憂患者背由嗜欲之不節勞慮之未息
若能染除情染令神氣清淨則自然安閑而恱豫爾)。】

「　恒持要久長」



【(夫為善者靡不有初鮮𠑽有終若守清靜以為恒德久而不渝則可以長保其元吉)。】


「　秋天澄皓色」


【(皓氣充乎內也則如秋天之色澹然而虛自矣)。】

「　春



【(靈秀發乎外也則如春日之輝藹然而滋麗矣)。】

「　一氣從初立」


【(冲和一氣亭毒萬類考其用則造化之權輿也原其體則恍惚
之精真也所以至於道者在乎抱一而巳)。】

「　三才位不忙」


【(三才既位四時
乃序生死由乎天榮枯繫乎命則埋不可以速而乱其常人不可以躁而濫其進也)。】


「　群生如似夢」


【(擾擾黎元孰明虛幻為色所盲為聲所聾同莊生之
化蝶頰鄭人之失度得非如夢𥧌乎)。】

「　甚處得馨香」


【(人不知道
則觸逢成滯至于歿齒無善可稽則馨香之德無所聞也)。】


「　道者強名立」


【(道本無名強名曰道既為萬化之祖爰開眾妙之門
始一氣以混成終三才而定位觸類而長何莫由斯)。】

「　清虛隱顯深」



【(清虛之理隱顯相生彼我玄同逍遙自得)。】

「　欲行千里路」


【(千里之程
起於足下萬殊之變唯道是㱕)。】

「　別探一般心」


【(真一之心玄之又玄以其
無為之功大也雖日用而不知是得言乎別探矣斯亦強而名之)。】

「　景象空天遠」


【(景中之景象外之象邈彼真空明茲妙理其或懵而求者猶天
之遠安能及哉)。】

「　何勞意馬尋」


【(既昧真源徒奔意馬沉劒刻舟勞而
無益)。】

「　玄談通智慧」


【(極玄妙之談本智慧而通也)。】

「　曩刧至從今」


【(夫聖人之心憑智慧之照得玄元之妙用悟虛無之至理則曩
刧至今不為遠也)。】


「　𡨋搜多景趣」


【(搜奇抉異景趣滋繁雖馳騖於襟靈亦未乖於冲默)。】


「　悟則䪺神清」


【(形者神之宅今既體道無外悟物非有不皦不昧寂
兮寥兮清思阜安頓超塵累矣)。】

「　拘束終難得」


【(德量曠然若其空谷
者方可希夷情性煙霞儔侶未有尚拘世態觸類成滯䔬遠於道得非難哉)。】

「　冲



【(大道之體本乎冲和俾之純純若嬰兒之未孜也)。】

「　知微通理性」


【(精義入神以致於用故能知微知章會通理性窮於妙本矣)。】

「　滋味好研精」


【(研精覃思探味道法行之不怠轉益清虛尓)。】

「　妙用天然法」


【(理自玄應生之無主雖修鍊之門有諸而自然之用顯著矣)。】

「　陰功蘊大名」


【(聖人得道之妙也功侔造化息被億兆百姓雖注其耳目而日
用不知故能成其大也)。】


「　氣類湏相假」


【(二儀氣分萬類萌漸皆相假借自道生之)。】

「　陰陽造化成」


【(陰中有陽陽中有陰自然之道變化之理以成於功也)。】

「　離凢知有聖」


【(得大藥者還神固身玉髓金骨名継三清列真人位矣)。】

「　得外更求精」


【(雖得玄元之道更湏採摭精微有無相生為金丹根蔕也)。】

「　先秘庚辛理」


【(至藥假陰陽含養以白金為始是以聖人秘而寶之)。】

「　後傳戊巳名」


【(土能生金王於四季雖無正位用且不窮子母之名傳之有自矣)。】

「　真鈆歸一體」


【(玄功歸一萬象始生妙用通靈而㱕大道也)。】

「　交感自長生」


【(金丹應日月交會感四氣循環成而得之證長生久視矣)。】


「　天界殊勝理」


【(三天之界既在杳𡨋之際仍居𠕀象之先究其理者
則超然殊異勝絕不可得而言也)。】

「　凡愚不易知」


【(仙凡路隔愚智
道殊故三天之理未易知也)。】

「　曩因誰可究」


【(鍊行之功動乎生刧
不知其道從何所生浩博先因其誰可究)。】

「　淺近故懷疑」


【(圓穹之廣滄海
之深窺管井蛙信之𠕀及是知淺近之智故懷疑也)。】

「　的實歸真性」



【(去華務實返朴陶真萬變雖殊可以執一而御之故歸真性矣)。】

「　區邪入正持」


【(行無私邪情絕貪慾故聖人取含受之腹去妄視之目執持大象善入無為)。】


「　樂耶親大道」


【(無為之樂體乎清靜除察察之政化淳淳之民故萬

物歸往也)。】

「　報應自然之」


【(契玄妙之功達自然之道迹超物表應用
無方矣)。】


「　知之修鍊法」


【(知彼聖道理極冲虛乃可修持法之為則矣)。】

「　便是聖中人」


【(達其修養之道可以駕鶴𩥵鸞陰符云知之修鍊謂之聖
人)。】

「　有象虛無境」


【(有象曰形虛無曰理是以聖人垂彼有象之儀
用顯無極之道矣)。】

「　邪緣不見真」


【(群動妄垢奔馳幻邪恣彼嗜欲之情
不覿真一之理矣)。】

「　恒持功行益」


【(夫至人者每自保守積行可持乃務
日益之功以至無為之道)。】

「　響應甚平均」


【(功濟道著其行必均
若谷傳聲應無不失)。】

「　渾然天地理」


【(道者混然而成乃在天地之內
故道經云有物混成先天地生)。】

「　大道徧𠃵坤」


【(希夷至理其大無極
况乎二儀不亦徧矣)。】


「　伏藏終不見」


【(至道幽微離於言象視之不可見聽之不可聞有如
伏藏難以觀聽者也)。】

「　巧拙是兼非」


【(天賦其性道且不違或巧有機
関或拙多魯鈍若非若是俱在陶鎔)。】

「　至道光懷古」


【(至真之道亘古
亘今欲慕而行者則懷其昔而師其道焉)。】

「　周通隱聖機」


【(智周萬物
曰聖不滯一隅曰通聖人雖觸物周通而尚隱機於所化爾)。】

「　清天何物性」


【(清氣為天自然之理於彼萬物何所滯其性乎)。】

「　濁地故相依」


【(濁氣
為地性本如然陰陽相符故可依矣)。】

「　修鍊真堪惜」


【(夫修之於內身
則感之於外氣澄澆漓而歸淳素息嗜慾而保長生在修鍊之門故真堪愛惜矣)。】


「　踈慵有順違」


【(道體恒周木無近遠若踈慵而不奉行者乃有
順違之事焉)。】


「　夢𥧌其中物」


【(以汞入鈆存亡未辯髣髴之狀其猶夢焉)。】

「　常人



【(真人至術遠俗出塵凡庶之流安可知也)。】

「　陰寒三[婁*殳]變」


【(極陰之氣制伏餘[婁*殳]力不自由三[婁*殳]皆變故云子水[婁*殳]一午火[婁*殳]二合而成三
者也)。】

「　陽暖兩相冝」


【(陽性剛暖能生至陰二氣相冝大藥功著矣)。】

「　宿世終皆有」


【(慕道求真背塵離俗若匪宿有妙性何能知乎長生者哉)。】

「　無緣勿強為」


【(欲臻至理必藉階緣儻鮮宿因皆為強立也)。】

「　真鈆歸至道」


【(借鈆為母丹成鈆謝靈汞獨存㱕乎至道尓)。】

「　龍虎笑華池」


【(陽龍也陰虎也二物共為變化神水華池妙道俻矣)。】


「　形影隨人得」


【(聖人取象考績玄功成陰之形取陽之[婁*殳]羽客隨之
久而挹持故冲虛經云形枉則影曲形端則影直也)。】

「　精修道不迷」



【(上士欲吐故納新怡神鍊氣所尚冲寞扣寂玄関徹視聽於希夷肆閑情於惚恍也)。】


「　研窮深理路」


【(唯恍唯惚物象難求香兮𡨋兮至理淵深其或研精
而覔探賾而尋亦尠得矣)。】

「　造化隱端倪」


【(太初之道真一之源權輿
天地茂養萬物不為長短之相生直為有形之類母)。】

「　雲引龍吟[宋-木+儿]」



【(雲行雨施龍吟霧起故易曰同聲相應同氣相求也)。】

「　風生虎嘯溪」



【(雲既從龍風亦從虎矣)。】

「　少知愚豈解」


【(夫希夷之理易象之微聡
明深智方乃探𧷤豈愚昧而知也)。】

「　南北與東西」


【(且顓蒙之軰迷
於方隅亦猶終年戴天不知天之高終日飲海不知海之闊豈可語道之玄妙乎)。】


「　鷰雀不生鳳」


【(修鍊之法品配有冝氣類不同枉施功用譬夫
鷰雀不產於鳳皇矣)。】

「　陰陽一氣全」


【(陰[慘-(彰-章)+小]陽舒雖分二位同本大道
而得發生故皆全於一氣也)。】

「　無窮深可及」


【(道之為體也虛而無
極妙而至深無以迎隨超乎視聽若存神以叩之則亦可及也)。】

「　境象理推遷」


【(在天成象在地成形而晝夜運行寒暑代謝間不容髮皆自然之理)。】

「　奚益



【(百歲光陰如駒過隙隨化如遷理之常也欲固益其形骸者非道何
以哉)。】

「　平章語默玄」


【(修養內丹商較至道或語或默皆叩玄関)。】

「　終身何所伇」


【(土之為道也苟能達其寵辱知其止足則可以優游度世
何勞伇之有哉)。】

「　道本豈私焉」


【(至理坦然本無私徇群情逐物自背
天真則知迷悟在人非道使之然也)。】


「　自在無拘束」


【(精通靈而感物神動氣而入微自然而然無在不在
道之行也一以貫之若然則復何拘束哉)。】

「　逍遙裏外安」


【(得逍遙之
妙者緘滕六志所以養其內也扃鋿百骸所以養其外也內則精神冲外則血氣
平和是得表裏彌康矣)。】

「　頑情終不見」


【(凡人之情不能固其形存
其神者終不能見彼長生之術也)。】

「　聖境類乎端」


【(端謂端倪也莊
子曰和之以天倪斯亦聖人窮神之絕境矣類此一言旁該萬化)。】

「　機智將何用」


【(既道有端倪乃自然之分也故聖人立法導民之心各使自然必背之者
非也夫如是則惟機與智將焉用之)。】

「　幽深大道寬」


【(道之大也幽而
不可見深而不可取然為萬物之宗是得言其寬矣)。】

「　區分能解意」



【(近取諸身遠取諸物區以別矣分而析之則可以懸解逍遙之旨也)。】

「　四季一周看」


【(節宣其氣動靜合冝信一指而有徵雖萬機而無擁至若歷周四序
曠然自得因時之妙愈有可觀)。】


「　求得真宗一」


【(抱守淳一冲和乃全為道之基養育由性故得一之
理契乎真宗也)。】

「　深知萬象清」


【(覆載既廣萬象居中既曉真宗遂能
齊物紛紜異類長短相形各復歸根咸資於道)。】

「　自然通感應」


【(陽唱
陰和感召自然成物之務以通乎幽奧者也)。】

「　意外別長生」


【(羽化
脫從之道在乎意出象外跡離區中不為物拘自陶時數何長生之理不明哉)。】

「　斆



【(効上古之風守知白之道舉措合度雍雅以居不怠行之
愈加光大)。】

「　輝華日瑩眀」


【(法道崇德立身揚名若果日麗天輝華燭
物洞然明瑩也)。】

「　誰窮天地理」


【(清濁異立剛柔不等法而象之體而
制之窮其根源者其唯聖人乎)。】

「　去住兩無成」


【(陰陽既敘𠃵坤並列或云
彼取此則物無成功若俯察仰觀則道何全濟盖造化交感去留無象也)。】


「　不醉迷如醉」


【(修鍊之者不考五行之精𠕀究真一之道不授訣於
至人如迷醉者也)。】

「　猶貪巨鼎扛」


【(九拔之金鑄鼎象物非可移也
令貪而扛之愚而舉之盖不度其力自耳其咎也)。】

「　通玄求得一」
