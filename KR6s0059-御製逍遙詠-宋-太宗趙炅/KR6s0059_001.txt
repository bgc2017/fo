










K. no. 1260

御製逍遙詠序　　　　　輕

「玄元大道理包深遠。」
【(玄元者虛極之妙本天地之元精淳
湛一真凝然本寂所以伯陽宗其至道教演五千囊括虛無可謂深遠矣)。】
「
逍遙
至論義貫精微。」
【(逍遙者懸解之名冲虛曠達之義大鵬翔乎寥
廓至人遊於無為放意於自得之場別物物任其性乃可妙契三無化周六合使澆
漓之俗復淳朴之風精微之𣅀其在於斯)。】
「
朕聽覽之餘留心
緗帙。」
【(万機之餘不倦披覽雖漢武乙夜觀書無以過也)。】
「
三墳五典
頗識指歸。」
【(三墳者三皇之書二皇之道大於五帝五典者五帝之書
五帝之道可以常行是以聖人皆達指歸妙盡其要者也)。】
「
金匱玉凾亦
探玄奧。」
【(金匱玉凾者太公黃石公所菩秘法也皆論六韜三略富國安
民養生鍊神機權要妙雖其淵契盡皆探賾矣)。】
「
嘗念寡昧於道
不達罔測津涯故發此詠題庶幾明
白坦然矣。」
【(夫欲明至道之極故著逍遙歌詠以暢宸襟而猶道成不
宰謙卑益光遂發以罔測之辤尚假以庶幾之語故老子云江海能為百谷王宥以
其善下故也)。】
「
夫道之妙也五行能之運化。」
【(道本
冲寂應用無方妙在卷舒成乎變化故在天曰五星在物曰五行經緯彝倫招蘓
彚也)。】
「
三才可以變通。」
【(三才著象皆從其道道變無窮乃道乎天
地人也)。】
「
虎嘯龍吟陽唱陰和。」
【(此明至道自然之理且龍
者水畜雲者水氣虎者陰物風者陰精以類相召自相符契故易云同聲相應同氣
相求此其義也)。】
「
窮其秘也神明可辯於情狀。」
【(道之
秘也神全形具體與物𡨋原始要終窮理盡性推其情狀何深奧之不達哉)。】
「
究

其精也剛柔自符於易簡。」
【(廓然無間妙極太虛
晝夜以之晦明寒暑因而來往所以𠃵剛坤柔自合易簡俾不為而善始不勞而善
成苟非道之至微孰能詳放此乎)。】
「
而惑者昧於修已攻乎
異端俟駕鶴而𩥵鸞猶繫風而捕影。」

【(蒙昧之人不達至道謂神仙之可學顧囂塵而欲棄或置盤承露或築臺通天求驂
駕鸞鶴亦猶繫風捕影不可得而及也)。】
「
老氏五千之旨尚不
述於神仙。」
【(老子所著五千言且不述神仙之事但欲顯道之玄極歸
乎真源者矣)。】
「
莊周九萬之談理更超於言象。」

【(南華云北溟有鵾化而為鵬九萬一程六月一息盖舉其至大以類至散皆齊物理
也是知事契精微理超言象誠哉莊生所諭實至道之樞要逍遙之妙旨也)。】
「
先
賢穎脫皆知道而不言。」
【(自昔達道之士既知道在
於我所以不言老子云知者不言斯不誣矣)。】
「
庸愚困蒙豈希
顏而議學。」
【(庸昧之者妄學神仙有類希顏欲齊亞聖不亦遠乎)。】
「
昔
秦始皇邪見所惑徒望蓬萊。」
【(始皇帝為邪所
惑道德不修妄學神仙欲求度世遂遣徐福往彼蓬萊採其靈藥徐福以無驗而不
返始皇猶未悟而懸望矣)。】
「
漢武帝民力是輕莫尋員
嶠。」
【(漢武神智獨穎英武無倫尤好方士禱祈山岳後感王母降於帝宮授以六甲
辰生之訣自謂必登仙品芥視黎甿崇峻臺館坈戮降服既失上仙之誡莫尋貪嶓
之靈矣)。】
「
斯皆不修仁德求利好貪。」
【(守位曰仁好生
曰德既殫民力復慕仙遊比轅適越良可悲矣)。】
「
事蓋不經良足
太息。」
【(事不師古動越典經百世之後斯過不泯誠堪太息者哉)。】
「
今之

所述其殆庶乎。」
【(殆近也庶幾道也今茲所製或象陰陽則𠃵坤
而演妙或明金水法离坎以談玄追莊老清淨之遐蹤継藏軒淳元之絕䡄近乎至
道樂矣大方且非慕彼神仙之術也)。】
「
或近取諸身。」
【(或內養純和之
氣靜修真一之心動合靈源咸歸妙本也)。】
「
或遠取諸物。」
【(或以日精
月華鈆水金液託其象而成其言也)。】
「
其詣理也不可以一途
執。」
【(窮其理也則必有所詣詣其理者乃能成天下之務也至若適時知變在乎幾
也又焉可以一途而執哉)。】
「
其筌意也不可以一理求。」

【(徵其意也則必因於筌筌其意者乃能通天下之志也至若窮神極化在乎深也又
焉可以一理而求哉)。】
「
探𧷤幽微。」
【(探𧷤之妙通幽洞微)。】
「
㕘詳法
度。」
【(極思㕘詳規矩法度)。】
「
智者見之謂之智仁者
見之謂之仁。」
【(俾夫智者覽逍遙之旨則動而無窮也仁者得逍遙之道
則澹乎無為也)。】
「
玄覽而可以照三清。」
【(上清玉清太清謂之
三清也玄覽心照也混尓真心朗然玄照越彼三天之上也)。】
「
旁行而可
以通萬有。」
【(不疾而速未兆而萌仁智之所莫拘動靜之所一致既周
行無礙又何萬有而不通哉)。】
「
夫詩頌歌辭華而不實。」

【(汎而論之不務其實則或傷於華也)。】
「
上不足以補時政之
闕失。」
【(時政之本務求其實必尚華言終無所補也)。】
「
次不足以
救蒼生之弊病。」
【(欲彼黎元咸躋富壽者義存道本也必尚
彼浮詞誠不足以救蒼生之弊病矣)。】
「
假有獨擅其名弗精
於用。」
【(辭也者所以通萬物之情也假有專其述作之名昧彼述作之用所
謂華而不實也)。】
「
義同畫餅棄不足珍。」
【(文彩燦然形容

儼若但堪恱目何可充膓就不足珍也)。】
「
爭如立至當之言
以濟於時世者哉。」
【(夫聖人之意其不可得而見也故觀之於
象而形之於言應時之所求合世之所望義存精當妙在長生也)。】
「
所以姑
務契理不取於文。」
【(且務契彼真一之理故不取於藻麗之文)。】

「貽尓方來悟夫懸解者矣。」
【(庶幾貽厥將來好古
博雅之上悟夫逍遙懸解之旨也)。
御製逍遙詠卷第一(五言)】


「　逍遙知語默」


【(聖人之御理也不凝滯於物不封執於教發言合道
守靜惟真所以語默一致逍遙自適也)。】

「　境外見真空」


【(真空之旨超越
塵境故須遊心於杳冥忘情於物象妙用無際綿綿亘古焉)。】

「　有位皆玄感」


【(處非常之位明非常之道皆符契上玄俯順庶品故使樂推長保永福也)。】


「　無緣不用功」


【(妙本冲寂修之仗緣其或真性未明塵心尚壅縱勤
功用事亦虛勞矣)。】

「　義說三才理」


【(天地剖判人倫攸依淳風既微
禮教斯著三才位而萬事彰矣)。】

「　幽深萬事通」


【(極深研幾退藏於密
此乃道體幽微雖通濟萬物而日用不知矣)。】

「　因依堪法則」


【(道經云執
古之道以御今之有蓋行無為之化以理有為之俗[竺-二+軌]而則之何淳素之不復哉)。】


「　荅謝向旻穹」


【(天之無恩而大恩生故以萬物為芻狗盖不責其報
然覆載之內誰不尊戴於穹旻矣)。】


「　大道虛無境」


【(大道冲粹凝湛一真應用無方本乎妙極虛通之要
盡契玄元矣)。】

「　信之則有靈」


【(上士勤行信而有徵神靈谷盈善得
道用者矣)。】

「　幽深玄妙理」


【(抱一之功幽隱深邃照用玄覽體括虛無

應物自然究理何極)。】

「　運轉幾曾停」


【(天地權與妙本見氣滋養
萬物運轉無窮動而愈出故未暫停也)。】

「　拘束從何益」


【(道之所宗
逍遙任性變通為用虛極為玄或見滯於情事有拘執則有何益矣)。】

「　周通靜杳冥」


【(杳冥之理生成之用周通萬彙動息常靜矣)。】

「　真空非作相」


【(真空之妙非色非形取相而觀背乎道矣)。】

「　善應保安寧」


【(善慶
之功旌乎感應天清地寧自然長久矣)。】


「　我自常安樂」


【(逍遙之道理契希夷達其深源自然安樂矣)。】

「　凡情勿笑耶」


【(道本淵奧宗旨甚深下士聞之而大笑故誡之矣)。】

「　雲生諸洞府」


【(洞府皆神仙棲隱之方故雲霞渺邈非洞府而不生)。】

「　𣗥不放仙花」


【(根性既珠仙凡自異諒其棘刺不秀名花盖明修道之士
品類有殊矣)。】

「　地卑三光照」


【(地本卑下而能厚載得一以寧功契
上玄合三光之照矣)。】

「　天開五色霞」


【(天法於道道法自然運用生成
於物不滯乃五色霞光縱橫分布)。】

「　未知何所是」


【(道本凝寂而昧之者
孰了其元故道經云吾不知其誰之子其斯之謂歟)。】

「　善願理交加」


【(於諸
法中而得至理者乃因善願矣願之與理俱不可廢矣)。】


「　我達逍遙理」


【(觸物無滯名曰逍遙舒卷任情乃達其理隨用
利物其惟聖人乎)。】

「　陰陽運五行」


【(一陰一陽動轉不息潛運五行
之氣用全二儀之功矣)。】

「　道從初一變」


【(混元之前澄寂無狀洎道
生乎一一生乎三變其淳留不出於道也)。】

「　德合混三清」


【(變澆漓之
風歸淳素之化是以大上立德德立道生可以脫三色之樊籠躋三清之聖境)。】

「　聖境堪依仗」


【(抱一還源窮神入聖故境界之超異也復何勝道哉誠堪

依仗耳)。】

「　狐疑事不成」


【(於事不决妄起迷疑如狐之渡冰猶預多
慮志既不一事亦難成)。】

「　異端違雅淡」


【(道性冲淡守之則真其或
攻乎異端是乃違於大道矣)。】

「　返禍自輕生」


【(既不純和貞絜仍聞
順佫背真真境未臻階先及自貽伊感可不謂輕生乎)。】


「　逍遙人控鶴」


【(優游自適至人不拘或駕鶴以凌虛顯逍遙之不滯)。】


「　碧落壓天津」


【(圓盖四垂壓乎四極誠為神仙勝賞之處也)。】

「　達理通含識」


【(物理虛通道性無別一以貫之其在茲矣)。】

「　虛無即是真」


【(虛無妙本體本即真物物周通何往弗濟)。】

「　河車誰肯信」


【(延齡妙術傳乎至人正氣河車信非凡庶)。】

「　鈆汞本相親」


【(鈆汞
相資陰陽共致於道非遠名曰相親)。】

「　造化皆如此」


【(虛而無心動而有應
造化萬彙功成一揆)。】

「　今緣論宿因」


【(積功累德詣道見真諒此今緣
本由曩植)。】


「　何處堪依仗」


【(法天為父稟地為母以德為弓以義為裘用仁作冠
兼信為冕自然堪依堪仗也)。】

「　經心便皺眉」


【(道濟天下功合玄穿
耻不義而取憤不仁而人所以皺眉尓)。】

「　世情多詭詐」


【(狐面狸德
言與行違既乖禮義之風難居仁壽之域故)。】

「　業重信愚癡」


【(易曰
履德之基謙德之柄本乎信崇乎仁豈愚昧而得也)。】

「　取次招他謗」



【(露才揚巳自是他非首招其謗直為怨府尓)。】

「　違非不自知」


【(老氏曰自
伐者無功自矜者不長故行乖德薄自不知尓)。】

「　貌恭無德行」


【(夫教恭柔
侫必外現禮容內荏惡行故宣尼云色厲而內荏也)。】

「　禍福在修持」



【(易曰善不積不足以成名惡不積不足以我身小人以小善為無益而弗為也以小

惡為無傷而弗去也此王在改過修持尓)。】


「　虛無知大道」


【(虛極無為絕於視聽達乎應用其猶橐籥虛而
不屈動而愈出達欺妙者所謂知大道矣)。】

「　極意思悠然」


【(意之極者
所謂忘其象也則神思悠然於物表非世態之所拘爾)。】

「　摩日雲中[序-予+(鴈-厂)]」


【(至於道者則情無所滯形無所羈如[序-予+(鴈-厂)]在空優游自適)。】

「　瀟湘水似天」


【(瀟湘之水凝碧如天况乎達人性其情而合道忘其象而入無得一混然
良在斯矣)。】

「　孤峯高遠近」


【(峯之高也則洞觀遐邇道之高也則妙入
有無達往知來如視諸掌)。】

「　寶月瑩團圓」


【(皓魄圓時無幽不燭况達
道之士玄覽周通則物無遺照矣)。】

「　信解明開悟」


【(至道冲虛超乎
言象非信解明白者不能開悟於玄奧爾)。】

「　青霄在目前」


【(九重霄漢
渺邈無涯一以貫之宛同目擊庶幾乎不出戶而知天下矣)。】


「　逍遙名有理」


【(夫天命之謂性也率性之謂道也雖道無常名而名
有常理以其綿日月而不老配天地而長存故得謂之逍遙矣)。】

「　得一便知空」


【(既窮真一之門便造真空之境夫如是則得無所得得之大也知無所知知之
至也)。】

「　道本背深邃」


【(源至道之來則持受異氣稟之自然非積
學所能及也信所謂深而且邃矣)。】

「　凡為不易窮」


【(崆峒絕境不可
以苟求汗澷真蹤不可以力致至若心期玉室志慕丹臺者几有所為未易窮也)。】


「　品流才雅淡」


【(飡朝霞而養素吸沆瀣以食精既曰清虛誠惟雅淡
若此之品流即仙才耳)。】

「　挌致意潛通」


【(格致玄同機關密契志以
道寧竟潛通也)。】

「　救世傳相益」


【(万年神衍良足以救世也傳而習之
則有以見復性命之源踐長注之踐斯可謂之益矣)。】

「　因成順八風」




【(吐故納新登神導氣內凝一真之性外調八節之風動順天和咸歸妙本)。】


「　誰測玄機聖」


【(玄機幽邃非凡情之所測惟聖人探賾索隱鉤
深致遠故能顯諸仁藏諸用也)。】

「　天功地久長」


【(範圍天地周備其理
故使能長且久不隨物變尓)。】

「　群生皆是性」


【(群生類聚營務且殊
其間巧拙不侔賢愚莫等論其幻情則有別推其道性則一同)。】

「　道德有圓方」


【(道者同於道德者同於德此乃功用兩全方圓一致也)。】

「　妍醜隨情意」


【(道非美惡情自妍醜老子云天下皆知美之為美斯惡已盖美
善無主俱是妄情矣)。】

「　於戲細酌量」


【(聖人欲令窮理盡性從無入有
故發歎辤誡勸學道之流審細叅詳達於玄奧尓)。】

「　達人知淡泊」



【(寂寥虛靜湛然常存達人體之獨立而不改周行而不殆也)。】

「　默默見非常」


【(非常之道在於無欲而觀是以古先哲王行不言之信處無為之事
玄默之見斯為要妙矣)。】


「　道德誰分別」


【(德者道之用志於道據於德異𣲖同源無分別也)。】


「　逍遙我自由」


【(優游自若放心於自得之場外物不能浼其中我自
曲矣)。】

「　時情人不達」


【(道即日用而不知一人用之不閒有餘天下用
之未聞不足盖人不達也)。】

「　深理少相投」


【(道者通幽洞微之玄盖不知其
妙而少相投也)。】

「　已是終無益」


【(修道之士若恃兄之能老子云自見
者不明自是者不彰終不為益也)。】

「　非他作逗留」


【(不緘其口非飾於他
不惟違理捐物兼於道懸遠也)。】

「　奈何為各異」


【(達士迷人情懷各異
君子依道小人隨利無奈何也)。】

「　未定復還休」


【(心尚猶豫不能勤行
往復思之却生墮𢢔)。】



「　奧義通玄化」


【(夫至道冲邈包乎奧義故能通於妙有叶於玄化矣)。】


「　精微妙入神」


【(杳冥之精通微應妙盖善究虛極以彰得一神授其
靈應用無竭也)。】

「　功程隨志願」


【(積德累功趣乎妙道行隨願感報應
斯眀)。】

「　貶降背於真」


【(道本無言依言顯道以是非而取相或貶降以
拘情動念乖真背於道矣)。】

「　靜默堪為法」


【(靜默之妙志守唯靜事可
䡄持善契真理)。】

「　無言事可親」


【(大音希聲無言契道理事俱寂息用
澄神惟德是修故可親也)。】

「　逢師傳口訣」


【(玄元秘旨妙出有無在三
之義既嚴得一之源可究矣)。】

「　金玉變埃塵」


【(富莫富於無貪貴莫
貴於得道道既可得心故無貪視金玉如埃塵誠可信焉)。】


「　清靜真空理」


【(妙本無極目之清靜塵滓不染是曰真空矣)。】


「　不無解者難」


【(至理眇邈有物非形而為學則日益為道則口
損得無難解乎)。】

「　縱橫依見性」


【(於物下滯任用無方其可洞明依乎
見性)。】

「　廓落智中觀」


【(理本寥廓而智者達之如指諸掌矣)。】

「　坦蕩歸仁德」


【(至道玄遠坦然寬大垂象示變人可則之皆歸仁德矣)。】


「　辛勤久遠看」


【(勤心慕道立志求真旦非動躁而可成在乎久
遠矣)。】

「　勿教為逆順」


【(逆即違理背道順乃負陰抱陽今制度冈𠍴修
暮應𠋫煥然可知也)。】

「　平地入雲端」


【(御氣乘虛在乎道成功著而浞
德合上玄仁資庶品自地昇灌豈為遠哉)。】


「　𠃵坤常運轉」


【(乹剛坤柔各亡其用晝行夜轉以成其功故曰
天道東南地道西北而運轉下息也)。】

「　兀兀見眾生」


【(眾類之生徧乎
天地盡受陰陽之覆燾皆從造化之發生然則兀兀陶陶日而不知也)。】

「　利物



【(內守於正外利於人普育黎元汎愛群庶必同聲相應無不合
於大道矣)。】

「　施為願要精」


【(欲立身行事剋巳安人須發大心以利萬
物施為作用故要精勤)。】

「　身慵何隱遁」


【(其或獨善乎一身慵於濟
物退處巖藪慕於寂寥斯乃違覆載之深仁行隱遁之小道耳)。】

「　舌辯謾縱橫」


【(其或巧言侫辯口給心非無實利以及人有虛飾而在巳終乖不言
之理謾施縱橫之機焉)。】

「　種性歸真理」


【(人之稟性維有不同至摭
實而為談須云非而取善不向真理何所歸乎)。】

「　因緣有重輕」


【(同
浩氣處此人倫盖修因立基已有善惡及稟質受性乃有重輕者焉)。】


「　仙經無限意」


【(丹臺妙訣玉簡靈書盖得乎修真養素之方豈獨靈
乘虛步雲之旨秘乎紫府可得而窺也)。】

「　龍虎返丹砂」


【(陽龍陰虎
二氣迭運成之大藥力同神速)。】

「　採綴陰陽理」


【(法奇偶之正理著變
化之玄功煥尓遐丹大哉妙術)。】

「　終期嵗月賖」


【(易朝菌之促為大椿
之齡有以見壺中日月物外風塵皆可以至道窮之者夫)。】

「　求來皆有應」



【(功行相資目足同濟志必感神之道聲必應響之談其來著矣)。】

「　不信固難誇」


【(釋以信為功德之母儒以信為景行之先如尚渧疑固難啔迪)。】

「　忙說閑清境」


【(不吝發揮開乎蒙昧三清勝槩如視諸掌)。】

「　真誠悟幾家」



【(信必由中言必干典真誠無妄群動可依)。】


「　陽與陰和合」


【(丹經叅同云陰者道之基陽者形之始故陽因陰有
陰稟陽孕能滋萬物為萬物基者也)。】

「　須從一字求」


【(考五行之精花真一
之道脫落塵世遊心冲寞故老氏云天得一以清地得一以寧神得一以靈是也)。】


「　分明皆是象」


【(聖人取象陽精陰魄積九數以先立攝万物以皆歸

故易云在天成象在地成形)。】

「　作相不能投」


【(夫大道者寂寥無聲
淡乎無味視之不見德之不聞豈作相而投何挹持而得也)。】

「　舉措先真實」


【(謙弱者德之柄柔和者道之用必處其厚而居其實也)。】

「　精麤各自柔」


【(學有精麤盖性有利鈍亦在守其柔弱棄彼堅強者矣)。】

「　恒持通覺悟」


【(道性虛極妙本湛然當須絕棄塵境覺悟玄解持守雌靜
而巳)。】

「　萬𣲖水東流」


【(百谷朝宗而東注天地之常道也易曰巽利東
南澤潤萬物者莫大乎水德以况於道之用也)。】


「　鈆汞精修鍊」


【(鈆汞之親雖同母子制之失度倐類參商理湏精研
然可修鍊)。】

「　陰陽恊物情」


【(得其真訣絲毫不差則自然陰陽相生變
化無所謂冥符於至誠之感爾)。】

「　花開黃道正」


【(黃牙白雪燦若
花開用之則可以變通餌之則可以久視黃者中之色也)。】

「　藥就紫煙輕」


【(丹之成也則有紫煙輕覆於金鼎斯則神功內圓靈氣外溢之象也)。】

「　海嶠浮雲合」


【(丹之神也呪之可以羽化則蓬素間苑廻合浮雲皆為往來所
踐之意)。】

「　神光照室明」


【(骨之換矣顏之駐矣則百骸調理三曰清
淨自有神光晶熒照室)。】

「　彩霞高麗日」


【(五色霞凝光吞麗日昔皆
由內景清虛感靈貺而外著矣)。】

「　細意話長生」


【(丹成道至功行兼齊
引平步三島高接九真夫如是則可以論大道長生之理)。】


「　照然明道德」


【(達妙一之本窮玄化之微既象帝之先乃為氣之祖
道之至也德亦生焉照然明白即可知矣斯乃深行之謂也)。】

「　萬象有其端」


【(雖萬象分歧而三元一致以其非道而不生故得言其端矣)。】

「　變化疑誰信」


【(與時消息應物變化隨之而不見其後迎之而不見其首

如是則惑之者其孰能信之乎)。】

「　超凡入聖看」


【(聖人不疑滯於物萬
物未嘗無然物不能為累者以其高超象表逈出凡機方能窮變化之源入希微之
妙上)。】

「　幽玄深自在」


【(非夫周旋至道自在成功者則無以洞其幽玄)。】


「　眇邈就中難」


【(玄津眇邈聖性凝深非耳目之可求豈形聲而能取
是曰難矣)。】

「　但味風塵裏」


【(冰雪其容風塵不雜道之勝也細而昧之)。】


「　方知宇宙寬」


【(雖宇宙之至大而道亦之乎其中矣方知道之寬甘
妙萬有而言則何物而不能生成哉)。】


「　世界無窮盡」


【(兩曜分照五緯標方又覆地載莫知窮極也)。】

「　論之遠更深」


【(𠃵坤分列尊卑定位論其遠則神無方言其深則道無在
至矣大矣惟聖明之)。】

「　陰陽分皂白」


【(日月運行陰陽生伏體之修鍊若緇
素之在目煥然自分尓)。】

「　瓦礫變黃金」


【(法遵五行丹成九轉化至
微之物成至貴之寶豈為難哉)。】

「　報應終還有」


【(至於蓻者無不考圖
𧨯順剛柔內蘊至誠外施陰德玄功暗助神丹必成故真人云若我陳偽謬之言則
天厭之是知報應不虛矣)。】

「　邪求不易尋」


【(求歸道而守迷欲履正而
懷曲徒經嵗月不究指歸期遇真人欲成至藥逮于白首轉益狐疑也)。】

「　忩忙誰正定」


【(動靜之制含育之𠋫固非忩遽求成其功正定不恒徒勤修鍊)。】


「　須是合天心」


【(真誠不差天心自合故崇効於天以濟萬物而成於
事何不克哉)。】


「　世利貪拘束」


【(人居浮世而貪為生之利則自羈縻而遠於道也)。】


「　欺心自不知」


【(外因貪利內即欺心生於自然而不自知也)。】

「　一真求意在」


【(道生於一而育萬物人能得一萬事斯畢矣)。】

「　四敘



【(通變万物莫大乎四時得道之妙者同乎四時絕其參差可以齊聖
也)。】

「　頓悟超諸聖」


【(人若悟道道亦通人人道若合超凡登聖矣)。】


「　娛情改換癡」


【(達人樂道可以自娛頓豁襟懷永無凝滯矣)。】

「　生前如似夢」


【(浮生瞬息不達抱一守真之道兀兀然皆如夢中)。】

「　願重細推之」


【(至士切於大道道不違願而得之細而推之皆由於願矣)。】


「　愚癡常似醉」


【(性昧於情智劣於識蠢蠹凡愚常猶醉矣)。】

「　何異夢中人」


【(迷道之人其昬若夢真心未達在理何醒)。】

「　欲達終無見」


【(妙道非遠而中智巳降欲達未能者何無頓悟之要識也)。】

「　懷疑道豈親」


【(真元之氣抱一之端若猶豫而求則踈遠矣)。】

「　非為貪閇口」


【(背道之貪非為之議靣惟緘口靜默為先)。】

「　郍是益於身」


【(擾擾塵寰區區動識迷者墜悟者升如能順理契真則於身益矣)。】

「　先生堪寂默」


【(占之遠道者不言而化故維摩之默然杜口仲尼曰子欲
無言)。】

「　種福不傳因」


【(世福猶種植而生道離色形諸相泯息故不傳
因也)。】
