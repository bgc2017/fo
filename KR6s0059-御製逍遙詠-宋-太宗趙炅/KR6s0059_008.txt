










御製逍遙詠卷第八(七言)　輕()


「　先賢所誡恐心邪」


【(先聖垂誠以警同邪正貞居中自然淳朴)。】


「　不說神眀道理耶」


【(夫道德之體神明之心應感不窮未嘗疲
於動用之境也)。】

「　豈信爐中生白雪」


【(大藥之功世人罕測或乃
四時可就或則九轉方成先見黃牙次飛白雪也)。】

「　是非汞內產丹砂」


【(夫陰陽抗行體性懸隔或則感氣而相得或乃變化以成形)。】

「　華池自有延齡藥」


【(黃帝云金丹之術百數其要在於神水華池內固其
身而外靜其志也)。】

「　龍虎相逢王幾家」


【(龍者陽位虎者陰位
若陰陽之契會猶龍虎以相從必用丹砂自能濟世)。】

「　十月陰陽須滿足」


【(丹房記云夫修大藥從春修制至十月畢功故云十月脫胎餘兩月沐浴)。】


「　餘殘審細勿教差」


【(華池之外白雪之餘更在用心勿令有成)。】


「　彩雲輕舉壓天津」


【(仙道巳成乃能變化雲凝五色馭以飛騰
縹緲陵虛高踰星漢)。】

「　舞鶴迎鸞自在身」


【(皓鶴彩鸞交翔
接舞任情驂跨輕若乘風十洞三清往復無阻)。】

「　蟾影枝開三島樹」


【(三山峭拔逈倚層霄琪樹蔢䓾影分蟾魄)。】

「　桃花風弄十洲春」


【(碧桃花發春滿十洲風動瓊枝香薰杳靄)。】

「　相傳靈異無虛說」


【(蓬壺異境杳隔雲濤仙者所傳應非虛誕而塵中下士難以信之)。】

「　道理分眀盡是真」


【(修鍊功至神仙道成理趣昭然無非真妙)。】

「　但信此中深護慎」


【(雖凡智昧劣未達真源但可仰而信之則無過矣)。】


「　修行所願勿因循」


【(夫修行者必先使其志立其誓然後勤而

行之則巧必成矣)。】


「　剛柔得所自安寧」


【(陰陽不差變化靡忒語剛柔則阻越言氣
類則相生故制伏安寧神聖可驗)。】

「　五色霞光景象清」



【(功著道成內丹照灼霞舒五彩光含萬象應用無竭莫之與京)。】

「　龍倚[宋-木+儿]吟雲雨至」


【(召雲者龍所以[宋-木+儿]吟而雲雨至盖雲者水氣也龍者水畜
也氣類交感而相應焉)。】

「　虎依巖嘯澗風生」


【(風從虎故依巖
嘯而谷風生此乃明陰中有陽㸦為主客遞相含養以至於道也)。】

「　從容海嶠非遙遠」


【(蓬洲閬苑皆依海而住境閑象外跡異人間今藥成九轉
名煥三清從容以遊故非遙遠)。】

「　倐忽煙蘿別有情」


【(煙蘿
掩暎巖竇曲盤得道之人於焉憩駕盖窒嗜慾絕煩慮物外人間曠然情意故無疑
滯也)。】

「　誰到真空閑密地」


【(道性無方體之則神能達此閑密運
茲妙用其唯真人乎)。】

「　火輪玉闕鎮常眀」


【(火輪陽精玉闕陰粹
光華相射明不暫歇既無私而恒在但有感而必通)。】


「　天非愛道許人修」


【(天稟至真固無私覆人守其道自合玄元
非求所愛自契希夷)。】

「　學士愚迷不到頭」


【(道雖無方學則有緒
而昧道者以噓吸為功藥餌為妙然可養形終非至理)。】

「　內境擬將何所用」


【(苟守內虛白不固外精既非元和安至乎道縱有所負抑亦何為)。】


「　杳知深邃故難求」


【(至真之理深邃之精以有名之筌通無象
之法而𠕀知者難可尋求)。】

「　丹霞隱密堪依仗」


【(靈光有照目以
丹霞內印無形爰彰隱密而達理通幽者誠堪依稟也)。】

「　雪月縱橫入品流」


【(夫道之妙者得用無滯凝明不窮况乎皓月凝霜青松戴雪可標品

格可任縱橫)。】

「　消息頓然潛世慮」


【(消息盈虛窮達一氣內精
外潔逸樂無為蕩蕩然何塵慮之可撓矣)。】

「　凡歧聖路有剛柔」



【(貞清者陽柔和至於聖道滓濁者陰剛佷極以凡途此兩者勝異雖殊制之在我也)。】


「　逍遙裏外混諸天」


【(內境外境諸天混然足可以暢彼逍遙足
可以遠乎塵慮)。】

「　澹蕩其中四敘間」


【(二境叅列萬象森然澹蕩
熙和四敘無間)。】

「　拂拭塵埃明印月」


【(精純慕道不染塵埃真性
融明猶乎朗月)。】

「　徧駈神鬼鎻驪山」


【(鞭駈山石使役鬼神𧬒塞
滄溟究理何益嗟乎妄動寂絕驪山)。】

「　裝添出沒何奇異」


【(凡所
裝成皆歸虛妄徒言奇異非道可稱)。】

「　秘隱藏真未是閑」


【(有為
合道徒自勞形秘隱真機未為閑適)。】

「　但好消詳清景內」


【(消詳
內境𡨋契虛無漱玄液於三瞧運清英於六氣)。】

「　勿生疑慮動機關」


【(體虛極之玄門必融通之上士狐疑塵慮不足可以奉行妄動機關詣道
逾遠也)。】


「　口稱千嵗少年顏」


【(心長禦氣氣與神會內既有主形乃長存
體合衰而不衰年當耄而不耄故雖稱千嵗而尚帶童顏也)。】

「　不是天人即地仙」


【(伶茲容貌儼若神仙望天則疑降自於三清指地則認來從於
十洞)。】

「　四海五湖皆徧歷」


【(靜無所著動無所繫或乘興浮於四
海或任運游於五湖命侶嘯儔故皆徧歷)。】

「　十洲三島去周遷」



【(雖十洲路𨗿三島雲深必周流遷陟無往不通故仙者遷也謂老而無變遷入於山
故稱仙矣)。】

「　道高舉措何優劣」


【(抱長生久視之道仰之而
彌高故稱高矣若觀夫容止舉措動合成䂓既出乎是非之外復何優劣者哉)。】

「　精



【(含和養氣懷玄抱真故精彩非常也抱朴
子曰求仙者要當以忠孝和順仁信為本若德不修而但務方術者終不得長生也
即此語之賢可以為國之紀可以副人之望也)。】

「　物理開襟通象外」


【(凡物之理豁開襟靈則幽通象外也)。】

「　未知幾許達深玄」



【(萬象歷然而人自昧必能頓兩妙理則逈遠深玄也)。】


「　鸞棲洞府影交加」


【(十洞九府鸞鳳所棲神仙所居無不翔集)。】


「　金鼎將成豈用誇」


【(成九還於金鼎就五色於仙經自恊冲虛
豈用誇誕)。】

「　向日迎陽爐上火」


【(日者陽之精也火者陽之華也
夫向日迎其大陽以伏其鈆汞極陰之氣也)。】

「　中秋望月桂開華」


【(月者陰之精也秋者金方肅殺之氣也月臨秋氣巳是純陰復至中秋桂華之
久即太陰得位之極正也故丹砂拒火在乎極陰之氣也)。】

「　丹砂伏定縱橫藥」


【(用日月之正𠋫就玄妙之靈功丹砂克成任縱橫萬藥不能動
也)。】

「　白雪飛騰不離家」


【(變緣河之製涌白雪之光勢告
飛騰然不離於天地之爐也)。】

「　一種為鈆鈆自化」


【(為金之鈆
亦與常鈆一種也盖制伏之術調𠋫之功只向此鈆倏然而自化矣)。】

「　靈胎母子笑天涯」


【(陰陽𠋫正母子道成神丹巳出於靈胎天際可期於笑樂)。】


「　离男坎女甚清廉」


【(日月英華陰陽大道育養群品生化萬靈
故老子云道生之德畜之各盡其清謹之用也)。】

「　金鼎分毫上火炎」


【(大藥在鼎用五行四象以陶鎔至道藏身以亡彼去我為玄妙差之纖毫必炎
熾歘作矣)。】

「　莫把餘心懷利用」


【(失道於動起念於心欲期不乏
之功永作濟世之用即去道遠矣)。】

「　古先學士誡無猒」


【(常思

往古學道高人尤誡無猒冀臻於玄要也)。】

「　深藏異寶叅詳理」



【(萬彚稟靈皆蘊道性若能詳究淵深必得達其妙旨)。】

「　巳漸成形旋要添」


【(日月一年十二度合聖人取象三十日月月次開鼎添洗藥物方漸成靈丹大
藥矣)。】

「　運轉但教經嵗月」


【(初自大易之始洎乎卦象火記之終
其間一飛一伏一佐一助陰陽互相含養遂經長久矣)。】

「　光揚溢目眾人瞻」


【(玄功既就光彩輝雖溢目動人燦然可重矣)。】


「　何人曉了是兼非」


【(至道無方空之懸遠洞達其旨孰曉是非)。】


「　鈆汞從凡隱聖機」


【(金鼎大藥道在一真鈆汞也必能自神合
聖道以修之考五行以鍊之是能變化通玄故聖機無隱也)。】

「　福壽丹砂隨日變」


【(丹藥功成服之不老五福之先遐壽為㝡驗其神用故隨日變
也)。】

「　長生藥鼎負龍威」


【(爐安八卦鼎具四神甲乙青龍
庚辛白虎丹必負威而服故能久視長生也)。】

「　秦皇漢武終邪見」


【(秦皇好道漢武慕仙徐福採藥以不還欒大訪術而受責徒思脫屣不務垂衣
上德迷真誠為邪見)。】

「　大道神仙在杳微」


【(大道虛無應之玄寂
以守真而抱一用絕念以造玄故在杳微視聽冈及)。】

「　學似牛毛成似角」


【(飡霞御氣捕影繫風者如牛毛而多矣還丹得就白日昇天者况祥麟之
一角舉世罕焉)。】

「　故然堪笑兩無依」


【(夫玄珠在水以𠕀象而得
之道在杳冥以無得而明之豈可以積學而求動念而取者去道懸遠故彼邪見者
堪笑無依也)。】


「　逍遙抱一豈相差」


【(聖人抱守淳一可以為天下式自然品類
無差其儀不忒矣)。】

「　方便門開智慧花」


【(万法曉了曰方便智

慧無滯曰門開得不果證於他年故顯花開於今日)。】

「　神室分形添石髓」


【(神者妙用難測室乃藏納有形洪潤流液故彰石髓矣)。】

「　華池巨燄隱靈砂」


【(夫跡叅洞府心契冲𡨋者可認華池之淺深靈砂之次第
也)。】

「　穹隆務實歸宗旨」


【(且形動而心靜神凝而跡移者
無為也必先肆意於穹隆歸旨於甚深矣)。】

「　釋意從容不是賖」



【(高豁性情興懷遠寄陶然於自得之境釋意於混茫之外)。】

「　滋味勿教貪過度」


【(夫佳味爽口美食爛膓故道經云五音令人耳聾五味令人口爽)。】


「　必然混世老天涯」


【(上士以棲真鍊魄冲虛靜默老於天涯混
於物表)。】


「　龍虎剛柔隱聖方」


【(龍為鈆而虎為汞剛為陽而柔為陰
旨趣玄微秘諸仙訣)。】

「　一時和合運陰陽」


【(金石之性各稟自然
合而用之互有違順若能消息會合陰陽變化之功通乎神妙)。】

「　周知配類無踈失」


【(愽達精微配其品類纖毫無失其功十全)。】

「　物性從來理自長」


【(萬物之性各有所長不可以論辛忌甘好丹非素也)。】

「　鼎裏花開裝五彩」


【(金鼎丹成如花爛澷二氣交感五色相鮮修鍊之功
驗於此矣)。】

「　心眀火𠋫制三黃」


【(三黃者則硫黃雄黃雌黃也純
陽之性擒制尤難若依彼神方明其火𠋫則亦可伏之矣)。】

「　更能通得華池妙」


【(華池之名何止百數制之在氣功用不同更在沉研以通其妙)。】


「　南北東西不可當」


【(功之成矣妙用縱橫掌視四方何往
不適矣)。】


「　神仙不死幾人存」


【(見素抱朴積德累功仙道既真谷神不死

然洞天遐阻鼇山杳隔得長生久視者幾人存焉)。】

「　欲界從頭是可論」


【(從凡入聖無不自欲界而昇諸天且修何妙用頗滌塵滓永處清虛
事匪常情故可細而論也)。】

「　真實自然歸至理」


【(至理淵曠非道
不達但虛白其心真實其行則曉悟玄門如指諸掌也)。】

「　狂顛逢見道須尊」


【(重為輕根靜為躁君且狂悖者良為未明玄旨正處勞生若見
道明真則顛沛自歇故知清淨之理可化不善之人)。】

「　分眀記取依經典」


【(老子云孔德之容唯道是從若依茲聖典佩服心腑則眾妙之門必
可履踐尓)。】

「　消息臨時度曉昬」


【(消息以時曉昬唯寂進退得度
用捨合冝矣)。】

「　四敘推遷由反覆」


【(歲時不駐所以偱環春秋
推遷寒暑然攝生養道者不隨物而改變也)。】

「　唯占日月順𠃵坤」


【(日御飛轡月魄輪空虧盈不差𠃵坤自順法而動靜其道乃光也)。】


「　逍遙密化道希夷」


【(得理息羈冲虛放曠乃冂逍遙潛憑
冈象寂寥無聲混漠無形乃希夷之道也)。】

「　宗習羣流我詠之」



【(宗彼玄虛習乎妙道雖探幽賾秘孰窺其元我以詠之令知乎要矣)。】

「　應是有情聽譬諭」


【(夫生我者道稟我者神猶性動為情情生昧道今可舉况
設諭皆令悟入也)。】

「　莫教虛度悟參差」


【(復垂慜誡再示玄虛
慮雖悟得性為宗恐不至鍊形為要乃參差之毫𨤲矣)。】

「　深窮動靜冝消息」


【(夫動舉靜移去道彌遠靜止動息乃合冲玄今可超動靜源出
有無表窮達要妙消息玄虛也)。】

「　惻隱玄元善所為」


【(太虛之先
寂寥何有至精感激日之玄元今可惻彼幽隱善所作為終極乎道矣)。】

「　性識更能通幻夢」


【(性可以洞神識可以鑒聖丁塵寰之幻夢不溺色聲

明妙本以全真何須修鍊)。】

「　恬然存相似生知」


【(虛白恬然樂於
內澡雪存相潔於外乃形儼性靜蕭灑出塵指万法似生知固一氣為大道也)。】


「　水精宮殿月華眀」


【(陽精注云水精宮殿帝后三妃九嬪
所居日有九華月有十華)。】

「　白玉為墻火不侵」


【(靈寶經云三清
聖境假使陽極水灾陰窮火運三界消壞而三境不侵)。】

「　八境風高駢羽駕」


【(八溟之府諸仙之境玄氣凝結清風益高羽駕爭行駢繁無盡)。】

「　九重天上絳霞深」


【(仙天九重赤霞掩暎淨色可觀故謂之深)。】

「　物情晃朗無遮隔」


【(神仙情性同日月之精英晃朗無遮明然自若)。】


「　非是凡間比類尋」


【(人間物象俾不解者可以喻知仙界幽通
比類奚及)。】

「　汗澷真遊誰得到」


【(洞府幽都神居靈域非夫
物外不拘之者曷遊真境之中)。】

「　桂花香潔影森森」


【(蔢䓾桂樹
異實名花影覆仙宮香浮神闕)。】


「　從凡入聖作梯媒」


【(人若通道道亦通人人道相通超凡入聖
故必假精修至鍊而作梯媒也)。】

「　紅燄爐深豈易裁」


【(夫修大藥
者必先置爐爐者鼎之丘郭也當灼灼紅燄之前宛有蓬萊之狀其間仰觀俯察明
陰洞陽千變萬化未易裁也)。】

「　點化縱橫名大藥」


【(惣五行而
合體括四象以成形點石化金縱橫妙用故名大藥)。】

「　催驅雜類謾成灰」


【(不達大丹之法者徒以世間雜類催促驅馳終無所益也)。】

「　真鈆即是機關理」


【(太陰真精謂之真鈆也機關運處理盡其中矣)。】

「　闇室皆明任眾猜」


【(自闇而明人莫能極故眾謾猜也)。】

「　未究前言勞自苦」


【(前言之妙乃聖賢之密用也豈造次而明之不達而求之

者徒自勞苦也)。】

「　精窮玄妙樂康哉」


【(道之至也即玄之又玄妙
之又妙必能精通窮究何樂如之而神用康也)。】


「　交結從頭有順違」


【(地天交泰二儀相通順之則物類攸行違
之則氣𠋫差㸦)。】

「　方知霧散若雲飛」


【(莊子云積水上溢故為霧
淮南子云山之出雲蒸柱潤礎斯皆積陰之盛也其或陰陽調暢象緯和平則霧之
散雲之飛理皆如也)。】

「　超凡入聖含金體」


【(抱素還朴儻積德而
累功以石為金即自凡而入聖)。】

「　拋棄迷蒙挂紫衣」


【(棄欺幻之
術得虛通之門丹田既離於迷蒙紫車盡期於變化)。】

「　細認消停無遠近」


【(差之毫𨤲失之千萬子細消息復何有於遠近乎)。】

「　分明意外隱玄機」


【(大道分明離於言象超然意慮之外故玄機隱而難見)。】

「　相逢配偶為閑事」


【(若陰陽和同則氣象諧偶萬彚通變俱不勞神雖庶事森
羅而自可閑適矣)。】

「　誰解調持得一歸」


【(夫天得一以清地得
一以寧人得一以靈然則知者雖眾其能調理任持歸于得一之體者誠為難矣)。】


「　昭宣形體思悠然」


【(覆載之內萬物生成皆得優游各遂
其性也)。】

「　郍是愚夫郍是仙」


【(仙凡雖殊妙道一致顯即
登真脫屣昧乃觸事靣墻)。】

「　不識饒教非遠近」


【(晦昧其道
汩沒於真雖遠近有殊皆不可求其及也)。】

「　悞來何事苦相懸」



【(或達虛旡之理只在𠛴郍前念猶自懵然後念杳無鄙隔)。】

「　含容裏外無諸物」


【(既達其理靜慮安神內即萬想俱捐外即一物無狀矣)。】

「　象法其由語話詮」


【(法顯形象言可詮量語默之間同歸大道)。】

「　品格調高奔世界」


【(神仙品格邈乎高尚所以乘雲躡虛游四荒八

極之外也)。】

「　任從他寶四周遷」


【(仙都日月難究遐長既得逍遙
任從遷改)。】


「　知之修鍊但辛勤」


【(修鍊之士今古尤多知之者可以造玄微
昧之者無以達其妙孜孜矻矻唯務辛勤)。】

「　苦志由來離六塵」



【(然以燒丹鍊石苦志勞神要在鄙弃諠譁蠲除嗜慾六塵既息万慮寂焉)。】

「　金鼎乍分紅燄火」


【(庚辛應象成兊鼎以含真丙丁助陽藉离火而養氣
輝華紅燄顥結玄霜)。】

「　月魂高鎻碧潭津」


【(金液成丹月華
流潤高陟煙霄之上深窮玄妙之津)。】

「　深思五嶺凡間路」


【(五嶺
路歧深思遐遠千谿万壑峭拔峯巒未出塵寰俱非聖轍)。】

「　好作三天物外人」


【(三天麗境万象之尊脫屣塵寰逍遙物外好真之侶善思齊焉)。】

「　萬里州通寬世俗」


【(四海混同九州萬里寬分宇宙通貫遐陬變世俗之
澆漓盡淳風之化洽)。】

「　清虛境象善求真」


【(海晏河清天寧
地肅二儀昭泰万彚清虛既属良時道冝進取)。】


「　如來秘法諭恒沙」


【(夫心印密傳靈符玄授演九藏之奧窮萬
法之源林葉不足以比其多恒河可以為其喻)。】

「　須是鈆開汞上花」


【(一㕛始生五陰方下花既生於汞上藥必熟於鼎中)。】

「　燒鍊勿令傷血脉」


【(夫欲鍊胎易質革穢除腐神形合契白日輕舉者務養其性命
有其血脉也)。】

「　凡情散亂損黃牙」


【(道者至靜為宗精思
為用自然神感有靈凡情不亂黃牙自成矣)。】

「　离男合度分明說」


【(夫金丹大鼎研味真奧八卦以成其序男女以別其位也)。】

「　坎女和同不用誇」


【(四神當位龍虎相逢丹砂必成谷神長在故不自矜失於道體)。】



「　大矣日中成九轉」


【(日中九轉十月玄功烹鍊既精名殊次第
即湛陰粉靣等也)。】

「　紅漿一飲到仙家」


【(道者萬物之奧羣情
之望肺膓既沃於紅漿形體必棲於碧落)。】


「　還源根本沓難窮」


【(歸根復命得道之常真曰大還全非
九轉斯之妙旨冲寞難窮)。】

「　秘密玄微造化中」


【(養素還淳功齊
造化玄開秘訣包括無遺)。】

「　白日昬迷常不見」


【(雖曰玄微其道
明白如日之照本自無私學者迷之瞑然無覩)。】

「　青龍霧曉大家同」


【(龍象太陽光充宇內五行有犯如霧昬蒙霧廓曉天無幽不燭故曰大同)。】

「　陰陽表裏和煙碧」


【(丹分內外二氣相資交泰混融纖塵不翳
如煙一色凝碧虛明)。】

「　天地相鮮似火紅」


【(陰陽相契功用自今
神光炳然交徹二位如火之盛紅燄晶熒)。】

「　直要審詳知道理」



【(差之秋毫天地懸隔未窮玄旨曷望成功)。】

「　丹成千載上昇功」



【(神丹既就妙用克彰足保遐齡上登仙界)。】
